# 公众号正文

本目录用于沉淀公众号系列文章草稿。

本项目不是简单写一篇文章，而是围绕“AI产业周期研究”形成一个系列。第一条主线是：

> 以史为鉴：新能源周期复盘。

---

## 一、写作原则

1. 不写短线复盘，要写产业周期；
2. 不只写产业，也要写 K 线、新闻流、资金参与度和估值周期；
3. 不只看赢家，也要看失败样本和炒作后阴影；
4. 每篇只解决一个核心问题；
5. 每篇最后都要映射 AI；
6. 不给买卖建议，只做产业周期研究。

## 三、每篇文章标准结构

1. 开头提出核心矛盾；
2. 一句话结论；
3. 拆解产业阶段；
4. 拆解标杆股走势；
5. 拆解当时新闻流；
6. 拆解资金参与度；
7. 写出事后验证；
8. 映射到 AI；
9. 结尾沉淀方法论。
