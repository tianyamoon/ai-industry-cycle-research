图表目标不是装饰，而是帮助读者快速理解产业周期。

## 二、图表设计原则

1. 每张图只表达一个核心观点；
2. 适合公众号竖版阅读；
3. 重要文字必须可读；
4. 不要把所有数据塞进一张图；
5. 信息层级要清楚：时间、阶段、产业主线、代表标的、AI 映射；
6. 图表服务于文章，不替代研究。
