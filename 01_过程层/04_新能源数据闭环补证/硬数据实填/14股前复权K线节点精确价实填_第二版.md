> 所属模块：01_过程层/04_新能源数据闭环补证 → 硬数据实填
> 生成日期：2026-06-27
> 数据区间：2020-04-01 至 2026-06-26
> 研究定位：用统一前复权日线口径，把 14 只新能源样本的启动、顶部、退潮和阴影节点推进到可复核的窗口法精确价。
> 重要口径：本表是“东方财富前复权日线 + 窗口法精确价”，不是 Wind / Choice 官方最终值；PE、PB、公募 / 北向持仓仍显式保留为“待补”。

## 一、数据源与方法

数据源为东方财富历史 K 线接口：

`https://push2his.eastmoney.com/api/qt/stock/kline/get`

统一参数：

- `klt=101`：日线；
- `fqt=1`：前复权；
- `beg=20200401`、`end=20260626`；
- `fields2=f51,f52,f53,f54,f55,f56,f57,f58,f59,f60,f61`。

字段解释：

| 字段 | 含义 |
|---|---|
| f51 | 日期 |
| f52 | 开盘价 |
| f53 | 收盘价 |
| f54 | 最高价 |
| f55 | 最低价 |
| f56 | 成交量 |
| f57 | 成交额 |
| f58 | 振幅 |
| f59 | 涨跌幅 |
| f60 | 涨跌额 |
| f61 | 换手率 |

窗口法精确价规则：

1. **启动**：取启动窗口内最低价；
2. **第一顶部**：取第一顶部窗口内最高价；
3. **二次顶部**：取二次顶部窗口内最高价；
4. **退潮确认**：取退潮窗口内最低价；
5. **阴影低点**：取阴影窗口内最低价；
6. **主升涨幅**：`max(第一顶部高点, 二次顶部高点) / 启动低点 - 1`；
7. **最大回撤**：`阴影低点 / max(第一顶部高点, 二次顶部高点) - 1`；
8. **顶部成交额、顶部换手率**：取最高峰值所在交易日的 f57、f61；成交额换算为亿元。

注意：阴影低点只表示截至 2026-06-26 的指定窗口内低点，不表示后续周期的最终低点。

## 三、按链条解释

### 1. 资源链

赣锋、天齐、华友的最大回撤均超过 80%，说明资源价格泡沫破裂后的估值回撤非常彻底。天齐的二次顶部高于第一顶部，赣锋和华友则在 2021 已先见峰，后续高利润更像兑现期而不是新增买点。

### 2. 材料链

通威因硅料瓶颈利润把峰值推迟到 2022；天赐、多氟多、恩捷、福斯特多在 2021 形成最高峰。隔膜、六氟、胶膜都呈现“高壁垒或高利润被周期化定价”的共性，其中恩捷窗口内最大回撤达到 -92.3%，是本表最极端样本。

### 3. 终端 / 核心供应商 / 平台链

阳光电源、比亚迪、宁德时代的回撤幅度明显低于多数资源和材料股，窗口法阶段性显示公司阿尔法、海外化、储能或终端平台能力缓释了贝塔回撤。但宁德仍在 2021 先见估值峰，不能把需求强度等同于股价继续创新高。

### 4. 设备订单链

先导智能的峰值出现在 2021-11-08，早于下游扩产利润的充分兑现。设备股后续应优先跟踪新签订单、合同负债、下游 CapEx 和现金流，而不是只等利润表确认。

## 五、本文件结论

第二版已完成 14 股前复权 K 线节点的窗口法精确价实填。

阶段性结论：

> **新能源样本的股价峰值不是同一天、也不是同一变量驱动。资源和材料更贴近价格周期，设备更贴近订单周期，核心资产和终端平台更贴近估值资金与公司阿尔法。窗口法精确价进一步确认：多数非穿越样本从峰值到阴影低点的回撤超过 80%。**

后续应把本表与 PE/PB、持仓和 Wind / Choice 复核结果合并，形成真正可引用的股价数据拐点对照表。
