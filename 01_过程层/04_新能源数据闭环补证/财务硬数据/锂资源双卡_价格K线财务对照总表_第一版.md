> 所属模块：01_过程层/04_新能源数据闭环补证 → 财务硬数据  
> 生成日期：2026-06-23  
> 研究对象：赣锋锂业、天齐锂业  
> 研究定位：把碳酸锂价格、赣锋 / 天齐前复权 K 线、财务季度表、资产负债表和估值验证路径合并成一张总表，作为锂资源周期的第一条完整投资闭环样板。

---

## 一、核心问题

锂资源周期要回答的核心问题是：

> **锂资源股的顶部，是价格顶部、股价顶部、利润顶部、估值顶部，还是资产负债表修复预期顶部？**

本总表用于验证：

1. 碳酸锂价格是否领先赣锋 / 天齐股价；
2. 赣锋 / 天齐股价是否领先利润；
3. 利润高点是否造成低 PE 陷阱；
4. 锂价下跌后，存货、减值、现金流和资产负债表压力如何滞后暴露；
5. 赣锋和天齐的股价弹性差异来自哪里。

---

## 二、锂资源周期总链条

```text
碳酸锂价格低位
→ 供需错配
→ 碳酸锂价格快速上涨
→ 锂盐毛利率上行
→ 资源股利润爆发
→ 市场外推高利润
→ 股价顶部 / 估值顶部
→ 碳酸锂价格见顶并下跌
→ 毛利率下行
→ 利润下行
→ 存货 / 减值 / 资产负债表压力暴露
→ 估值中枢重估
```

## 四、赣锋 vs 天齐核心差异

| 维度 | 赣锋锂业 | 天齐锂业 | 研究含义 |
|---|---|---|---|
| 核心属性 | 锂盐加工 + 全球资源布局 + 电池延伸 | 优质资源权益 + 锂盐 + 资产负债表弹性 | 天齐更偏资源权益和负债表弹性，赣锋更偏锂盐平台 |
| 价格敏感性 | 高 | 极高，待验证 | 要比较区间涨幅和回撤 |
| 杠杆敏感性 | 中高，待补 | 高，重点补资产负债表 | 天齐必须额外看负债率和财务费用 |
| 利润弹性 | 锂盐价格和销量驱动 | 锂价 + 资源权益 + 资产负债表修复 | 两者利润质量不同 |
| 下行风险 | 存货、项目回报、减值 | 权益资产、负债、现金流、减值 | 天齐退潮风险可能更复杂 |
| 估值口径 | PE + PB + 资源项目 | PB + 资产负债表 + 资源权益 | 资源股不能只看 PE |

## 九、本文件结论

锂资源双卡已经形成第一版闭环框架：

> **赣锋代表锂盐平台和资源利润泡沫，天齐代表资源权益和资产负债表弹性。二者都必须以碳酸锂价格为第一变量，但验证重点不同：赣锋更看锂盐毛利率和存货减值，天齐更看资产负债表修复、权益资产和 PB 重估。**

当前还不能下最终结论，因为关键硬数据仍待补：

1. 前复权 K 线；
2. 碳酸锂月度均价；
3. 季度毛利率和利润；
4. 存货和减值；
5. PE / PB / 持仓。

但下一步路径已经明确：

> 先闭合锂资源，再复制到硅料、六氟、组件三条价格链。
