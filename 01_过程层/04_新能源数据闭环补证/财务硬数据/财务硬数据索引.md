## 一、为什么要做财务硬数据

价格曲线和 K 线节点只能回答：

- 价格什么时候涨跌；
- 股价什么时候启动和见顶。

财务硬数据要回答：

> 股价见顶时，公司财务到底是已经恶化，还是仍然很好？

这决定了周期复盘中最关键的问题：

1. 是基本面见顶，还是估值和资金先见顶；
2. 是价格先见顶，还是利润先见顶；
3. 是需求证伪，还是利润率证伪；
4. 报表最好时，是否反而接近周期尾部；
5. 退潮后，存货、减值、现金流和 ROIC 如何暴露问题。

## 三、统一字段

### 1. 利润表字段

| 字段 | 用途 |
|---|---|
| 营业收入 | 判断需求和价格共同作用 |
| 归母净利润 | 判断利润兑现 |
| 扣非净利润 | 判断主业利润质量 |
| 毛利率 | 判断价格和定价权传导 |
| 分业务收入 | 判断利润来源 |
| 分业务毛利率 | 判断哪个环节贡献利润 |

### 2. 现金流字段

| 字段 | 用途 |
|---|---|
| 经营现金流 OCF | 判断利润现金含量 |
| OCF / 净利润 | 判断利润质量 |
| 资本开支 CapEx | 判断扩张压力 |
| 自由现金流 | 判断穿越能力 |

### 3. 资产负债表字段

| 字段 | 用途 |
|---|---|
| 存货 | 判断价格下跌风险 |
| 存货跌价准备 | 判断退潮后果 |
| 资产减值 | 判断高景气扩张后的损失 |
| 资产负债率 | 判断杠杆压力 |
| 有息负债 | 判断债务压力 |
| 货币资金 | 判断安全垫 |
| 财务费用 | 判断利息压力 |

### 4. 资本回报字段

| 字段 | 用途 |
|---|---|
| ROE | 判断权益回报 |
| ROIC | 判断资本开支回报 |
| 总资产周转率 | 判断资产效率 |
| 毛利率 - 费用率 | 判断利润结构 |

## 四、第一批锂资源要验证的问题

### 1. 价格顶部和利润顶部关系

要验证：

> 碳酸锂价格顶部是否早于赣锋 / 天齐利润顶部？

如果成立，则资源股报表最好时，可能已经接近周期尾部。

### 2. 股价顶部和利润顶部关系

要验证：

> 股价顶部是否早于利润顶部？

如果成立，则市场会提前交易价格和利润拐点，而不是等年报确认。

### 3. 高利润低 PE 陷阱

要验证：

> 2022 年资源股利润高点时，PE 是否看起来很低，但随后利润下滑导致估值重估？

### 4. 存货和减值滞后暴露

要验证：

> 锂价下跌后，存货、减值、现金流压力是否滞后于价格和股价。

### 5. 赣锋和天齐的差异

要验证：

- 赣锋：锂盐加工 + 全球资源布局；
- 天齐：资源权益 + 资产负债表弹性；
- 哪个对锂价更敏感；
- 哪个对资产负债表修复更敏感；
- 哪个在低价周期中压力更大。
