# 新闻政策原文表｜字段模板

> 所属模块：01_过程层/04_新能源数据闭环补证  
> 生成日期：2026-06-21  
> 用途：把政策、新闻、公告和行业事件从“叙事”转化为可复核证据。

---

## 一、模板目标

新闻政策原文表用于解决：

> 当时市场到底在交易什么？事后看这些政策和新闻是成立、部分成立，还是被证伪？

投资复盘不能只写“受政策利好刺激”，必须落到：

- 日期；
- 发文单位；
- 原文标题；
- 原文关键句；
- 当时市场理解；
- 对应 K 线表现；
- 事后验证。

---

## 二、字段模板

| 字段 | 说明 | 示例 |
|---|---|---|
| 日期 | 精确到日 | 2018-05-31 |
| 事件类型 | 政策 / 新闻 / 公司公告 / 行业数据 / 研报观点 | 政策 |
| 标题 | 原文标题 | 关于 2018 年光伏发电有关事项的通知 |
| 发布主体 | 发改委 / 能源局 / 公司 / 协会等 | 发改委、财政部、能源局 |
| 原文链接 | 官方或可信来源 | 待补 |
| 原文关键句 | 不超过必要摘录 | 待补 |
| 当时市场理解 | 利好 / 利空 / 分歧 | 利空光伏装机 |
| 对应标的 | 隆基、通威等 | 隆基、通威 |
| 对应 K 线阶段 | 出清 / 主升 / 顶部 / 退潮 | 第一次出清 |
| 后续数据验证 | 装机、销量、价格、财务 | 2018 装机下降 |
| 事后结论 | 成立 / 部分成立 / 被证伪 | 成立，但龙头被筛选出来 |

## 四、新闻事件判断规则

| 情况 | 判断 |
|---|---|
| 新闻推动股价，但后续数据没有兑现 | 题材炒作，需警惕 |
| 新闻推动股价，后续价格和财务兑现 | 产业催化成立 |
| 利空导致下跌，但龙头份额提升 | 出清型利空，可能是长期机会 |
| 利好不断但股价不涨 | 预期已经充分反映 |
| 新闻发生在价格高点之后 | 可能是滞后叙事 |
| 新闻发生在股价高点之后 | 可能不是领先指标 |

## 五、最终输出要求

每个关键事件最终必须链接到：

1. 原文；
2. 股价；
3. 价格；
4. 财务；
5. 事后结论。

最终目标：

> 分清楚哪些新闻是领先催化，哪些只是高位叙事，哪些是退潮确认。
