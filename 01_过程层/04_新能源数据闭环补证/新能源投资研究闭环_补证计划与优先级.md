# 新能源投资研究闭环｜补证计划与优先级

> 所属模块：01_过程层/04_新能源数据闭环补证  
> 生成日期：2026-06-21  
> 研究定位：暂停 AI 单股扩展，回到新能源周期本身，把 K 线、价格、财务、估值、持仓和新闻原文补齐，形成可用于股票投资研究的闭环。

---

## 一、为什么必须补证

前面已经完成的是新能源十年周期的逻辑骨架：

**产业阶段 → 标杆股路径 → 新闻政策 → 价格曲线 → 财务指标 → AI 映射。**

但对股票投资来说，仅有逻辑骨架还不够。

投资研究必须回答：

1. 当时市场为什么买；
2. 股价什么时候启动；
3. 哪个数据在股价启动前已经变化；
4. 哪个数据在股价见顶前已经恶化；
5. 顶部时财务到底是好还是坏；
6. 退潮时是需求证伪、价格证伪、估值证伪，还是资金证伪；
7. 下一轮 AI 映射时，应该盯哪些领先指标。

因此，当前阶段不能继续只扩 AI 单股。必须先把新能源复盘做成投资闭环。

## 三、投资闭环所需六张底表

### 1. K 线节点表

目的：确定每只股票真正的启动、第一次顶部、二次顶部、退潮和当前状态。

字段：

| 字段 | 说明 |
|---|---|
| 标的 | 公司名称 |
| 阶段 | 概念、出清、主升、顶部、二次冲顶、退潮、阴影 |
| 起止日期 | 精确到交易日或月份 |
| 前复权价格 | 用统一口径 |
| 后复权价格 | 辅助验证长期收益 |
| 区间涨跌幅 | 判断行情强弱 |
| 最大回撤 | 判断退潮力度 |
| 成交额 / 换手 | 判断资金拥挤 |
| 备注 | 关键事件 |

优先标的：

1. 隆基绿能；
2. 通威股份；
3. 赣锋锂业；
4. 天齐锂业；
5. 天赐材料；
6. 比亚迪；
7. 宁德时代。

### 2. 价格曲线表

目的：判断股价是否由价格驱动，价格是否领先财务。

必须补的价格：

| 品种 | 对应标的 | 用途 |
|---|---|---|
| 碳酸锂 | 赣锋、天齐、天赐、宁德、比亚迪 | 锂资源和电池成本核心 |
| 氢氧化锂 | 赣锋、天齐 | 高端锂盐需求 |
| 锂辉石 | 赣锋、天齐 | 上游资源成本 |
| 硅料 | 通威、隆基 | 光伏上游利润核心 |
| 硅片 | 隆基、通威 | 光伏主链价格传导 |
| 组件 | 隆基、福斯特、阳光 | 主链价格战核心 |
| 六氟磷酸锂 | 天赐、多氟多 | 材料涨价泡沫核心 |
| 电解液 | 天赐 | 电解液盈利验证 |
| 隔膜 | 恩捷 | 高壁垒材料周期化 |
| 胶膜 / EVA / POE | 福斯特 | 辅材价格传导 |
| 钴 / 镍 | 华友 | 三元资源材料周期 |

字段：

| 字段 | 说明 |
|---|---|
| 日期 | 月度 |
| 品种 | 碳酸锂 / 硅料 / 六氟等 |
| 价格 | 元 / 吨、美元 / 吨或行业口径 |
| 环比 | 月度变化 |
| 同比 | 年度变化 |
| 所处阶段 | 低位、上行、高位、下行、底部 |
| 对应标的 | 受益或受损公司 |
| 股价阶段 | 启动、主升、顶部、退潮 |

---

### 3. 财务指标表

目的：判断股价变化是否被财务验证，顶部时财务是领先、同步还是滞后。

字段：

| 指标 | 用途 |
|---|---|
| 营业收入 | 需求验证 |
| 归母净利润 | 利润兑现 |
| 扣非净利润 | 主业质量 |
| 毛利率 | 定价权和价格周期 |
| 净利率 | 盈利质量 |
| ROE | 股东回报 |
| ROIC | 资本开支回报 |
| 经营现金流 | 利润质量 |
| CapEx | 扩产风险 |
| 存货 | 价格下跌风险 |
| 减值 | 退潮后果 |
| 资产负债率 | 杠杆压力 |
| 有息负债 | 资源股和重资产压力 |
| 业务分部收入 | 拆分真实增长来源 |
| 业务分部毛利率 | 判断价格和产品结构 |

优先做年度表，再补季度表。

年度区间：2015–2025。

季度重点：2020Q1–2025Q4。

---

### 4. 估值和资金拥挤表

目的：判断股价顶部是否来自基本面恶化，还是估值和资金拥挤。

字段：

| 字段 | 用途 |
|---|---|
| PE TTM | 判断利润估值 |
| PB | 判断资产估值 |
| PS | 判断收入估值 |
| 市值 | 判断预期总量 |
| 公募重仓市值 | 判断机构拥挤 |
| 公募持股占流通股比例 | 判断筹码集中 |
| 北向持股 | 判断外资拥挤 |
| 融资余额 | 判断杠杆资金 |
| 成交额 | 判断市场热度 |
| 换手率 | 判断筹码切换 |

重点问题：

- 隆基 2021 年第一次顶部时财务是否已经恶化？
- 宁德 2021 高位是否主要是估值和持仓拥挤？
- 资源 / 材料 2022 高点是否是利润和估值同时高？
- 2023 后是杀估值还是杀利润？

---

### 5. 新闻政策原文表

目的：把政策和新闻从“叙事”变成“证据”。

| 字段 | 说明 |

| 当时市场理解 | 利好 / 利空 / 分歧 |
| 事后验证 | 成立 / 部分成立 / 被证伪 |
| 对应标的 | 相关公司 |
| 对应 K 线阶段 | 启动 / 顶部 / 退潮 |

1. 新能源车补贴政策；
2. 动力电池白名单；
3. 双积分；
4. 补贴退坡；
5. 光伏 531；
6. 平价上网；
7. 双碳；
8. 2021 资源和材料涨价新闻；
9. 2022 锂价和硅料高点新闻；
10. 2023–2025 价格战和产能过剩新闻。

### 6. 股价—数据拐点对照表

目的：形成最终投资闭环。

字段：

| 字段 | 说明 |
|---|---|
| 标的 | 公司 |
| 股价拐点 | 启动 / 顶部 / 退潮 / 底部 |
| 日期 | 精确日期或月份 |
| 股价表现 | 涨跌幅 / 回撤 |
| 领先数据 | 哪个指标先变化 |
| 同步数据 | 哪个指标同步变化 |
| 滞后数据 | 哪个指标后变化 |
| 市场当时叙事 | 当时买卖理由 |
| 事后真实原因 | 复盘后的核心原因 |
| AI 映射 | 对 AI 研究的启示 |

最终要回答：

> 新能源每一轮股价大涨和大跌，分别是由什么数据先提示的？

## 四、补证优先级

### 第一阶段：隆基闭环

必须先做隆基，因为它是整个新能源周期的锚点。

隆基要补：

1. 2015–2025 前复权 K 线节点；
2. 2016–2025 财务年度表；
3. 硅料、硅片、组件价格曲线；
4. 531、平价上网、双碳、2023 价格战新闻；
5. 估值和基金持仓；
6. 股价顶部和财务 / 价格拐点对应。

核心问题：

> 隆基 2021 年 1 月底 / 2 月初第一次见顶，是基本面见顶，还是估值和资金先见顶？

### 第二阶段：通威闭环

通威要补：

1. 硅料价格曲线；
2. 硅料毛利率；
3. 2021–2022 利润高点；
4. 2023–2025 硅料价格下跌和利润反噬；
5. 扩产公告和资本开支。

核心问题：

> 通威是不是典型的瓶颈利润后见顶？

### 第三阶段：锂资源闭环

标的：赣锋、天齐。

要补：

1. 碳酸锂、氢氧化锂、锂辉石月度价格；
2. 2016–2017 第一轮锂价行情；
3. 2018–2020 锂价低谷；
4. 2021–2022 第二轮锂价泡沫；
5. 2023–2025 锂价崩塌；
6. 财务和资产负债表。

核心问题：

> 资源股顶部是价格顶部、利润顶部，还是估值顶部？

### 第四阶段：材料闭环

标的：天赐、多氟多、恩捷。

要补：

1. 六氟、电解液、隔膜价格；
2. 毛利率变化；
3. 扩产和客户压价；
4. 存货和减值；
5. 股价顶部与价格高点的关系。

核心问题：

> 高壁垒材料是否真的能免疫价格周期？

### 第五阶段：穿越样本闭环

标的：比亚迪、宁德、阳光。

要补：

1. 销量 / 出货；
2. 单车利润 / 动力电池毛利率 / 储能毛利率；
3. 海外收入；
4. 现金流；
5. ROIC；
6. 价格战阶段表现。

核心问题：

> 哪些公司从赛道贝塔切到公司阿尔法？

### 第六阶段：设备和辅材闭环

标的：先导、福斯特。

要补：

1. 新签订单 / 在手订单；
2. 应收账款和合同负债；
3. 胶膜价格和毛利率；
4. 主链价格战传导；
5. 第二曲线兑现。

核心问题：

> 设备和辅材是不是比主链更早给出退潮信号？

## 六、当前行动原则

从现在开始，新能源研究不再继续写“泛逻辑”。

每个结论都必须落到：

1. K 线节点；
2. 价格曲线；
3. 财务表；
4. 估值和持仓；
5. 新闻政策原文；
6. 事后验证。

投资研究闭环标准：

> 任何一句“为什么涨、为什么跌”，都必须能在至少两个数据源中找到支撑。
