> 研究定位：为新能源十年周期总表、14 张标杆股 K 线图谱卡和后续 AI 映射提供价格变量底座。

## 一、为什么价格曲线比销量更重要

新能源周期最大的误区是：只看需求，不看价格。

2023–2025 最典型的教训是：

**新能源车销量仍然增长，光伏装机仍然增长，但锂价、硅料、组件、电解液、六氟、隔膜等价格大幅下跌，导致资源、材料、光伏制造公司股价和估值持续下行。**

所以价格曲线的作用是解释：

1. 为什么 2021–2022 资源 / 材料股后半段最强；
2. 为什么 2022 是利润最好但风险最高的阶段；
3. 为什么 2023–2025 需求仍高增但股价长期阴跌；
4. 为什么 AI 周期也必须盯 ASP、毛利率和供给扩张，而不是只看订单和需求。

统一框架：

**价格上涨 → 利润上移 → 股价强化 → 扩产 → 价格下跌 → 利润回落 → 估值坍塌。**

## 三、碳酸锂价格曲线

### 1. 产业含义

碳酸锂是新能源车周期中最重要的价格变量之一。

它解释了：

- 赣锋、天齐 2016–2017 第一轮锂资源行情；
- 2021–2022 第二轮锂价泡沫；
- 2023–2025 资源股阴影期；
- 天赐、多氟多等材料公司成本和利润变化；
- 宁德、比亚迪等中下游企业的成本压力。

### 2. 阶段划分

| 阶段 | 价格状态 | 股价含义 |
|---|---|---|
| 2015–2017 | 第一轮上涨，新能源车补贴和电池需求推动锂资源被重估 | 赣锋、天齐第一次进入资源主线 |
| 2018–2019 | 锂价回落，资源股估值收缩 | 第一轮资源行情结束，周期属性暴露 |
| 2020 | 低位修复，新能源车需求重新启动 | 第二轮行情前夜 |
| 2021 | 快速上涨，供需错配强化 | 锂资源股进入后半段强势行情 |
| 2022 | 接近极端高位，市场报价一度接近 60 万元 / 吨 | 利润最好、风险最高，资源泡沫阶段 |
| 2023–2024 | 大幅回落，2024 年约 10 万元 / 吨附近波动 | 赣锋、天齐估值和利润双杀 |
| 2025–至今 | 低位波动，局部供给扰动带来反弹 | 周期修复观察，不等于高点估值恢复 |

### 3. 对应标的

- 直接受益：赣锋锂业、天齐锂业；
- 成本受压：宁德时代、比亚迪、亿纬锂能；
- 间接受影响：天赐材料、多氟多、华友钴业。

### 4. 对 AI 的映射

碳酸锂对应 AI 中的上游瓶颈价格：

- HBM；
- 光模块；
- 高速铜连接；
- 变压器；
- 数据中心电力设备；
- 液冷核心材料。

核心经验：

**价格暴涨带来的利润，不等于长期成长利润；高价会刺激扩产，高利润会召唤供给。**

## 四、六氟磷酸锂与电解液价格曲线

### 1. 产业含义

六氟磷酸锂是电解液核心材料，是天赐材料、多氟多股价弹性的核心变量。

电解液和六氟价格解释了：

- 2016–2017 第一轮锂电材料行情；
- 2021–2022 材料涨价泡沫；
- 2023–2025 材料股利润坍塌。

### 2. 阶段划分

| 阶段 | 六氟 / 电解液状态 | 股价含义 |
|---|---|---|
| 2015–2017 | 第一轮供需紧张和涨价交易 | 天赐、多氟多第一次进入材料主线 |
| 2018–2019 | 补贴退坡后价格回落 | 材料股周期属性显现 |
| 2020 | 需求修复，价格底部企稳 | 第二轮材料行情前夜 |
| 2021 | 六氟价格暴涨，电解液毛利率上行 | 天赐、多氟多成为后半段弹性标的 |
| 2022 | 高位后市场开始交易扩产和价格见顶 | 报表利润好，但股价风险升高 |
| 2023–2024 | 价格大幅回落，产能过剩，毛利率下行 | 材料股进入阴影期 |
| 2025–至今 | 低位修复观察 | 需要看价格、开工率和行业出清 |

### 3. 对应标的

- 天赐材料：电解液 / 六氟一体化龙头；
- 多氟多：六氟单品价格弹性更强；
- 宁德、比亚迪、亿纬：电池成本端受影响。

### 4. 对 AI 的映射

六氟 / 电解液对应 AI 中的单品材料涨价弹性：

- 液冷材料；
- 高端 CCL；
- 高速连接材料；
- 光模块辅材；
- 电源核心部件。

核心经验：

**单品价格弹性越强，高点越要警惕扩产和价格回落。**

## 五、隔膜价格曲线

### 1. 产业含义

隔膜不是最强价格弹性品种，但它是高壁垒材料的代表。

恩捷股份的股价核心不是单纯价格暴涨，而是：

- 高端湿法隔膜国产化；
- 客户认证；
- 工艺和良率；
- 规模和设备壁垒；
- 后期价格竞争和毛利率下行。

### 2. 阶段划分

| 阶段 | 隔膜状态 | 股价含义 |
|---|---|---|
| 2016–2017 | 国产替代和高端湿法隔膜认知形成 | 恩捷进入高壁垒材料候选池 |
| 2018–2019 | 补贴退坡，价格和估值承压 | 龙头客户和工艺优势被筛选 |
| 2020 | 动力电池需求修复 | 恩捷进入主升浪准备 |
| 2021 | 供需偏紧，龙头重估 | 高壁垒材料被机构抱团 |
| 2022 | 高位维持，但扩产预期增强 | 市场开始担心未来价格竞争 |
| 2023–2024 | 产能释放，价格竞争，毛利率下行 | 恩捷进入阴影 / 修复观察 |
| 2025–至今 | 等待毛利率和开工率企稳 | 不能只看出货量，要看 ASP 和折旧压力 |

### 3. 对 AI 的映射

隔膜对应 AI 中高壁垒材料 / 部件：

- 高端 PCB；
- 高端 CCL；
- 先进封装材料；
- 高可靠性液冷部件；
- 电源核心器件。

核心经验：

**有客户认证和工艺壁垒，不代表不会被扩产、折旧和客户压价周期化。**

## 六、钴、镍与三元材料价格曲线

### 1. 产业含义

钴、镍是华友钴业的重要价格变量，也解释了三元路线和磷酸铁锂路线变化对股价的影响。

### 2. 阶段划分

| 阶段 | 钴 / 镍状态 | 股价含义 |
|---|---|---|
| 2016–2017 | 钴价第一轮上涨，三元电池预期强化 | 华友成为钴资源行情核心 |
| 2018–2019 | 钴价回落，三元预期降温 | 第一轮资源行情退潮 |
| 2020 | 新能源车需求修复，三元材料重估 | 华友从资源向材料平台切换 |
| 2021–2022 | 镍钴资源、前驱体和三元预期强化 | 资源 + 材料双属性高位 |
| 2023–2024 | 磷酸铁锂占比提升，钴镍波动，材料利润承压 | 华友估值承压 |
| 2025–至今 | 路线和价格再验证 | 看三元需求、资源价格和前驱体利润 |

### 3. 对 AI 的映射

钴镍价格对应 AI 中技术路线影响资源价值：

- 铜连接 vs 光连接；
- 风冷 vs 液冷；
- GPU vs ASIC；
- 云端推理 vs 端侧推理；
- 传统封装 vs 先进封装。

核心经验：

**资源价值不只取决于需求，还取决于技术路线。**

## 七、硅料价格曲线

### 1. 产业含义

硅料是光伏周期里最重要的上游价格变量，直接解释通威股份的后半段强势和 2023–2025 退潮。

### 2. 阶段划分

| 阶段 | 硅料状态 | 股价含义 |
|---|---|---|
| 2015–2017 | 光伏装机高增，上游认知形成 | 通威进入光伏上游候选池 |
| 2018–2019 | 531 后承压，行业出清 | 龙头低成本能力被筛选 |
| 2020 | 需求修复，供需趋紧预期 | 通威进入主升浪 |
| 2021 | 硅料价格快速上涨，利润向上游集中 | 通威后半段强于部分组件 / 硅片公司 |
| 2022 | 硅料高利润阶段 | 利润最好，扩产风险同步积累 |
| 2023–2024 | 产能释放，硅料价格大幅下跌 | 通威从稀缺成长切回周期制造 |
| 2025–至今 | 低位 / 反内卷观察 | 看供给出清和价格修复 |

### 3. 对应标的

- 直接：通威股份；
- 间接：隆基、福斯特、组件链、光伏设备链。

### 4. 对 AI 的映射

硅料对应 AI 中瓶颈产能：

- HBM；
- 电力设备；
- 变压器；
- 液冷核心部件；
- 高端封装产能。

核心经验：

**瓶颈利润最强的环节，一旦扩产完成，也会成为利润反噬最剧烈的环节。**

## 八、硅片、电池片、组件价格曲线

### 1. 产业含义

这三条曲线解释隆基绿能和光伏主链为什么进入长期阴影。

需求和装机仍然增长，但组件、硅片、电池片价格下跌，导致制造端盈利和估值下行。

### 2. 阶段划分

| 阶段 | 主链价格状态 | 股价含义 |
|---|---|---|
| 2015–2017 | 装机高增，产业链景气 | 隆基等主链龙头第一轮认知 |
| 2018 | 531 后价格下跌 | 第一轮出清和低点形成 |
| 2019 | 平价上网推动成本下降，龙头集中 | 主升浪前夜 |
| 2020 | 装机修复和双碳推动重估 | 隆基主升 |
| 2021 | 高景气，核心资产估值打满 | 隆基第一次见顶和二次高点 |
| 2022 | 高装机和高预期延续，但扩产加速 | 未来过剩风险积累 |
| 2023–2024 | 组件价格大跌，产能过剩严重 | 光伏核心资产长期阴跌 |
| 2025–至今 | 低价和反内卷观察 | 看行业减产、出清、价格修复 |

### 3. 对应标的

- 隆基绿能：硅片、组件、电池片；
- 通威股份：电池片、硅料；
- 福斯特：辅材被主链价格战传导；
- 阳光电源：受光伏贝塔影响，但相对独立。

### 4. 对 AI 的映射

光伏主链对应 AI 中硬件制造环节：

- AI 服务器；
- 光模块；
- 交换机；
- 数据中心设备；
- 端侧硬件；
- 机器人硬件。

核心经验：

**需求高增长但供给扩得更快，硬件价格会下跌，股价也会从成长切回制造。**

## 九、EVA / POE 胶膜与光伏辅材价格曲线

### 1. 产业含义

胶膜解释福斯特的辅材周期。

福斯特比主链制造更稳，但并非不受周期影响。组件价格战和客户压价最终会向胶膜等辅材传导。

### 2. 阶段划分

| 阶段 | 胶膜 / 辅材状态 | 股价含义 |
|---|---|---|
| 2015–2017 | 光伏装机高增，胶膜需求提升 | 福斯特被认知为辅材隐形冠军 |
| 2018 | 531 后订单和估值承压 | 随光伏板块回撤 |
| 2019 | 平价逻辑修复，龙头份额稳定 | 主升前夜 |
| 2020 | 光伏主升，胶膜需求高增 | 福斯特进入核心资产池 |
| 2021–2022 | EVA / POE 胶膜需求和产品结构升级 | 高盈利和高估值阶段 |
| 2023–2024 | 主链价格战传导，毛利率承压 | 福斯特半穿越 / 周期承压 |
| 2025–至今 | 等待胶膜盈利中枢和电子材料第二曲线 | 看毛利率和第二曲线兑现 |

### 3. 对 AI 的映射

胶膜对应 AI 辅材和耗材：

- CCL；
- PCB 材料；
- 液冷辅材；
- 光模块辅材；
- 服务器结构材料；
- 数据中心建设耗材。

核心经验：

**辅材龙头更稳，但不能脱离主产业链价格周期。**

## 十、价格曲线与 14 只标的对应表

| 标的 | 最关键价格变量 | 解释的股价阶段 |

### 核心结论

新能源价格曲线最重要的结论是：

**需求增长只决定空间，价格和毛利率决定利润，利润和估值共同决定股价。**

2021–2022 的资源 / 材料行情，本质是价格上涨带来的利润泡沫；2023–2025 的新能源阴影，本质是供给扩张导致价格和毛利率回落。
