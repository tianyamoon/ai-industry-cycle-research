## 一、为什么要做公司财务指标索引

前面已经建立了：

1. 新闻政策时间轴；
2. 价格曲线索引；
3. 政策原文索引；
4. 新闻源清单。

但产业周期最终必须落到公司财务上。

价格上涨不一定带来利润，收入增长不一定带来现金流，出货增长不一定带来 ROIC。

公司财务指标索引要解决的问题是：

1. 股价主升时，财务是否同步改善？
2. 第一次见顶时，收入、利润、毛利率是否已经见顶？
3. 二次冲顶时，是利润兑现还是估值外推？
4. 退潮时，是收入下滑、价格下跌、毛利率下降，还是资本开支和减值压力？
5. 当前状态是穿越、半穿越、阴影，还是出清失败？

核心框架：

**收入 → 毛利率 → 净利率 → 归母净利润 → ROE / ROIC → 经营现金流 → 资本开支 → 存货 / 减值 → 资产负债表。**

## 二、统一财务指标模板

每只标的后续都按以下字段补：

| 指标 | 解释 | 用途 |
|---|---|---|
| 营业收入 | 公司规模和出货扩张 | 判断需求是否兑现 |
| 归母净利润 | 股东利润 | 判断价格和毛利率是否转化为利润 |
| 扣非净利润 | 剔除非经常损益后的利润 | 判断主业质量 |
| 毛利率 | 产品定价权和成本能力 | 判断价格战和利润中枢 |
| 净利率 | 费用、成本、竞争综合结果 | 判断盈利质量 |
| ROE | 股东回报 | 判断成长是否带来回报 |
| ROIC | 投入资本回报 | 判断重资产扩张是否有效 |
| 经营现金流 | 真实现金创造能力 | 判断利润质量 |
| 资本开支 | 扩产投入 | 判断供给扩张和折旧压力 |
| 存货 | 库存和价格风险 | 判断价格下跌时的减值风险 |
| 减值损失 | 高景气扩张后的反噬 | 判断阴影期严重程度 |
| 资产负债率 | 杠杆水平 | 判断低景气承压能力 |
| 有息负债 | 真实财务压力 | 对天齐、重资产公司尤其重要 |
| 海外收入占比 | 全球化能力 | 判断穿越能力 |
| 研发费用 | 技术迭代能力 | 判断第二曲线和技术路线 |
| 产品出货量 | 量的兑现 | 与价格和收入交叉验证 |

## 四、按产业链的财务关注点

### 1. 光伏主链：隆基、通威

核心问题：

- 装机高增是否转化为利润？
- 价格战如何压缩毛利率？
- 扩产后折旧和减值如何影响净利润？
- 龙头是否真的拥有成本优势？

必补指标：

- 硅料 / 硅片 / 电池片 / 组件收入；
- 各业务毛利率；
- 产能和出货量；
- 资本开支；
- 存货；
- 减值；
- 经营现金流；
- 海外收入。

对应股价阶段：

- 2019：低位筛选；
- 2020：主升；
- 2021：核心资产高点；
- 2022：利润和扩产高点；
- 2023–2025：价格战、减值和阴影。

### 2. 光伏电力电子与辅材：阳光、福斯特

核心问题：

- 是否比主链更稳定？
- 海外化是否提供高毛利？
- 储能是否是真第二曲线？
- 辅材是否被主链压价传导？

必补指标：

- 逆变器收入和毛利率；
- 储能收入和毛利率；
- 海外收入占比；
- 胶膜收入和毛利率；
- 电子材料收入；
- 经营现金流；
- 存货。

对应股价阶段：

- 2020–2021：重估；
- 2021H2–2022：储能 / 胶膜延续；
- 2023–2025：半穿越验证。

### 3. 整车与电池：比亚迪、宁德、亿纬

核心问题：

- 价格战是否压缩利润？
- 龙头是否能用成本和规模换份额？
- 储能第二曲线是否有利润？
- 资本开支和 ROIC 是否匹配？

必补指标：

- 新能源车销量；
- 单车收入；
- 单车利润；
- 汽车业务毛利率；
- 动力电池收入和毛利率；
- 储能收入和毛利率；
- 资本开支；
- 研发费用；
- 海外收入；
- 经营现金流。

对应股价阶段：

- 2020：产品和全球化重估；
- 2021–2022：渗透率和销量兑现；
- 2023–2025：价格战和公司阿尔法验证。

### 4. 锂资源：赣锋、天齐

核心问题：

- 锂价如何影响收入和利润？
- 高价利润是否被当成长期成长利润？
- 低价时存货和减值压力如何？
- 天齐的有息负债和财务费用如何随锂价变化？

必补指标：

- 锂盐收入；
- 锂盐毛利率；
- 锂资源权益；
- 存货；
- 减值；
- 经营现金流；
- 有息负债；
- 财务费用；
- 资产负债率。

对应股价阶段：

- 2016–2017：第一轮资源行情；
- 2018–2019：价格回落；
- 2021–2022：第二轮锂价泡沫；
- 2023–2025：阴影和周期修复。

### 5. 锂电材料：天赐、多氟多、恩捷、华友

核心问题：

- 产品价格和毛利率是否同步？
- 高壁垒材料是否仍会被扩产和客户压价？
- 三元路线和磷酸铁锂路线如何影响华友？
- 单品价格弹性和平台能力如何区分？

必补指标：

- 电解液收入和毛利率；
- 六氟收入和毛利率；
- 隔膜收入和毛利率；
- 三元前驱体 / 正极收入和毛利率；
- 钴镍资源收入；
- 资本开支；
- 存货；
- 减值；
- 现金流。

对应股价阶段：

- 2016–2017：第一轮材料认知；
- 2021–2022：材料涨价和利润高点；
- 2023–2025：价格崩塌和毛利率下行。

### 6. 设备：先导智能

核心问题：

- 设备订单高峰是否可持续？
- 下游扩产放缓后，订单和回款如何变化？
- 应收账款和合同负债是否领先股价？
- 新技术路线能否带来新一轮设备周期？

必补指标：

- 新签订单；
- 在手订单；
- 收入；
- 毛利率；
- 应收账款；
- 合同负债；
- 存货；
- 经营现金流；
- 前五大客户收入占比。

对应股价阶段：

- 2015–2017：设备国产化；
- 2020–2022：下游扩产高峰；
- 2023–2025：订单周期退潮。

## 七、对 AI 的财务映射

新能源财务指标对 AI 的启示：

| 新能源指标 | AI 对应指标 | 研究意义 |
|---|---|---|

| 储能毛利率 | AI 第二曲线毛利率 | 判断第二曲线是收入还是利润 |
| 资本开支和 ROIC | 算力投入回报 | 判断扩产是否创造价值 |
| 存货和减值 | AI 硬件库存和跌价准备 | 判断供给过剩是否显性化 |
| 海外收入 | 全球客户能力 | 判断是否具备穿越能力 |
| 研发费用 | 技术路线迭代 | 判断是否有第二增长曲线 |

核心经验：

> AI 产业链不能只看订单和收入，最终要看毛利率、现金流、ROIC、库存和减值。
