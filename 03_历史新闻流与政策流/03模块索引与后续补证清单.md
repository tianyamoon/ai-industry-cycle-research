> 生成日期：2026-06-20

---

## 一、模块定位

03 模块不是普通新闻汇总，而是产业周期证据库。

每条新闻 / 政策 / 数据都要服务于以下问题：

1. 它解释了哪一段行情的启动？
2. 它解释了哪一次调整或出清？
3. 它解释了哪一个价格或利润拐点？
4. 当时市场如何理解？
5. 事后看是否被证实？
6. 对应哪些标的？
7. 对 AI 产业链有什么映射？

统一框架：

**日期 → 事件 → 影响环节 → 当时市场理解 → 事后验证 → 相关标的 → 来源。**

## 六、下一步建议

03 模块底稿已经支撑 14 张单股回填卡第一版。下一步不再扩散新闻索引，而是把新闻 / 政策入口和 06 硬数据实填联动。

优先顺序：

1. 每只样本保留“当时市场在交易什么”的新闻 / 政策起点；
2. 资源链优先对齐锂价、钴镍价格、利润和资产负债表；
3. 材料链优先对齐硅料、组件、六氟、隔膜、胶膜和毛利率；
4. 终端 / 设备 / 平台链优先对齐销量、订单、现金流、ROIC 和第二曲线；
5. 失败 / 阴影样本后续补政策、财报、退市 / 重整 / 业务失败证据。

联动格式：

**新闻 / 政策起点 → K 线阶段 → 第一变量 → 财务拐点 → 事后验证 → 当前状态。**
