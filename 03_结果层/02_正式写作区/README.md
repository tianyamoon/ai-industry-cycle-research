# 正式写作区

本目录属于：

`结果层`

职责不是做证据生产，也不是做桥接卡，而是承接已经成熟的：

1. 正式正文
2. 结果层长文
3. 基金风格输出
4. 可读性优先的最终成稿

---

## 使用边界

只有在以下条件基本满足时，内容才应进入本目录：

1. 过程层已有足够证据支撑。
2. 桥接层已有清晰研究骨架或样本验证。
3. 关键判断能被还原为 `事实 -> 变量 -> 传导 -> 验证 -> 映射`。
4. 主要缺口已被显式标注，而不是被隐藏。

## 写作纪律

1. 不把原始证据堆回正文。
2. 不把桥接层观察池直接升级成正式结论。
3. 先组织读者路径，再组织作者研究过程。
4. 正文内优先保留结论、证据摘要、风险与边界。
5. `AI交易策略日报` 这类按日文件，根目录只保留稳定入口；dated 日报与附录统一归档到 `90_历史归档/`。
