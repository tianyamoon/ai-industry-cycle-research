## 1. 一句话最优策略

今天最优策略是 `防守性持有审计 + 等右侧`：全市场宽度明显走弱，AI 组内只剩半导体设备、先进封装和少数存储 / 国产芯片强者维持强势，新增进攻不追高；`NVIDIA 800VDC` 提升供配电 / 电源链催化权重，但 A 股订单、收入、OCF 还未闭环，只能作为观察触发器。

执行含义：

1. 已持有高分方向，只做 `持有审计 / 分歧承接后再评估`。
2. 新开仓只允许小仓、右侧、计划内；不因单日反弹追高。
3. AIDC 供配电 / 800VDC 只盯右侧条件，不因叙事新鲜直接升级为主攻。

## 3. Top 策略展开

### 3.1 半导体设备：强势持有审计，不追高

为什么入选：

1. 事实：半导体设备样本组当日均值 `+1.69%`，20日 `+52.34%`，60日 `+116.82%`，组内 4/4 月线趋势向上。
2. 变量：市场仍愿意为国产设备订单、利润兑现、存储扩产和先进制程国产替代支付溢价。
3. 传导：订单 / 在手订单 -> 收入确认 -> 利润率 / OCF -> 估值维持。
4. 验证：盛美上海 `2026-07-07` 收盘 `+6.30%`，长川 / 中微仍在高位强趋势区。
5. 应对：这是持有审计，不是无脑新增买入。

可执行动作：

1. 已持有：保留核心仓位，日内冲高不加，回撤看承接和成交钝化。
2. 未持有：只等分歧承接后小仓试错；不得在连续加速后追买。
3. 仓位上限：单一方向不超过组合 `25%`；单票不超过 `8%-10%`。

加仓 / 减仓 / 退出触发：

1. 加仓：组内龙头回踩后不破 20日结构，成交缩量，且公告 / 中报继续验证订单或利润。
2. 减仓：组级 5日继续转弱，距60日高点回撤扩大到 `-20%` 且反弹无量。
3. 退出：中报显示收入 / 利润 / OCF 无法承接当前估值，或主锚利润兑现不扩散。

### 3.2 先进封装 / 封测 / 材料：只看强者留存

为什么入选：

1. 事实：组级当日 `+4.34%`，20日 `+32.95%`，长电科技当日 `+6.20%`、华天科技涨停，资金仍在先进封装方向聚集。
2. 变量：AI 芯片高算力封装瓶颈、HBM / CoWoS / 先进封装扩产预期继续支撑交易。
3. 传导：CapEx 增加 -> 封测 / 材料订单 -> 收入确认 -> 利润率和现金流验证。
4. 验证：资金榜显示“先进封装”仍有主力净流入，但组内已高位分化。
5. 应对：只看强者留存，不做低位补涨臆测。

可执行动作：

1. 已持有强者：以 `持有 + 动态减仓线` 管理。
2. 新仓：只允许在强势标的分歧后缩量承接，且不得追涨停后排。
3. 仓位上限：方向上限 `20%`，高弹性材料小票单票不超过 `5%`。

触发条件：

1. 加仓：平台龙头和材料弹性同步修复，而不是只有单只小票冲高。
2. 降级：只剩小票拉升、平台龙头掉队，或 20日斜率明显钝化。
3. 退出：收入侧无法验证 AI 相关订单，或估值进一步失真并出现放量滞涨。

### 3.3 AIDC供配电 / NVIDIA 800VDC：催化升级，但动作仍是等待

为什么入选：

1. 事实：项目内 `NVIDIA_800VDC供电架构与A股映射评分_2026-07-07.md` 已确认 NVIDIA 官方 800VDC 路线和生态伙伴是新增变量。
2. 变量：AI factory 从多级 AC / 低压 DC 向 800VDC / HVDC 架构迁移，提升机柜电源、DC/DC、保护、母线和供配电系统重要性。
3. 传导：官方路线 -> 海外生态 / 国内客户导入 -> A股公司产品 / 订单 -> 收入 / 毛利率 / OCF。
4. 验证：麦格米特 / 中恒电气 / 欧陆通在专题评分里靠技术路径和预期靠前，科士达 / 科华 / 金盘靠经营质量补分。
5. 应对：当前仍是 `B/C 阶段样本`，不是正式 A 级交易结论。

可执行动作：

1. 当前动作：观察，不主动追。
2. 右侧触发：组级20日重新转正且 60日突破 `30%`；至少一条路径出现 `订单 -> 收入 -> OCF` 强化；月线和周线同时向上。
3. 仓位上限：触发前 `0%-5%` 观察仓；触发后再讨论 `10%-15%` 方向仓。

证伪条件：

1. 800VDC 只停留在路线图和研报预期，A 股公司无订单 / 客户 / 收入披露。
2. 核心样本继续弱于科创50和半导体设备。
3. 订单增长伴随应收、存货、OCF 恶化。
