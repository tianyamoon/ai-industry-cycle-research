### 2.3 机构态度拆解

#### 亿欧智库｜人工智能OPC模式商业洞察

1. `机构态度`
   - 更接近 `结构梳理`，不是产业验证或订单前瞻。
   - 它在强化 `AI 把创业组织形态从小团队压缩到单人+工具链` 的应用层叙事。
2. `它真正强调的变量`
   - `AI 工具协同` 对产品设计、研发、运营、客服等标准化环节的替代。
   - 细分场景里 `项目制 -> 产品化 -> 平台化` 的演进路径。
3. `对仓库的意义`
   - 有助于补应用层的方法边界：`AI应用` 不只看软件公司收入，也要看组织效率重构是否真的发生。
   - 但它没有给出 A 股映射公司的订单、ARPU、留存或利润兑现，不能上升为事实层。
4. `仍待核`
   - `单人+AI` 组织形态是否能在国内可投资映射标的上形成可验证的商业模式，而不只是概念拆解。

#### 头豹研究院｜存储芯片洞察

1. `机构态度`
   - 更接近“赛道结构梳理 + 存储认知升级”，不是交易型强观点。
   - 它在强化 `HBM 是 AI 时代关键存储形态`，但没有把结论直接落到 A 股具体公司盈利兑现。
2. `它真正强调的变量`
   - 存储形态分层：DRAM / NAND / HBM / LPDDR / 嵌入式存储各自分工。
   - HBM 的高带宽与 3D 堆叠属性，使其在 AI / HPC 场景中从“配角”变成“战略资源”。
3. `对仓库的价值`
   - 适合补强桥接层里 `HBM -> 存储 -> 端侧存储` 的概念边界。
   - 不足以单独上调样本权重，因为没有订单、产能利用率、毛利率或 CapEx 兑现链。
4. `仍待核`
   - HBM 产业价值重估是否已经传导到国内映射公司的订单、ASP、利润率和估值中枢。

#### 东吴证券｜存储测试机专题

1. `机构态度`
   - 是明显偏积极的设备链观点，核心不是泛泛看多存储，而是押注 `HBM 将把测试环节抬成新的瓶颈与弹性点`。
   - 这是本轮行业研报里最接近“可继续追踪经营变量”的一篇。
2. `它真正强调的变量`
   - DRAM / NAND 现货涨价、两存扩产、长鑫存储高产能利用率。
   - HBM 测试道数从传统 DRAM 的 `3-4 道` 提升到 `15 道以上`，测试机需求约为传统 DRAM 的 `5-6 倍`。
   - 存储测试机国产化率仅 `8%-10%`，远低于其他测试设备细分赛道。
3. `机构的隐含押注`
   - 不是押整个半导体设备普涨，而是押 `HBM扩产 -> KGSD等前移测试 -> 测试设备国产替代` 这条更窄、更硬的链。
   - 推荐口径已经明确落到 `长川科技 / 精智达 / 联讯仪器`，说明它在用个股映射承接行业判断。
4. `对仓库的价值`
   - 可把 `半导体设备全景组` 中测试设备的优先级抬高一档。
   - 也说明 `HBM` 不应只看存储本体，还要看 `测试/封装/材料` 的二阶受益链。
5. `仍待核`
   - 设备推荐是否已经出现 `订单 -> 验收 -> 收入 -> 利润` 的连续兑现，而不是只停留在技术方向正确。

#### 开源证券｜AIDC量价或通胀

1. `机构态度`
   - 态度非常激进，核心不是“算力建设继续”，而是直接推进到 `AIDC 供需错配 + 存量资产定价权提升`。
   - 这是典型的“产业判断先行，经营兑现待后证”的强叙事研报。
2. `它真正强调的变量`
   - 国内 CSP CapEx 上修、豆包日均 Token 调用量大幅增长、国产超节点加速部署。
   - 交换芯片与 GPU 配比抬升，交换网络在系统级互连里的重要性提升。
   - 以新意网案例推演上架率提升与供需剪刀差，进而得出 AIDC 长期涨价逻辑。
3. `机构的隐含押注`
   - 押注的不是单一公司，而是 `土地/电力/机柜/液冷/交换网络` 全链条资产重估。
   - 对仓库最重要的提醒是：市场可能已经从“算力设备”往“算力基础设施定价权”迁移叙事。
4. `需要警惕的地方`
   - `字节CapEx、Token调用量、上架率、涨价逻辑` 中有相当部分是引述口径或机构推演，不能直接当仓库事实层。
   - 它对 `AIDC涨价` 的判断目前更像假设，不是已经被 A 股报表广泛验证的事实。
5. `对仓库的价值`
   - 值得继续追 `光环新网 / 奥飞数据 / 宝信软件 / 润泽科技 / 科华数据` 这条观察链。
   - 但不能因此跳过 `上架率 / 利用率 / 单柜价格 / OCF` 的硬核验。

#### 中银证券｜AI基建周期或将长达数十年

1. `机构态度`
   - 更偏中观和长周期叙事，态度积极，但不是短线交易建议。
   - 它在强化 `Token经济 + AI工厂 + 长周期基础设施重置` 这个顶层框架。
2. `它真正强调的变量`
   - 豆包 2.1 Pro 的多模态、Agent、性价比与企业级稳定性。
   - 黄仁勋关于 `AI数据中心是制造 token 的工厂` 的表述。
3. `机构的隐含押注`
   - 押的是 `模型能力提升 -> Token吞吐增长 -> AI工厂长期扩张` 的总叙事。
   - 相比开源，它更少落到资产涨价，更强调“长赛道成立”。
4. `对仓库的价值`
   - 适合放进 `方法论/预期层`，作为为什么 AI 基建不应被单季度波动轻易证伪的背景材料。
   - 不适合作为当期 A 股样本已经兑现的证据。
5. `仍待核`
   - 长周期成立，不代表每个 A 股映射样本都能享受到同等盈利兑现；这仍要回到个股财务和订单。

### 3.3 机构态度拆解

#### 东吴证券｜芯碁微装

1. `机构态度`
   - 明显偏积极，且不是普通设备股推荐，而是试图把公司抬到 `AI资本开支底层工具` 的位置。
   - 这更接近 `已有主线加强 + 候选扩容线索` 的中间态。
2. `机构最看重什么`
   - 公司披露 `PLP2000` 板级封装直写光刻设备拿到先进封装头部客户订单。
   - 设备最大支持 `600×600mm` 大板加工，可覆盖 `CoPoS / FOPLP / 玻璃基板封装 2μm` 量产制程需求。
   - 同日机构把 `鹏鼎控股` 的 `AI服务器 / 高速光模块高阶HDI基板` 扩产，作为下游需求扩张旁证。
3. `机构真正押注的变化`
   - 押 `AI服务器与高速光模块基板扩产 -> PCB光刻与先进封装曝光设备需求抬升`。
   - 更深一层是押公司从 `PCB LDI` 向 `先进封装曝光设备` 延伸成功，而不是只吃传统 PCB 周期。
4. `对仓库的意义`
   - 它提示 `先进封装` 不应只盯封测厂和材料，也要单列 `曝光/光刻设备` 这一层。
   - 现阶段更适合定义为 `候选扩容线索`，不能直接抬成核心样本，因为订单还只是单点验证。
5. `仍待核`
   - 头部客户订单是否形成持续复购与更大批量交付。
   - 公司 `先进封装设备` 收入占比、毛利率和现金回收情况是否持续改善。

#### 东吴证券｜并行科技

1. `机构态度`
   - 这是明显的首次覆盖型积极观点，核心标签是 `低估 + 算力调度平台稀缺性`。
   - 它不是只把并行科技当机柜租赁商，而是试图把公司抬到 `算力租赁 + MaaS 平台 + 运维服务` 的复合叙事。
2. `机构最看重什么`
   - 2025 年营收 `11.10 亿元`、同比 `+69.56%`，2026Q1 收入同比 `+117%`。
   - `200万 CPU 核心 + 5万+ GPU 卡 + 1300+ 企业客户`。
   - 今年以来公告算力资产购买总额 `12.78 亿元`，说明公司在逆周期扩张供给能力。
3. `机构真正押注的变化`
   - 押 `算力资源供给能力 + 调度平台能力 + AI需求上升` 共同推升估值修复。
   - 它认为并行科技与传统服务器/IDC 公司的差别在于调度层和 MaaS 生态，而不是单纯资产堆砌。
4. `对仓库的意义`
   - 值得纳入 `算力租赁 / 跨界算力转型` 的重点观察队列。
   - 但现阶段更像 `预期差样本`，还不是已经闭环的强样本。
5. `仍待核`
   - 资产投入之后能否形成 `上架 -> 起租 -> 回款 -> OCF` 闭环。
   - GPU 卡的新购置是否伴随租赁利用率与单卡回报率同步改善。

#### 国信证券｜特锐德

1. `机构态度`
   - 是“主业稳健 + 新叙事打开估值空间”的典型增持逻辑。
   - 真正吸引机构的，不只是 AIDC，而是 `电力设备出海 + 算电岛新品` 双逻辑叠加。
2. `机构最看重什么`
   - 海外收入 `13.62 亿元`、同比 `+189%`，国际业务中标额超过 `12 亿元`。
   - 2026 年 6 月首发 `算电岛`，切入高压交直流预制舱供电站方案。
   - 与伊顿电源合作，补强算力与新能源场景一体化电力方案。
3. `机构真正押注的变化`
   - 押注 `AI 用电需求上升 -> 供电架构升级 -> 特锐德凭电力设备底子切入算力中心供电`。
   - 本质上是“电力设备龙头跨到算力供配电”，不是纯 AI 原生公司。
4. `对仓库的意义`
   - 适合放进 `AI算力基建大组扩展观察池` 的旁证层。
   - 可作为 `供配电并非只看金盘/科华/科士达` 的样本扩容提醒。
5. `仍待核`
   - `算电岛` 是否进入正式订单与收入拆分。
   - 市场是否愿意持续给这类“低 AI 纯度、高工程属性”公司更高估值。

#### 中邮证券｜长电科技

1. `机构态度`
   - 这是标准的“平台型封测龙头重新定价”逻辑，态度积极且偏长期。
   - 它不是押单一订单，而是押 `先进封装平台能力 + 临港扩产 + HPC/存储/CPO 多场景覆盖`。
2. `机构最看重什么`
   - 2025 年营收 `388.71 亿元`、同比 `+8.09%`，毛利率 `13.95%`、同比提升 `1.07pct`。
   - 上海临港高端先进封测工厂总投资 `78 亿元`。
   - `XDFOI` 已量产，硅光引擎完成客户样品验证，存储封测与全球前三大存储器制造商合作。
3. `机构真正押注的变化`
   - 押 `先进封装平台化能力`，而不是纯赌某一个 AI 芯片单点客户。
   - 也在押长电能从传统封测估值框架，向更偏先进封装 / HPC 的框架切换。
4. `对仓库的意义`
   - 它强化了 `先进封装` 不能只看情绪，需要看平台能力、扩产节奏与客户导入。
   - 对 `HBM / 存储 / CPO / 功率封装` 的交叉映射价值高。
5. `仍待核`
   - 临港扩产是否按节奏推进。
   - 高端先进封装收入占比、盈利质量、客户集中度是否持续改善。

#### 山西证券｜鼎龙股份

1. `机构态度`
   - 典型的材料成长股买入逻辑，押的是 `高端光刻胶放量 + 第二成长曲线外延`。
   - 相比讲赛道，它更强调订单验证已经开始出现。
2. `机构最看重什么`
   - 两家头部晶圆厂新增订单近 `1000 加仑`。
   - 高端光刻胶、CMP、半导体材料多品类推进。
   - 收购皓飞，强化第二成长曲线。
3. `机构真正押注的变化`
   - 押 `从送样/验证走向批量订单`，说明逻辑已经从“国产替代故事”往“收入兑现前夜”迁移。
   - 对材料股而言，这比单纯讲政策扶持更有价值。
4. `对仓库的意义`
   - 可以把鼎龙从普通候选池往 `重点回源队列` 再前推一步。
   - 也提醒仓库在半导体材料里优先看 `订单批量化`，而不是只看产品谱系是否齐全。
5. `仍待核`
   - 批量订单能否延续成季度收入与毛利改善。
   - 外延并购对主业现金流和资本开支节奏是否形成压力。

#### 开源证券｜锐捷网络

1. `机构态度`
   - 是明显的业绩上修型积极观点，核心在于 `交换机需求已经从叙事进入利润兑现`。
   - 相比行业研报，它更像对 AIDC 网络侧的一次财务验证。
2. `机构最看重什么`
   - 公司业绩超预期，并上调 2026-2028 年归母净利润预测。
   - `800G` 交换机继续放量，推出 `128口 800G` 高性能交换机。
   - `NPO / CPO` 技术路线并行，JDM 白盒方式渗透大客户。
3. `机构真正押注的变化`
   - 押注互联网大客户的数据中心交换机需求已经进入业绩兑现。
   - 同时押公司未来能从商用交换机经验迁移到白盒/JDM 体系。
4. `对仓库的意义`
   - 它是 `AIDC网络侧` 里比纯行业叙事更硬的旁证。
   - 说明如果后续还要扩样，`交换机 / 光互连 / 网络设备` 应保持独立观察，不应被液冷或供配电淹没。
5. `仍待核`
   - 业绩高增是否具有可持续性，还是集中出货导致的短期利润释放。
   - CPO / NPO 从样机、小规模适配走到规模化，还需要时间。

#### 东吴证券｜雷赛智能

1. `机构态度`
   - 明显偏积极，核心不只是半年度业绩超预期，而是把公司讲成 `工控景气恢复 + 机器人第二曲线放量` 的双轮驱动。
   - 这是机器人链里少数把订单、利润、产能一起讲出来的条目。
2. `机构最看重什么`
   - 2026H1 归母净利润预计 `1.84-1.96 亿元`、同比 `+55%-65%`。
   - 26Q2 归母净利润预计 `1.12-1.24 亿元`、同比 `+78%-97%`。
   - 无框力矩电机 25 年交付超过 `12 万台`，同比 `20倍+`，在建产能 `200 万台`。
3. `机构真正押注的变化`
   - 押 `3C/电子/半导体复苏` 对工控主业的拉动。
   - 同时押 `人形机器人零部件` 进入更实质的订单和批量供货阶段。
4. `对仓库的意义`
   - 比单纯“机器人概念扩散”更扎实，适合放到 `机器人 / 端侧 / 具身` 观察层靠前位置。
   - 但它依然不是仓库长周期主线，因为工控主业与机器人业务的估值混合仍较重。
5. `仍待核`
   - 机器人零部件收入占比、毛利率、客户粘性是否持续改善。
   - 工控主业高增是否只是景气修复，而非长期中枢抬升。

#### 金元证券｜中国电影

1. `机构态度`
   - 偏温和积极，本质是“内容龙头 + AI 工具应用”而非纯 AI 成长股。
   - AI 在这里是效率改善因子，不是业绩核心矛。
2. `机构最看重什么`
   - AI 导演创作平台、AI 配音译制辅助系统、AI 审核辅助系统。
   - 影片储备与发行份额，而不是单独的 AI 变现数据。
3. `机构真正押注的变化`
   - 押 `AI 降本增效`，叠加未来内容供给改善，带动创作和发行两端修复。
   - 不是押订阅制或 SaaS 式 AI 收费模式。
4. `对仓库的意义`
   - 它提醒应用层要分清：有 AI 工具接入，不等于形成 AI 经营主线。
   - 更适合放在 `应用叙事延伸`，不适合抬升为应用兑现证据。
5. `仍待核`
   - AI 工具对成本率、项目周期、票房转化的真实贡献。

#### 补充观察｜中邮证券｜天准科技

1. `机构态度`
   - 是“检测设备平台化 + 半导体前道检测 + 具身智能控制器”的复合型积极观点。
   - 它不在原先主表重点内，但值得作为新增旁证观察。
2. `机构最看重什么`
   - 苏州矽行在手销售和 Demo 订单合计 `6887 万元`。
   - 人形机器人核心域控制器新增订单 `4650.71 万元`。
   - 同时覆盖光模块检测、PCB 检测、晶圆前道缺陷检测。
3. `对仓库的意义`
   - 它连接了 `算力检测设备` 与 `具身智能控制器` 两条链，说明视觉检测设备平台型公司可能是漏掉的交叉样本。
4. `仍待核`
   - 晶圆检测设备订单能否持续扩张并形成更稳定收入占比。
   - 机器人控制器订单是否能走向规模化、重复化。

1. `并行科技` 的低估值修复逻辑，仍是机构估值判断，不等于未来必然兑现。

2. `特锐德` 需要继续核验 `算电岛` 是否进入正式订单、收入拆分和现金流层。
3. `长电科技` 需要继续核验新封测厂建设节奏、中报利润兑现与高端封装收入占比。
4. `鼎龙股份` 需要继续核验光刻胶批量订单是否延续，以及是否进入更大规模收入兑现。
5. `芯碁微装` 需要继续核验 `先进封装设备订单 -> 交付 -> 收入 -> 毛利率` 是否走成连续闭环。

### 4.3 海外机构态度拆解

#### Citi｜China's 2H outlook

1. `机构态度`
   - 不是简单“看多中国”，而是认为 `AI超级周期` 正在切入中国 `K 型分化` 的上半段资产。
   - 这更接近 `宏观结构分化 + 赛道强化`。
2. `它真正押注的变量`
   - AI 资本开支与科技投资对增长结构的边际支撑。
   - 地产与传统链条拖累仍在，说明不是全面景气抬升。
3. `对仓库的意义`
   - 强化了仓库当前对 `AI强、旧经济弱` 的结构分化镜头。
   - 但它仍是宏观与资产配置层判断，不能替代 A 股样本的订单、收入和利润验证。
4. `仍待核`
   - 中国 AI 投资是否能形成更广泛企业盈利兑现，而不只是少数核心资产受益。

#### Citi｜Overcoming Gridlock 2.0

1. `机构态度`
   - 不是泛泛谈 AI 用电，而是明确把 `电力瓶颈` 提到与芯片、算力并列的位置。
   - 更接近 `赛道强化 + 风险约束重估`。
2. `它真正押注的变量`
   - 电网审批、输电能力、发电新增投资与数据中心负荷增长。
   - AI 负荷对能源安全与公用事业资本开支的二次抬升。
3. `对仓库的意义`
   - 与仓库当前的 `AIDC供配电` 主线高度共振，说明不能把供电侧只当配角。
   - 但目前还缺 `2家全球投行 + 1家资管/宏观机构 + 1条原始事实源` 的完整校验链，因此只能停在 `观察层`。
4. `仍待核`
   - 国内 `单柜功率、上架率、电价、并网周期` 是否已出现类似瓶颈证据。
