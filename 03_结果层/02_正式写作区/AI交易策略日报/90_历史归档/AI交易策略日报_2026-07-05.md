## 3. Top 1-3 策略

### Top 1：终端 / 具身智能

**为什么入选**

1. 价格层：样本组同时满足周线、月线正斜率，是本轮候选池里最接近新增右侧观察条件的方向。
2. 事实层：雷赛智能 2026H1 业绩预告与补充说明公告已出现，今日研报扫描中也把机器人 / 具身相关收入线索列为可跟踪对象。
3. 传导层：若 AI 资金从高位算力硬件切向低位或中位应用 / 终端，具身智能可能成为承接方向之一。
4. 验证层：必须看 2026-07-06 是否继续强于创业板 / 科创50，并且不是单一高标脉冲。

**可执行动作**

只做 `观察 -> 右侧确认 -> 小仓试错` 的研究纪律：若 2026-07-06 组内至少两个样本继续强于指数，且强势股分歧后承接有效，可列入盘前观察；若高开冲高回落，不追。

**仓位上限**

新增试错上限 `1成`；若只是研究跟踪，不产生交易动作。已有仓位以 `2成以内趋势仓` 为上限，跌破周线趋势或核心样本放量长阴时降级。

**加仓 / 减仓 / 退出触发**

1. 加分：组内至少两个样本继续周线向上，且雷赛智能 / 奥比中光 / 汇川技术等不出现高位放量回落。
2. 降级：只有单一高标冲刺，或强势股 2026-07-06 放量回撤超过 `-5%` 且不能收回。
3. 退出：月线转负、周线破位，或公告 / 中报无法拆出机器人 / AI 相关收入与利润贡献。

### Top 2：新增研报线索 / 设备材料

**为什么入选**

1. 价格层：芯碁微装、鼎龙股份、锐捷网络组成的可得样本 20 日均值 `+28.97%`，但月线仍 `-3.27%`，说明是短期线索而非完整趋势。
2. 事实层：今日商业研报扫描显示，芯碁微装被机构连接到 `PCB光刻 / 先进封装曝光设备`，鼎龙股份连接到 `高端光刻胶订单验证`，锐捷网络连接到 `数据中心交换机业绩兑现`。
3. 传导层：这条线索把 AI CapEx 从算力卡外溢到设备、材料、网络侧，但当前仍以研报假设和单点订单为主。

**可执行动作**

只列 `候选扩容 / 回源队列`，不列主动买入。明日重点核验是否有公告、调研纪要或中报预告能把研报观点转成 `订单 -> 交付 -> 收入 -> 毛利率 / OCF`。

**仓位上限**

未过月线门槛，不设新增交易仓位；若后续月线转正且公告回源成立，再允许进入 `1成以内试错`。

**加仓 / 减仓 / 退出触发**

1. 加分：芯碁微装先进封装设备订单出现正式公告或收入拆分；锐捷网络业绩兑现继续被财报确认。
2. 降级：研报热度升温但公告 / 财报无验证，或样本 20 日强度开始明显回落。
3. 退出：月线继续下行，或只剩题材语料没有经营变量。

### Top 3：半导体设备

**为什么入选**

1. 价格层：20 日 `+42.23%`、60 日 `+149.21%`，说明仍是强资金参与方向；但周线 `-4.25%`、月线 `-12.64%`，高位分化已经出现。
2. 基本面层：长川科技仍是利润验证主锚，盛美上海是订单强化补强位，中微 / 华峰测控是连续审计样本。
3. 风险层：当前不是低位赔率，而是高位兑现审计；华峰测控等强势样本若回撤扩散，会先压缩板块风险偏好。

**可执行动作**

持有者做 `不追高 + 观察回撤承接`；空仓者等待周线重新转正。不能把设备全景组泛化成所有设备股共同高景气。

**仓位上限**

已有趋势仓不超过 `2成`；新增仓位暂不开放。若 2026-07-06 继续高位放量滞涨，应先降仓而不是加仓。

**加仓 / 减仓 / 退出触发**

1. 加分：长川 / 盛美 / 中微至少两个样本重新周线转正，且出现订单或利润预告扩散。
2. 减仓：强势样本距 60 日高点回撤扩大到 `-20%` 附近，或高开低走后成交放大。
3. 退出：中报不能延续订单、收入、利润和 OCF 的同步强化。
