## 6. 机构态度拆解

### 6.1 爱建证券｜PCB设备

1. `机构到底在强化什么`
   - 强化的是 `AI服务器 / 高速光模块 / 高阶HDI` 扩产，先传导到 `设备端`，而不是等板厂报表兑现后再看。
2. `它真正押注的核心变量是什么`
   - 鹏鼎控股定增、HDI 与 mSAP 产能扩张、光模块 PCB 工艺升级、设备环节的先行资本开支。
3. `对当前仓库结论的意义是什么`
   - 继续强化 `PCB高端扩产 -> 设备金铲子` 主线，适合和 `芯碁微装 / 天准科技 / 大族数控` 一类前置设备样本联动观察。
4. `哪些地方仍然只是推演`
   - 设备景气能否持续并不等于已兑现；当前更多是扩产前瞻。
5. `哪些必须继续追证`
   - 订单、交付、收入、毛利率和 OCF。

### 6.2 国金证券｜AMHS / 晶圆厂物流

1. `机构到底在强化什么`
   - 强化的是 `AI存储扩产 -> 晶圆厂建设 -> 厂务自动化物流` 的前置资本开支传导，而不是末端芯片销量故事。
2. `它真正押注的核心变量是什么`
   - 300mm 晶圆厂扩产、先进制程/先进封装共振、AMHS 市场扩容、海外双寡头垄断与国产替代缺口。
3. `对当前仓库结论的意义是什么`
   - 这是本轮最值得补写的 `可能被忽略的变化`，因为它把硬件景气向更前置的基础设施环节再上溯了一层。
4. `哪些地方仍然只是推演`
   - 国产化窗口、景气外溢和盈利弹性仍主要是研报推演。
5. `哪些必须继续追证`
   - 晶圆厂项目落地、AMHS 中标、洁净室/EPC 订单、收入与回款。

### 6.3 万联证券｜玻璃基 / CPO / 以太网互联升级

1. `机构到底在强化什么`
   - 强化的是头部 AI 厂商开始把竞争延伸到 `网络生态 + 互连路线`，玻璃基与 CPO 是更远端但更关键的系统升级方向。
2. `它真正押注的核心变量是什么`
   - GlassWorks AI、Spectrum-X、数据中心以太网交换机收入增速、CPO/NPO 商业化时间窗。
3. `对当前仓库结论的意义是什么`
   - 说明 `交换机 / 互连 / 玻璃基` 不能拆开看，而应作为同一条系统级互联升级主线。
4. `哪些地方仍然只是推演`
   - 玻璃基与 CPO 的 A 股映射大部分还停留在路线期权。
5. `哪些必须继续追证`
   - 量产、客户导入、订单、ASP 与正式财务传导。

### 6.4 万联证券｜AI应用商业化验证

1. `机构到底在强化什么`
   - 强化的是 AI 应用开始从“高投入叙事”往“付费验证和收入覆盖”迁移。
2. `它真正押注的核心变量是什么`
   - AI 收入与折旧关系、Agent 协同标准、模型调用量、企业/开发者付费转化。
3. `对当前仓库结论的意义是什么`
   - 应用层的确在变强，但更接近 `成熟样本加强 + 广义应用仍待核`，不能据此把整个应用层误升格。
4. `哪些地方仍然只是推演`
   - 海外平台和国内大模型的调用数据，不等于 A 股公司经营兑现。
5. `哪些必须继续追证`
   - 合同、收入、续费、合同负债、OCF。
