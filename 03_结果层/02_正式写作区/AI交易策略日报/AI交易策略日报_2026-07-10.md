## 3. Top 1-3 策略

### 3.1 Top1：服务器 / ODM

**为什么入选**：价格层 20日涨幅居前，且 `000977 浪潮信息` 已在巨潮披露 `2026年半年度业绩预告` 与 `股票交易异常波动公告`。这条链比纯题材更接近 `订单/收入 -> 利润 -> 中报验证`。

**可执行动作**：不做盘中追高；只在下个交易日出现分歧后仍能维持相对强度时，作为右侧观察方向。已有仓位可按趋势纪律持有审计。

**仓位上限**：研究口径建议单方向进攻仓位不超过 `2-3成`；若指数与科创50继续回撤，降到 `0-1成观察`。

**加仓触发**：浪潮 / 紫光 / 曙光至少两只继续强于科创50，且浪潮业绩预告后的承接不失效。

**减仓 / 退出触发**：浪潮业绩预告后高开低走并放量失守，或服务器组 5日强度快速转负。

### 3.2 Top2：半导体设备

**为什么入选**：20日与60日强度最高，且当日商业研报补跟踪显示 `AI存储扩产 -> 量检测/测试设备` 的传导开始被卖方强化。

**可执行动作**：只做强趋势持有审计，不新增追击。今日组内长川、盛美、中微、北方普遍大跌，说明风险经理视角优先级高于进攻视角。

**仓位上限**：已有仓位不超过 `2成`，新增仓位为 `0`，等回撤后周线承接。

**加仓触发**：长川或盛美回撤后重新站回 5日/10日短均并带动设备组修复，同时中报前公告继续给出订单或利润兑现。

**减仓 / 退出触发**：20日趋势转负，或设备公告仍停留在融资、并购、补助、风险提示而无经营兑现扩散。

### 3.3 Top3：先进封装 / 封测 / 材料

**为什么入选**：20日 `+29.35%`、60日 `+100.91%`，仍是硬件强势组；但今日组内上涨占比为 `0%`，说明已进入高位分歧。

**可执行动作**：持有审计，不做接力。只观察长电、通富、华海诚科能否把回撤控制在约 `-10%` 附近。

**仓位上限**：已有仓位 `1-2成` 审计；新增仓位为 `0`，除非高位平台重新确认。

**加仓触发**：平台龙头与材料弹性样本同步修复，且不是小票孤立冲高。

**减仓 / 退出触发**：平台龙头掉队、小票冲高回落，或收入 / AI占比 / CapEx 证据无法跟上估值。
