## 4. Top 策略展开

### 4.1 Top 1：AI服务器 / ODM

为什么入选：

1. 事实：样本池中 `浪潮信息` 当日 +10.00%，组均 20 日 +26.84%，5 日成交额 / 20 日成交额约 `1.35`，说明不是低量脉冲。
2. 变量：AI 服务器从“总钱包大”进入“收入和利润兑现”再定价，早间商业研报跟踪也记录了 `浪潮信息二季度业绩超预期增长` 相关公开研报线索。
3. 传导：高密度服务器订单 -> 收入确认 -> 毛利率改善 -> OCF 质量，是比单纯算力概念更短的验证链。
4. 验证：浪潮创 60 日高点，紫光股份接近 60 日高点，工业富联相对滞后；组内不是全面低位，而是进入右侧验证。
5. 应对：作为今日最平衡策略，只能在回踩不破 5 日 / 10 日趋势且成交不失真的情况下考虑小仓右侧；涨停后不追单。

可执行动作：

1. 已持有：继续持有审计，盯 5 日线、成交额和中报正式披露。
2. 未持有：不追涨停；只等分歧后仍能缩量承接、再放量转强。
3. 仓位上限：单方向最高 `20%-25%` 研究仓位，单标的最高 `8%-10%`；若已有半导体硬件仓位，合并计算。

加仓 / 减仓 / 退出触发：

1. 加仓：浪潮 / 紫光分歧后不破 5 日线，且工业富联不继续掉队；同时中报或公告能补出毛利率 / 现金流改善。
2. 减仓：涨停次日高开低走且放量，或组内只剩单一标的强。
3. 退出：中报显示收入强但毛利率、营运资本和 OCF 不改善，或价格跌回启动平台。

### 4.2 Top 2：HBM / 存储代理

为什么入选：

1. 事实：样本池组均 20 日 +26.08%、60 日 +84.97%，澜起科技、佰维存储、江波龙均在 20 日和 60 日正斜率。
2. 变量：海外存储周期、HBM 供需和长鑫科技 IPO 共同强化存储链资金语言。
3. 传导：DRAM / HBM 紧平衡 -> 价格和订单预期 -> 存储接口、模组、封测与国产替代估值重估。
4. 验证：当日存储芯片概念 +6.16%、主力净流入靠前，兆易创新涨停、澜起科技 +15.64%。
5. 应对：这是强趋势，但已经是高拥挤，不是低位赔率。策略是持有审计，不是追高买入。

可执行动作：

1. 已持有：保留核心强者，弱者反弹不续强则切出。
2. 未持有：等待 5 日均线附近承接或 20 日线级别分歧，不在单日大阳后新开大仓。
3. 仓位上限：方向最高 `15%-20%`，且不得与先进封装、半导体设备一起叠加到超风险预算。

加仓 / 减仓 / 退出触发：

1. 加仓：高位分歧后 2 个交易日内收复分歧高点，且成交没有放量失控。
2. 减仓：20 日涨幅继续扩到极端区，但成交额钝化或高位长上影。
3. 退出：距 60 日高点回撤扩大到 `-15%` 以下且不能快速收复，或利润好看但 OCF / 存货恶化。

### 4.3 Top 3：先进封装 / 封测 / 材料

为什么入选：

1. 事实：样本池组均 20 日 +33.25%、60 日 +107.34%，长电科技涨停、通富微电接近涨停、华海诚科继续高弹性。
2. 变量：AI 芯片封装瓶颈、国产替代和存储 / HBM 外溢共同驱动。
3. 传导：先进封装需求 -> CapEx / 订单 -> 收入确认 -> 良率与现金流审计。
4. 验证：先进封装概念 +5.63%、主力净流入靠前；但估值和行情斜率已经把未来预期提前反映。
5. 应对：只做强者持有和分歧承接，不把所有封测、材料、TGV 旁证都泛化成确定性主线。

可执行动作：

1. 已持有：保留强者，弱者反弹不创新高时减。
2. 未持有：等第一次明显分歧承接；不能在高位材料小票连续加速时追。
3. 仓位上限：方向最高 `15%`；如果同时持有 HBM / 存储，合并最高 `30%`。

加仓 / 减仓 / 退出触发：

1. 加仓：长电 / 通富平台龙头与华海等弹性样本同步强，而不是只剩小票冲刺。
2. 减仓：只剩小票冲高，平台龙头掉队。
3. 退出：订单 / 收入 / AI 占比和现金流不能跟上估值，或 20 日强势破坏。
