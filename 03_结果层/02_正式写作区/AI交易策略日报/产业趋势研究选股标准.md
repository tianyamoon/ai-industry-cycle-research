## 一句话结论

当前标准不是把此前九条纪律推倒重来，而是在原有九条纪律上引入外部样本校准：`先排方向，再确认映射角色和可验证路径，再看经营证据及其增量，再反推市场隐含预期与赔率，再识别资金属性、拥挤和价格位置，最后用技术门槛、排除法、反方条件和执行条件决定能不能进入主动候选`。

## 一、不可删除的九条纪律

当前标准必须完整保留此前九条纪律：

1. `方向优先于个股`
2. `经营验证大于故事`
3. `高位强票，不无脑追，审慎评价`
4. `低位方向不能因为没涨就买`
5. `必须有反方条件`
6. `买的是可验证的路径，不是概念标签`
7. `月线向上`
8. `周线向上`
9. `月线乖离率不过大`

外部样本只负责补强这些纪律，不能把它们改成“机构喜欢什么就买什么”“游资买什么就跟什么”。

## 二、固定决策链

后续所有周一进攻方向、日报、周更和临时选股回答，统一按下面顺序判断：

`方向排序 -> 映射角色 -> 可验证路径 -> 经营验证 -> 证据增量 -> 隐含预期 -> 估值赔率 -> 资金属性/拥挤 -> 技术门槛 -> 排除法 -> 反方条件 -> 执行条件`

### 1. 方向排序

先判断方向是否在当前产业和资金主线前排，而不是从单只股票倒推方向成立。

必须说明：

1. 该方向是 `主攻 / 备选 / 观察 / 回避`；
2. 排序依据来自经营变量、行情强弱、组内扩散，还是仅来自题材热度；
3. 如果方向低位，不能只因为没涨就上调。

### 2. 可验证路径

必须把概念拆成可验证传导链，例如：

1. `订单 -> 收入 -> 利润 -> OCF`
2. `价格上行 -> 毛利率改善 -> 利润兑现`
3. `CapEx 增加 -> 设备 / 基建订单 -> 交付 -> 回款`
4. `客户导入 -> 份额提升 -> 规模效应 -> ROIC 改善`

说不清路径的方向，只能放进观察池。

### 3. 经营验证

经营验证必须大于故事。至少说明当前证据来自：

1. 订单或合同；
2. 收入和利润；
3. OCF、回款、合同负债、应收和存货；
4. 客户导入或价格变量；
5. 官方公告、财报、交易所问询回复或可靠行业数据。

如果只有政策、概念、同业映射、短线情绪热度或榜单异动，不得写成主动买入理由。

### 4. 估值预期

好行业也可能是差买点。必须反问：

1. 当前价格隐含了几年高增长；
2. 当前估值隐含的利润率和 ROIC 是否可持续；
3. 市场是否已经把远期订单和远期利润提前打满；
4. 若中报、三季报不及预期，估值是否有足够容错。

### 4.1 证据增量、隐含预期与赔率

证据等级和行动吸引力必须分开。`A级证据`只能说明逻辑更真，不能自动说明价格更便宜或赔率更好。

每次选股必须同时回答：

1. 当前新增的是 `证据等级提升`，还是市场早已知道的旧事实；
2. 证据正在从 `C -> B`、`B -> A` 加速，还是已经进入高共识、低增量阶段；
3. 当前价格隐含的收入增速、利润率、ROIC、订单和市场份额是多少；
4. 下一项公开信息需要好到什么程度，才能超过而不是仅仅符合市场预期；
5. 若逻辑正确但估值、拥挤和价格位置已透支，是否应归类为 `持有审计 / 等待消化`，而不是新增主动候选。

固定使用四象限：

| 研究状态 | 价格/预期状态 | 标准处理 |
|---|---|---|
| 证据改善 | 预期尚未充分反映 | 优先进入预期差候选，继续过经营、资金和技术门槛 |
| 证据确凿 | 价格已高位透支 | 持有审计或等待业绩消化，不用确定性覆盖赔率不足 |
| 证据不足 | 价格低位 | 观察，不把便宜或没涨当成逻辑 |
| 证据转弱 | 价格仍高 | 回避或降级，防止盈利和估值双杀 |

### 5. 资金属性

必须说明当前主要由哪类资金语言主导：

| 资金属性 | 本标准中的使用方式 |
|---|---|
| 机构主导 | 看长期利润池、现金流、ROIC、估值预期和资本周期 |
| 私募主导 | 看产业景气、公司弹性、风格风险和组合容错 |
| 量化主导 | 看趋势、因子、容量、拥挤和资金退潮 |
| 游资主导 | 看情绪周期、主线地位、辨识度、标准买点、卖点纪律、仓位控制和退潮信号 |

资金属性不是买入理由本身，而是解释上涨、退潮和执行风险的语言。

游资主导的含义不是“某个席位上榜”，而是“短周期交易资金的选股与执行方法在主导价格”：先看情绪周期和主线地位，再看辨识度、容量、标准买点、卖点和仓位纪律。龙虎榜或席位数据只能作为案例复盘材料，不能作为买入理由。

### 6. 技术门槛

主动买入候选必须同时满足：

1. `月线向上`
2. `周线向上`
3. `月线乖离率不过大`

继承既有初始阈值：

1. `6月线乖离率 > 60%`，原则上不得新增主动买入；
2. `12月线乖离率 > 100%`，原则上不得新增主动买入；
3. `6月线乖离率在 40%-60%`，只能等分歧承接或平台消化后再评估；
4. 日线只负责择时，不能替代月线和周线。

技术面合格只是资格，不代表一定买；方向、经营、估值、资金和排除法仍要通过。

### 7. 排除法

排除法优先级高于乐观叙事。出现硬排除项时，即使方向强，也要降级。

### 8. 反方条件

每个方向和个股必须写清楚什么情况说明判断错了。反方条件必须可观察，不能写成“长期逻辑不变”。

### 9. 执行条件

最后才写能不能进攻：

1. 允许进攻的触发；
2. 只能观察的原因；
3. 等待右侧确认的条件；
4. 放弃或退出的证伪条件。

## 三、方向输出模板

当用户要求选择方向、周一进攻方向或方向排序时，每个方向必须说明十项。

| 字段 | 必须说明的逻辑 |
|---|---|
| 方向地位 | 是主攻、备选、观察还是回避；为什么排在这个位置 |
| 映射角色 | 是直接经营受益、订单承接、成本受益、负向暴露、条件性映射还是题材旁证 |
| 经营验证 | 当前证据来自订单、收入、利润、OCF、客户、价格，还是只有故事 |
| 证据增量 | 证据是在升级并产生预期差，还是已经成为高共识旧事实 |
| 隐含预期与赔率 | 当前价格已经写入什么增长、利润率和份额；未来还需什么超预期才能继续重估 |
| 位置与拥挤 | 是高位强趋势、低位未启动、修复反弹还是高位分化；是否有持仓、成交或估值拥挤 |
| 低位纪律 | 如果低位，必须说明为什么不能仅因没涨就买，以及需要什么拐点 |
| 反方条件 | 哪些变量反向会证明方向判断错了 |
| 可验证路径 | 买的是哪条传导路径，不能只写概念标签 |
| 技术门槛 | 月线、周线、月线乖离率是否满足；待核时必须写 `月周线待核 / 月线乖离率待核` |

输出时必须先给方向逻辑，再给个股逻辑；方向优先于个股。

## 四、个股输出模板

当用户要求推荐个股、解释个股或给进攻标的时，每个个股必须说明四项。

| 字段 | 必须说明的逻辑 |
|---|---|
| 角色 | 主锚、补强位、弹性票、平台票、旁证票、观察票或反证票 |
| 为什么看 | 它验证了方向里的哪个变量，如订单、收入、利润、OCF、客户、价格或承接 |
| 为什么不无脑买 | 估值、隐含预期、位置、拥挤、现金流、AI纯度、订单兑现、月线乖离率或技术形态限制 |
| 研究/行动分层 | 分别给出研究有效性和行动吸引力，禁止用高研究分覆盖低赔率 |
| 触发 / 证伪 | 什么情况下允许进攻，什么情况下必须放弃、降级或退出 |

如果个股逻辑无法说明它在方向中的角色，就不能只因为涨得强而入选。

## 五、排除法清单

| 排除条件 | 为什么排除 | 处理 |
|---|---|---|
| 无经营验证 | 没有订单、收入、利润、OCF、客户或价格变量，无法证明产业路径 | 观察或剔除 |
| 估值透支 | 价格隐含的增长和利润率明显高于可验证斜率 | 降级为持有审计或等待消化 |
| 证据确凿但无新增预期差 | 公开信息只是确认市场共识，没有提高未来盈利或现金流上限 | 不作为新增主动候选 |
| 高研究分掩盖低赔率 | 产业和公司都优秀，但估值、拥挤、价格位置或下行空间不匹配 | 研究保留，行动降级 |
| 月线乖离率太大 | 价格远离中长期成本区，新增买入变成追高 | 不新增主动买入 |
| 低位无趋势 | 低位只说明没涨，不说明拐点成立 | 等月线 / 周线转强 |
| 高位无消化 | 高位强票没有分歧承接、平台整理或估值消化 | 不追高，等承接 |
| 短线情绪只有热度无基本面 | 情绪热度、榜单异动或席位标签不能证明订单和利润 | 只作交易行为线索 |
| 游资方法论买点不成立 | 没有主线地位、辨识度、容量或标准买点，属于盘中冲动 | 不进入主动买入候选 |
| 游资方法论卖点触发 | 不及预期、断板无修复、低开无承接、高位放量滞涨、梯队断裂 | 降低执行等级或退出 |
| 量化 / 短线资金退潮 | 负反馈可能快于基本面变化 | 降低执行等级 |
| 资金属性错配 | 用游资语言解释长期投资，或用机构语言解释超短波动 | 重写逻辑或剔除 |
| 资本周期反噬 | CapEx、融资和供给释放可能压缩未来回报 | 加入资本周期审计 |
| 宏观信用收缩 | 重资产和长久期方向对融资成本敏感 | 降低估值容忍度 |
| 只有概念标签 | 无法拆成可验证路径 | 剔除 |
| 没有反方条件 | 无法判断何时认错 | 不进入进攻候选 |

## 六、证据强度与采纳纪律

| 证据等级 | 使用方式 |
|---|---|
| A | 可支撑强采纳条款 |
| B | 可支撑条件采纳和样本边界 |
| C | 只作风险提示或待核线索 |
| D | 不进入正式结论，只作反面提醒 |

强采纳条款必须至少有 A/B 级来源支撑。游资方法论条款必须回到公开复盘、连续案例或可记录的买卖标准；龙虎榜、席位和买卖额只作为案例验证材料，不作为方法论样本主体，更不能写成买入理由。

## 七、实战校验口径：AI算力方向

在不重新抓取最新 K 线和财报的前提下，按本仓库已有结果层口径，当前标准能够解释此前的三类分层：

| 方向 | 当前标准解释 | 当前处理 |
|---|---|---|
| 半导体设备 | 有利润兑现主锚和订单强化补强位，属于经营验证相对更强的进攻候选；但仍需逐只过月线、周线和乖离率 | 若技术门槛合格，可作为主攻候选；若乖离过大，降为持有审计 |
| HBM / 高性能存储代理 | 产业地位强，但部分标的可能已高位验证，必须做估值预期和高位消化审计 | 备选或高位审计，等待承接和估值消化 |
| AIDC供配电 / 智算电力基础设施 | 有经营线索和低位接力可能，但必须先确认订单、收入、OCF 闭环和月周线趋势 | 观察或等待右侧确认，不能因为低位就买 |

这个校验说明：当前标准并不会简单追最强，也不会简单买低位；它买的是可验证路径与技术资格同时成立。

## 八、后续接入规则

1. 后续所有周一进攻方向，先按当前标准写 `方向排序`，再写个股；
2. 后续所有日报和周更，必须显式说明 `方向地位 / 经营验证 / 位置拥挤 / 低位纪律 / 反方条件 / 可验证路径 / 技术门槛`；
3. 后续所有个股输出，必须显式说明 `角色 / 为什么看 / 为什么不无脑买 / 触发或证伪`；
4. 若行情、财报或公告数据待核，必须写 `待核`，不得用确定语气；
5. 当前标准执行后，应定期回看排除法是否减少误判，并把失败案例写回证伪表。

## 九、推荐输出顺序

当用户问“周一只能选一个方向”或“推荐 3 个股票”时，推荐输出顺序固定为：

1. 一句话结论；
2. 方向总排序；
3. 主攻方向的十项逻辑；
4. 备选和观察方向为什么暂不主攻；
5. 个股角色、为什么看、为什么不无脑买、触发 / 证伪；
6. 研究有效性 `R` 与行动吸引力 `A`；
7. 周一执行条件；
8. 反方撤退条件。

最后一句必须说明：这不是概念标签选择，也不是用证据确定性替代价格纪律，而是对可验证路径、经营证据、隐含预期、估值赔率、资金属性和技术门槛的综合选择。

## 十、研究有效性与行动吸引力双评分

总分不能把“研究正确”和“当前值得参与”混为一项。后续统一保留两个相互独立的分数，任何一个不合格都不得进入主动候选。

### 1. 研究有效性 `R`：满分 60

| 维度 | 分值 | 核心问题 |
|---|---:|---|
| 方向与映射角色 | 10 | 产业变量是否清楚；公司是直接受益、承接、成本受益还是旁证 |
| 证据等级与增量 | 15 | 来源是否可靠；证据是否正在升级，而不是重复市场共识 |
| 经营传导 | 20 | 订单、收入、利润、毛利率、回款和 OCF 是否形成闭环 |
| 资本周期与现金流质量 | 15 | CapEx、供给、融资、库存和 ROIC 是否支持长期回报 |

建议口径：`R < 36` 只观察；`36-47` 条件研究；`R >= 48` 才取得研究资格。研究资格不等于主动买入资格。

### 2. 行动吸引力 `A`：满分 40

| 维度 | 分值 | 核心问题 |
|---|---:|---|
| 隐含预期与预期差 | 10 | 新信息能否超过价格已写入的增长与利润假设 |
| 估值与下行容错 | 10 | 估值分位、盈利兑现和悲观情景是否保留安全边际 |
| 拥挤、容量与资金属性 | 10 | 同类资金是否过度集中；退潮时谁先撤、流动性是否承受得住 |
| 技术位置与执行条件 | 10 | 月周线、乖离率、分歧承接、触发位和证伪位是否合格 |

建议口径：`A < 16` 不新增；`16-23` 持有审计/等待；`24-31` 主动候选；`A >= 32` 才可列为高吸引力候选。涉及技术分值时，必须使用统一技术数据源和既有技术选股规则，不得主观补分。

### 3. 双评分动作矩阵

| 研究有效性 | 行动吸引力 | 统一动作 |
|---|---|---|
| 高 | 高 | 主攻候选，仍需触发和证伪条件 |
| 高 | 低 | 逻辑保留、持有审计或等待估值/价格消化 |
| 低 | 高 | 低位或强势都只作交易观察，不升级产业结论 |
| 低 | 低 | 回避 |

## 十一、高证据高价格的处理纪律

当证据已经确凿、市场形成共识且价格处于高位时，只允许等待以下四类新条件之一：

1. `业绩消化`：盈利和现金流增长追上估值，预期不再继续上修也能降低估值压力；
2. `非基本面回撤`：流动性、风格或短期扰动带来回撤，但核心经营变量未恶化；
3. `新增超预期`：订单、价格、份额、利润率、ROIC 或现金流上限被新的 A/B 级证据上修；
4. `二阶扩散`：寻找尚未充分定价的承接环节，但必须拥有自身订单、收入、利润和 OCF 证据，不能只靠海外或龙头映射。

四类条件均不满足时，允许结论为 `看对但不参与`。不行动是完整的研究结果，不是研究失败。
