# AI全球芯片制造深链与A股映射总表
> 所属模块：03_结果层/01_总表整合与AI映射
> 生成日期：`2026-07-04`
> 研究定位：把 `日本材料簇 / 韩国 HBM 配套簇 / InP 光器件簇 / ABF-玻璃基板簇 / 技术路线之争` 回落到 A 股研究对象，形成全球制造深链到本土承接位的正式总表。
> 使用边界：本表回答的是 `产业位置映射` 与 `变量传导映射`，不等于证明 A 股样本已经获得同等级客户认证或经营兑现；未拿到客户、订单、收入或财务连续验证的，一律只写 `映射候选`。

---

## 一句话结论

全球制造深链补到这一层后，A 股最值得继续下钻的，不再是泛泛的“先进制程国产替代”，而是五条更具体的映射链：`日本光刻胶/显影液/掩模/pellicle/前驱体/清洗 -> A股高端材料与设备`、`韩国 HBM / bonding / bridge interposer / substrate -> A股先进封装与测试`、`InP -> A股光芯片与化合物半导体`、`ABF / glass -> A股载板与玻璃基期权`、`路线之争 -> A股预期差排序`。

---

## 一、使用纪律

1. 先看海外样本在制造链中的真实位置，再看 A 股谁是同一节点的 `映射候选`。
2. A 股层级固定分为 `正式承接者 / 优先补证池 / 观察池`：
   - `正式承接者`：已有客户 / 收入占比 / 毛利率 / 现金流至少两项连续验证。
   - `优先补证池`：产业位置和产品方向已被官方来源钉住，但经营验证仍不够。
   - `观察池`：只有技术映射或路线期权，缺客户与经营证据。
3. 只要缺 `客户认证 / 订单 / 收入 / 毛利率 / 现金流` 连续验证，就不能把“映射候选”升级成“正式承接者”。
4. `技术相似` 不等于 `客户相同`，`产品名相似` 不等于 `利润池相同`。
5. 这张表优先解决“下一步该补谁”，不是直接给出“谁一定最受益”的结论。

## 三、日本材料簇 -> A 股映射

### 1. 日本材料簇的真实拆法

1. `EUV 光刻胶 / developer`
2. `MOR 等下一代材料`
3. `photomask blank / photomask`
4. `pellicle`
5. `silicon wafer`
6. `CMP slurry / pad`
7. `ALD / CVD precursors`
8. `高纯特气 / 湿电子化学品`
9. `湿法清洗 / selective wet etch`

### 2. A 股映射候选表

| 日本链节点 | 海外样本 | A股映射候选 | 映射逻辑 | 当前证据边界 | 当前优先级 |
|---|---|---|---|---|---|
| EUV 光刻胶 / developer | TOK、JSR、Fujifilm | `彤程新材`、`晶瑞电材`、`上海新阳`、`南大光电`、`雅克科技` | 都属于高端光刻材料或先进前驱体/配套化学品候选池 | 当前多为产业位置映射，不等于已完成同级 EUV 客户验证 | 高 |
| MOR / 下一代材料 | JSR | `彤程新材`、`南大光电`、`雅克科技` | 对应“材料代际升级”而非传统低端替代 | 多数仍缺下一代路线经营级验证 | 中高 |
| photomask / mask 配套 | HOYA、TOPPAN | `路维光电`、`清溢光电` | 对应掩模制造与版图载体，不等于 blank 本体 | A 股掩模能力已存在，但高端 blank 仍弱 | 中高 |
| pellicle | Mitsui Chemicals | `清溢光电`、`路维光电`、相关材料链待补 | 对应掩模保护膜及掩模配套观察位 | 当前正式对应样本仍偏弱，需保留 `待补` | 中 |
| silicon wafer | Shin-Etsu | `沪硅产业`、`立昂微` | 对应半导体硅片与高端规格升级 | 需继续补先进节点规格、客户与价格验证 | 高 |
| CMP slurry / pad | Fujifilm | `安集科技`、`鼎龙股份` | 对应 CMP 浆料与抛光材料平台化 | 这条映射最清晰，但仍需继续补高端客户与品类结构 | 高 |
| 前驱体 / 特气 | ADEKA | `雅克科技`、`南大光电`、`华特气体`、`金宏气体` | 对应 ALD / CVD 前驱体、高纯电子气与先进化学品 | 需要更细到具体前驱体和先进节点导入 | 高 |
| 湿法清洗 | SCREEN | `盛美上海` | 对应单片清洗与湿法工艺升级 | 这是“设备兑现链”，证据强于多数材料映射 | 高 |

### 3. 判断

1. 日本材料簇映射到 A 股后，最强的不是“所有材料一起涨”，而是 `高端化学品平台 + 清洗设备 + CMP + 硅片` 四条更容易形成经营验证的分支。
2. `pellicle`、`blank`、`MOR` 属于更前沿也更高门槛的节点，适合持续补证，不宜直接当成普遍兑现主线。
3. 如果后续先进制程扩产继续强化，而 A 股仍只能拿出“泛国产替代叙事”，那这条映射链就仍停留在研究层而不是投资层。

6. `FCBGA / ABF / glass`

### 2. A 股映射候选表

| 韩国链节点 | 海外样本 | A股映射候选 | 映射逻辑 | 当前证据边界 | 当前优先级 |

| 封测平台 | Nepes | `长电科技`、`通富微电`、`甬矽电子` | 对应异构集成、先进封装平台、桥接式系统级封装 | 逻辑成立，但与 NPU+HBM 同级客户绑定仍待补 | 高 |

| contactor / probe head | ISC `iSP` | `华峰测控`、`精测电子`、`长川科技` | 对应高频高速测试治具、probe head、RF / wafer probing 接触件体系 | 海外已补出官方产品样本，但 A 股侧仍缺同口径客户与经营拆分 | 中 |
| hybrid bonding | SEMES | `长电科技`、`通富微电`、`甬矽电子`、`精测电子` | 对应 2.5D / 3D 封装平台、先进互连与后续 HBM 代际工艺观察位 | 海外已补出官方开发样本，但当前仍只是 `节点已封口、经营待补` | 中 |
| 热管理 / 底填 / 封装材料 | HBM 全链 | `华海诚科`、`天承科技`、`中石科技` | 对应底填、材料与热路径协同 | 需要更细的客户、品类和 HBM 相关收入拆分 | 中高 |

1. 韩国 HBM 配套簇对 A 股最有用的启发，不是“谁也能做 HBM”，而是 `谁在 HBM 系统链上更容易兑现配套利润`。
2. A 股更值得持续跟踪的，是 `先进封装平台 / 高端基板 / 测试设备 / 热管理材料`，而不是简单寻找“国产 SK hynix”。
3. `memory test socket / burn-in socket / AI system IC socket + package final test` 已通过 `ISC + Okins Electronics` 补出第一版正式样本；`contactor` 与 `hybrid bonding` 也已分别通过 `ISC iSP` 与 `SEMES` 完成节点级官方补口，但两者都仍停留在 `经营待补`。

2. 这一层目前仍要防止把“光模块股上涨”误判成“InP 上游已经完成验证”。

## 六、ABF / glass 基板簇 -> A 股映射

| 节点 | 海外样本 | A股映射候选 | 当前口径 |
|---|---|---|---|
| FCBGA / 高端 package substrate | SEMCO、Ibiden、TOPPAN | `兴森科技`、`深南电路`、`沪电股份` | 更接近现实主线，应持续看客户认证、ASP 和产能利用率 |
| glass substrate / glass core | Intel、SEMCO、TOPPAN | `沃格光电`、`莱宝高科` | 仍属高阶期权，先看量产时点和客户验证 |
| TGV / 玻璃加工 | glass 路线观察层 | `沃格光电`、更多玻璃工艺链待补 | 容易被题材先行，经营验证最慢 |

判断：

1. `ABF / FCBGA` 是现实主线，`glass` 是期权主线。
2. 结果层不应把二者写成同一阶段，否则会把成熟资产与前沿观察混在一起。

## 七、技术路线之争对 A 股排序的影响

| 路线之争 | 若强化，A股更值得上调的方向 | 若弱化，A股最该下调的方向 |
|---|---|---|
| `MOR / 下一代材料` | 高端光刻材料、平台型化学品、先进前驱体 | 只停留在低端替代叙事的材料股 |
| `pellicle / photomask 配套升级` | 掩模、掩模配套、先进材料观察池 | 只靠“光刻概念”但缺实质产品的映射 |
| `HBM4 / 异构集成强化` | 先进封装、测试设备、基板、热管理材料 | 纯通用存储修复、泛封测低质映射 |
| `glass substrate / TGV` | 玻璃基、玻璃加工、载板升级期权 | 只有概念没有客户和量产的题材外溢 |
| `InP 光源强化` | 光芯片、化合物半导体、检测封装 | 仅依赖模块价格弹性的被动配套 |

## 八、轻量复评

按 `$industry-logic-scorer` 口径，这张表新增后补强了三层：

1. `传导链完整度`
   - 首次把“海外制造瓶颈”系统回落到 `A 股节点 -> 变量 -> 待补证`。
2. `研究对象优先级`
   - 不再停留在“全球样本名单”，而是形成下一步 A 股补证顺序。
3. `证伪边界`
   - 每条映射都区分 `现实主线 / 高阶期权 / 观察池`，降低题材误判。

仍未补满的部分：

1. 韩国测试链虽然已完成 `contactor / hybrid bonding` 节点补口，但经营连续验证仍偏弱。
2. 大多数 A 股映射仍缺 `客户 / 订单 / 收入占比 / 毛利率` 连续验证。
3. 历史分位系统尚未把全部制造深链优先对象纳入。

当前可给这张映射表本身的研究质量打到 `88/100` 左右；仍不宜宣称达到 `90/100`，原因不是结构，而是经营级连续验证和历史分位接入还没有同步跟上。
