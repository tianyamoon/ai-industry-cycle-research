## 一句话结论

截至 `2026-07-03`，AI 算力行情的主线仍在 `硬件`，但口径应从上一版的“高位加速并存”下调为 `硬件主线未结束 + 高位分化审计加深 + 基建接力仍未确认`。`HBM / 先进封装` 还没彻底退潮，但 `光模块 / PCB` 的 20 日斜率明显走弱，`AIDC供配电` 的组级 20 日与 60 日仍未过接力线，`IDC / AIDC运营` 继续停留在修复观察而非主线重返。

## 二、当前市场在交易什么

### 1. 正在交易的

1. `华海诚科 / 佰维存储 / 江波龙` 这类高弹性硬件强者留存。
2. `长电科技 / 生益科技 / 海光信息` 这类已兑现硬件龙头的高位审计。
3. `润泽科技` 一类修复型个股反弹，但尚未扩散成资源运营组级修复。

### 2. 还没完整交易的

1. `AIDC供配电`
2. `液冷` 的组级扩散
3. `IDC / AIDC运营` 的全面修复

其中 `AIDC供配电` 当前要拆成三条路径看：

1. `金盘科技`
   - 订单型高弹性设备
2. `科华数据`
   - 平台型智算供电基础设施
3. `科士达`
   - 标准化稳态兑现样本

扩组成 `AI算力基建大组` 后，还要额外区分：

1. `核心供配电`
   - 仍负责三路径经营闭环。
2. `电力设备旁证`
   - 只能用来观察 AIDC 电力需求是否外溢，不能替代核心样本。
3. `液冷 / 温控`
   - 用来观察高功率机柜散热需求是否组级扩散。
4. `IDC / AIDC运营`
   - 用来验证下游上架率、利用率、租赁价格和 OCF 是否修复。

### 3. 市场暂时不愿意为它买单的

1. 没有 20 日斜率和组级扩散支持的低位基建方向。
2. 只有长期空间、没有短中期现金流验证的重资产运营方向。
3. 只剩高位弹性、没有新增经营变量验证的小票跟风逻辑。

## 四、下周最重要的三个切换点

### 1. HBM / 先进封装会不会从高位加速切到高位分化

这是本轮周更最关键的问题。  
上一版最强方向还是“高位加速”，本版已经变成“高位分化审计”；如果最强方向继续掉队，整个 AI 硬件的风险偏好会先被审计。

### 2. 光模块 / PCB 龙头会不会重新夺回资金注意力

如果老主线重新转强，说明市场从“高位弹性外溢”回到“高确定性兑现”。  
这会直接改变我们对板块风险偏好的判断。

### 3. AIDC供配电能不能走出第一段明确接力

如果它还是低位震荡，就说明基建接力还没来。  
如果它开始转强，说明资金在尝试从高位硬件切向低位基础设施。

这条线现在不能只看一个总标签，而要同时看三条内部路径：

1. `金盘科技`
   - 看订单斜率能否继续兑现成收入和 OCF。
2. `科华数据`
   - 看平台分部收入、应收效率和 OCF 是否继续同步改善。
3. `科士达`
   - 看数据中心核心业务稳增能否持续跑赢新能源波动，并逐步拆出 AI 纯度。

## 五、周更最小动作

每周只要先更新下面 10 项，就能快速重算阶段判断：

1. `各细分 20日涨跌幅`
2. `各细分 60日涨跌幅`
3. `各细分距60日高点`
4. `组内上涨占比`
5. `组内最强标的是否变化`
6. `是否出现新的订单 / 财务 / 现金流验证`
7. `AIDC供配电` 内部三条路径谁先出现 `订单 / 收入 / OCF` 同步强化
8. `AI算力基建大组` 四个子层是否至少两个同步改善
9. `月线趋势` 是否向上
10. `周线趋势` 是否向上

截至 `2026-07-03` 的额外纪律：

1. 本轮巨潮 `2026-07-01 ~ 2026-07-03` 查询里，`科华数据 / 金盘科技 / 科士达 / 润泽科技 / 奥飞数据 / 数据港 / 光环新网 / 宝信软件 / 首都在线 / 申菱环境 / 高澜股份 / 英维克` 未见新增正式公告，因此桥接层经营评分表本周维持不动。
2. 本周结果层的变动，主要来自东方财富公开日线把最新交易日推进到 `2026-07-03` 后的行情结构再分类，而不是经营层新证据升级。

如果这 10 项没变，结论通常也不会大变。  
如果其中 2 到 3 项同时反向，才值得把阶段标签下调或上调。

## 六、当前执行优先级

1. `最高优先`
   - HBM / 高性能存储代理
   - 先进封装 / 封测 / 材料
   - 目的：判断高位加速是否结束
2. `第二优先`
   - PCB / CCL / 高速互连辅材
   - 光模块 / 高速互联
   - 目的：判断老主线是否二次转强
3. `第三优先`
   - AIDC供配电 / UPS / 变压器
   - AI算力基建大组观察池
   - 目的：判断基建接力是否开始、三条兑现路径谁先跑出来，以及扩组后是否出现跨层共振
4. `第四优先`
   - 液冷 / 温控
   - 目的：判断是否从单点弹性升级为组级行情
5. `最低优先`
   - IDC / AIDC运营 / 算力租赁
   - 目的：只做修复观察，不提前升级

## 七、本版结论

如果只看这一张看板，当前最重要的结论是：

1. `硬件主线还没结束，但已经从全面加速切到高位分化审计。`
2. `最强位置不是最安全位置，当前更值得盯的是谁还能在高位审计里留下来。`
3. `供配电不是单一设备链，而是订单型、平台型、标准化三条路径并行；本周经营层无新公告，行情层也未拿到接力资格。`
4. `资源运营层还没有修复，液冷也还没有从单点弹性升级成组级主线。`

换成更直接的话说：

> 现在不是在问“AI算力行不行”，而是在问“高位硬件的分化审计会不会继续加深、供配电什么时候拿到接力资格、IDC运营什么时候才配重新进入主线讨论”。
