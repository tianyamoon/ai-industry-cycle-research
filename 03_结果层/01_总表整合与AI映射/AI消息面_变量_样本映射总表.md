# AI消息面_变量_样本映射总表

> 形成日期：2026-07-04
> 所属层级：结果层
> 所属模块：03_结果层/01_总表整合与AI映射
> 研究定位：把消息面从“新闻堆叠”改成“事件 -> 变量 -> 样本 -> 升降级动作”的结果层映射表。
> 使用边界：本表不替代公告、财报或 IR 原文核验；来源未达到 `A级 / B级` 的消息只能进入观察层，不得支撑主结论；只有真正改变变量或样本等级的消息才进入本表。

---

## 一句话结论

AI 消息面必须按细分产业和变量归档，而不是按热点归档；每条消息都要回答它改变的是需求、供给、价格、CapEx、现金流、估值、技术路线还是监管，且必须绑定样本和证伪点。

---

## 一、来源分级

| 来源等级 | 定义 | 结论权限 |
|---|---|---|
| `A级` | 公司公告、定期报告、交易所文件、官方 IR、官方财报材料、监管文件 | 可进入变量验证 |
| `B级` | Reuters、AP、FT、Nikkei、证券时报、财联社等主流媒体或行业协会 | 可用于消息面和情绪层，需交叉验证 |
| `C级` | 市场新闻、二手转述、主题电报、论坛传播 | 只能进入观察层 |
| `待核` | 传闻、未见原始出处、无法回源 | 不得支撑结论 |

### 5. 验证

1. `价格验证`：DRAM、NAND、企业级 SSD、近线 HDD 价格是否继续上行，库存是否同步下降。
2. `订单验证`：Applied Materials、Lam Research、KLA 等设备公司订单和指引是否继续上修，A股设备公司订单是否转收入。
3. `财务验证`：AMD 数据中心收入、Applied Materials 半导体系统收入、存储厂收入和毛利率是否同步改善。
4. `A股验证`：江波龙、佰维存储、澜起科技、长电科技、通富微电、长川科技、盛美上海等是否出现收入、毛利率、OCF 和客户结构的连续兑现。
5. `证伪触发`：若目标价上修后只表现为短线估值扩张，而价格、订单、收入和 OCF 无法跟上，应维持预期层，不得升级为经营层结论。

### 6. AI映射

这条消息更像新能源周期里的 `上游和设备后半段重估`：当市场确认主需求后，资金开始从最核心资产外扩到存储、设备和配套环节。但新能源经验也提示，后半段上修最容易同时伴随估值透支和 CapEx 反噬，所以本事件的仓库处理应是 `提高验证优先级`，不是 `提高结论强度`。

4. 因此本事件证据等级定为 `C级用户线索 / 待核原研报全文`，只进入 `预期层 / 估值层 / 消息面映射层`。

### 3. 变量

| 变量 | 当前含义 | 证据权限 |
|---|---|---|
| 远期收入斜率 | 卖方把中际旭创的景气窗口从 `2026` 继续往 `2027-2028` 外推 | 只能作为预期层线索；需要原研报假设表和后续财报验证 |
| 远期利润斜率 | 净利润上修幅度大于绝对收入扩张，意味着市场更愿意交易龙头的利润久期 | 不能替代毛利率、OCF 和客户结构验证 |
| EPS长期化定价 | EPS 上修会直接抬高“核心资产可被长期化定价”的市场想象 | 只能进入估值层；不能直接抬升样本评级 |
| 1.6T量价假设 | 若预测成立，背后大概率对应更激进的 `1.6T份额 / ASP / 出货` 假设 | 需要正式来源或后续经营数据回源 |

### 4. 传导

| 海外投行预期变化 | 产业链传导 | A股映射位置 |
|---|---|---|
| 中际旭创 `2027-2028` 远期预测大幅上修 | 市场把 `800G兑现` 继续外推到 `1.6T放量 + 更长久期海外CapEx` | 光模块组龙头久期锚 |
| 龙头久期继续被抬高 | 组内资金可能更集中到龙头，而不是平均分配到二线和上游平台 | 中际旭创相对新易盛、天孚通信的估值锚更强 |
| 长久期定价继续抬高 | 一旦 `ASP / 毛利率 / OCF` 低于预期，估值压缩也会更敏感 | 光模块主线高位赔率审计 |

### 5. 验证

1. `原研报验证`：补野村原研报全文、发布日期和假设表。
2. `经营验证`：看 `2026H1 / 2026Q3` 是否把 `出货 -> 收入 -> 毛利率 -> OCF` 同步推高。
3. `产品验证`：看 `1.6T` 认证、出货、客户、份额和 ASP 是否有正式来源支撑。
4. `背景验证`：看海外云厂 `CapEx` 和 AI 网络资本开支是否继续上修。
5. `证伪触发`：若远期预测继续抬高，但订单、收入、毛利率和 OCF 没有跟上，应维持预期层，不得升级为经营层结论。

### 6. AI映射

这更像新能源里 `瓶颈利润龙头从当期高盈利被进一步外推到两年后` 的阶段。好处是龙头更容易被市场长期化定价；风险是只要 `价格 / 毛利率 / 扩产 / 客户压价` 中任一变量先掉队，先被压缩的往往不是产业方向，而是远期久期。
