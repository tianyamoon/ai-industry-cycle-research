## 一句话结论

新能源 2015-2025 的核心，不是一个线性成长故事，而是一轮由需求扩散、供给瓶颈、资本开支扩张、估值拥挤、上游接力、价格战、过剩出清和龙头分化共同构成的完整产业周期；在这轮周期里，价格或订单通常先转弱，股价往往先交易预期，毛利率和利润随后确认，现金流与资本回报最后验证，而真正能穿越的公司，最终必须靠现金流、ROIC、客户结构和资本配置证明自己。

## 一、研究边界与方法

### 1. 研究边界

本总结覆盖的是新能源十年母体研究，重点围绕：

1. 光伏主链；
2. 锂资源与锂电材料链；
3. 设备链；
4. 终端平台与核心供应商链；
5. 资源材料复合链。

样本边界固定为 14 只核心公司，不再扩样本：

- 资源链：赣锋锂业、天齐锂业、华友钴业
- 材料链：通威股份、隆基绿能、福斯特、天赐材料、多氟多、恩捷股份
- 终端 / 设备 / 平台链：比亚迪、宁德时代、阳光电源、先导智能、亿纬锂能

### 2. 研究方法

本版采用统一顺序：

`边界 -> 第一变量 -> 传导 -> 阶段 -> 估值与预期 -> 穿越验证`

对应到实际研究中，先看：

1. 价格、订单、供需与 CapEx；
2. 再看毛利率、利润、现金流和 ROIC；
3. 最后看估值、资金拥挤和市场预期是否已经提前透支。

### 3. 当前证据基础

本总结主要基于三层硬证据：

1. 价格层：`01_过程层/04_新能源数据闭环补证/硬数据实填/价格曲线月度实填总表_第二版.md`
2. 财务层：`01_过程层/04_新能源数据闭环补证/硬数据实填/14股季度财务实填总表_第二版.md`
3. 资金层：`01_过程层/04_新能源数据闭环补证/硬数据实填/14股公募北向持仓与交叉校验_第二版.md`

三层口径已经分别收口为：

1. 价格层：价格先行，利润滞后，主链压力再向辅材和高壁垒材料传导；其中碳酸锂、硅料、六氟已可直接引用，组件 / 胶膜 / 隔膜已足以支撑阶段判断；
2. 财务层：价格或订单先转弱，毛利率和利润随后确认，现金流与资本回报最后验证；其中 14 股利润顶部类型、第一领先变量和验证顺序已经成立，但逐季度全实数仍属增强层；
3. 资金层：公募拥挤已验证，北向和 Wind / Choice 仍待核验，重要机构持仓只能作辅助线索。

当前仍保留但不阻塞新能源总结收官的缺口包括：

1. 硅料早期全月连续化；
2. 组件 / 胶膜 / 隔膜全月化；
3. 14 股逐季度全实数；
4. 北向与 Wind / Choice 交叉校验。

## 二、九阶段周期叙事

新能源十年并不是一条匀速上升曲线，而是一轮完整的产业周期。  
如果把它压缩成可复用的机构叙事，可以拆成九阶段。每一阶段都对应不同的领先变量、不同的市场交易重点、也对应不同的投资错误。

### 第一阶段：主题导入期

这是产业故事开始成立、但证据仍然稀薄的阶段。

市场主要在交易：

1. 政策方向；
2. 技术替代；
3. 渗透率想象空间；
4. 龙头稀缺性。

这一阶段最该看的领先变量，不是利润，而是：

1. 政策强度；
2. 技术路线是否跑通；
3. 最早一批龙头是否先于行业启动；
4. 是否已经出现真实客户试单或试点。

这一阶段的本质，是市场先给行业一张“未来入场券”。  
最容易犯的错误，是把概念热度误当作产业闭环，把股价上涨误当作需求成立。

### 第二阶段：产业验证期

这一阶段的核心，是需求从叙事走向现实。

市场主要在交易：

1. 首批订单是否真实；
2. 客户导入是否能持续；
3. 量产节点是否兑现；
4. 哪些环节开始出现瓶颈。

最该看的领先变量是：

1. 订单；
2. 出货；
3. 客户结构；
4. 产能利用率；
5. 上下游交期。

这一阶段真正重要的，不是“有没有故事”，而是“有没有真实客户、真实订单、真实交付”。  
龙头如果能在这一阶段率先证明交付和份额，后面才有资格被市场长期化定价。

### 第三阶段：订单放量期

行业从“证明能做”进入“证明能放量”。

市场主要在交易：

1. 订单转收入的斜率；
2. 产能利用率提升；
3. 龙头与二线的份额差；
4. 上游设备、材料、资源是否开始同步受益。

最该看的领先变量是：

1. 新签订单；
2. 合同负债；
3. 收入增速；
4. 产能释放节奏；
5. 龙头和二线相对收益差。

这一阶段标志着市场从主题投资转向产业链投资。  
谁能把订单变成收入，谁的扩产更克制、交付更稳定，开始决定后面谁是核心资产，谁只是景气贝塔。

### 第四阶段：主升浪扩散期

这是新能源最像“全民景气”的阶段，也是最容易产生线性外推的阶段。

市场主要在交易：

1. 盈利上修；
2. 估值扩张；
3. 赛道广度扩散；
4. 机构持仓持续加码。

最该看的领先变量是：

1. 行业覆盖从龙头扩散到二线的速度；
2. 公募持仓和主题资金扩散路径；
3. 上游价格是否同步走强；
4. 全行业 CapEx 是否开始升温。

这一阶段最危险的认知偏差，是把“景气扩散”误判成“所有公司都会长期变好”。  
很多二线和边缘资产就是在这一阶段被高估，因为市场开始为贝塔、故事和弹性同时付费。

### 第五阶段：核心资产估值顶期

这一阶段经常发生在基本面依然很强的时候，所以最有迷惑性。

市场主要在交易：

1. 多年高增长；
2. 长期高份额；
3. 稳定高回报率；
4. 龙头的确定性溢价。

最该看的领先变量是：

1. 估值分位；
2. 机构持仓拥挤度；
3. 龙头相对二线的强弱变化；
4. 股价是否开始先于行业景气转弱。

这一阶段的本质，不是公司先变差，而是预期先打满。  
隆基、宁德这类核心资产之所以常在高景气中先见顶，就是因为市场对它们写入了过多远期好消息。

### 第六阶段：上游与高弹性接力期

这一阶段是利润在产业链内部重新分配的阶段。

市场主要在交易：

1. 资源价格；
2. 材料价格；
3. 稀缺产能；
4. 单品利润弹性。

最该看的领先变量是：

1. 碳酸锂、硅料、六氟等价格曲线；
2. 上游毛利率；
3. 利润弹性与估值压缩是否同时被忽略；
4. 资金是否从核心资产外溢到高弹性环节。

这就是“上游后半段更强”的来源。  
当核心资产的确定性已经被充分定价，市场会寻找更直接、更极致的利润放大器，于是资源、材料和瓶颈环节成为新的主战场。

### 第七阶段：资本开支失控期

这一阶段是周期尾部最关键、也最容易被忽视的前兆。

市场主要在交易：

1. 远期扩产；
2. 未来份额；
3. 稀缺利润可复制性；
4. 资本市场对产能故事的提前买单。

最该看的领先变量是：

1. 扩产公告密度；
2. 新进入者数量；
3. 在建工程和 CapEx；
4. 融资环境是否仍然宽松；
5. 设备交付和产能释放的时间窗是否开始集中。

高利润会吸引供给进入，这是资本周期最核心的反身性机制。  
真正危险的不是企业扩产，而是全行业在高利润幻觉下同步扩产，并假设高价会长期存在。

### 第八阶段：价格战与去库存期

这一阶段的核心，不是需求突然消失，而是供给释放快于需求兑现。

市场主要在交易：

1. ASP 下行；
2. 库存和应收抬升；
3. 减值和盈利下修；
4. 现金流恶化。

最该看的领先变量是：

1. 主产品价格；
2. 辅材 ASP；
3. 存货和跌价准备；
4. OCF / 净利润；
5. 合同负债与新签订单是否同步走弱。

价格战通常先伤主链核心产品，再沿着辅材、材料、二线和弱资产传导。  
这时市场从成长叙事切换到现金流与回报率审视，很多“好公司”第一次被迫回答自己到底是不是穿越者。

### 第九阶段：出清与分化期

这一阶段决定谁只是牛市受益者，谁是真正能跨越周期的公司。

市场主要在交易：

1. 资产质量；
2. 现金流韧性；
3. ROIC；
4. 份额再集中；
5. 管理层资本配置能力。

最该看的领先变量是：

1. 弱者是否退出、停产或边缘化；
2. 龙头是否逆势扩大份额；
3. 现金流和利润是否重新匹配；
4. 海外化、客户结构和第二曲线是否真实成立。

穿越者的标准，不是没跌过，而是跌完后还能重新证明自己。  
真正的阿尔法，不是在赛道最热时出现，而是在赛道退潮后才显形。

## 三、把代表样本嵌回九阶段

如果只写阶段，很容易变成抽象方法论；如果只写公司，又容易回到公司卡片。  
真正可用于结题正文的写法，是把代表样本重新嵌回九阶段，形成“阶段 - 样本 - 变量 - 结论”的统一叙事。

### 1. 主题导入到产业验证：先涨的是方向感，不是利润表

这一段最能代表的是隆基、宁德、比亚迪这类最先被市场赋予长期叙事的公司。  
它们的意义，不在于当时利润有多高，而在于它们最早承接了市场对新能源方向的长期定价。

这时研究重点不是追问静态估值，而是看：

1. 谁最早拿到真实客户；
2. 谁最早形成规模化交付；
3. 谁最早从政策受益者变成产业组织者。

所以，龙头最早的上涨并不是财务验证，而是对产业方向和竞争格局的提前占位。

### 2. 订单放量到主升扩散：龙头定价开始外溢到全产业链

这一段最能代表的是隆基、宁德、比亚迪的继续强化，以及先导、福斯特、恩捷等链上关键环节的跟随上行。  
市场开始从“买一个方向”转向“买一整条产业链”。

这时真正重要的是区分两类公司：

1. 被龙头订单和行业扩产真实带动的公司；
2. 只是跟着赛道热度享受估值抬升的公司。

设备、辅材、材料股在这一阶段被系统性重估，背后的核心逻辑不是“它们都是好公司”，而是市场开始相信新能源将从点状景气变成面状景气。

### 3. 核心资产估值顶：最强的公司最先被写满预期

这一段最有代表性的，是隆基和宁德这类核心资产。  
它们往往不是先在报表上出问题，而是先在估值和相对收益上见顶。

这时市场默认：

1. 高增长会持续多年；
2. 龙头份额足够稳固；
3. ROIC 可以长期维持高位；
4. 竞争恶化不会显著侵蚀回报。

一旦这些假设里有任何一条开始松动，股价就会先于财务调整。  
所以核心资产先见顶，本质上是“预期打满后的再定价”，不是“公司突然变差”。

### 4. 上游接力：利润弹性开始压倒确定性溢价

这一段最有代表性的，是赣锋、天齐、通威、天赐、多氟多。  
市场的偏好从“最确定”切到“最弹性”，从核心资产切到价格驱动资产。

原因很简单：

1. 核心资产已经不便宜；
2. 上游价格开始成为最强信号；
3. 利润弹性在报表里快速放大；
4. 市场更愿意追逐短期业绩爆发而非长期稳态价值。

这就是为什么新能源后半段最强的往往不是最稳的龙头，而是资源和材料股。

### 5. 资本开支失控：全行业一起对高利润作出反应

这一段不是某一家公司单独的问题，而是全产业链共同的问题。  
通威、天赐、多氟多、恩捷、先导都能从不同角度证明这一点：

1. 上游高利润引发扩产；
2. 设备链订单同步繁荣；
3. 材料公司开始以前景而不是现状扩张；
4. 市场愿意为远期产能预支估值。

从资本周期角度看，这一阶段的真正风险不是“现在赚不到钱”，而是“现在赚得太多，以至于所有人都决定继续投”。

### 6. 价格战与去库存：主链先坏，再向辅材和高壁垒传导

这一段最有代表性的，是隆基、福斯特、恩捷。  
主链价格战不是孤立事件，而是利润池收缩对整条产业链的重新分配。

这一阶段常见的传导顺序是：

`主产品价格下行 -> 主链毛利率承压 -> 辅材 ASP 下行 -> 高壁垒材料被重估 -> ROIC 下修`

这正是为什么“高壁垒也会周期化”必须成为正式结论的一部分。  
如果只研究上游价格而忽略主链价格战，就会低估辅材和高壁垒材料的退潮幅度。

### 7. 订单滞后与报表错觉：设备股的风险总是来得更晚

这一段最有代表性的，是先导智能。  
设备股在高景气后期经常看起来最稳，因为它还有在手订单、还有合同负债、还有收入确认。

但这份“稳”，本质上是滞后性，不是免疫力。

研究先导的意义在于提醒我们：

1. 设备股不能先看利润；
2. 下游 CapEx 放缓才是最早风险信号；
3. 新签订单和合同负债转弱，比利润回落更重要。

所以，设备股在这轮周期里不是例外，而是“报表滞后”的典型样本。

### 8. 出清与再证明：平台型公司开始和纯周期弹性资产分化

这一段最有代表性的，是比亚迪、阳光、宁德、亿纬，以及一部分华友。  
赛道退潮后，市场不再给所有新能源资产统一估值，而是强迫它们重新回答公司质量问题。

这时最重要的分水岭不再是题材，而是：

1. 现金流是否真实；
2. ROIC 是否仍高于资本成本；
3. 客户结构和海外能力是否能支撑份额；
4. 管理层是否能控制 CapEx 和资本错配。

这也是为什么穿越者不是“没跌过”的公司，而是“跌完后还能重新证明自己”的公司。

## 四、14股样本分组与角色

为了避免只看单个明星公司，本研究把 14 股放进五组角色里看。  
这种分组不是为了美观，而是为了回答五个不同问题：

1. 谁最容易先被长久期定价；
2. 谁最依赖价格弹性；
3. 谁看起来有壁垒但依然会被周期重估；
4. 谁必须先看订单而不是利润；
5. 谁要在赛道退潮后重新证明阿尔法。

### 1. 核心资产先见顶组

代表样本：

- 隆基绿能
- 宁德时代的部分阶段

这一组的共同特征是：

1. 行业地位强；
2. 市场份额高；
3. 曾被按多年高增长和高回报率定价；
4. 机构持仓和一致预期最容易集中。

这一组最重要的研究结论是：

1. 它们往往在景气最强时先被长久期定价；
2. 股价先于财务见顶；
3. 后续主要风险来自估值压缩，而不是立即的报表崩塌。

研究这组公司时，最该盯的不是短期利润增速，而是：

1. 估值分位；
2. 持仓拥挤度；
3. 龙头相对收益是否开始走弱；
4. 价格战和竞争格局是否开始侵蚀长期假设。

这一组解决的是一个核心问题：  
为什么行业最好的公司，反而常在行业最好的时候先跌。

### 2. 上游后半段更强组

代表样本：

- 赣锋锂业
- 天齐锂业
- 通威股份
- 天赐材料
- 多氟多

这一组的共同特征是：

1. 利润对单品价格高度敏感；
2. 报表弹性大；
3. 周期后半段往往最受追捧；
4. 价格下行时也最容易遭遇双杀。

这一组最重要的研究结论是：

1. 价格是第一变量；
2. 利润顶部大多滞后于价格顶部；
3. 报表最好时，往往已经接近周期尾部。

研究这组公司时，核心不是争论“公司好不好”，而是回答三件事：

1. 当前利润是成本优势赚来的，还是价格泡沫送来的；
2. 全行业是否正在用高利润刺激未来供给；
3. 估值是否把周期利润错当成永续利润。

这一组解决的是另一个核心问题：  
为什么很多周期股会在利润最好时最危险。

### 3. 高壁垒也会周期化组

代表样本：

- 恩捷股份
- 福斯特

这一组的共同特征是：

1. 曾被市场视为高壁垒材料资产；
2. 毛利率、客户结构和份额一度看起来更稳；
3. 估值通常高于普通周期材料股；
4. 但最终仍要接受 ASP、客户压价、折旧与 ROIC 的检验。

这一组最重要的研究结论是：

1. 壁垒可以延缓周期传导，但不能消灭周期；
2. ASP、客户压价、折旧和扩产一样会重估这些公司；
3. “高壁垒”不等于“估值永远安全”。

研究这组公司时，最关键的是拆掉一个常见误区：  
高壁垒并不意味着脱离产业主链。只要主链价格战和供给扩张足够深，辅材和高壁垒材料也会被传导重估。

### 4. 订单滞后组

代表样本：

- 先导智能

这一组的共同特征是：

1. 它的利润并不领先；
2. 它的生意本质上由下游 CapEx 驱动；
3. 订单、合同负债和回款质量比利润更重要。

这一组最重要的研究结论是：

1. 设备股不能先看利润；
2. 必须先看下游 CapEx、订单和合同负债；
3. 收入和利润顶部通常滞后于订单顶部。

研究这组公司时，分析顺序必须固定为：

`下游扩产意愿 -> 新签订单 -> 合同负债 -> 收入确认 -> 利润兑现 -> 现金流与应收质量`

这一组解决的是周期研究里一个很容易错位的问题：  
如果先看利润，往往已经晚了一个周期。

### 5. 阿尔法验证组

代表样本：

- 比亚迪
- 阳光电源
- 宁德时代
- 亿纬锂能
- 华友钴业的一部分平台叙事

这一组的共同特征是：

1. 不完全是单一价格驱动；
2. 具备平台化、全球化、客户结构或第二曲线叙事；
3. 在赛道退潮后仍有机会靠经营质量再证明自己。

这一组最重要的研究结论是：

1. 赛道退潮后，市场不再为贝塔买单；
2. 必须靠现金流、ROIC、客户结构、海外化和第二曲线证明自己；
3. 真穿越者与阴影样本，会在这一阶段被真正区分出来。

研究这组公司时，最该问的不是“它是不是龙头”，而是：

1. 收入能否稳定转成现金流；
2. ROIC 能否在下行期保持高于资本成本；
3. 客户质量、全球化、渠道和产品结构是否能抵消赛道退潮；
4. 管理层会不会在高景气后犯资本配置错误。

这一组最终解决的是最重要的问题：  
谁是赛道赢家，谁只是牛市里的幸运儿。

## 五、八条投资闭环链

新能源十年不是一个统一节奏，而是八条不同的投资闭环链并行。  
所谓“闭环链”，指的是一条可以从产业变量出发，一路跟踪到股价、报表、估值、资金和退潮验证的完整链条。

### 1. 锂资源链

代表样本：

- 赣锋锂业
- 天齐锂业

第一变量：

- 碳酸锂价格

典型传导：

`碳酸锂价格 -> 锂盐毛利率 -> 归母净利润 -> 存货 / 减值 / 资产负债表压力`

市场为何会买这条链：

1. 资源价格是最直接的稀缺利润；
2. 利润弹性大，报表爆发快；
3. 周期后半段最容易成为资金追逐对象。

顶部最早看什么：

1. 锂价是否先拐头；
2. 高利润是否诱发全行业扩产；
3. 股价是否在利润仍强时先弱化。

退潮后怎么验证：

1. 看存货与减值；
2. 看资产负债表压力；
3. 看利润中枢是否被永久下修。

核心结论：

1. 价格顶部通常先于利润顶部；
2. 资源股后半段弹性最强；
3. 价格坍塌后估值和利润中枢都会被迅速重估。

### 2. 硅料链

代表样本：

- 通威股份
- 隆基绿能

第一变量：

- 硅料价格

典型传导：

`硅料价格 -> 硅料毛利率 / 主链成本 -> 利润 -> ROIC 与估值重定价`

市场为何会买这条链：

1. 硅料一旦成为瓶颈，利润放大极快；
2. 龙头既可享受瓶颈利润，也能享受主链景气；
3. 这条链能同时观察上游利润泡沫和核心资产估值顶。

顶部最早看什么：

1. 硅料价格高位是否松动；
2. 组件价格战是否开始侵蚀主链；
3. CapEx 是否从少数龙头扩散到全行业。

退潮后怎么验证：

1. 看通威类高利润瓶颈能否维持；
2. 看隆基类核心资产是否进入估值与 ROIC 重定价；
3. 看高利润是否被后续供给反噬。

核心结论：

1. 通威代表高利润瓶颈的放大与反转；
2. 隆基代表核心资产先见顶与主链价格战承压；
3. 上游高利润会很快吸引 CapEx，最终反噬自己。

### 3. 六氟链

代表样本：

- 天赐材料
- 多氟多

第一变量：

- 六氟磷酸锂价格

典型传导：

`六氟价格 -> 电解液 / 六氟毛利率 -> 利润弹性 -> 估值压缩`

市场为何会买这条链：

1. 单品涨价最容易形成利润爆发；
2. 材料股体量适中、弹性显著；
3. 高景气期容易被当作“成长 + 周期”双重资产。

顶部最早看什么：

1. 六氟价格是否见顶；
2. 电解液和材料毛利率是否开始回落；
3. 库存和客户压价是否开始显性化。

退潮后怎么验证：

1. 看存货减值；
2. 看利润弹性是否迅速回落；
3. 看市场是否把周期利润重新压回正常盈利中枢。

核心结论：

1. 这是最典型的单品涨价利润模型；
2. 不能把周期利润外推成永续利润；
3. 报表最好时，往往意味着价格已经非常高。

### 4. 组件 / 胶膜链

代表样本：

- 隆基绿能
- 福斯特

第一变量：

- 组件价格
- 胶膜 ASP

典型传导：

`组件价格 -> 主链毛利率 -> 胶膜 ASP -> 辅材利润与 ROIC`

市场为何会买这条链：

1. 主链龙头具备规模、品牌和份额优势；
2. 辅材公司在景气上行时看起来既稳又有成长；
3. 这条链能清楚展示价格战如何从主链向辅材传导。

顶部最早看什么：

1. 组件价格战是否提前出现；
2. 主链毛利率是否先走弱；
3. 胶膜 ASP 和辅材盈利能力是否开始被挤压。

退潮后怎么验证：

1. 看辅材利润是否滞后下行；
2. 看 ROIC 是否重估；
3. 看“高壁垒辅材”是否仍能独立于主链。

核心结论：

1. 价格战先发生在主链核心产品；
2. 然后才向辅材利润传导；
3. 辅材公司并不会完全独立于主链周期。

### 5. 隔膜链

代表样本：

- 恩捷股份

第一变量：

- 隔膜 ASP
- 扩产折旧

典型传导：

`ASP -> 毛利率 -> 折旧与产能利用率 -> 利润与 ROIC`

市场为何会买这条链：

1. 隔膜曾被视为高壁垒核心材料；
2. 龙头份额、技术门槛和客户结构使其享有高估值；
3. 这条链最适合观察“高壁垒也会周期化重估”。

顶部最早看什么：

1. 客户压价是否启动；
2. ASP 是否松动；
3. 扩产带来的折旧压力是否提前累积。

退潮后怎么验证：

1. 看 ROIC 是否下滑；
2. 看产能利用率是否承压；
3. 看现金流能否跟上利润。

核心结论：

1. 高壁垒材料依旧会被周期重估；
2. 扩产与折旧对 ROIC 的伤害不可忽视；
3. “龙头地位”不能替代“回报率验证”。

### 6. 设备订单链

代表样本：

- 先导智能

第一变量：

- 下游 CapEx
- 新签订单
- 合同负债

典型传导：

`CapEx -> 新签订单 -> 合同负债 -> 收入 -> 利润 -> 现金流与应收`

市场为何会买这条链：

1. 设备股天然承接下游扩产；
2. 在产业上行期，订单能提前锁定未来景气；
3. 只要扩产继续，设备股就会显得“业绩很稳”。

顶部最早看什么：

1. 下游 CapEx 是否放缓；
2. 新签订单是否先回落；
3. 合同负债是否失速。

退潮后怎么验证：

1. 看收入确认还能持续多久；
2. 看应收和回款质量；
3. 看 OCF 是否开始弱于利润。

核心结论：

1. 订单比收入更领先；
2. 收入比利润更领先；
3. 利润比回款质量更领先。

### 7. 终端 / 核心供应商穿越链

代表样本：

- 比亚迪
- 宁德时代
- 阳光电源

第一变量：

- 毛利率
- OCF
- ROIC
- 海外化

典型传导：

`赛道贝塔 -> 股价高位 -> 估值消化 -> 财务阿尔法验证`

市场为何会买这条链：

1. 这类公司兼具赛道贝塔与公司阿尔法叙事；
2. 在高景气期容易同时享受成长溢价和龙头溢价；
3. 在退潮期仍有机会靠经营质量再证明自己。

顶部最早看什么：

1. 估值与持仓是否先拥挤；
2. 股价是否先于报表走弱；
3. 竞争、价格战和海外节奏是否开始挑战长期假设。

退潮后怎么验证：

1. 看 OCF；
2. 看 ROIC；
3. 看客户结构与全球化；
4. 看第二曲线是否真实贡献利润质量。

核心结论：

1. 这组公司不适合只按周期股框架看；
2. 退潮后真正决定股价的，是公司质量而不是赛道热度；
3. 现金流、ROIC 和客户结构才是终极验证指标。

### 8. 多业务平台 / 资源材料复合链

代表样本：

- 亿纬锂能
- 华友钴业

第一变量：

- 分业务毛利率
- 资源价格
- CapEx
- ROIC

典型传导：

`多业务叙事 -> 分业务分化 -> 利润结构重估 -> ROIC 与资本配置检验`

市场为何会买这条链：

1. 多业务平台给市场提供更长的叙事想象；
2. 不同业务可以在景气高点掩盖彼此问题；
3. 平台化、资源化、材料化的复合结构容易被高估。

顶部最早看什么：

1. 分业务毛利率是否开始分化；
2. 资源价格和平台叙事是否同时承压；
3. CapEx 是否过度前置。

退潮后怎么验证：

1. 看减值和现金流；
2. 看 ROIC 是否还能支撑平台故事；
3. 看资本配置是否能证明多业务不是复杂化而是真协同。

核心结论：

1. 多业务不自动等于穿越；
2. 平台叙事只有在现金流和回报率兑现时才成立；
3. 复杂公司更需要拆分，而不是合并看故事。

## 六、四条正式主结论

### 1. 股价通常领先财务

新能源周期里，股价通常先交易：

1. 价格；
2. 订单；
3. 估值；
4. 资金预期。

而毛利率、利润、现金流在报表中的体现，往往滞后于市场定价。

这意味着：

1. 只看财报，通常会在顶部之后才意识到风险；
2. 要研究周期，必须先看第一变量，而不是先看报表。

可直接写入正文的版本是：

> 新能源价格层已经足以支撑总结，核心顺序是“价格先行、利润滞后、股价往往先交易预期，主链压力再向辅材和高壁垒材料传导”。

### 2. 核心资产常在高景气中先见顶

核心资产最容易被长久期定价。

当市场已经默认它们会多年维持高增长、高份额和高回报时，任何增长放缓、竞争恶化或回报率下降，都会先触发估值压缩。

所以：

1. 核心资产先见顶，并不代表它们先变差；
2. 它们是先被“预期打满”，再被“现实校正”。

### 3. 上游资源和材料股常在后半段更强

行业从“需求验证”走向“利润放大”时，利润最容易向上游资源和材料集中。

这时市场最想交易的，不再是最稳的龙头，而是利润弹性最大的环节。

所以：

1. 上游后半段更强，不是偶然；
2. 但也因为它们最依赖价格高位，所以退潮时往往更剧烈。

### 4. 真正穿越者必须靠现金流、ROIC、客户与资本配置证明

赛道贝塔退去之后，公司必须重新回答四个问题：

1. 能不能把收入变成现金流；
2. 能不能维持高于资本成本的回报率；
3. 客户、渠道和海外能力是否真实存在；
4. 管理层能不能克制扩产和资本错配。

这四个问题，决定了谁是穿越者，谁只是牛市受益者。

可直接写入正文的版本是：

> 新能源财务层已经足以支撑总结，关键顺序不是“利润先坏”，而是“价格或订单先转弱，毛利率和利润随后确认，现金流与资本回报最后验证”。

> 新能源资金层已经足以支撑“公募拥挤存在、北向和 Wind / Choice 仍待核验”的总结口径，不能把重要机构持仓误写成公募拥挤已验证。

## 七、为什么新能源十年对 AI 研究重要

新能源母体研究最大的价值，不是简单告诉我们“AI 会不会像新能源”，而是提供了一套可迁移的机构化研究顺序。

这套顺序告诉我们：

1. 先找第一变量；
2. 先看价格、订单、ASP、供需和 CapEx；
3. 再看毛利率、利润、现金流和 ROIC；
4. 最后看估值、机构拥挤和预期是否透支；
5. 不要在赛道最热时把周期利润误判成长期能力。

新能源给 AI 的真正启发，不是题材类比，而是方法论复制。

## 八、当前仍保留的缺口

本版可以正式成立，但仍保留少量缺口，不把缺口写成全闭环。当前仍保留的缺口主要有：

1. 组件、胶膜、隔膜连续月度曲线仍未全补满；
2. 硅料早期月度区间仍待补；
3. 多数公司季度财务仍待继续补实数；
4. 北向持仓与 Wind / Choice 交叉校验尚未完全闭环。

这些缺口当前不阻塞主结论，但会影响后续更细的顺序分析和量化强度。

## 十、本版结论

新能源十年研究现在已经可以进入正式结论系统。

可直接写入正文的主句只有四条：

1. 价格或订单先转弱，股价往往先交易预期，毛利率和利润随后确认；
2. 核心资产常在高景气中先见顶，上游资源和材料股常在后半段更强；
3. 公募拥挤已验证，但北向和 Wind / Choice 仍要保留缺口口径；
4. 真正的穿越者，最后必须靠现金流、ROIC、客户结构和资本配置证明。

其余内容，留给后续定向补证，不再留在正文最后一层反复展开。
