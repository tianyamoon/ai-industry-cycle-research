## 三、当前主线判断

### 1. 硬件主线是否结束

结论：`没有结束，但已经从高位加速切到高位分化审计。`

事实：

1. `澜起科技` 近 `60日涨幅` `109.65%`，`长电科技` `137.97%`，`生益科技` `210.99%`。
2. 但 `中际旭创` 近 `20日涨幅` `-12.81%`，`新易盛` `-4.97%`，`天孚通信` `-27.02%`。
3. `佰维存储` 近 `20日涨幅` 仍有 `25.45%`，`华海诚科` `35.14%`，说明最强弹性资产仍在高位区。

判断：

1. 当前不能写成“硬件主线结束”，因为 HBM 与先进封装里仍有强势高位样本。
2. 但也不能继续沿用上一版“高位加速并存”的写法，因为组级 20 日斜率已明显回落，老主线转弱信号增多。
3. 更稳妥的正式口径是：`主线未破，但高位内部正在审计谁能继续留在主线。`

### 2. AIDC供配电是否开始接力

结论：`还没有。`

事实：

1. `科华数据 / 金盘科技 / 科士达` 近 `20日涨幅` 分别为 `-4.91% / -16.00% / -6.70%`。
2. 三者近 `60日涨幅` 分别为 `3.89% / 11.53% / 9.76%`，远未达到结果层定义的 `30%` 接力升级线。
3. 巨潮 `2026-07-01 ~ 2026-07-03` 查询中，三家公司未检出新增正式公告，经营层本周没有新的硬证据推动评分迁移。

判断：

1. 这条线当前仍应维持 `未充分演绎 / 低位观察`。
2. 它的研究价值还在，但本周既没有新的经营层正式升级，也没有行情层走强。
3. 因此不能把“经营层先验证”误写成“行情层已确认接力”。

### 3. 资源运营层是否修复

结论：`没有。`

事实：

1. `润泽科技` 近 `20日` 为 `4.48%`、`60日` 为 `5.48%`，是组内唯一相对稳一点的样本。
2. `奥飞数据 / 数据港 / 光环新网 / 宝信软件 / 首都在线` 近 `60日涨幅` 分别为 `-3.55% / -17.37% / -22.89% / -18.01% / -26.64%`。
3. 组内 `60日上涨占比` 仍显著低于结果层要求的修复线。

判断：

1. 当前资源运营层最多算“个别样本反弹”，还不是组级修复。
2. 周更里仍应把它放在 `修复观察区`，而不是重新抬回主线讨论。

### 4. 液冷是否从单点弹性升级为组级行情

结论：`没有。`

事实：

1. `申菱环境` 近 `20日涨幅` `14.36%`、`60日涨幅` `81.59%`，继续保持组内最强。
2. `英维克 / 高澜股份` 近 `20日` 仅 `3.19% / -4.31%`，`60日` 为 `11.26% / 8.41%`。
3. 巨潮 `2026-07-01 ~ 2026-07-03` 查询中，`申菱环境 / 高澜股份 / 英维克` 未见新增正式公告。

判断：

1. 当前仍应维持“申菱环境单点强势、液冷组级未确认”的正式口径。
2. 只有当第二个经营锚点出现，或者组内至少两只标的同步强化，才适合把液冷从个股弹性上调为组级行情。
