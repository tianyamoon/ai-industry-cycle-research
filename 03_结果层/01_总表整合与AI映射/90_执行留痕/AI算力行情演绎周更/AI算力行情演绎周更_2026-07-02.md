## 一句话结论

截至 `2026-07-01`，本轮 AI 算力行情已经走到 `硬件主线高位演绎、基建接力尚未确认` 的阶段。`HBM / 先进封装` 是当前最强位置，`PCB / 光模块` 进入高位分化和滞涨审计，`AIDC供配电` 仍是低位观察位，但内部已经分成 `金盘科技`、`科华数据`、`科士达` 三条兑现路径，经营层先有验证、行情层尚未确认接力，`IDC / AIDC运营` 仍不具备回到主线的条件。

## 一、本周最重要的变化

由于本期是第一期周更，本周最重要的变化不是“和上周相比谁更强”，而是把当前行情结构第一次压成固定口径。

当前最重要的三个事实是：

1. `HBM / 高性能存储代理` 与 `先进封装 / 封测 / 材料` 的 `20日涨幅` 都显著高于科创50的 `27.36%`，说明最强位置还在高位加速。
2. `光模块 / 高速互联` 与 `PCB / CCL` 的 `60日涨幅` 仍强，但短线斜率开始弱化，说明老主线已进入高位分化审计。
3. `AIDC供配电` 的 `60日均涨幅` 只有 `9.40%`，显著弱于硬件主线；同时其内部又已经分化出 `订单型高弹性设备`、`平台型智算供电基础设施`、`标准化稳态兑现` 三条路径，说明经营层证据已经开始出现，但市场还没有把它交易成统一接力主线。

## 三、当前主线判断

### 1. 主线仍在硬件

当前市场最愿意给溢价的，仍然是 `HBM / 先进封装 / PCB / 光模块` 这四条硬件链。

其中：

1. `HBM / 先进封装` 更像高位加速。
2. `PCB / 光模块` 更像高位主升后的分化和滞涨审计。

这意味着我们当前不能用一句“AI算力都强”来概括，而要明确区分：

1. 哪些方向已经涨得很深；
2. 哪些方向只是低位观察；
3. 哪些方向市场暂时并不买账。

### 2. 基建接力还没有确认

`AIDC供配电` 现在只是“最值得盯的低位方向”，还不是已经确认的主线接力。

更重要的是，这条链已经不能再被当成单一设备链处理，而要拆成三条内部路径：

1. `金盘科技`
   - `订单型高弹性电力设备`
   - 关键观察变量是 `AIDC订单 -> 收入确认 -> OCF闭环`
2. `科华数据`
   - `平台型智算供电基础设施`
   - 关键观察变量是 `数据中心分部收入 -> 交付效率 -> OCF改善`
3. `科士达`
   - `标准化智算基础设施 / 稳态兑现`
   - 关键观察变量是 `数据中心核心业务收入 -> AI纯度拆分 -> OCF持续性`

当前它距离升级为接力主线，还差两步：

1. 行情上要看到 `20日转正` 和 `60日突破30%`；
2. 基本面上要看到三条路径中至少一条出现 `订单 / 收入 / OCF` 的同步验证。

### 3. IDC / AIDC运营当前仍不在主线讨论里

当前这条线更适合放在修复观察层，而不是配置优先层。

原因不是它没长期空间，而是：

1. `60日上涨占比` 太低；
2. `距60日高点` 回撤太大；
3. 市场还没有看到上架率、利用率和现金流的系统修复。

## 四、本期最该回答的三个问题

### 1. 高位加速会不会直接切成高位分化

最需要盯的是：

1. `HBM / 先进封装` 的 `20日涨幅`
2. `距60日高点`

如果 20 日斜率明显钝化，且回撤扩大到 `-15%` 以下，就要从“高位加速”降级到“高位分化审计”。

### 2. 老主线会不会重新夺回注意力

最需要盯的是：

1. `光模块`
2. `PCB / CCL`

如果龙头重新转强，说明资金在从高弹性边缘重新回到高确定性核心。

### 3. 供配电能不能拿到接力资格

这条线是当前最值得继续跟踪的“没被完整演绎”的方向。
但只有当行情和报表一起改善，且 `金盘 / 科华 / 科士达` 三条路径里至少有一条先跑出连续兑现，才能从观察位升级成真正接力位。

## 五、下周最重要的动作

1. 更新 `HBM / 先进封装` 的 `20日涨幅` 与 `距60日高点`。
2. 更新 `光模块 / PCB` 龙头是否出现二次转强。

4. 继续把 `液冷` 按个股强势处理，不提前升级为组级主线。
5. 继续把 `IDC / AIDC运营` 按修复观察处理，不提前抬升优先级。

## 六、当前不该做的误判

1. 不要把 `HBM / 先进封装` 的强势，误判成“现在最安全”。
2. 不要把 `供配电` 的低位，误判成“已经确认接力”。
3. 不要把 `IDC / AIDC运营` 的长期空间，误判成“当下行情已经在交易它”。
4. 不要把 `光模块 / PCB` 的高位滞涨，误判成“主线已经结束”。

## 七、本期结论

这期周更的核心，不是告诉我们“哪条线最好”，而是固定当前的市场结构基线：

1. `最强主线`：`HBM / 先进封装`
2. `高位审计区`：`PCB / 光模块`
3. `观察接力区`：`AIDC供配电（三路径待确认）`
4. `个股弹性区`：`液冷`
5. `修复观察区`：`IDC / AIDC运营`

如果后面几期周更里，这五个分层开始变化，我们就能更清楚地判断：主线是在轮动，还是在退潮，还是在准备第二段结构性切换。对 `AIDC供配电` 来说，下一步不是继续笼统看“供配电行不行”，而是跟踪 `金盘 / 科华 / 科士达` 里谁先把经营验证推进成行情确认。
