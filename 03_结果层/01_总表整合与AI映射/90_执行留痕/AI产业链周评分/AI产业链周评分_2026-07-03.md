## 产业逻辑层

- 当前最强仍是 `计算芯片 / 存储 / 高速互联 / 先进封装 / 海外硬件主链`，但逻辑从“主线普涨”切到“订单、毛利率、良率、CapEx、客户导入的验证分化”。
- `AIDC / 供配电 / 散热 / 数据中心资源` 更像主线承接层，只有交付、利用率、回款和 OCF 连续改善，才值得上修。
- `模型 / 数据 / 工具 / 安全 / 治理` 大方向成立，但多数细分仍缺稳定商业化，不能把正确方向直接写成高分结果。
- `终端 / 具身 / AI 玩具 / 车端 AI / 前沿观察 / 影子资产` 目前仍偏中长期映射、主题扩散或验证池，保持低置信度更客观。

## 估值资金层

- 项目内历史估值分位与机构持仓表显示，`硬件核心层` 仍处高位再平衡状态，赔率不再来自低位，而来自分化后仍能兑现的环节。
- `基础设施层` 和 `部分成熟应用层` 的拥挤度相对更低，但“没那么热”不等于“已经便宜”，仍要看经营闭环。
- `模型 / 数据 / 工具 / 安全 / 前沿观察` 的资金层多为中性或低热度，只有商业化真正转入报表，资金才可能从观望切到持续配置。

## 技术结构层

- 本周技术层统一按项目内 `20日/60日斜率`、`距60日高点回撤`、`组内强弱`、`是否组级共振` 代理口径处理。
- `AI算力硬件主线` 仍有强势样本，但整体已进入高位分化审计，不能再把“还能涨”直接写成“适合买入”。
- `AIDC / 应用 / 终端 / 具身` 更像等待组级共振的观察状态；本周没有使用直接 MCP 数据去硬做月线、周线和乖离率脑补。
