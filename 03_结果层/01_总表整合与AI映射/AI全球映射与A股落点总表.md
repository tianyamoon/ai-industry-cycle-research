## 一句话结论

当前全球映射最重要的正式口径不是“A 股全面对标海外 AI 龙头”，而是：`算力硬件与 AIDC 配套` 已形成较清晰的全球变量传导链，`企业软件与 MaaS` 处于经营修复验证期，`终端 / 机器人 / AI玩具` 仍主要停留在前沿观察与早期映射层。

### 1.1 映射后的二级门槛

本表中的“强 / 中强 / 中 / 弱”只表示全球变量到 A 股产业角色的映射强度，不代表主动买入等级。每条映射进入选股或策略输出前，必须补齐以下字段：

| 二级门槛 | 必答问题 | 允许输出 |
|---|---|---|
| 影响方向 | 对 A 股样本是收入受益、订单承接、成本受益、负向暴露还是条件性影响 | 不允许只写“相关”或“受益” |
| 证据等级与增量 | 是 A/B/C 哪一级；最新信息是否提高经营上限，还是仅确认共识 | 共识确认不得自动上调动作 |
| 隐含预期 | 当前价格已经写入多少增长、利润率、份额和 ROIC | 写明还需什么超预期 |
| 估值与拥挤 | 估值分位、持仓、成交和价格位置是否透支 | 高位高拥挤降为持有审计 |
| 技术与执行 | 月周线、乖离率、承接、触发和证伪是否合格 | 技术合格只是资格，不是逻辑 |

固定动作：`映射强 + 赔率高 -> 主动候选`；`映射强 + 赔率低 -> 持有审计/等待消化`；`映射弱 + 价格低 -> 观察`；`映射转弱 + 价格高 -> 降级/回避`。

## 二、当前三条固定判断

### 1. 全球映射已不能继续内嵌处理

如果不把全球映射单独拉出，后续周更和结果层总论就会持续把：

1. 海外主链变量
2. A 股盈利承接
3. 纯主题映射

混写成同一个结论。

### 2. A股最强的仍是“全球主链的配套与承接”

当前更清晰的不是“谁复刻海外平台估值”，而是：

1. 算力硬件
2. AIDC 基建协同
3. 部分平台与企业软件本土化落地

### 3. 机器人和 AI玩具都应纳入应用维护链，但仍保守放在观察层

1. `机器人`
   - 是 AI 应用和终端向真实物理世界延伸的重要方向。
2. `AI玩具`
   - 是消费级陪伴硬件与内容订阅结合的重要早期方向。
3. 但两者当前都不能跳过 `出货/订单 -> 收入 -> 现金流` 的验证顺序。

## 三、下一轮补证优先级

1. `最高优先`
   - 海外云 CapEx、GPU / HBM 供需变化与 A 股硬件 / 基建传导联动。
2. `第二优先`
   - 企业软件、MaaS 与办公协同的续费、AI 收入占比、合同负债和 OCF。
3. `第三优先`
   - 机器人 / 具身智能与 AI玩具的量产、出货、收入占比、订阅或复购证据。
