## 一句话结论

当前最值得再向上提分的，不是继续加样本，而是把 `资源运营 / 液冷 / AIDC供配电` 统一拉回 `收入 -> 利润 -> OCF -> 资本结构 -> 证伪点` 的连续硬字段审计。现阶段更接近经营兑现锚点的是 `润泽科技 / 奥飞数据 / 宝信软件 / 科华数据 / 科士达`，最需要继续严审现金流和回款链条的是 `光环新网 / 首都在线 / 申菱环境 / 高澜股份 / 金盘科技`。

## 一、固定审计口径

本表固定使用以下字段：

`旧总分 / 当前状态 / 2025收入 / 2025归母 / 2025 OCF / 2026Q1收入 / 2026Q1归母 / 2026Q1 OCF / 当前经营锚点 / 最敏感反证点 / 下一验证窗口 / 来源状态`

审计纪律固定为：

1. 不把订单、上架率、签约机柜或项目口径直接等同于收入、利润和 OCF。
2. 不把液冷二线、资源运营平台和供配电三路径混写成同一类风险暴露。
3. 若 `收入 / 利润` 改善但 `OCF / 回款 / 存货 / 合同负债 / 长期应付款` 恶化，优先视作“兑现后审计”，而不是直接上调长期结论。
4. 本表解决的是“经营硬字段连续化”问题，不替代 `北向最新逐只持股`、`终端 / 具身智能连续 BOM` 和 `中报后证伪表首次实战回写`。

## 三、当前可正式上收的结论

1. `润泽科技 / 奥飞数据 / 宝信软件 / 科华数据 / 科士达` 已经不只是“有逻辑”，而是至少把部分经营链条推进到了 `收入 / 利润 / OCF` 可连续审计的阶段，因此更适合写成“经营兑现审计位”而不是纯题材观察位。
2. `光环新网 / 首都在线 / 申菱环境 / 高澜股份 / 金盘科技` 的共同点，不是方向错，而是 `OCF / 回款 / 存货 / 长债 / 折旧 / 融资结构` 里至少有一项仍在压制结论上收，所以它们更像“经营验证中或兑现后审计位”。
3. 资源运营层当前最关键的分水岭，不是谁拥有更多机柜，而是谁能把 `资源投放 -> 上架率 / 利用率 -> 收入 -> OCF -> 资本结构` 解释成闭环。
4. 液冷二线当前不能只看订单和毛利率，必须连着看 `收入确认 -> 回款 -> OCF`；否则容易把项目制波动误写成持续高景气。
5. AIDC供配电当前的真正优势，是经营层先进入验证、行情层仍未充分演绎；因此这条线的赔率来自“接力前的审计期”，而不是“历史估值绝对低位”。

---

## 四、当前仍未闭环的点

1. `北向逐只持股最新日度` 仍未形成稳定、可系统回填的公开口径。
2. `2026H1` 尚未大面积披露，本表解决的是“连续硬字段入口”，还没有完成“中报后第一次实战迁移”。
3. `上架率 / 利用率 / GPU租赁价格 / 液冷收入占比 / AI订单纯度` 仍有不少字段停留在 `待补 / 待核 / 待确认`。
4. `终端 / 具身智能连续 BOM 与订阅 ARPU` 仍未像训练、推理、AIDC 三条线一样形成同强度闭环。
