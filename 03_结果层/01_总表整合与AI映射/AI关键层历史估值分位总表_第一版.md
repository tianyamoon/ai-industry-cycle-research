## 一句话结论

核心硬件、液冷二线、半导体设备与部分平台承接样本，当前大多已经处在 `5年70分位以上` 的高位区间；资源运营与应用成熟层分化更大，其中 `润泽科技 / 光环新网 / 科大讯飞 / 用友网络` 并不都处在同样的高位，说明“产业逻辑成立”和“赔率是否还便宜”已经明显分离。

1. `INDICATOR_TYPE / INDEX_TYPE = 1 / 2 / 3 / 4` 分别对应 `PE(TTM) / PB(MRQ) / PS(TTM) / 市现率(TTM)`。

4. 对负 PE、负市现率或明显失真的样本，不沿用失真指标，改用更稳定的 `PB(MRQ)` 或 `PS(TTM)`。

## 三、当前可正式上收的结论

1. 光模块、PCB/CCL、液冷二线、半导体设备利润兑现层，代表样本多数已经处在 `5年70分位以上`，它们更像高位赔率审计区，而不是低位赔率区。
2. HBM / 存储代理与先进封装并不便宜，尤其 `佰维存储`、`江波龙`、`华海诚科` 的 `PS(TTM)` 已经进入高分位；因此这条线更适合写成高弹性交易区，而不是安全低位。
3. 国产 AI 芯片 / 平台层内部并不完全一致：`寒武纪` 的 `PS(TTM)` 更接近 `5年50-70`，但 `海光信息` 的 `PE(TTM)` 已在 `5年70` 上方，`中科曙光` 则更接近中高位而非极端高位。
4. 资源运营与应用成熟层分化明显：`润泽科技` 更接近中位附近，`数据港` 仍在高位，`光环新网` 属低到中位修复区；应用层里 `科大讯飞 / 用友网络` 偏低位，而 `同花顺` 的 `PB(MRQ)` 已重新回到高分位。
5. AIDC供配电三样本并非低估值洼地，`金盘科技 / 科华数据 / 科士达` 当前大多位于 `5年50-70` 或 `>70`，所以这条线的核心优势是“位置低于主线情绪拥挤”，不是“历史估值绝对便宜”。

2. 若样本盈利转负、现金流大幅失真或业务模型变化，允许从 `PE(TTM)` 切换到 `PB(MRQ)` 或 `PS(TTM)`，但必须显式记录切换原因。
3. 周更和中报回写时，若当前值连续跨越 `30 / 50 / 70` 阈值，应同步回写赔率判断，不得只更新股价不更新分位。
4. 本表解决的是“历史估值分位”缺口，不替代 `北向最新持股`、`公募拥挤时序` 和 `经营兑现` 审计。
