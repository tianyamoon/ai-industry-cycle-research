# AI全链条估值_预期_拥挤审计总表

> 所属模块：03_结果层/01_总表整合与AI映射
> 生成日期：2026-07-04
> 研究定位：把 AI 产业链从“产业逻辑成立”推进到“价格、估值、资金和预期已经隐含什么”的统一审计表。
> 边界说明：本表不构成个股买卖建议；公开源无法稳定取得的历史分位、公募明细、北向最新日度口径，一律保留 `待核`，不为完整性伪造数据。

---

## 一句话结论

AI 当前不能只问“产业方向对不对”，还必须问“当前价格已经为哪个环节付了多少预期”。硬件主线已经进入高位赔率审计，AIDC 供配电仍处在低位接力观察，资源运营和液冷二线必须先证明现金流与回款，应用成熟层则需要用真实付费和续费证明估值久期。

---

## 一、口径与来源规则

### 1. 固定字段

本表固定使用以下字段：

`当前估值口径 / 相对分位 / 行情斜率 / 成交拥挤 / 公募持仓 / 北向边界 / 价格隐含假设 / 最先被压缩的预期 / 投资赔率标签`

### 2. 数据来源优先级

1. 行情、估值、财务：东方财富行情与财务公开接口、同花顺公开页、交易所公告。
2. 公司事实：巨潮资讯、上交所、深交所公告、投资者关系活动记录、定期报告。
3. 资金拥挤：公募季报公开重仓、成交额、换手率、北向公开口径。
4. 在北向逐只持股最新日度不可稳定取得时，当前统一以 `机构持仓变化代理（公募为主口径） + 成交额 / 换手 + 北向边界` 作为结果层资金验证主口径。
5. 若北向最新日度不可得，统一写为 `北向最新待核`，只保留旧口径边界。

## 三、当前正式判断

### 1. 产业强弱与赔率强弱已经明显分离

核心算力硬件仍是产业兑现最清晰的方向，但估值、行情斜率和拥挤已经抬高。AIDC 供配电和部分资源运营层反而因为行情没有充分演绎，具备继续观察接力的赔率意义，但前提是经营闭环必须补上。

### 2. 最先被压缩的不是“AI 需求”，而是“长期高斜率兑现”

若后续出现回撤，最可能先被压缩的是：

1. 核心硬件的高毛利率持续性；
2. 国产 AI 芯片的久期估值；
3. 先进封装和 HBM 外溢样本的高弹性预期；
4. 液冷二线从订单到 OCF 的兑现速度；
5. 资源运营层从资源厚度到现金回报的转化效率。

### 3. 北向和资金拥挤仍不能写成完整闭环

当前可写成：`机构持仓变化代理、公募、成交和行情斜率已经足以提示拥挤风险`。
当前不能写成：`北向最新日度已完整交叉验证`。

但本轮可以把边界再往前推一步：

根据东方财富市场数据公开摘要，`2026-07-02` 北向资金当日总成交约 `4240.18亿元`，约占两市总成交额 `12.29%`；当日公开摘要中，`中微公司`、`寒武纪`、`中天科技` 位列沪股通成交前三。

这说明：

1. 最新北向“交易热度”公开摘要是可以看到的；
2. 但逐只样本的最新 `持股量 / 持股占比 / 连续增减仓` 仍未系统回填；
3. 因此当前口径应升级为：`北向最新成交热度可见，个股持仓最新日度仍待逐只核验`；
4. 与此同时，`机构持仓变化代理（公募为主口径）` 已经足以承担当前主研究的核心资金验证职责。

### 4. 制造深链赔率摘要

1. 制造深链专表已新增：
   [AI全球芯片制造深链估值_预期_拥挤审计表_第一版](</F:/股票/ai-industry-cycle-research/03_结果层/01_总表整合与AI映射/AI全球芯片制造深链估值_预期_拥挤审计表.md>)
2. 当前可正式回写的摘要只有三点：
   - `盛美上海 / 长电科技 / 华海诚科` 已在高分位审计区，不能被写成“低位制造深链赔率”。
   - `通富微电` 更接近中高位，不是绝对低估值洼地。
   - `安集科技 / 雅克科技 / 兴森科技 / 华峰测控 / 沪硅产业 / 鼎龙股份 / 沃格光电 / 莱宝高科` 当前更大的问题仍是 `历史分位待补 + 经营连续验证不完整`；其中 `华峰测控 / 沪硅产业 / 鼎龙股份` 已完成第一轮经营字段补口，但仍不宜直接上收为赔率占优结论。

## 四、下一轮补证顺序

1. 对所有核心样本继续补 `近 20 日 / 60 日成交额分位、换手率分位`，并把未覆盖样本的 `PE / PS / PB 历史分位` 继续补齐。
2. 用公募季报公开重仓补 `持仓集中度、加仓 / 减仓方向、同业拥挤`，并把它正式视为机构持仓变化代理主口径。
3. 若能继续找到公开机构持仓口径，再逐步扩到 `公募之外的重要机构类别`。
4. 若北向公开口径恢复，优先补 `核心算力、国产算力、AIDC供配电、资源运营` 四层。
5. 把本表中的 `价格隐含假设` 回写到周更模板，作为每周赔率检查项。
