## 一、研究目标

本模块不是做单纯股价复盘，而是建立一套可复用的产业周期图谱：

**早期概念启动 → 调整筛选 → 主升浪 → 第一次见顶 → 二次冲顶 / 补涨 → 退潮路径 → 当前状态 → AI 映射。**

重点不是“哪只股票涨了多少”，而是回答：

1. 股价什么时候先于产业启动？
2. 哪些产业催化推动了第一轮上涨？
3. 第一次见顶时产业是否已经见顶？
4. 二次冲顶是新周期，还是牛市余波？
5. 后续退潮是估值杀、利润杀、供给杀，还是三者叠加？
6. 哪些公司穿越，哪些公司半穿越，哪些公司进入阴影？
7. 这些路径如何映射 AI 产业链？

## 三、统一图谱卡模板

每只股票均按以下字段写：

1. 基本信息；
2. 第一轮概念启动时间；
3. 第一轮上涨逻辑；
4. 第一轮见顶与回撤；
5. 调整筛选期；
6. 2020–2021 主升浪启动点；
7. 第一次见顶时间；
8. 是否二次冲顶；
9. 历史最高点对应产业逻辑；
10. 之后如何退潮；
11. 当前状态；
12. 对 AI 的映射。

统一状态定义：

| 状态 | 含义 |
|---|---|
| 穿越 | 产业退潮后仍能维持较强盈利、份额、现金流或新增长曲线 |
| 半穿越 | 龙头地位还在，但盈利和估值仍受周期压制 |
| 阴影 | 产业需求还在，但股价、估值、利润、供给压力仍未修复 |
| 出清失败 | 公司或方向被长期证伪，难以回到主线 |

## 五、14 只标的的周期角色总结

| 类型 | 标的 | 核心启示 |
|---|---|---|
| 光伏核心资产阴影 | 隆基绿能 | 产业继续增长，不代表股价不会长期阴跌 |
| 上游瓶颈利润 | 通威股份 | 后半段最强的利润环节，扩产后也会被周期反噬 |
| 相对穿越 | 阳光电源 | 海外化、储能和电力电子平台带来穿越可能 |
| 终端平台穿越 | 比亚迪 | 技术、产品、成本、品牌和渠道形成公司阿尔法 |
| 核心基础设施 | 宁德时代 | 产业核心地位不等于估值永远上涨 |
| 多业务平台 | 亿纬锂能 | 第二曲线必须兑现为利润和现金流 |
| 锂资源泡沫 | 赣锋锂业 | 不要把周期利润当成长利润 |
| 资源杠杆周期 | 天齐锂业 | 高景气并购和扩产会在低景气放大风险 |
| 材料涨价泡沫 | 天赐材料 | 材料股核心是价格和毛利率，不只是需求 |
| 单品周期弹性 | 多氟多 | 单品涨价弹性强，但穿越难度更高 |
| 高壁垒材料 | 恩捷股份 | 工艺壁垒也会被扩产和客户压价周期化 |
| 资源材料复合 | 华友钴业 | 资源、材料、技术路线和资本开支共同决定估值 |
| 设备资本开支 | 先导智能 | 设备股本质是下游资本开支周期 |
| 光伏辅材 | 福斯特 | 辅材龙头更稳，但仍受主链价格战传导 |

## 六、下一步

02 模块第一版已完成，下一阶段建议进入：

**03｜历史新闻流与政策流**

目标：为 14 张 K 线图谱卡补充逐年新闻、政策、价格和产业数据依据。

优先补充顺序：

1. 2015–2017：新能源车补贴、动力电池白名单、锂钴和六氟第一轮涨价、光伏领跑者、单晶替代；
2. 2018：光伏 531、新能源车补贴退坡、锂钴价格回落、行业出清；
3. 2019：平价上网、特斯拉上海工厂、动力电池集中、龙头低位布局；
4. 2020：双碳、特斯拉国产、刀片电池、欧洲新能源车、机构抱团；
5. 2021：核心资产见顶、材料涨价、六氟 / PVDF / 碳酸锂 / 硅料上涨；
6. 2022：锂价和硅料高点、资源利润高点、市场开始交易未来过剩；
7. 2023–2025：碳酸锂下跌、光伏价格战、材料利润坍塌、反内卷、穿越与阴影。
