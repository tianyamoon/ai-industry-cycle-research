## 一、回填目标

赣锋锂业是新能源周期中最典型的资源股样本之一。

前一版主卡已经建立主线：

**2015–2017 第一轮锂资源行情 → 2018–2019 锂价回落和资源股退潮 → 2020 低位修复 → 2021–2022 第二轮锂价泡沫 → 2023–2025 锂价坍塌和资源股阴影。**

本补证卡要解决的问题是：

> **为什么新能源车需求继续增长，赣锋这类资源龙头却可以在 2022 高点后持续承压？**

核心证据链：

**新能源车需求高增 → 动力电池排产上修 → 锂盐供需错配 → 碳酸锂 / 氢氧化锂价格上涨 → 锂盐毛利率和利润爆发 → 全球资源扩产 → 锂价下跌 → 存货、减值和利润中枢重估 → 股价进入阴影。**

## 三、阶段一：2015–2017 第一轮锂资源认知

### 1. 政策和产业背景

2015–2017 年，新能源车补贴和销量高增长带动动力电池需求扩张。市场第一次系统性把锂资源作为新能源车产业链的瓶颈环节来定价。

赣锋这一阶段的市场关注点：

- 电池级碳酸锂和氢氧化锂需求增长；
- 锂盐加工能力；
- 上游资源布局；
- 动力电池客户需求；
- 新能源车补贴和双积分政策强化长期空间。

### 2. 待补价格数据

| 数据 | 口径 | 用途 | 待补来源 |
|---|---|---|---|
| 碳酸锂价格 | 电池级 / 工业级 | 验证第一轮锂价上涨 | SMM / 百川 / Wind |
| 氢氧化锂价格 | 电池级 | 验证高镍预期和锂盐需求 | SMM / 百川 / Wind |
| 锂辉石精矿价格 | 6% 精矿 | 验证上游资源成本 | Fastmarkets / Benchmark / SMM |
| 中国新能源车销量 | 年度 | 验证需求端 | 中汽协 |
| 动力电池装机 | 年度 / 月度 | 验证电池需求 | GGII / 中汽协 / 工信部 |

### 3. 待补财务数据

需补赣锋 2015–2017：

- 营业收入；
- 归母净利润；
- 锂盐业务收入；
- 锂盐业务毛利率；
- 锂资源权益；
- 资本开支；
- 经营现金流；
- 主要客户和产品结构。

### 4. 事后验证

这一阶段的逻辑部分成立：锂确实成为电动化时代的核心资源之一。但第一轮行情后，2018–2019 的锂价回落证明：

> 锂是战略资源，但不是永久高价资源。

### 5. AI 映射

对应 AI 中的第一轮上游瓶颈认知：

- HBM；
- 光模块；
- 高速铜连接；
- 变压器；
- 电力设备；
- 液冷核心材料；
- 先进封装产能。

早期被发现是对的，但不能把“被发现”直接外推成“长期高价格”。

## 四、阶段二：2018–2019 锂价回落和资源股退潮

### 1. 产业背景

2018–2019，新能源车补贴退坡，低端车型和补贴依赖产能出清，动力电池和材料链需求预期阶段性下修。

对赣锋的影响：

- 锂价回落；
- 锂盐毛利率下降；
- 资源股估值收缩；
- 市场重新认识锂资源的周期属性；
- 第一轮锂资源行情结束。

### 2. 当时市场理解

市场从：

**新能源车长期需求 → 锂资源长期稀缺**

切换到：

**补贴退坡 → 需求放缓 → 锂价下跌 → 资源股利润回落。**

### 3. 待补财务验证

| 指标 | 用途 |
|---|---|
| 2018–2019 营业收入 | 看需求和价格对规模的影响 |
| 2018–2019 毛利率 | 看锂价回落的盈利冲击 |
| 归母净利润 / 扣非净利润 | 看利润下行幅度 |
| 存货 | 看价格下跌风险 |
| 经营现金流 | 看利润质量 |
| 资本开支 | 看是否逆势布局资源 |

### 4. 事后验证

这阶段证明资源股研究不能只看终端需求。

关键变量是：

- 价格；
- 供给；
- 库存；
- 下游议价；
- 资本开支；
- 成本曲线。

## 五、阶段三：2020 第二轮锂价行情前夜

### 1. 政策和产业背景

2020 年，特斯拉国产、欧洲新能源车需求、双碳政策共同推动新能源车链条复苏。

对赣锋而言，重要变化是：

- 新能源车真实需求恢复；
- 电池厂排产上修；
- 锂盐需求预期改善；
- 锂价底部开始被市场关注；
- 资源股从低位进入第二轮行情前夜。

### 2. 待补证据

| 证据 | 用途 | 来源 |
|---|---|---|
| 2020 新能源车销量 | 验证需求恢复 | 中汽协 |
| 2020 动力电池装机 | 验证排产恢复 | GGII / 中汽协 |
| 2020 碳酸锂价格底部 | 验证价格反转前夜 | SMM / 百川 |
| 赣锋 2020 年报 | 验证低位财务 | 年报 / Wind |
| 2020 扩产 / 资源公告 | 验证资源布局 | 公司公告 |

### 3. 事后验证

2020 是“价格底 + 需求恢复 + 估值修复”的组合。

资源股在这一阶段的买点不是利润已经爆发，而是价格和需求拐点即将形成。

## 六、阶段四：2021–2022 第二轮锂价泡沫

### 1. 产业背景

2021–2022，新能源车渗透率快速提升，动力电池排产强劲，锂资源供给释放滞后，碳酸锂和氢氧化锂价格快速上涨。

市场对赣锋的定价从“周期修复”转向“战略资源长期稀缺”。

### 2. 价格逻辑

这一阶段需要重点补三条价格曲线：

| 价格 | 意义 |
|---|---|
| 碳酸锂 | 资源股利润核心 |
| 氢氧化锂 | 高镍电池和高端锂盐需求 |
| 锂辉石精矿 | 上游原料成本和资源端供需 |

核心判断：

> 2022 年锂价接近极端高位时，赣锋的利润弹性最大，但风险也最大。

### 3. 财务验证

待补赣锋 2021–2022：

- 锂盐收入；
- 锂盐毛利率；
- 归母净利润；
- 扣非净利润；
- ROE；
- 经营现金流；
- 存货；
- 资本开支；
- 锂资源权益和扩产公告。

### 4. 当时市场理解

市场高点附近相信：

- 新能源车长期高增长；
- 锂资源长期稀缺；
- 赣锋全球资源布局具备平台价值；
- 氢氧化锂需求长期强；
- 固态电池等远期叙事提供估值延展。

### 5. 事后验证

被证伪的不是新能源车长期需求，而是：

- 锂价长期维持极端高位；
- 周期利润可以按成长利润估值；
- 上游资源不会快速扩产；
- 下游车企和电池厂会长期接受高锂价。

### 6. AI 映射

AI 上游瓶颈未来也可能在某个阶段出现类似外推：

- 需求长期高增长；
- 供给短期不足；
- 价格大涨；
- 利润爆发；
- 市场把短期供需错配当作永久稀缺。

需要警惕“报表最好即顶部”。

---

## 七、阶段五：2023–2025 锂价坍塌与资源股阴影

### 1. 产业背景

2023–2025，新能源车销量继续增长，但锂资源和材料链进入价格下行周期。

原因包括：

- 上游锂资源扩产释放；
- 下游车企价格战；
- 电池厂去库存；
- 高锂价刺激替代和降本；
- 市场对锂盐长期高价假设破灭；
- 资源股估值从成长切回周期。

### 2. 对赣锋的影响

赣锋承压来自：

- 锂盐价格下跌；
- 毛利率下降；
- 存货跌价风险；
- 减值压力；
- 资本开支回报率被质疑；
- 资源项目盈利预期下修；
- 股价进入旧周期阴影。

### 3. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 收入、利润、毛利率、存货、现金流 | 看锂价下跌第一阶段影响 |
| 2024 | 扣非净利润、减值、锂盐毛利率 | 看低价环境下盈利中枢 |
| 2025 | 中报 / 年报、价格低位、资源项目进展 | 看是否进入周期底部修复 |

### 4. 事后结论

赣锋不是出清失败，而是资源龙头进入阴影期。

准确表述：

> 需求仍在，资源仍有价值，但高点估值体系已破裂；必须等待价格、利润和供给出清重新平衡。

## 九、赣锋回填后的核心判断

赣锋锂业的路径可以概括为：

1. **2015–2017：第一次战略资源重估。** 市场发现锂是新能源车时代的重要资源。
2. **2018–2019：第一轮周期退潮。** 补贴退坡和供给释放让锂价回落，资源股周期属性显现。
3. **2020：第二轮行情前夜。** EV 真实需求恢复，锂价底部反转预期形成。
4. **2021–2022：第二轮锂价泡沫。** 供需错配带来极致利润，市场把周期利润成长化外推。
5. **2023–2025：资源股阴影。** 锂价坍塌后，利润、估值、存货和资本开支全部重估。
