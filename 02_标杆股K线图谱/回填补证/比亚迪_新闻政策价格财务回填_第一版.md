## 一、回填目标

比亚迪是新能源十年周期中最重要的穿越观察样本。

它和隆基、通威、赣锋、天赐不同：

- 隆基代表核心资产先见顶；
- 通威代表瓶颈利润扩产后反噬；
- 赣锋 / 天齐代表资源价格泡沫；
- 天赐 / 多氟多 / 恩捷代表材料价格和毛利率周期；
- 比亚迪代表终端平台型公司能否从赛道贝塔切到公司阿尔法。

本补证卡要验证的问题是：

> **为什么新能源赛道整体退潮后，比亚迪仍然具备穿越观察价值？**

核心证据链：

**补贴驱动早期放量 → 补贴退坡筛选 → 刀片电池和 DM-i 产品周期 → 规模制造和垂直整合 → 2021–2022 股价和销量共振 → 2023–2025 价格战检验成本、品牌、渠道、海外化和现金流 → 从赛道贝塔转向公司阿尔法验证。**

## 三、阶段一：2015–2017 补贴驱动和第一轮认知

### 1. 政策和产业背景

2015–2017 年，新能源车补贴和推广政策推动行业快速放量。比亚迪凭借整车和电池一体化，成为 A 股新能源车主题的核心标的。

早期市场关注：

- 新能源车销量；
- 插混和纯电车型；
- 动力电池能力；
- 补贴车型目录；
- 公交和专用车等政策需求；
- 整车 + 电池垂直整合。

### 2. 待补数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 比亚迪新能源车销量 | 验证第一轮放量 | 公司公告 / 中汽协 |
| 新能源车补贴政策 | 验证政策驱动 | 财政部 / 工信部 |
| 汽车业务收入 | 验证业务增长 | 年报 |
| 动力电池相关收入 | 验证垂直整合 | 年报 / 公司资料 |
| 汽车业务毛利率 | 验证补贴期盈利质量 | 年报 |

### 3. 事后验证

这一阶段的新能源方向是对的，但补贴驱动模式并不稳定。

真正能穿越的不是“拿补贴”，而是后续形成：

- 产品力；
- 成本力；
- 技术路线；
- 规模制造；
- 品牌和渠道。

## 四、阶段二：2018–2019 补贴退坡和旧逻辑被证伪

### 1. 产业背景

2018–2019，补贴退坡提高技术门槛，低续航、补贴套利、低端客车和部分专用车链条承压。

对比亚迪而言，市场担心：

- 补贴减少影响利润；
- 传统车型竞争力不足；
- 新能源车销量结构变化；
- 汽车业务毛利率承压；
- 公司仍需证明产品力。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 新能源车销量 | 看补贴退坡影响 |
| 汽车收入 | 看规模变化 |
| 汽车毛利率 | 看盈利压力 |
| 归母净利润 / 扣非净利润 | 看主业质量 |
| 研发费用 | 看产品周期投入 |
| 经营现金流 | 看低谷韧性 |

### 3. 事后验证

补贴退坡不是比亚迪被证伪，而是旧模式被证伪。

后续真正打开估值的不是补贴，而是：

- 刀片电池；
- 汉；
- DM-i；
- 王朝 / 海洋产品矩阵；
- 垂直整合降本。

## 五、阶段三：2020 刀片电池、汉和主升启动

### 1. 产业背景

2020 年是比亚迪从补贴逻辑切向产品逻辑的重要年份。

关键节点包括：

- 刀片电池发布；
- 汉车型推出；
- 双碳目标提出；
- 新能源车需求恢复；
- 市场重新定价比亚迪的电池技术和整车平台。

### 2. K 线含义

比亚迪主升不是单纯跟随新能源车赛道，而是产品周期和技术周期共振。

市场开始相信：

- 比亚迪电池安全性和成本优势；
- 车型产品力改善；
- 混动路线有爆发潜力；
- 垂直整合可转化为单车成本优势。

## 六、阶段四：2021–2022 DM-i 放量和主升重估

### 1. 产业背景

2021–2022，比亚迪进入产品大周期。

核心变化：

- DM-i 超级混动放量；
- 纯电和插混双轮驱动；
- 新能源车渗透率加速；
- 比亚迪停止燃油车生产；
- 垂直整合带来供应链韧性；
- 锂价高位下仍能凭成本能力争夺份额。

### 2. 财务验证

待补 2021–2022：

- 新能源车销量；
- 纯电 / 插混销量结构；
- 汽车业务收入；
- 汽车业务毛利率；
- 单车收入；
- 单车利润；
- 归母净利润；
- 扣非净利润；
- 经营现金流；
- 研发费用；
- 海外销量；
- 电池外供或电池业务进展。

### 3. 事后验证

这一阶段比亚迪的上涨与材料股不同。

材料股依赖产品价格上涨；比亚迪更依赖：

- 产品力；
- 规模；
- 成本；
- 渠道；
- 垂直整合；
- 品牌扩展。

这就是它具备穿越观察价值的原因。

## 七、阶段五：2023–2025 价格战和公司阿尔法验证

### 1. 产业背景

2023–2025，中国新能源车进入价格战和成熟竞争阶段。

比亚迪面对的考验：

- 车企价格战；
- 冠军版等降价策略；
- 新势力和传统车企竞争；
- 智能化短板被市场讨论；
- 高端品牌能否突破；
- 海外化能否贡献增量；
- 单车利润和毛利率能否维持。

### 2. 穿越验证框架

比亚迪能否穿越，不看单一销量，而看：

1. 销量增长是否同时伴随现金流；
2. 降价后毛利率是否仍有韧性；
3. 海外收入是否提升；
4. 高端品牌是否形成利润贡献；
5. 研发费用是否转化为智能化和平台能力；
6. 价格战是否扩大份额而不是只牺牲利润。

### 3. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 销量、汽车毛利率、单车利润、现金流 | 看价格战第一阶段穿越能力 |
| 2024 | 海外销量、毛利率、净利率、研发费用 | 看出海和智能化投入 |
| 2025 | 中报 / 年报、高端品牌、海外收入、价格策略 | 看成熟期公司阿尔法 |

### 4. 事后判断

比亚迪不是传统意义上的“阴影期材料股”。更准确的状态是：

> 赛道贝塔退潮后，比亚迪进入公司阿尔法验证期。它能否穿越取决于价格战中毛利率、现金流、海外化和产品结构。

## 九、比亚迪回填后的核心判断

比亚迪路径可以概括为：

1. **2015–2017：补贴驱动的第一轮新能源认知。**
2. **2018–2019：补贴旧逻辑被证伪。**
3. **2020：刀片电池和汉带来产品 / 技术周期。**
4. **2021–2022：DM-i 和规模放量，完成主升重估。**
5. **2023–2025：价格战下的公司阿尔法验证。**

比亚迪真正区别于资源和材料股的是：

> 它不是依靠上游价格上涨获得利润，而是依靠产品、成本、规模、供应链和品牌竞争。

## 十、对 AI 的映射

比亚迪对应 AI 中的终端平台型穿越样本。

AI 里类似公司可能是：

- 终端 AI 设备平台；
- 机器人整机平台；
- AI 应用平台；
- 软硬件一体化公司；
- 具备自研模型 / 芯片 / 应用 / 渠道闭环的公司。

核心判断：

> 终端平台能否穿越，不看概念热度，而看产品力、成本、用户、渠道、现金流和持续迭代能力。
