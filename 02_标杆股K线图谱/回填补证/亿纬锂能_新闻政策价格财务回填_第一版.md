## 一、回填目标

亿纬锂能是新能源周期中非常典型的第二梯队多业务电池平台样本。

它既不是宁德时代那样的动力电池绝对龙头，也不是比亚迪那样的整车终端平台，而是：

**消费电池基本盘 + 动力电池扩张 + 储能第二曲线 + 圆柱电池路线 + 客户结构验证。**

本补证卡要验证的问题是：

> **多业务平台到底是第二增长曲线，还是多个估值想象的叠加？**

核心证据链：

**消费电池提供底盘 → 新能源车和储能打开动力 / 储能电池空间 → 2020–2022 多业务成长被市场重估 → 产能扩张和客户验证进入关键期 → 2023–2025 电池价格竞争、储能毛利率和资本开支回报成为核心验证。**

## 三、阶段一：2015–2017 多业务锂电认知

### 1. 产业背景

2015–2017 年，新能源车补贴推动动力电池需求，锂电产业链整体被市场关注。亿纬的特殊性在于，它并不是从零开始做动力电池，而是已经有消费电池和锂原电池业务基础。

市场早期关注：

- 消费电池稳定收入；
- 动力电池扩张想象；
- 锂电平台化；
- 新能源车补贴；
- 材料和电池国产化。

### 2. 待补数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 消费电池收入 | 验证基本盘 | 年报 |
| 动力电池收入 | 验证新业务 | 年报 |
| 新能源车销量 | 验证需求 | 中汽协 |
| 动力电池装机 | 验证行业需求 | GGII / 工信部 |
| 毛利率和现金流 | 验证盈利质量 | 年报 |

### 3. 事后验证

亿纬早期逻辑不是单一动力电池，而是多业务延展。它的研究重点不能只看某一项业务，而要看业务组合是否能形成现金流和利润闭环。

---

## 四、阶段二：2018–2019 补贴退坡和客户筛选

2018–2019，补贴退坡和动力电池集中度提升，市场对二线电池厂更谨慎。

亿纬需要验证：

- 消费电池是否稳定；
- 动力电池客户是否真实；
- 新业务是否拖累利润；
- 资本开支是否过重；
- 经营现金流是否足以支撑扩张。

财务重点：

| 指标 | 用途 |
|---|---|
| 消费电池收入和毛利率 | 看基本盘 |
| 动力电池收入和毛利率 | 看新业务质量 |
| 客户结构 | 看订单确定性 |
| 资本开支 | 看扩张压力 |
| 现金流 | 看利润质量 |
| 存货 | 看周期风险 |

事后看，这一阶段是二线电池平台的筛选期。能活下来的不是讲故事的公司，而是有客户、资金、技术和旧业务底盘的公司。

## 五、阶段三：2020 主升启动与平台重估

2020 年，特斯拉国产、双碳和新能源车需求恢复推动动力电池链条重估。亿纬被市场重新定价为多业务锂电平台。

催化因素：

- 动力电池需求恢复；
- 储能需求萌芽；
- 圆柱电池路线预期；
- 消费电池提供现金流底盘；
- 机构寻找宁德之外的电池弹性标的。

待补证据：

| 证据 | 用途 | 来源 |
|---|---|---|
| 2020 动力电池装机 | 验证需求恢复 | GGII / 中汽协 |
| 2020 储能电池出货 | 验证第二曲线萌芽 | GGII / EVTank |
| 亿纬 2020 年报 | 验证财务修复 | 年报 |
| 圆柱电池和客户资料 | 验证技术路线 | 公告 / 研报 |
| 资本开支和产能规划 | 验证扩张 | 公告 / 年报 |

## 六、阶段四：2021–2022 多业务重估和高位定价

2021–2022，亿纬的多条曲线被市场充分定价：

- 动力电池；
- 储能电池；
- 消费电池；
- 圆柱电池；
- 客户拓展；
- 产能扩张。

高点需要验证的不是收入增长，而是利润质量：

| 指标 | 验证问题 |
|---|---|
| 动力电池收入和毛利率 | 动力业务是否有利润 |
| 储能收入和毛利率 | 储能是利润曲线还是收入曲线 |
| 消费电池毛利率 | 基本盘是否稳定 |
| 资本开支 | 扩张是否过重 |
| ROIC | 投入是否创造回报 |
| 现金流 | 报表利润是否真实 |
| 客户结构 | 订单是否集中或不稳定 |

事后看，2021–2022 的逻辑有真实产业基础，但高估值隐含了过多业务曲线同时兑现的假设。

---

## 七、阶段五：2023–2025 半穿越验证

2023–2025，动力电池和储能需求仍增长，但电池价格竞争、储能系统价格下行、客户议价和产能利用率成为核心压力。

亿纬的核心问题：

1. 动力电池是否能获得稳定客户和毛利率；
2. 储能收入增长是否能转化为利润；
3. 圆柱路线是否兑现为订单；
4. 资本开支是否形成折旧压力；
5. 消费电池基本盘能否对冲周期；
6. 现金流是否支撑多业务扩张。

准确状态：

> 亿纬是半穿越样本，不是简单阴影股。它能否真正穿越，取决于动力和储能业务的利润质量以及资本开支回报。

## 九、亿纬回填后的核心判断

亿纬路径可以概括为：

1. **2015–2017：从消费电池延伸到锂电平台想象。**
2. **2018–2019：补贴退坡下验证客户和基本盘。**
3. **2020：动力和储能需求恢复，平台价值重估。**
4. **2021–2022：多业务曲线被充分定价。**
5. **2023–2025：半穿越验证，核心看毛利率、现金流和资本开支回报。**

## 十、对 AI 的映射

亿纬对应 AI 中第二梯队多业务平台公司。

类似公司可能有：

- 传统硬件公司切入 AI 服务器；
- 传统电源公司切入数据中心电力；
- 散热公司切入液冷；
- 工业软件公司切入 AI 应用；
- 多业务机器人零部件平台。

核心经验：

> 多业务不是天然优势。第二曲线必须兑现为毛利率、现金流和 ROIC，否则只是估值想象。
