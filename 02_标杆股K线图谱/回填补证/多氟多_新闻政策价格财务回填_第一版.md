## 一、回填目标

多氟多是锂电材料组中“单品周期弹性”的典型样本。

天赐材料更像：

**电解液 / 六氟一体化材料平台。**

多氟多更像：

**六氟磷酸锂单品价格弹性 + 氟化工和新材料延伸。**

本补证卡要验证的问题是：

> **单一材料价格暴涨带来的利润，能不能转化成长期平台价值？**

核心证据链：

**动力电池需求高增 → 六氟供需紧张 → 六氟价格暴涨 → 多氟多利润弹性释放 → 高利润刺激扩产 → 六氟供给释放 → 价格回落 → 毛利率和利润中枢下移 → 股价从单品成长切回周期材料。**

## 三、阶段一：2015–2017 第一轮六氟认知

### 1. 产业背景

2015–2017 年，新能源车补贴推动动力电池需求增长，电解液需求上升，六氟磷酸锂作为电解液关键溶质被市场重估。

多氟多早期关注点：

- 六氟产能；
- 六氟价格；
- 氟化工技术底盘；
- 锂电材料国产化；
- 下游电池和电解液客户。

### 2. 待补价格数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 六氟磷酸锂价格 | 验证单品涨价弹性 | SMM / 百川 / Wind |
| 电解液价格 | 验证下游传导 | SMM / 百川 |
| 碳酸锂价格 | 验证原材料成本 | SMM / 百川 |
| 动力电池装机 | 验证需求 | GGII / 中汽协 |
| 新能源车销量 | 验证终端 | 中汽协 |

### 3. 待补财务数据

需补 2015–2017：

- 营业收入；
- 归母净利润；
- 六氟业务收入；
- 六氟业务毛利率；
- 氟化工业务收入；
- 资本开支；
- 存货；
- 经营现金流。

### 4. 事后验证

第一轮行情说明六氟确实是锂电材料重要品种，但不是永久稀缺品。

这为 2021–2022 第二轮泡沫埋下了参考坐标。

## 四、阶段二：2018–2019 补贴退坡和六氟周期退潮

### 1. 产业背景

2018–2019，新能源车补贴退坡，动力电池产业链降温，六氟价格回落。

多氟多承压来自：

- 六氟价格下跌；
- 毛利率下降；
- 单品利润回落；
- 材料估值收缩；
- 市场重新认识六氟的周期属性。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 六氟业务收入 | 看价格下跌对规模影响 |
| 六氟毛利率 | 看利润弹性下行 |
| 归母净利润 | 看报表冲击 |
| 存货 | 看跌价风险 |
| 经营现金流 | 看低谷韧性 |
| 新材料收入 | 看是否有第二曲线 |

### 3. 事后验证

这一阶段证明：

> 单品价格弹性不能当作长期护城河。价格一旦回落，利润和估值会迅速收缩。

---

## 五、阶段三：2020 第二轮六氟行情前夜

### 1. 产业背景

2020 年，EV 需求修复和双碳政策推动锂电材料链复苏，六氟价格底部企稳。

多氟多第二轮行情基础：

- 六氟价格低位；
- 供给释放慢；
- 动力电池排产上修；
- 市场寻找材料价格弹性；
- 公司具备六氟技术和产能基础。

### 2. 待补证据

| 证据 | 用途 | 来源 |
|---|---|---|
| 2020 六氟价格 | 验证底部反转 | SMM / 百川 |
| 2020 动力电池装机 | 验证需求 | GGII / 中汽协 |
| 多氟多 2020 年报 | 验证财务底部 | 年报 / Wind |
| 六氟扩产公告 | 验证供给计划 | 公司公告 |
| 当时研报 | 验证市场如何理解六氟弹性 | 券商研报 |

### 3. 事后验证

2020 是第二轮六氟行情的准备期，真正的股价和利润弹性在 2021 年体现。

---

## 六、阶段四：2021–2022 六氟价格暴涨与利润弹性释放

### 1. 产业背景

2021–2022，动力电池排产快速提升，六氟供需错配明显，六氟价格快速上涨。

多氟多在这一阶段弹性强，原因是：

- 六氟单品价格弹性集中；
- 报表对六氟毛利率高度敏感；
- 市场偏好材料涨价标的；
- 单品价格上涨带来利润想象空间。

### 2. 财务验证

待补 2021–2022：

- 六氟业务收入；
- 六氟毛利率；
- 归母净利润；
- 扣非净利润；
- ROE；
- 经营现金流；
- 存货；
- 六氟产能和扩产计划；
- 新材料 / 电子化学品收入。

### 3. 当时市场理解

市场高点附近相信：

- 动力电池需求长期高增；
- 六氟供需紧张持续；
- 公司六氟技术和产能有定价权；
- 新材料业务能延展估值。

### 4. 事后验证

成立的是：

- 六氟确实在 2021–2022 出现供需错配；
- 多氟多利润确实对价格高度敏感。

被证伪的是：

- 六氟高价长期维持；
- 单品毛利率可以永久高位；
- 单品弹性可以自动转化为平台护城河。

### 5. AI 映射

对应 AI 中单品价格弹性股：

- 某类 CCL；
- 某类高速连接材料；
- 某类液冷材料；
- 某类光模块辅材；
- 某类电源核心器件。

这类公司上涨最猛，但最需要警惕扩产和替代。

## 七、阶段五：2023–2025 六氟价格回落和周期弹性观察

### 1. 产业背景

2023–2025，六氟供给释放，电解液产业链价格下行，电池厂压价，材料利润下降。

多氟多承压来自：

- 六氟价格回落；
- 六氟毛利率下行；
- 单品利润中枢下降；
- 存货跌价和减值风险；
- 原有高估值体系破裂；
- 新材料第二曲线尚需验证。

### 2. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 六氟收入、毛利率、存货、现金流 | 看价格下跌初期影响 |
| 2024 | 扣非净利润、减值、新材料收入 | 看低价环境与转型成效 |
| 2025 | 中报 / 年报、六氟价格、电子化学品进展 | 看周期修复和第二曲线 |

### 3. 事后结论

多氟多不是完全出清失败，但原有“六氟高景气成长股”估值被打破。

未来需要看两点：

1. 六氟价格是否企稳；
2. 新材料 / 电子化学品是否成为真正第二曲线。

## 九、多氟多回填后的核心判断

多氟多路径可以概括为：

1. **2015–2017：第一轮六氟认知。** 六氟从化工品被重估为锂电核心材料。
2. **2018–2019：单品价格退潮。** 六氟价格下跌，周期属性显现。
3. **2020：第二轮行情前夜。** EV 需求修复，六氟价格企稳。
4. **2021–2022：单品价格泡沫。** 六氟价格暴涨，利润弹性极强。
5. **2023–2025：周期重估。** 六氟价格回落，单品高景气估值破裂，等待新材料第二曲线。

### 核心结论

多氟多最重要的教训是：

**单品涨价弹性可以带来极强行情，但如果缺乏不可复制的护城河和多产品平台能力，高景气后的回撤也会非常猛烈。**

### 下一步建议

继续回填：

**恩捷股份。**

原因：

- 天赐、多氟多解释材料价格泡沫；

- 三者合起来才能完整解释锂电材料组。
