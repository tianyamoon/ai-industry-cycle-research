## 一、回填目标

阳光电源是光伏组中最重要的相对穿越样本。

它和隆基、通威、福斯特不同：

- 隆基受光伏主链价格战和产能过剩压制；
- 通威受硅料价格下跌和利润反噬压制；
- 福斯特受主链价格战向辅材传导；
- 阳光则因逆变器、电力电子、储能和海外化，具备更强的半穿越 / 穿越观察价值。

本补证卡要验证的问题是：

> **为什么同样处在光伏产业链，阳光比部分主链制造公司更具相对穿越能力？**

核心证据链：

**光伏装机增长 → 逆变器需求提升 → 平价上网和海外化强化电力电子龙头价值 → 2020–2021 主升重估 → 2021H2–2022 储能第二曲线打开 → 2023–2025 光伏主链过剩但储能和海外化提供相对韧性 → 看储能毛利率、海外收入和现金流决定穿越质量。**

## 三、阶段一：2015–2017 逆变器认知形成

### 1. 产业背景

2015–2017 年，光伏装机快速增长，逆变器作为光伏系统核心设备，进入 A 股市场认知。

阳光早期关注点：

- 光伏逆变器国产化；
- 光伏装机增长；
- 电力电子技术能力；
- 海外市场拓展；
- 从设备商向系统方案商延伸。

### 2. 待补数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 中国新增光伏装机 | 验证需求 | 国家能源局 / CPIA |
| 全球光伏装机 | 验证海外空间 | CPIA / IEA / BNEF |
| 阳光逆变器出货 | 验证份额 | 年报 / 研报 |
| 逆变器收入和毛利率 | 验证盈利质量 | 年报 |
| 海外收入 | 验证全球化 | 年报 |

### 3. 事后验证

这一阶段阳光的优势不是单纯光伏贝塔，而是电力电子能力和全球化潜力。

## 四、阶段二：2018 光伏 531 出清

### 1. 产业背景

2018 年光伏 531 新政压缩补贴和新增规模，国内光伏装机承压。

阳光受影响：

- 国内逆变器需求阶段性下修；
- 光伏板块估值收缩；
- 市场担心订单和利润；
- 但海外业务和电力电子能力仍保留。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 逆变器收入 | 看国内需求冲击 |
| 毛利率 | 看价格竞争影响 |
| 海外收入 | 看是否具备对冲能力 |
| 经营现金流 | 看低谷韧性 |
| 存货 | 看订单波动风险 |
| 研发费用 | 看技术持续投入 |

### 3. 事后验证

531 没有证伪阳光。它的意义是筛选光伏链条中能够从国内补贴装机转向全球平价需求的公司。

## 六、阶段四：2020–2021 主升重估

### 1. 产业背景

2020–2021，双碳、全球光伏装机预期和能源转型推动阳光进入主升。

市场相信：

- 光伏平价打开长期需求；
- 逆变器是高质量电力电子环节；
- 阳光全球份额提升；
- 储能开始提供第二曲线；
- 公司比部分主链制造环节更轻资产、更具平台潜力。

### 2. 财务验证

待补 2020–2021：

- 逆变器收入；
- 逆变器毛利率；
- 储能收入；
- 海外收入；
- 归母净利润；
- 扣非净利润；
- 经营现金流；
- 存货；
- 研发费用；
- 全球份额。

### 3. 事后验证

阳光主升逻辑成立，但 2021 高估值同样会受到新能源抱团影响。

真正区别于隆基、通威的是：

- 逆变器价格周期压力相对不同；
- 海外化更强；
- 储能第二曲线可延续估值；
- 电力电子属性高于纯制造主链。

---

## 七、阶段五：2021H2–2022 储能第二曲线强化

### 1. 产业背景

2021H2–2022，储能需求开始成为阳光估值的重要延展。

催化包括：

- 新能源装机提高对储能需求；
- 海外能源价格和能源安全；
- 大储和户储需求；
- 光储一体化；
- 储能系统集成能力；
- 海外渠道和认证。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 储能收入 | 看第二曲线规模 |
| 储能毛利率 | 看第二曲线质量 |
| 海外收入占比 | 看全球化能力 |
| 经营现金流 | 看利润质量 |
| 存货 | 看储能订单和库存风险 |
| 逆变器毛利率 | 看主业稳定性 |

### 3. 事后验证

储能是阳光相对穿越的关键，但不能只看收入增长。

需要看：

- 毛利率是否稳定；
- 现金流是否跟上；
- 是否存在低价竞争；
- 海外收入是否可持续；
- 储能系统集成是否带来真实利润。

---

## 八、阶段六：2023–2025 相对穿越验证

### 1. 产业背景

2023–2025，光伏主链进入过剩和价格战，隆基、通威等承压明显。

阳光的相对优势在于：

- 逆变器需求仍随全球装机增长；
- 储能业务提供第二曲线；
- 海外收入提供利润弹性；
- 电力电子平台属性强于单一制造价格周期；
- 但储能系统竞争和库存风险需要验证。

### 2. 穿越验证框架

阳光能否穿越，看五个指标：

1. 逆变器毛利率是否稳定；
2. 储能收入是否继续增长；
3. 储能毛利率是否真实；
4. 海外收入和现金流是否匹配；
5. 存货和应收是否可控。

### 3. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 储能收入、海外收入、毛利率、现金流 | 看主链退潮中的韧性 |
| 2024 | 储能毛利率、逆变器毛利率、存货 | 看半穿越质量 |
| 2025 | 中报 / 年报、海外订单、现金流 | 看是否从半穿越走向穿越 |

### 4. 事后判断

阳光不是完全脱离周期，而是相对穿越。

准确表述：

> 光伏主链过剩背景下，阳光的电力电子、储能和海外化使其比部分主链制造更有韧性，但最终仍要用储能毛利率、海外现金流和存货质量验证。

## 十、阳光回填后的核心判断

阳光电源路径可以概括为：

1. **2015–2017：逆变器认知。** 光伏装机推动逆变器国产化和电力电子龙头认知。
2. **2018：531 出清。** 国内装机承压，但全球化和技术能力未被证伪。
3. **2019：平价上网前夜。** 成本下降和全球需求强化逆变器长期空间。
4. **2020–2021：主升重估。** 双碳和光伏平价推动阳光进入核心资产池。
5. **2021H2–2022：储能第二曲线。** 储能强化相对穿越逻辑。
6. **2023–2025：半穿越验证。** 光伏主链过剩背景下，阳光要用储能利润、海外收入和现金流证明韧性。

## 十二、对 AI 的映射

阳光对应 AI 中的基础设施配套平台型公司。

类似方向：

- 数据中心电力设备；
- UPS / 电源系统；
- 液冷系统集成；
- 配电和变压器；
- 能源管理平台；
- 服务器基础设施系统商。

核心经验：

> 在主产业链价格战中，相对穿越公司通常不是需求最大的一环，而是能提供系统能力、海外渠道、第二曲线和现金流的环节。
