## 一、回填目标

恩捷股份是锂电材料组里区别于天赐、多氟多的重要样本。

天赐、多氟多更偏价格弹性：

- 六氟；
- 电解液；
- 单品涨价；
- 材料利润泡沫。

恩捷更偏高壁垒制造：

- 湿法隔膜；
- 客户认证；
- 良率和设备；
- 规模和成本；
- 扩产、折旧和客户压价。

本补证卡要验证的问题是：

> **有客户认证、工艺壁垒和龙头份额的材料公司，为什么仍会在 2023–2025 被价格竞争和盈利中枢下移压制？**

核心证据链：

**动力电池需求高增 → 隔膜供需偏紧 → 恩捷份额和毛利率被重估 → 高估值和高扩产 → 产能释放、客户压价和 ASP 下行 → 毛利率和 ROIC 回落 → 高壁垒材料从成长切回周期制造。**

## 三、阶段一：2016–2017 高端隔膜认知

### 1. 产业背景

2016–2017 年，新能源车补贴推动动力电池增长，湿法隔膜国产替代成为锂电材料的重要方向。

恩捷早期被市场关注的变量：

- 湿法隔膜技术；
- 客户认证；
- 良率和设备能力；
- 动力电池客户；
- 高端隔膜国产替代；
- 与普通材料不同的工艺壁垒。

### 2. 待补数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 隔膜出货量 | 验证需求增长 | GGII / EVTank / 公司年报 |
| 湿法隔膜占比 | 验证技术路线 | GGII / 券商研报 |
| 隔膜 ASP | 验证价格 | GGII / 券商研报 / Wind |
| 动力电池装机 | 验证需求 | GGII / 中汽协 |
| 主要客户认证 | 验证壁垒 | 公司公告 / 年报 |

### 3. 待补财务数据

需补 2016–2017：

- 营业收入；
- 隔膜收入；
- 隔膜毛利率；
- 出货量；
- 资本开支；
- 研发费用；
- 客户结构；
- 现金流。

### 4. 事后验证

隔膜确实具备比普通材料更强的工艺壁垒和客户认证壁垒。

但后续也证明：

> 有壁垒不代表不会周期化。

## 四、阶段二：2018–2019 补贴退坡与龙头筛选

### 1. 产业背景

2018–2019，新能源车补贴退坡，动力电池行业集中度提升，低端电池和材料厂承压。

恩捷需要验证的是：

- 客户是否稳定；
- 出货是否增长；
- 毛利率是否有韧性；
- 资本开支是否合理；
- 是否逆势扩大份额。

### 2. 当时市场理解

市场对隔膜的理解从：

**锂电材料高成长**

逐步转向：

**高壁垒材料龙头，低迷期强者更强。**

### 3. 财务验证重点

| 指标 | 用途 |
|---|---|
| 隔膜收入 | 看需求和份额 |
| 隔膜毛利率 | 看价格压力和壁垒 |
| 出货量 | 看客户绑定 |
| 资本开支 | 看扩产节奏 |
| 经营现金流 | 看盈利质量 |
| 客户集中度 | 看下游议价风险 |

### 4. 事后验证

恩捷在这一阶段没有被证伪，反而证明隔膜龙头具备更强筛选能力。

## 五、阶段三：2020 EV 需求恢复与主升启动

### 1. 产业背景

2020 年，特斯拉国产、双碳和新能源车需求恢复带动动力电池排产上修，隔膜需求随之改善。

恩捷受益于：

- 动力电池装机恢复；
- 储能需求开始形成远期想象；
- 头部电池厂扩产；
- 高端湿法隔膜产能价值提升；
- 机构资金配置锂电材料龙头。

### 2. 待补证据

| 证据 | 用途 | 来源 |
|---|---|---|
| 2020 动力电池装机 | 验证需求恢复 | GGII / 中汽协 |
| 2020 隔膜出货量 | 验证订单增长 | GGII / 公司年报 |
| 2020 隔膜 ASP | 验证价格状态 | GGII / 研报 |
| 恩捷 2020 年报 | 验证财务修复 | 年报 |
| 扩产公告 | 验证资本开支 | 公司公告 |

### 3. 事后验证

2020 是恩捷从材料龙头进入核心资产池的阶段，但高估值真正集中发生在 2021–2022。

## 六、阶段四：2021–2022 高壁垒材料抱团与扩产高峰

### 1. 产业背景

2021–2022，动力电池和储能需求高增，隔膜需求快速增长，高端湿法隔膜供需偏紧。

恩捷被市场赋予高估值，原因是：

- 隔膜是电池核心材料；
- 客户认证周期长；
- 工艺、良率和设备壁垒；
- 龙头份额和客户结构；
- 产能扩张支撑收入增长；
- 高毛利率被市场外推。

### 2. 财务验证

待补 2021–2022：

- 隔膜收入；
- 隔膜出货量；
- 隔膜 ASP；
- 隔膜毛利率；
- 归母净利润；
- 扣非净利润；
- ROE；
- 资本开支；
- 产能规划；
- 现金流；
- 客户结构。

### 3. 当时市场理解

市场高点附近相信：

- 隔膜龙头壁垒强；
- 头部电池客户稳定；
- 产能紧张可维持高毛利；
- 恩捷能够享受锂电材料核心资产估值；
- 隔膜不完全像六氟那样容易周期化。

### 4. 事后验证

恩捷的壁垒确实高于很多单品材料，但高点假设仍有问题：

- 扩产最终会释放；
- 电池厂会压价；
- ASP 下降会影响毛利率；
- 折旧压力会影响 ROIC；
- 高壁垒不能消灭供需周期。

### 5. AI 映射

对应 AI 中高壁垒核心材料 / 部件：

- 高端 PCB；
- 高端 CCL；
- 先进封装材料；
- 高可靠性液冷部件；
- 电源核心器件；
- 高速连接器。

关键问题不是“有没有壁垒”，而是：

> 壁垒是否足以抵抗扩产、折旧和客户压价？

## 七、阶段五：2023–2025 扩产、客户压价和高壁垒周期化

### 1. 产业背景

2023–2025，锂电材料整体退潮，隔膜行业也进入价格竞争和盈利中枢下移阶段。

压力来自：

- 行业产能释放；
- 电池厂议价增强；
- 隔膜 ASP 下行；
- 毛利率下降；
- 折旧和资本开支压力；
- 存货和减值风险；
- 市场从高壁垒成长股切回周期制造龙头。

### 2. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 隔膜收入、出货量、ASP、毛利率 | 看价格竞争第一阶段影响 |
| 2024 | 扣非净利润、资本开支、折旧、现金流 | 看盈利中枢下移和投资压力 |
| 2025 | 中报 / 年报、产能利用率、客户结构 | 看是否具备修复条件 |

### 3. 事后结论

恩捷不是出清失败，但进入高壁垒材料阴影期。

准确表述：

> 客户认证和工艺壁垒仍有价值，但不能让公司完全脱离材料价格、产能扩张、折旧和下游压价周期。

## 九、恩捷回填后的核心判断

恩捷股份路径可以概括为：

1. **2016–2017：高端隔膜认知。** 湿法隔膜和客户认证让恩捷区别于普通材料股。
2. **2018–2019：龙头筛选。** 补贴退坡后，客户、工艺和良率优势保留。
3. **2020：主升启动。** EV 需求恢复和电池排产上修推动隔膜龙头重估。
4. **2021–2022：高壁垒材料抱团。** 供需偏紧、客户认证和高毛利被充分定价。
5. **2023–2025：高壁垒周期化。** 扩产、客户压价、ASP 下行和折旧压力让估值从成长切回制造。
