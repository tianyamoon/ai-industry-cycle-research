# AI产业周期研究

本仓库不再按历史编号理解，而是按三层研究生产链理解：

1. `过程层`
   研究方法论、历史复盘、时间轴资料、硬证据补证、图表素材。
2. `桥接层`
   标准行业研究卡、图谱坐标、样本验证卡、组比较、滚动验证清单。
3. `结果层`
   顶级基金风格的正式总结、总表整合、阶段上收与正式写作区。

`00_总控台` 负责控制面与方法校准，`99_交接包` 负责当前接管入口与历史归档。

## 维护纪律

1. 状态变化先回写 `00_总控台/项目状态面板.md`。
2. 新增样本、组比较或滚动清单，先回写桥接层主索引与总映射索引。
3. 新证据优先回写过程层主表，不优先平行新建解释文档。
4. 正式结论只在结果层上收，不在桥接层提前伪装成终稿。
5. `99_交接包` 保留历史轨迹，但不替代状态源与正式主文。
