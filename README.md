# AI产业周期研究

本仓库用于沉淀“产业周期研究”项目成果，核心目标是：

> 以新能源产业周期为历史模板，研究 AI 产业链从概念导入、产业验证、主升重估、资本开支扩张，到过剩退潮和龙头分化的完整路径。

---

## 当前主线

### 01｜以史为鉴：新能源周期复盘

通过 2015–2025 年 A 股新能源产业周期，建立可迁移到 AI 的研究框架。

重点研究：

- 第一轮概念炒作如何发生；
- 第一轮退潮后如何筛选真正龙头；
- 2020–2021 主升浪为什么形成；
- 2021 年产业利好最多时，为什么股价反而陆续见顶；
- 上游资源行情为什么最后最疯狂；
- 资本开支如何反噬；
- 谁真正穿越周期，谁仍在炒作后的阴影里；
- 这些经验如何映射当前 AI 产业链。

## 推荐使用顺序

1. 先读 `00_总控台/研究框架.md`；
2. 再读 `00_总控台/分会话提示词库.md`；
3. 按照提示词逐一创建分会话；
4. 每个分会话结束后，输出交接包；
5. 把交接包保存到 `99_交接包/`；
6. 后续总结和公众号正文优先读取交接包。
