# AI产业周期研究

本仓库现在按“少入口、少重复、强分层、好追踪”治理。

- 研究状态只看 `00_总控台/项目状态面板.md`
- 整理状态只看 `00_总控台/项目治理看板.md`
- 活文档统一使用稳定文件名，不再在活跃目录并行保留 `第一版/第二版/第三版`
- 历史留痕只进入 `90_历史归档/` 或 `90_执行留痕/`
- 临时、缓存、实验文件只进入 `tmp/`

## 顶层分工

- `00_总控台`：方法论、状态源、治理规则、总映射索引
- `01_过程层`：原始证据、闭环补证、历史复盘、图表素材
- `02_桥接层`：研究卡、样本池、组比较、滚动验证
- `03_结果层`：正式总结、总表整合、正式写作
- `99_交接包`：当前接管入口与历史交接归档

## 当前阅读路径

1. 先确认研究状态：`00_总控台/项目状态面板.md`
2. 再看当前评分：`03_结果层/01_总表整合与AI映射/AI产业链当前评分总览.md`
