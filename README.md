# AI产业周期研究

本仓库用于沉淀产业周期研究资料。

研究主线：

> 以新能源产业周期为历史样本，梳理产业从概念导入、产业验证、主升重估、资本开支扩张，到供需再平衡和龙头分化的完整路径，并建立可迁移到 AI 产业链研究的分析框架。

---

## 当前主线

### 01｜以史为鉴：新能源周期复盘

通过 2015–2025 年新能源产业周期，建立可迁移到 AI 产业链研究的分析框架。

重点研究：

- 第一轮概念扩散如何发生；
- 第一轮退潮后如何筛选真正龙头；
- 主升浪为什么形成；
- 产业利好较多时，为什么部分核心资产会率先见顶；
- 上游资源和材料行情为什么在后半段更强；
- 资本开支如何影响利润和估值；
- 哪些公司真正穿越周期，哪些公司仍处在周期阴影中；
- 这些经验如何迁移到 AI 产业链研究。
