## 三、04 模块核心结论

### 1. 新能源给 AI 的最大启示

**产业会长期增长，但股价不会线性增长。**

需求之外，还必须看：

- 价格；
- 毛利率；
- 供给；
- 资本开支；
- 现金流；
- 存货和减值；
- 资金拥挤度；
- ROIC；
- 第二曲线利润质量。

### 2. AI 的八阶段模型

| 阶段 | 含义 |
|---|---|
| 第一轮概念认知 | 政策、模型突破、算力建设启动 |
| 第一次出清 | ROI 质疑、CapEx 放缓、伪概念退场 |
| 低位筛选 | 真客户、真订单、真龙头形成 |
| 主升浪 | 订单、业绩、政策和资金共振 |
| 核心资产见顶 | 产业仍强，但估值和持仓先打满 |
| 瓶颈利润泡沫 | HBM、光模块、电力设备等利润最强 |
| 过剩和价格战 | ASP 下行、扩产释放、客户压价 |
| 穿越 / 阴影 | 从赛道贝塔转向公司阿尔法 |

### 3. AI 产业链角色映射

| 新能源角色 | AI 对应角色 |
|---|---|
| 比亚迪 | AI 终端平台、机器人整机、AI 应用平台 |
| 宁德时代 | AI 核心供应商，如芯片、HBM、光模块 |
| 隆基 | AI 硬件制造龙头 |
| 通威 / 赣锋 / 天齐 | AI 瓶颈产能和稀缺资源 |
| 天赐 / 多氟多 / 恩捷 / 福斯特 | AI 材料、辅材和耗材 |
| 先导智能 | AI 设备和资本开支链 |
| 阳光电源 | AI 基础设施配套平台，如电力、液冷、能源管理 |
| 亿纬 / 华友 | 多业务复合平台 |
