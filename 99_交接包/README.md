# 交接包

本目录用于保存每个研究分会话结束后的交接摘要。

交接包是整个项目最重要的上下文传递机制。后续总结会话、正文会话、图表会话，都应该优先读取这里的内容。

---

## 一、交接包命名规则

建议命名：

```text
01_新能源周期总表交接包.md
02_K线图谱交接包.md
03_历史新闻流交接包.md
04_产业数据交接包.md
05_资金参与度交接包.md
06A_隆基绿能案例交接包.md
06B_通威股份案例交接包.md
06C_比亚迪案例交接包.md
07_正文交接包.md
08_图表配图交接包.md
09_最终总结交接包.md
```

---

## 二、每份交接包必须包含

1. 本会话研究目标；
2. 已完成内容；
3. 核心结论；
4. 关键证据；
5. 重要表格；
6. 待补内容；
7. 对后续会话的建议；
8. 可直接复用段落。

详细模板见：

```text
00_总控台/交接包模板.md
```

---

## 三、使用方式

每完成一个分会话：

1. 让 ChatGPT 输出“本会话交接包”；
2. 将交接包保存为 Markdown 文件；
3. 放入本目录；
4. 后续总结时优先读取本目录。
