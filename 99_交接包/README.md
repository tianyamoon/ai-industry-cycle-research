## 新增交接包的条件

只在以下情况新增交接包：

1. 主线顺序发生明显变化
2. 某个专题形成阶段收口
3. 某组样本完成后，下一步接管路径明显改变
4. 如果不留交接包，下一会话很容易误判当前状态

如果只是补一条证据、补一个字段、补一张卡，优先回写原文件，不默认新建交接包。
