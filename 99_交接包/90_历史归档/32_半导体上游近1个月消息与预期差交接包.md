## 三、本轮核心结论

1. 当前最有预期差的，不是“所有国产替代”，而是 `已出现正式订单 / 业绩兑现的高端材料与测试 / 量检测设备`。
2. 其中更高确定性的两条线是：
   - `半导体测试 / 量检测设备`
   - `已拿订单的高端光刻材料 / CMP材料`
3. `300mm半导体硅片` 与 `先进封装 / 封装材料` 逻辑成立，但当前仍更依赖后续客户验证、产能利用率和中报 / 三季报继续兑现，因此归入 `中确定性预期差`。
4. `AI存储链` 在本窗口里更适合写成 `景气背景`，而不是“最有预期差”的一级主结论，因为市场关注度已经很高。
5. 近1个月中日互卡链条里，`已证实的新变化` 更接近中国侧对关键矿产 / 受管制物项执法强化的外溢，而不是日本侧新增一轮全面半导体材料断供升级。
6. `日本对华全面断供ArF/EUV光刻胶` 这类说法，在本时间窗内没有足够强的官方或权威媒体交叉验证，只能留在 `传闻层 / 待核层`，不能进入主结论。
