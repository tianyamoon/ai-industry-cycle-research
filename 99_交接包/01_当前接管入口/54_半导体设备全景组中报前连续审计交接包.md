## 三、当前最重要的已成立判断

1. 半导体设备方向当前最合理的主线，不是继续横向扩设备公司，而是固定按 `长川科技 -> 盛美上海 -> 旁证扩展位` 的顺序做连续审计。
2. `长川科技` 仍是最硬的利润兑现主锚，且已新增 `2026H1` 业绩预告验证。
3. `盛美上海` 仍是订单强化补强位，但本轮新增披露更多偏治理与资本动作，仍待中报把订单继续传导到收入、利润与 OCF。
4. `中微公司 / 北方华创 / 华峰测控 / 华兴源创 / 中科飞测 / 拓荆科技 / 芯源微` 已完成首轮官方回指，但当前只能作为连续验证样本，不能提前抬成新主锚。
5. 先进封装 / TGV 相关样本继续只做旁证，不直接升级为设备主结论位。
6. 本组当前最准确的阶段定义是：`第一轮官方回指完成 -> 连续验证启动 -> 阶段切换待审计`。

## 四、当前未决问题

1. 除 `长川科技` 外，设备组尚未看到更广泛的 `2026H1` 利润预告扩散。
2. 旁证层近期新增披露较多是 `融资 / 并购 / 补助 / 风险提示 / 说明会触发`，不能直接等同经营兑现升级。
3. `华兴源创` 的半导体检测收入、毛利率、订单与利润贡献拆分仍待正式披露。
4. `中微公司 / 北方华创 / 华峰测控 / 中科飞测 / 拓荆科技 / 芯源微` 仍需中报、三季报继续验证 `订单 / 客户 / 交付 -> 收入 / 利润 / OCF` 的连续传导。
5. 先进封装 / TGV 旁证层仍需防止“工艺验证”被误写成“设备利润主线”。

## 五、下一步建议

1. 固定按 `先更新滚动验证清单 -> 再更新旁证证据表 -> 最后回写组级主文` 的顺序维护。
2. 下一轮优先盯 `2026中报 / 三季报 / 业绩预告 / 订单与客户验证 / OCF / 存货 / 应收 / 合同负债 / 预付款 / 在建工程`。
3. 任何融资、并购、补助、风险提示类公告，都必须先过资本周期审计，再决定是否上收为阶段结论。
4. 若后续标签、顺序或权重变化，再同步回写 `AI标杆股研究卡_进度索引.md`、`AI标杆股研究卡_索引与模板.md` 与 `00_总控台/项目状态面板.md`。
5. 当前不建议继续新增设备样本，除非出现能改变组级排序的新正式兑现链。
