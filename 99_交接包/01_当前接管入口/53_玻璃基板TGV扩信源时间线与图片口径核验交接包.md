## 三、本轮核心结论

1. 玻璃基板 / TGV 题材的外部产业路线，已不再只是 `Intel 2023` 单点叙事，而是至少可以补到：
   - `Intel` 官方路线定调
   - `Corning` 官方把 `CPO / 光互连 / Glass Bridge` 讲清楚
   - `TrendForce` 把 `TSMC CoPoS / glass interposer` 写进时间表
   - `SEMI` 把 `Glass Core Substrate` 升格为独立研究对象
2. 因此，这条线现在更像：
   - `海外路线持续强化`
   - `国内公司分化式映射`
   - `A股主题先板块化、再扩散、再被公告反复降温`
3. 图片里的公司不能再笼统写成“都在玻璃桥主线上”，而必须拆成：
   - `主线验证 / 平台锚`
   - `重估池`
   - `高波动扩散池`
   - `弱映射 / 已降温`
4. 图片里的很多句子，本质只是 `传播口径`，不是 `正式兑现口径`。
5. 这轮补强后，最重要的研究边界更清楚了：
   - `康宁链` 不等于 `Glass Bridge兑现`
   - `显示玻璃基板` 不等于 `先进封装玻璃载板`
   - `有送样 / 有立项 / 有加工能力` 也不等于 `有订单 / 有量产 / 有收入`

## 八、可直接复用段落

> 玻璃基板 / TGV 这条线当前最容易犯的错误，不是看不见题材，而是把传播层的句子直接当成公司已经兑现的事实。很多图片和市场转述会把“立项、送样、加工能力、合作关系、显示玻璃产线”写成“玻璃桥主线受益”，但回到正式披露后，绝大多数公司仍处于研发早期、送样早期，甚至与先进封装主线并不是同一条利润链。因此，这条线最重要的工作不是再堆更多公司，而是把传播口径和正式口径拆开。

> 从题材演进看，玻璃基板 / TGV 并非突然产生。它先由 Intel 等海外大厂把玻璃基板写进先进封装路线图，再由 Corning 把 Glass Bridge 与 CPO、光互连场景讲清楚，随后 TrendForce 与 SEMI 等机构把这条路线进一步写进时间表和产业研究框架。A 股层面，则是先有少数公司零散露出技术储备，再由莱宝高科和帝尔激光提供样品与设备验证，最终京东方A与康宁合作把主题推向板块化交易。

> 这条线的市场扩散顺序也已经比较清楚：先有平台锚出现，再回头强化纯工艺验证，然后外溢到传统低估值平台重估，最后才扩散到高弹性小票和弱映射公司。这个顺序本身，就是后续判断买点和预期差的重要边界。真正值得重仓研究的，不是最后一棒谁弹性最大，而是谁有机会把下一条正式公告升级成新的预期差。
