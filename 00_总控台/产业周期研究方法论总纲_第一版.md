# 产业周期研究方法论总纲_第一版

## 一、目的

本文件是对顶级机构产业周期方法论的压缩总结。

目标不是重复大量方法论条目，而是提炼出一套适合本仓库长期使用的产业周期研究总纲，让后续所有研究都能在同一框架下推进。

## 二、本仓库采用的方法论总公式

本仓库的产业周期研究，默认遵循以下总公式：

`结构力量 -> 宏观金融条件 -> 产业变量 -> 公司回报质量 -> 市场定价 -> 市场心理 -> 阶段映射 -> 结论`

这条链路的含义是：

1. 先判断这是短周期现象，还是长期结构变化。
2. 再判断利率、信用、流动性是否在放大或压制该产业。
3. 再拆需求、供给、成本、价格、CapEx、库存等产业变量。
4. 再看公司层面能否把景气转成 ROIC、现金流和可持续优势。
5. 再看市场价格已提前定价了多少预期。
6. 再看市场心理和资金拥挤是否已造成偏离。
7. 最后才做“当前所处阶段”的判断，并映射到 AI。

## 三、必须同时使用的六个方法论镜头

### 1. 宏观机器镜头

解决的问题：

1. 这轮行情有多少是产业逻辑，有多少是流动性和利率推动？
2. 当前信用扩张还是收缩？
3. 宏观金融条件是在放大产业趋势，还是压制产业趋势？

### 2. 周期位置镜头

解决的问题：

1. 当前位于早期、中期、后期、泡沫期还是出清期？
2. 市场是在交易改善，还是在透支改善？
3. 情绪是否已经从理性乐观进入过度外推？

### 3. 生命周期镜头

解决的问题：

1. 行业处在导入、成长、成熟、震荡还是衰退？
2. 公司当前更应该看渗透率、份额，还是看回报率、现金流？
3. 产业景气是否真的在创造股东价值？

### 4. 资本周期镜头

解决的问题：

1. 当前高利润是否会引来过度扩产？
2. 资本开支是否正在埋下未来的价格战和利润塌陷？
3. 哪些环节受益于资本约束，哪些环节将受害于资本过剩？

### 5. 结构力量镜头

解决的问题：

1. 当前变化是短期景气，还是更大的产业重构？
2. 技术、地缘、能源、政策是否在改变利润池归属？
3. 这是普通轮动，还是长期中枢上移或下移？

### 6. 预期与估值镜头

解决的问题：

1. 市场已经把多少好消息写进股价？
2. 当前价格在反映现实，还是在反映更远期的理想状态？
3. 结论即使正确，投资结果会不会仍然很差？

## 四、产业周期研究的九步流程

### 第一步：先判断问题类型

1. 这是宏观问题、行业问题、公司问题，还是市场定价问题？
2. 这是景气波动，还是结构转移？
3. 这是需求驱动、供给驱动，还是流动性驱动？

### 第二步：先建立变量表，而不是先写结论

所有行业研究先列变量：

1. 需求
2. 供给
3. 成本
4. 价格
5. CapEx
6. 库存
7. 融资条件
8. 资金参与度
9. 估值水平

### 第三步：判断变量之间的主次关系

要判断：

1. 当前谁是主导变量
2. 谁是滞后变量
3. 谁是验证变量
4. 谁只是噪音

### 第四步：把变量放进传导链

标准传导链是：

`变量变化 -> 产业景气 -> 公司收入/利润 -> 资本开支 -> 估值 -> 股价`

### 第五步：判断行业所在阶段

必须明确写出当前更接近哪一段：

1. 概念导入
2. 产业验证
3. 主升扩散
4. 资本拥挤
5. 核心资产先见顶
6. 上游接力
7. 扩产过剩
8. 价格战与利润回落
9. 出清与分化
10. 穿越与阴影并存

### 第六步：判断资本周期是否正在反噬

至少回答四个问题：

1. 当前盈利是否足以诱发大规模扩产？
2. 当前扩产是否已经快于真实需求？
3. 当前融资环境是否仍支持资本继续涌入？
4. 新增产能释放后，谁的利润最先受损？

### 第七步：判断市场是否已经透支预期

至少回答：

1. 市场当前是在交易现实，还是交易远期理想状态？
2. 当前估值对应的利润和份额假设是否过于乐观？
3. 当前持仓、成交、主题热度是否说明交易已拥挤？

### 第八步：判断哪些公司能穿越，哪些公司只是顺周期受益

区分标准至少包括：

1. 竞争优势是否可持续
2. ROIC 是否能长期高于资本成本
3. 资产负债表是否健康
4. 资本配置是否克制
5. 现金流是否真实
6. 退潮后是否仍有客户、技术、成本或渠道优势

### 第九步：做 AI 映射，但禁止机械复制

映射时必须强调：

1. 机制相似比时间相似更重要
2. 变量相似比图形相似更重要
3. 资本开支约束、供给释放、估值透支，是最该优先映射的内容

## 五、研究时必须回答的十二个问题

1. 这轮行情最底层的驱动是什么？
2. 它属于结构力量、宏观力量、产业力量，还是市场心理力量？
3. 利率、信用和流动性是在强化它，还是削弱它？
4. 真正的主导变量是需求、供给、成本、价格还是 CapEx？
5. 当前阶段利润改善的质量如何？
6. 资本回报是否高于资本成本？
7. 当前扩产是否会反噬未来利润？
8. 当前市场是否已把好消息定价过头？
9. 股价领先的到底是产业事实，还是市场情绪？
10. 哪些公司具备穿越能力，哪些只是景气 beta？
11. 哪些结论已经被证据支持，哪些仍然只是推断？
12. 这一规律如何映射到 AI 产业链？

## 六、本仓库最应警惕的十类误区

1. 先有观点，再找资料拼接
2. 把股价上涨直接当成基本面验证
3. 把高利润率直接外推成长期高回报
4. 忽略资本开支的滞后杀伤力
5. 忽略利率和信用环境的影响
6. 忽略估值与预期已经透支
7. 把结构性主题热度误判成所有环节同步受益
8. 用后视角把历史讲得过于线性
9. 只研究龙头上涨，不研究失败样本和阴影资产
10. 只做新能源和 AI 的图形类比，不做机制类比

## 七、落到本仓库后的执行要求

后续所有重要研究任务，默认都要尽量满足以下要求：

1. 先列变量表
2. 再写阶段判断
3. 再补价格、财务、政策、持仓和 K 线证据
4. 再判断资本周期和估值是否反噬
5. 最后才做 AI 映射和文字总结

如果一个研究结论无法被还原成：

`事实 -> 变量 -> 传导 -> 阶段 -> 验证 -> 映射`

那它就还不算完成。
