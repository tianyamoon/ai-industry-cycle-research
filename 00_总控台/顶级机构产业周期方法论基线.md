# 顶级机构产业周期方法论基线

## 一、目的

本文件用于给本仓库提供“外部方法论基线”。

与本文件配套使用的两份文档：

1. `00_总控台/产业周期研究方法论总纲.md`
   - 用于日常研究执行，属于压缩后的主工作框架
2. `00_总控台/顶级机构产业周期方法论语句库.md`
   - 用于深挖方法论条目，属于扩展语句库

原因很简单：

1. 第一性原理能帮助我们回到底层变量。
2. 但第一性原理本身不自动提供完整的研究框架。
3. 如果没有成熟的方法论镜头，研究容易变成“堆数据”或“先有结论再找证据”。

因此，本仓库默认参考全球顶级投资机构和长期投资研究框架，把它们压缩成适合本项目的研究工具箱。

## 二、方法论总览

本仓库当前采用六个核心镜头：

1. Bridgewater：宏观机器与债务周期
2. Oaktree / Howard Marks：周期位置与市场心理
3. Morgan Stanley / Mauboussin：生命周期、ROIC-WACC、现金流阶段
4. J.P. Morgan：商业周期结构与衰退形态变化
5. BlackRock：结构性长期力量与利润池迁移
6. Goldman Sachs：资本开支波次与产业链阶段轮动

这六个镜头不是互相替代，而是互相校验。

## 三、Bridgewater 镜头：先看宏观机器

适用问题：

1. 当前是流动性驱动，还是真实需求驱动？
2. 当前更像短周期波动，还是长债务周期压力？
3. 产业上行是基本面扩张，还是信用扩张放大？

核心启发：

1. 经济不能只看增长，要同时看生产率、短期债务周期、长期债务周期。
2. 很多资产大行情并不只是产业故事，也叠加了信用、利率和流动性环境。
3. 周期位置判断不能脱离宏观金融条件。

## 四、Oaktree 镜头：先判断自己站在周期哪里

适用问题：

1. 我们现在处于周期早期、中期还是末期？
2. 市场是理性定价，还是已经进入情绪化外推？
3. 资产价格反映的是改善，还是过度乐观？

核心启发：

1. 预测拐点很难，但判断“身处何处”非常重要。
2. 周期会自我强化，也会自我反转。
3. 市场价格短期主要受投资者心理和估值参数变化影响，而不只是现金流变化。
4. 最危险的错误之一，是把暂时的高增长当成永久状态。

## 五、Morgan Stanley / Mauboussin 镜头：产业之外，还要看生命周期与回报质量

适用问题：

1. 公司处于导入、成长、成熟、震荡还是衰退阶段？
2. 高增长是否真的在创造价值？
3. 资本开支到底是在扩大护城河，还是在摧毁回报率？

核心启发：

1. 公司和行业都有生命周期，不能用一个估值框架看所有阶段。
2. ROIC 和 WACC 的关系，是判断竞争优势和价值创造的重要抓手。
3. 现金流结构可以帮助判断企业所处阶段。
4. 市场价格反映的不只是当前利润，更反映了对未来回报的预期。

## 六、J.P. Morgan 镜头：商业周期会重复，但形态会变化

适用问题：

1. 当前这轮周期是不是简单重复上一轮？
2. 库存、地产、财政、服务业结构是否改变了衰退形态？
3. 为什么这轮回撤可能更缓、但更长？

核心启发：

1. 商业周期没有消失，但每轮周期的“解剖结构”会变化。
2. 库存管理、财政结构、地产波动、服务业占比等因素会改变周期的烈度和节奏。
3. 不能机械照搬旧周期长度与斜率。

## 七、BlackRock 镜头：先判断这是周期，还是更大的结构性力量

适用问题：

1. 当前变化是短期行情，还是长期结构转移？
2. 人口、AI、地缘、能源转型是否正在重塑利润池？
3. 哪些行业的长期盈利中枢可能上移或下移？

核心启发：

1. 有些变化不是普通商业周期，而是更长的结构性力量。
2. 结构力量会改变长期增长、通胀、资本开支和行业利润分配。
3. 如果忽略结构力量，就容易把长期趋势误判成短期波动。

## 八、Goldman Sachs 镜头：资本开支波次和受益链条会轮动

适用问题：

1. 当前市场在交易基础设施、平台层，还是生产率受益层？
2. 大规模资本开支是否已经超前？
3. 投资者是否开始区分“有收入兑现的支出”和“高负债堆出来的支出”？

核心启发：

1. 技术投资浪潮通常存在阶段轮动，不会所有环节一直同步受益。
2. 基础设施先涨，不等于基础设施永远最强。
3. 市场最终会追问：资本开支能否转化为收入、利润和自由现金流。
4. 资本开支过快扩张，本身就可能孕育后续估值压力和供给过剩。

## 九、本仓库的综合研究框架

结合以上镜头，本仓库默认按以下顺序研究产业周期：

1. 先判断结构性背景
2. 再判断宏观金融环境
3. 再判断产业变量
4. 再判断生命周期
5. 再判断市场定价
6. 再判断市场心理和拥挤度
7. 最后做 AI 映射

## 十、写研究结论时必须回答的七个问题

1. 最底层驱动变量是什么？
2. 它属于结构力量、宏观变量、产业变量，还是市场心理变量？
3. 它如何传导到收入、利润、估值和股价？
4. 这个阶段的资本开支是扩大战果，还是埋下过剩？
5. 市场当前的价格已经隐含了什么预期？
6. 这个判断被哪些事实验证，哪些地方仍待补证？
7. 这一规律如何迁移到 AI 产业链？

## 十一、不应犯的典型错误

1. 用单一叙事解释完整周期
2. 把股价上涨直接当成基本面验证
3. 把利润高增长直接当成长期价值创造
4. 忽略估值、拥挤和信用条件
5. 忽略资本开支的反噬
6. 用后视角把历史讲得过于线性
7. 只看技术革命，不看供给扩张和资本回报塌陷

## 十二、当前建议

对本项目而言，后续研究建议默认同时保留五条主线：

1. 产业主线：需求、供给、价格、盈利
2. 资本主线：CapEx、融资、扩产、出清
3. 市场主线：估值、拥挤度、风格偏好、资金参与
4. 结构主线：技术扩散、政策、地缘、能源、人口
5. 映射主线：新能源历史规律如何迁移到 AI
