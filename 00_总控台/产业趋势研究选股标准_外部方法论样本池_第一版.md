## 一、证据等级

| 等级 | 来源类型 | 使用方式 |
|---|---|---|
| A | 机构官网、股东信、年报、监管披露、论文原文、可复核的连续交易案例 | 可直接支撑方法论条款 |
| B | 权威媒体、行业榜单、私募排排网 / 中基协等公开转述、正式访谈 | 可支撑样本入池与初步归纳 |
| C | 数据聚合站、券商整理、市场广泛称谓 | 只能作为线索，必须二次核验 |
| D | 论坛、传闻、未署名总结 | 只能记录，不能写入正式结论 |

外部样本进入最终结论前，至少需要 `A/B` 级来源之一；游资研究尤其要避免身份和席位失真，正式样本应写成 `方法单元`，如情绪周期、主线确认、分歧转一致、不及预期卖出、退潮空仓等。

## 二、研究维度

每个样本后续统一实填六列：

1. `选择理念`
   - 看方向、商业质量、估值、赔率、流动性，还是短线辨识度。
2. `技术选择方法`
   - 看趋势、相对强弱、均线、突破、放量、平台、回撤，还是量化信号。
3. `风险偏好`
   - 长期低换手、集中持仓、多空中性、多策略、趋势交易、超短接力等。
4. `排除法`
   - 明确不碰什么：无现金流、高估值无兑现、趋势破坏、乖离过大、无承接、无流动性、监管风险等。
5. `证伪 / 退出`
   - thesis 失效、基本面不兑现、价格跌破关键结构、流动性反噬、情绪退潮等。
6. `映射到本标准`
   - 对应本仓库的哪条纪律：方向优先、经营验证、位置拥挤、低位纪律、反方条件、可验证路径、技术门槛。

## 六、第一版先验归纳

### 1. 顶级机构给出的硬约束

1. 好行业不等于好买点，必须反推价格里隐含了多少增长。
2. 产业趋势必须回到现金流、ROIC、资本配置和资产负债表。
3. 结构性机会也需要起点估值纪律。
4. 资本周期是核心排除法：高利润吸引扩产，扩产会反噬利润。
5. 长期资金更重视“下行时是否活得好”，不是只看上行弹性。

### 2. 顶级私募给出的硬约束

1. 主观私募更强调产业趋势、公司质量、估值赔率和组合容错。
2. 量化私募更强调信号一致性、容量、回撤、换手、风险预算。
3. 宏观 / CTA 类私募提醒：趋势成立也要看流动性与宏观 regime。
4. 私募的共同排除法是：不做没有可重复验证变量的故事。

### 3. 游资方法论给出的硬约束

1. 短线资金首先看情绪周期、主线地位和市场辨识度，不等于长期产业验证。
2. 游资方法论的核心是 `选股标准 + 买点标准 + 卖点标准 + 仓位纪律 + 退潮识别`。
3. 龙虎榜只能作为案例复盘数据源，不能替代方法论，也不能替代基本面。
4. 对产业趋势研究最重要的借鉴不是追板，而是识别 `无主线 / 无辨识度 / 无标准买点 / 不及预期 / 退潮期` 的排除条件。

### 4. 技术方法给出的硬约束

1. CAN SLIM 类方法强调基本面和技术面结合，尤其是领先行业、盈利增长、机构参与和市场方向。
2. Minervini / SEPA 类方法强调强趋势模板，弱趋势标的直接排除。
3. AQR 趋势研究强调趋势信号有长期证据，但必须处理拥挤、回撤和跨周期失效。
4. CMT 技术分析体系强调趋势、量价、行为金融、风险管理和交易系统，不把单个图形当万能答案。

## 七、对《产业趋势研究选股标准》的修正方向

后续标准应从当前九条继续升级为：

1. `方向筛选`
   - 先确认产业方向是否处在资金与经营变量共同改善阶段。
2. `路径筛选`
   - 必须说清楚买的是哪条可验证路径，不买概念标签。
3. `经营验证`
   - 订单、收入、利润、OCF、回款、客户、价格至少要有明确主变量。
4. `估值与预期`
   - 反推价格中隐含的增长和利润率，防止好行业买贵。
5. `技术图形`
   - 月线向上、周线向上、月线乖离率不过大。
6. `资金属性`
   - 区分机构主导、游资方法论主导、量化主导，不能把不同资金语言混写。
7. `排除法`
   - 明确不买：无经营验证、无承接、月线乖离过大、估值透支、低位无趋势、高位无消化、短线情绪只有热度无基本面。
8. `反方条件`
   - 每个方向必须写出下调或退出条件。
