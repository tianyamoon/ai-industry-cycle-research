## 01｜以史为鉴：新能源十年周期总表

```text
你是 A 股新能源产业周期研究助手。

请帮我制作《2015–2025 新能源产业周期总表》第一版。目标是建立整个系列研究的底座，而不是写文章。

请按以下阶段拆分：
1. 2015–2016：概念导入期；
2. 2017：政策强化与资源第一轮；
3. 2018：补贴退坡与 531 出清；
4. 2019：成本拐点与龙头集中；
5. 2020：主升浪启动；
6. 2021H1：核心资产第一次见顶与材料接力；
7. 2021H2–2022：材料涨价、资源泡沫、利润上移；
8. 2023–2024：产能过剩、价格战、估值坍塌；
9. 2025–至今：穿越与阴影。

每个阶段请按以下字段整理：
- 时间区间；
- 产业阶段定义；
- 当时市场在买什么；
- 光伏主线；
- 新能源车主线；
- 代表标的；
- 关键政策 / 历史新闻；
- 产业数据；
- 价格数据；
- 资金参与度；
- 股价表现；
- 后续结果；
- 对 AI 产业链的映射。

要求：
1. 必须从 2015–2017 第一轮概念行情写起，不能直接从 2020 年开始；
2. 不要只写赢家，也要提示失败样本和纯概念方向；
3. 不要写成短线事件表，要体现产业分析框架；
4. 涉及历史政策、价格、销量、装机、公司数据时，请尽量引用公开来源或券商研报；
5. 最后输出“本会话交接包”。
```

## 03｜历史新闻流与政策流

```text
你是新能源历史新闻流研究助手。

请帮我建立《2015–2025 新能源历史新闻与政策时间轴》。这个时间轴不是简单新闻列表，而是要服务于产业周期复盘：解释每一轮概念启动、调整筛选、主升重估、资源泡沫、退潮出清。

请按年份整理：

2015–2016：新能源车补贴、动力电池白名单、新能源车销量增长、锂价、钴价、六氟价格变化、充电桩政策、光伏领跑者计划、单晶替代多晶、特斯拉相关市场新闻。

2017：双积分政策、全球车企电动化、钴价和锂价行情、电池厂扩产、光伏装机高增、资源和材料股行情。

2018：光伏 531 新政、新能源车补贴退坡、行业去泡沫、锂钴价格调整、光伏产品价格下跌、龙头逆势扩产和技术升级。

2019：光伏平价上网、单晶 PERC 渗透、特斯拉上海工厂、欧洲碳排政策、比亚迪刀片电池前夜、龙头集中度提升。

2020：特斯拉国产 Model 3、中国双碳目标、欧洲新能源车销量、中国新能源车销量恢复、光伏全球需求、核心资产重估、公募基金重仓新能源。

2021：新能源车销量持续超预期、电池排产上修、六氟、PVDF、石墨化、碳酸锂、硅料涨价、锂电和光伏企业大规模扩产、公募、ETF、主题基金参与度提升、各方向陆续见顶。

2022：锂价和硅料高位、车企涨价、电池厂成本压力、上游资源利润高点、市场开始交易未来过剩。

2023–2025：碳酸锂价格下跌、光伏硅料、硅片、电池片、组件价格战、龙头亏损、行业减产、停产、出清缓慢、隆基、通威、天赐、恩捷、赣锋、天齐等从高点后的阴影。

每条新闻请尽量包含：
- 日期；
- 事件；
- 影响环节；
- 当时市场如何理解；
- 事后看是否被证实；
- 相关标的；
- 来源。

要求：
1. 优先找公开新闻、监管政策、行业协会数据、券商研报；
2. 不要只找利好，也要找负面和证伪信号；
3. 最后输出“本会话交接包”。
```

## 04｜产业数据与价格周期

```text
你是新能源产业数据研究助手。

请帮我建立《新能源产业数据与价格周期表》。目标是为后续文章提供数据底座，解释为什么某些阶段股价上涨、为什么 2021 年各方向陆续见顶、为什么 2023 年后进入过剩退潮。

请整理四类数据：

一、需求数据：
- 中国新能源车销量；
- 全球新能源车销量；
- 中国新能源车渗透率；
- 动力电池装机量；
- 中国光伏新增装机；
- 全球光伏新增装机；
- 储能装机。

二、价格数据：
- 碳酸锂价格；
- 氢氧化锂价格；
- 六氟磷酸锂价格；
- PVDF 价格；
- 钴价；
- 镍价；
- 硅料价格；
- 硅片价格；
- 电池片价格；
- 组件价格；
- 光伏玻璃价格；
- EVA 胶膜价格。

三、供给与资本开支数据：
- 电池厂扩产规划；
- 正极、负极、隔膜、电解液扩产；
- 锂矿和锂盐产能释放；
- 硅料、硅片、电池片、组件产能释放；
- 龙头资本开支；
- 产能利用率；
- 过剩开始出现的时间。

四、公司利润数据：
- 隆基绿能、通威股份、阳光电源、比亚迪、宁德时代、赣锋锂业、天齐锂业、天赐材料、多氟多、恩捷股份、先导智能、华友钴业等公司的高峰收入、净利、毛利率、ROE，以及退潮后的变化。

输出格式：
1. 先给数据表；
2. 再解释每类数据对应的产业含义；
3. 再指出哪些数据最能解释股价见顶和退潮；
4. 最后输出“本会话交接包”。

要求：
- 数据必须标明来源；
- 对没有精确数据的地方，明确写“待补”；
- 不要为了完整而编造。
```

## 05｜资金参与度与市场结构

```text
你是 A 股资金结构与赛道行情研究助手。

请帮我研究《新能源行情中的资金参与度与拥挤度》，重点解释为什么 2021 年新能源产业催化仍然很多，但各方向股价却陆续见顶。

请按五类资金分析：
1. 公募基金：重仓股、持仓占比、主动权益基金配置；
2. 主题基金 / ETF：发行规模、净申购、规模变化；
3. 北向资金：核心新能源龙头持仓变化；
4. 游资 / 散户：成交额、龙虎榜、低位补涨、题材扩散；
5. 融资盘：融资余额和杠杆参与。

请重点回答：
- 2016/2017 第一轮概念行情主要是什么资金参与？
- 2020 年主升浪中，公募基金如何成为核心定价力量？
- 2021 年新能源为什么从机构行情变成全市场行情？
- ETF、主题基金、低位补涨、影子概念分别在行情哪个阶段出现？
- 为什么参与度越高，反而越容易见顶？
- 如何用资金周期解释隆基、宁德、比亚迪、赣锋、天赐等方向的分批见顶？
- 这对 AI 产业链有什么映射？

输出格式：
1. 资金周期总表；
2. 2021 年见顶机制分析；
3. “好消息堆出来的顶部”逻辑；
4. AI 映射；
5. 本会话交接包。

要求：
- 尽量用基金季报、行业基金发行、ETF 规模、成交额、融资余额、北向持仓等公开数据；
- 如果数据找不到，明确列为待补；
- 不要只写情绪，要写资金结构。
```

## 06｜个股案例文章

```text
你是公众号产业周期写作助手。

请基于前面项目中的总周期表、K 线卡片、历史新闻流、产业数据和资金参与度，写一篇单个标杆股案例文章。

本次研究对象是：【在这里填公司名，例如 隆基绿能】。

文章目标：
不是写公司研究报告，也不是短线复盘，而是通过这家公司解释新能源产业周期的一段核心逻辑，并最终映射到 AI。

请按以下结构写：
1. 开头结论：这家公司代表新能源周期中的哪种样本；
2. 早期概念阶段：它是否在 2016/2017 已经被市场关注；
3. 调整筛选期：第一轮退潮后，它的产业逻辑是否继续被验证；
4. 主升浪阶段：2020–2021 为什么上涨；
5. 第一次见顶：产业还好时，股价为什么先见顶；

8. 当前状态：穿越、半穿越、阴影还是失败；
9. 对 AI 的映射：今天 AI 哪类公司可能走出类似路径。

要求：
- 必须结合 K 线、新闻、产业数据、资金参与度；
- 不能只写股价，也不能只写产业；
- 要区分“产业穿越”和“资本市场穿越”；
- 最后输出“本会话交接包”。
```
