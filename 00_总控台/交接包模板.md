# 交接包模板

每个研究分会话结束时，都要输出一份交接包，并保存到 `99_交接包/`。

交接包的作用：

1. 让后续会话快速接上上下文；
2. 防止长期项目在多会话之间丢失结论；
3. 把每批研究成果沉淀为可复用资料；
4. 为最终总结和正文写作提供素材。

---

# 本会话交接包

## 1. 本会话研究目标

用 3–5 句话说明本会话原本要解决什么问题。

示例：

> 本会话目标是建立 2015–2025 年新能源产业周期总表，明确每个阶段市场在买什么、产业主线是什么、代表标的有哪些、资金参与度如何变化，并为后续 K 线图谱和 AI 映射提供底座。

---

## 2. 已完成内容

列出本会话已经完成的表格、框架、结论、案例、数据。

格式：

- 已完成：……
- 已完成：……
- 已完成：……

---

## 3. 核心结论

用 5–10 条总结最重要的发现。

要求：

- 不要空泛；
- 必须有时间、阶段、产业链位置或代表标的；
- 对不确定内容标记“待核验”。

---

## 4. 关键证据

列出支撑结论的证据，包括：

- 历史政策；
- 历史新闻；
- 产业数据；
- 价格数据；
- 券商研报；
- 公司公告；
- K 线节点；
- 资金数据。

建议表格：

|---|---|---|---|---|
| 政策 |  |  |  |  |
| 新闻 |  |  |  |  |
| 数据 |  |  |  |  |
| 研报 |  |  |  |  |
| K线 |  |  |  |  |

---

## 5. 重要表格

把本会话产出的核心表格重新列出来，方便后续会话直接复用。

如果表格很长，可以只放摘要，并注明原文位置。

---

## 6. 待补内容

明确哪些地方数据不完整、来源不足、需要后续补充。

建议表格：

| 待补项 | 优先级 | 用途 | 备注 |
|---|---|---|---|
|  | 高 / 中 / 低 |  |  |

---

## 7. 对后续会话的建议

告诉下一个会话应该接着做什么。

示例：

> 下一会话建议进入“标杆股 K 线图谱”，优先研究隆基绿能、通威股份、阳光电源、比亚迪、宁德时代、赣锋锂业、天赐材料等代表样本。

---

## 8. 可直接复用段落

输出 3–5 段可以直接放入正文或总报告的文字。

要求：

- 语言可以稍微成文；
- 不要过度口语化；
- 适合后续直接引用或改写。

---

99_交接包/02_K线图谱交接包.md

99_交接包/04_产业数据交接包.md
