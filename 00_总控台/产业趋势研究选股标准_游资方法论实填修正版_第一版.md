> 所属模块：00_总控台  
> 生成日期：2026-07-05  
> 修正原因：此前把第三批样本误收窄为 `游资 / 龙虎榜席位与风格样本`，容易把研究对象从“选股方法论”带偏到“席位、榜单、身份和传闻”。本文件按用户纠偏要求，把第三批改为 `游资选股方式方法、买入标准、卖出标准、仓位与排除法` 的方法论实填。

## 一句话结论

游资方法论对《产业趋势研究选股标准》的价值，不是“看谁买了”，而是提供一套短周期执行纪律：`先判市场周期 -> 再判主线地位 -> 再选辨识度最高的标的 -> 只在标准买点出手 -> 不及预期立即降级 -> 退潮期主动空仓或大幅降仓`。

## 三、对第二版标准的修正条款

1. `游资主导` 不再定义为“龙虎榜或席位主导”，而定义为“短周期情绪资金和交易型资金主导”。
2. 游资方法论进入标准的位置是 `资金属性 / 技术门槛 / 排除法 / 执行条件`，不进入 `经营验证`。
3. 龙虎榜只作为事后复盘数据源，不作为买入理由。
4. 第二版排除法中的 `龙虎榜只有情绪无基本面` 应改写为 `短线情绪只有热度无基本面`。
5. 后续研究游资样本时，优先填 `方法单元`，其次才用公开案例验证；不得反过来用席位标签倒推方法。

## 四、后续补证顺序

1. 先为 50 个方法单元各找 3-5 个公开案例；
2. 每个案例只记录 `市场周期 / 主线地位 / 买点 / 卖点 / 仓位假设 / 次日反馈 / 错误归因`；
3. 不记录传闻身份，不把席位归属写成事实；
4. 最终把有效方法回写到第二版标准，把无效或不可复现方法放入 `不采纳`。
