# 财务指标表｜字段模板

> 所属模块：06｜新能源数据闭环补证  
> 生成日期：2026-06-21  
> 用途：统一记录新能源标杆股 2015–2025 年财务数据，用于验证股价、价格和利润周期。

---

## 一、模板目标

财务指标表用于解决：

> 股价上涨是否被财务验证？顶部时财务是已经恶化，还是仍然很好？退潮时是杀估值、杀价格，还是杀利润？

投资研究不能只看收入和净利润，还必须看：

- 毛利率；
- 净利率；
- 经营现金流；
- 资本开支；
- 存货；
- 减值；
- ROE；
- ROIC；
- 分业务收入和毛利率。

---

## 二、年度财务表字段

| 字段 | 说明 | 用途 |
|---|---|---|
| 标的 | 公司名称 | 统一索引 |
| 股票代码 | A 股代码 | 统一索引 |
| 年份 | 2015–2025 | 年度复盘 |
| 营业收入 | 总收入 | 需求和规模验证 |
| 收入同比 | 收入增长率 | 判断成长斜率 |
| 归母净利润 | 净利润 | 利润兑现 |
| 归母净利润同比 | 利润增长率 | 判断利润弹性 |
| 扣非归母净利润 | 主业利润 | 排除非经常项目 |
| 毛利率 | 总毛利率 | 定价权和价格周期 |
| 净利率 | 总净利率 | 盈利质量 |
| ROE | 股东回报 | 判断高估值支撑 |
| ROIC | 投入资本回报 | 判断资本开支质量 |
| 经营现金流 | OCF | 利润质量 |
| OCF / 净利润 | 现金流匹配度 | 判断利润真实性 |
| 资本开支 | CapEx | 扩产和周期风险 |
| 存货 | 库存风险 | 判断价格下跌冲击 |
| 存货周转天数 | 周转效率 | 判断库存压力 |
| 减值损失 | 存货 / 资产减值 | 退潮后果 |
| 资产负债率 | 杠杆 | 判断资产负债表压力 |
| 有息负债 | 债务压力 | 资源 / 重资产公司重点 |
| 海外收入 | 全球化 | 穿越能力验证 |
| 研发费用 | 技术投入 | 第二曲线和竞争力 |
| 市值区间 | 年内高低市值 | 估值对照 |
| K 线阶段 | 主升 / 顶部 / 退潮等 | 与股价对应 |
| 结论 | 一句话判断 | 投资复盘 |

## 五、财务拐点判断规则

| 现象 | 初步判断 |
|---|---|
| 股价见顶时收入和利润仍高增 | 可能是估值 / 资金先见顶 |
| 股价见顶时毛利率已见顶 | 可能是价格 / 利润周期先见顶 |
| 股价见顶后利润继续创新高 | 典型财务滞后于股价 |
| 毛利率下行但收入继续增长 | 需求仍在，但价格 / 竞争恶化 |
| OCF 弱于净利润 | 利润质量下降，关注应收和库存 |
| CapEx 高增但 ROIC 下行 | 扩产周期风险上升 |
| 存货和减值上升 | 退潮阶段确认 |

---

## 六、最终输出要求

每只标的最终至少形成：

1. 2015–2025 年度财务表；
2. 2020–2025 季度财务拐点表；
3. 分业务收入和毛利率表；
4. 股价顶部与财务拐点对照表。

最终目标：

> 明确股价到底领先财务多久，以及哪个财务指标最先提示风险。
