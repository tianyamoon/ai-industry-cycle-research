# 六氟磷酸锂价格曲线｜月度待补表

> 所属模块：06｜新能源数据闭环补证 → 价格曲线硬数据  
> 生成日期：2026-06-22  
> 研究定位：建立六氟磷酸锂价格月度数据表框架，用于验证天赐材料、多氟多的价格—毛利率—利润—股价关系。

---

## 一、研究目标

六氟磷酸锂是电解液产业链中最关键的价格变量之一，也是天赐材料和多氟多 2021–2022 材料行情的核心待验证变量。

本表要解决：

> 天赐材料、多氟多的股价顶部，到底领先、同步还是滞后于六氟价格顶部？六氟价格下跌后，毛利率和利润中枢下移滞后多久？

---

## 二、重要说明

六氟价格历史高低点存在不同口径：

- 工业级 / 电池级；
- 含税 / 不含税；
- 出厂价 / 市场价；
- 周度均价 / 月度均价；
- 现货成交价 / 报价；
- 不同数据源的统计口径差异。

因此本文件第一版只建立框架，不写死未经核验的具体价格高点和低点。

后续实填必须优先使用：

1. SMM；
2. 百川盈孚；
3. Wind；
4. 公司年报 / 问询回复 / 投资者交流；
5. 券商研报中明确标注来源的价格序列。

## 六、价格阶段判断

| 阶段 | 判断口径 | 投资含义 |
|---|---|---|
| 低位 | 价格接近行业低利润区间 | 关注供给出清和低位修复 |
| 上行 | 价格连续上涨 | 天赐、多氟多利润弹性释放 |
| 极端高位 | 价格显著高于历史中枢 | 警惕单品利润被长期化外推 |
| 下行 | 价格连续回落 | 毛利率和利润中枢重估 |
| 修复观察 | 价格低位企稳或反弹 | 观察是否形成新周期，而非单纯反弹 |

| 多氟多前复权 K 线 | 判断单品弹性股和价格高点关系 |
| 天赐电解液 / 六氟毛利率 | 判断平台型材料利润传导 |
| 多氟多六氟毛利率 | 判断单品利润弹性和下行风险 |
| 存货和减值 | 判断价格下跌后的退潮后果 |

## 九、本文件结论

六氟磷酸锂价格曲线是天赐材料和多氟多投资闭环的第一优先级。

当前只能形成待验证判断：

> 天赐、多氟多 2021–2022 的强势大概率与六氟价格和电解液毛利率上行高度相关。

但最终必须补齐：

1. 六氟月度价格；
2. 天赐、多氟多前复权 K 线；
3. 分业务毛利率；
4. 存货和减值；
5. 估值和持仓。

否则不能精确判断价格顶部、股价顶部和利润顶部的先后顺序。
