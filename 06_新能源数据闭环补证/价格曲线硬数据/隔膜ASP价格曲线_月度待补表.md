# 隔膜 ASP 价格曲线｜月度待补表

> 所属模块：06｜新能源数据闭环补证 → 价格曲线硬数据  
> 生成日期：2026-06-24  
> 研究定位：建立隔膜 ASP 月度数据表框架，用于验证动力电池需求、隔膜供需、恩捷股份毛利率、客户压价、扩产折旧、ROIC 和估值重估。

---

## 一、研究目标

隔膜 ASP 是恩捷股份高壁垒材料周期化的第一价格变量。

本表要解决：

> 隔膜作为高壁垒材料，为什么仍可能在扩产、客户压价和产能释放后出现 ASP 下行、毛利率下行和 ROIC 重估？

进一步验证：

1. 隔膜 ASP 是否领先恩捷毛利率下行；
2. 隔膜 ASP 是否领先利润中枢下移；
3. 隔膜出货增长是否能对冲 ASP 下行；
4. 涂覆膜和海外客户是否带来价格韧性；
5. ASP 下行后，折旧和产能利用率如何影响 ROIC。

---

## 二、价格口径说明

隔膜价格必须后续统一口径，不写死未经核验数值。

后续实填必须区分：

- 湿法隔膜 / 干法隔膜；
- 基膜 / 涂覆膜；
- 国内客户 / 海外客户；
- 平方米价格 / 其他价格单位；
- 公司 ASP / 行业 ASP / 市场报价；
- 含税 / 不含税；
- 价格、出货量和收入推算口径。

优先数据源：

1. 恩捷股份年报：隔膜收入、出货量、毛利率，推算 ASP；
2. Wind / Choice / 行业数据库；
3. SMM / 百川 / 行业协会；
4. 公司问询回复和投资者关系；
5. 券商研报中明确标注来源的价格序列。

## 五、价格阶段判断

| 阶段 | 判断口径 | 投资含义 |
|---|---|---|
| 需求修复 | 出货增长、ASP 稳定或温和下行 | 龙头份额和规模效应重估 |
| 供需偏紧 | ASP 和毛利率韧性较强 | 高壁垒材料估值扩张 |
| 扩产压力 | 行业新增产能释放，客户压价预期上升 | 警惕毛利率和估值先行回落 |
| ASP 下行 | 隔膜 ASP 连续下跌 | 利润中枢重估 |
| 低价竞争 | ASP 低位，产能利用率和折旧压力上升 | ROIC 重估和出清观察 |
| 修复观察 | ASP 企稳或反弹 | 判断新周期还是短期修复 |

## 六、后续必须补齐的数据源

优先顺序：

1. 恩捷股份年报：隔膜收入、出货量、毛利率，推算 ASP；
2. Wind / Choice 行业价格；
3. SMM / 百川隔膜价格；
4. 公司问询回复和投资者交流；
5. 券商研报中明确来源的隔膜价格表；
6. 同行公司披露数据交叉验证。

补证顺序：

1. 先用恩捷年报和半年报，把 `2019-01~2025-12` 至少压成年中 / 年末锚点；
2. 再用 Wind / Choice / SMM / 百川，把 `2021-01~2025-12` 细化成行业 ASP 变化节奏；
3. 最后再把 CapEx、折旧、产能利用率并到同一观察框架，验证 ROIC 重估。

优先落表月份：

1. `2021-06` 和 `2021-12`：验证高景气和高壁垒溢价；
2. `2022-06` 和 `2022-12`：验证扩产高峰和顶部消化；
3. `2023-06` 和 `2023-12`：验证 ASP 下行和客户压价；
4. `2024-06` 和 `2024-12`：验证低价竞争和折旧压力；
5. `2025-06` 和 `2025-12`：验证修复观察是否成立。

## 八、本文件结论

隔膜 ASP 是隔膜链投资闭环的核心变量。

当前只能形成待验证判断：

> 高壁垒隔膜龙头并不天然免疫周期。若隔膜 ASP 下行、客户压价、扩产折旧和产能利用率压力同时出现，恩捷的毛利率、ROIC 和估值中枢仍会重估。

最终需要补齐：

1. 隔膜月度 ASP；
2. 恩捷隔膜出货量、收入和毛利率；
3. 恩捷前复权 K 线；
4. CapEx、折旧、产能利用率和 ROIC；
5. 存货、减值和现金流。
