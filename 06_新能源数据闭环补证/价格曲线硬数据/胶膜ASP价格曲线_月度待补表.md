# 胶膜 ASP 价格曲线｜月度待补表

> 所属模块：06｜新能源数据闭环补证 → 价格曲线硬数据  
> 生成日期：2026-06-23  
> 研究定位：建立胶膜 ASP 月度数据表框架，用于验证组件价格战如何传导到福斯特胶膜 ASP、胶膜毛利率、利润、ROIC 和估值中枢。

---

## 一、研究目标

胶膜 ASP 是福斯特辅材价格传导的第一变量。

本表要解决：

> 光伏组件价格战发生后，胶膜 ASP 是同步下跌、滞后下跌，还是因辅材壁垒保持韧性？

进一步验证：

1. 胶膜 ASP 是否滞后于组件价格下行；
2. 胶膜 ASP 是否领先福斯特胶膜毛利率；
3. 胶膜 ASP 是否领先福斯特利润下行；
4. EVA / POE 原料价格是否影响胶膜毛利率；
5. 胶膜价格低位后，福斯特估值中枢如何重估。

---

## 二、价格口径说明

胶膜价格必须后续统一口径，不写死未经核验数值。

后续实填必须区分：

- EVA 胶膜 / POE 胶膜 / EPE 胶膜；
- 含税 / 不含税；
- 出厂价 / 市场价；
- 周度均价 / 月度均价；
- ASP / 报价 / 成交价；
- 胶膜价格与 EVA / POE 原料价格的口径差异。

优先数据源：

1. 公司年报中的胶膜销售均价或收入 / 出货量推算；
2. PVInfoLink / CPIA / Wind；
3. 百川 / SMM 原料和辅材价格；
4. 明确标注来源的券商研报。

## 六、价格阶段判断

| 阶段 | 判断口径 | 投资含义 |
|---|---|---|
| 平价前夜 | 组件和胶膜价格下降但需求扩张 | 辅材龙头份额提升 |
| 高景气 | 胶膜出货增长、ASP 相对稳定 | 毛利率和估值抬升 |
| 原料波动 | EVA / POE 原料涨跌影响毛利率 | 成本传导能力验证 |
| 价格战传导 | 组件价格下跌，客户压价胶膜 | 辅材毛利率重估 |
| 低价竞争 | 胶膜 ASP 和毛利率低位 | ROIC 和估值中枢重估 |
| 修复观察 | ASP 企稳或反弹 | 判断新周期还是短期修复 |

## 七、后续必须补齐的数据源

优先顺序：

1. 福斯特年报：胶膜收入、出货量、毛利率，推算 ASP；
2. PVInfoLink / CPIA：胶膜和组件价格；
3. Wind：胶膜、EVA、POE 价格；
4. 百川 / SMM：EVA / POE 原料价格；
5. 券商研报中明确标注来源的价格序列。

补证顺序：

1. 先用福斯特年报和半年报，把 `2019-01~2025-12` 至少压成年中 / 年末锚点；
2. 再用 PVInfoLink / CPIA / 券商研报，把 `2023-01~2025-12` 细化到价格战阶段的月度锚点；
3. 最后再把 EVA / POE 原料价格并入，验证胶膜毛利率是 ASP 端还是成本端主导。

优先落表月份：

1. `2021-06` 和 `2021-12`：验证高景气期 ASP 韧性；
2. `2022-06` 和 `2022-12`：验证高位消化和过剩预期；
3. `2023-06` 和 `2023-12`：验证组件价格战向胶膜传导；
4. `2024-06` 和 `2024-12`：验证低价竞争和利润中枢重估；
5. `2025-06` 和 `2025-12`：验证修复观察是否成立。

## 九、本文件结论

胶膜 ASP 是组件 / 辅材链投资闭环的核心变量。

当前只能形成待验证判断：

> 组件价格战会向胶膜辅材传导，但传导节奏、幅度和福斯特毛利率韧性，必须用胶膜 ASP、EVA / POE 原料价格和福斯特财务数据共同验证。

最终需要补齐：

1. 胶膜月度 ASP；
2. EVA / POE 原料价格；
3. 福斯特胶膜出货量和毛利率；
4. 福斯特前复权 K 线；
5. 电子材料第二曲线数据。
