# 锂电设备订单 / 合同负债｜待补表

> 所属模块：06｜新能源数据闭环补证 → 财务硬数据  
> 生成日期：2026-06-25  
> 研究定位：建立锂电设备订单、合同负债、在手订单、下游电池厂 CapEx 和现金流的对照表，用于验证先导智能设备订单周期。

---

## 一、研究目标

设备股最重要的领先指标不是收入，而是订单和下游资本开支。

本表要解决：

> 先导智能股价和估值的拐点，是不是领先于收入和利润，更多由新签订单、合同负债、在手订单和下游电池厂 CapEx 决定？

---

## 二、核心变量定义

| 变量 | 含义 | 领先 / 滞后关系 |
|---|---|---|
| 下游电池厂 CapEx | 宁德、比亚迪、亿纬等扩产投入 | 设备订单最前置变量 |
| 扩产公告 | 电池厂新基地、新产线、新产能规划 | 订单预期变量 |
| 新签订单 | 设备公司新增订单 | 领先收入 |
| 在手订单 | 尚未确认收入的订单池 | 领先未来收入 |
| 合同负债 | 预收款和订单确认强度 | 领先收入确认 |
| 发出商品 | 已发货未确认收入 | 连接订单与收入 |
| 应收账款 | 已确认收入但未回款 | 反映回款压力 |
| OCF | 经营现金流 | 反映利润质量 |
| 坏账 / 减值 | 退潮后果 | 滞后暴露 |

## 五、领先 / 滞后关系验证

| 关系 | 当前假设 | 需要补的数据 |
|---|---|---|
| 下游 CapEx → 新签订单 | CapEx 是设备订单前置变量 | 电池厂 CapEx、扩产公告、先导新签订单 |
| 新签订单 → 合同负债 | 新签订单领先合同负债变化 | 新签订单、合同负债 |
| 合同负债 → 营业收入 | 合同负债领先收入确认 | 合同负债、收入 |
| 收入 → 利润 | 收入确认后利润兑现 | 收入、毛利率、利润 |
| 利润 → OCF | 退潮期利润可能领先现金流恶化 | 利润、OCF、应收、存货 |
| 股价 → 财务 | 股价可能领先订单和利润恶化 | K 线节点、订单、财务 |

## 六、风险信号表

| 信号 | 含义 | 投资意义 |
|---|---|---|
| 下游 CapEx 放缓 | 电池厂扩产意愿下降 | 设备订单前置转弱 |
| 新签订单下降 | 新增需求减少 | 设备股主升逻辑弱化 |
| 合同负债下降 | 未来收入确认基础变弱 | 收入高增可持续性下降 |
| 应收账款上升 | 客户回款压力 | 利润质量下降 |
| OCF / 净利润下降 | 利润现金含量变差 | 财务质量转弱 |
| 存货 / 发出商品上升 | 交付和确认风险 | 退潮后果滞后暴露 |
| 毛利率下降 | 竞争或议价压力 | 估值重估 |

1. 先导智能年报和季报；
2. 先导智能订单公告、投资者关系和问询回复；

6. 券商研报中明确引用的订单和 CapEx 数据。

## 八、本文件结论

设备订单链的核心不是产品价格，而是：

> 下游 CapEx → 新签订单 → 在手订单 / 合同负债 → 收入确认 → 利润 → 回款和现金流。

当前只能形成待验证判断：

> 设备股股价通常不会等收入和利润转弱才见顶，而会提前交易新签订单、合同负债和下游 CapEx 的拐点。

最终需要补齐：

1. 下游电池厂 CapEx；
2. 先导新签订单和在手订单；
3. 合同负债；
4. 收入和利润；
5. OCF、应收账款、存货和发出商品；
6. 前复权 K 线和估值。
