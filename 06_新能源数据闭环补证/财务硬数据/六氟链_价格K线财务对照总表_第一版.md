---

## 一、核心问题

六氟链要回答的核心问题是：

> **锂电材料股的高景气，到底是长期平台能力，还是单品 / 材料价格上行带来的阶段性利润泡沫？**

本总表用于验证：

1. 六氟价格是否领先天赐 / 多氟多股价；
2. 股价顶部是否领先毛利率和利润顶部；
3. 多氟多是否比天赐更贴近六氟单品价格周期；
4. 天赐一体化是否提高韧性但不能消除周期；
5. 价格下跌后，存货、减值、ROIC 和估值如何重估。

---

## 二、六氟链总逻辑

```text
动力电池排产上修
→ 电解液需求增长
→ 六氟供需错配
→ 六氟价格上涨
→ 天赐 / 多氟多毛利率上行
→ 利润弹性释放
→ 市场外推材料长期成长
→ 股价顶部 / 估值顶部
→ 六氟价格见顶并下跌
→ 毛利率下行
→ 利润下行
→ 存货 / 减值 / ROIC 压力暴露
→ 估值中枢重估
```

## 三、价格口径说明

当前六氟价格暂不写死未经核验的数值。

原因：

- 电池级 / 工业级口径不同；
- 含税 / 不含税口径不同；
- 周度均价 / 月度均价口径不同；
- 报价 / 成交价口径不同；
- 不同数据源统计范围不同。

后续必须统一使用：

1. SMM；
2. 百川盈孚；
3. Wind；
4. 公司公告 / 年报 / 问询回复；
5. 明确标注来源的券商研报。

## 四、天赐 vs 多氟多核心差异

| 维度 | 天赐材料 | 多氟多 | 研究含义 |
|---|---|---|---|
| 核心属性 | 电解液 / 六氟一体化平台 | 六氟单品价格弹性更强 | 天赐更偏平台，多氟多更偏单品弹性 |
| 2021–2022 核心驱动 | 六氟 + 电解液价格 + 一体化优势 | 六氟价格暴涨 | 两者都受六氟影响，但利润结构不同 |
| 顶部类型 | 平台型材料泡沫 | 单品价格弹性泡沫 | 多氟多股价可能更贴近六氟价格顶部 |
| 退潮机制 | 电解液 / 六氟价格下行、客户压价、毛利率重估 | 六氟价格下行直接压利润 | 多氟多下行可能更直接 |
| 第二曲线 | 一体化和新材料平台 | 新材料 / 电子化学品验证 | 第二曲线必须看收入、毛利率和现金流 |
| AI 映射 | AI 材料平台 | AI 单品涨价股 | 平台股和单品股估值逻辑不同 |

## 九、本文件结论

六氟链已经形成第一版闭环框架：

> **天赐代表平台型材料利润泡沫，多氟多代表六氟单品价格弹性。二者都必须以六氟价格为第一变量，但验证重点不同：天赐要看一体化和电解液平台韧性，多氟多要看六氟收入占比和单品毛利率弹性。**

当前还不能下最终结论，因为关键硬数据仍待补：

1. 六氟月度价格；
2. 天赐 / 多氟多前复权 K 线；
3. 分业务毛利率；
4. 存货和减值；
5. PE / PB / 持仓。

但下一步路径已经明确：

> 锂资源链、硅料链之后，六氟链是第三条完整投资闭环样板。
