# 估值与持仓表｜字段模板

> 所属模块：06｜新能源数据闭环补证  
> 生成日期：2026-06-21  
> 用途：记录估值、机构持仓和资金拥挤度，用于判断股价顶部是否来自估值和资金，而不是基本面恶化。

---

## 一、模板目标

估值与持仓表用于解决：

> 龙头股见顶时，是基本面先坏了，还是估值、持仓和资金拥挤先见顶？

新能源周期中，很多核心资产顶部并不是财务已经坏了，而是：

- 估值打满；
- 机构重仓过度集中；
- 资金拥挤；
- 股价对利好钝化；
- 后续增速斜率无法继续超预期。

---

## 二、估值字段

| 字段 | 说明 | 用途 |
|---|---|---|
| 标的 | 公司名称 | 统一索引 |
| 日期 | 月末 / 季末 / 关键交易日 | 与 K 线节点对应 |
| 市值 | 总市值 | 判断预期总量 |
| PE TTM | 滚动市盈率 | 利润估值 |
| PE 分位 | 近 3 / 5 / 10 年分位 | 判断高低估 |
| PB | 市净率 | 资产估值 |
| PB 分位 | 历史分位 | 判断泡沫程度 |
| PS | 市销率 | 收入估值 |
| EV / EBITDA | 如可得 | 重资产公司辅助 |
| 股息率 | 如适用 | 成熟期参考 |
| 对应净利润 | TTM 利润 | 估值分母 |
| K 线阶段 | 主升 / 顶部 / 退潮 | 对照股价 |
| 判断 | 高估 / 合理 / 低估 | 复盘结论 |

---

## 三、持仓字段

| 字段 | 说明 | 用途 |
|---|---|---|
| 标的 | 公司名称 | 统一索引 |
| 日期 | 季末 | 与公募披露同步 |
| 公募重仓市值 | 公募基金重仓金额 | 判断机构拥挤 |
| 公募持股占流通股比例 | 如可得 | 判断筹码集中 |
| 重仓基金数量 | 持有基金数 | 判断共识强度 |
| 北向持股市值 | 外资持仓 | 判断外资拥挤 |
| 北向持股比例 | 占流通股比例 | 判断边际资金 |
| 融资余额 | 杠杆资金 | 判断交易拥挤 |
| ETF 持仓 | 如适用 | 被动资金影响 |
| 成交额 | 日均 / 月均成交额 | 判断热度 |
| 换手率 | 月度 / 阶段换手 | 判断筹码交换 |
| 判断 | 拥挤 / 正常 / 低配 | 复盘结论 |

---

## 四、顶部判断规则

| 信号 | 含义 |
|---|---|
| PE / PB 分位极高 | 估值已经打满 |
| 公募重仓快速上升 | 机构拥挤度提高 |
| 北向持仓高位 | 外资边际买盘可能减弱 |
| 成交额放大但股价滞涨 | 高位筹码交换 |
| 股价对利好钝化 | 预期已充分反映 |
| 估值高位但财务仍好 | 典型核心资产先见顶 |
| 利润高位 + 估值高位 | 资源 / 材料泡沫高风险 |

---

## 五、优先验证问题

### 隆基绿能

- 2021 年 1 月底 / 2 月初第一次见顶时，PE / PB 是否处在历史高位？
- 公募和北向是否高度拥挤？
- 当时收入和利润是否仍然强？

### 宁德时代

- 2021 高位是否主要来自核心资产估值和机构抱团？
- 2022–2023 下跌是杀估值还是杀利润？

### 资源 / 材料股

- 2022 高点是否同时出现利润高点和估值高点？
- 公募持仓是否在利润最好时达到高位？

---

## 六、最终输出要求

每只标的最终至少形成：

1. 关键顶部日期的估值表；
2. 2019–2025 季度持仓表；
3. 股价顶部与资金拥挤度对照；
4. 财务顶部与估值顶部对照。

最终目标：

> 判断顶部到底是基本面顶部、价格顶部、估值顶部，还是资金顶部。
