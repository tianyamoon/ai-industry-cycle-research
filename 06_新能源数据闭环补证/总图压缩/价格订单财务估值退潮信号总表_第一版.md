## 一、核心问题

本表回答：

> **一轮产业行情退潮时，到底是哪里先坏？需求、价格、订单、毛利率、利润、现金流、估值，谁领先谁滞后？**

总原则：

```text
价格和订单先坏
→ 毛利率和合同负债确认
→ 收入利润滞后变差
→ 现金流、应收、存货开始暴露
→ 减值和资产负债表最晚确认
```

## 二、五类退潮信号总表

| 信号类别 | 核心指标 | 领先 / 滞后 | 适用链条 | 代表标的 | 退潮含义 |
|---|---|---|---|---|---|
| 价格信号 | 碳酸锂、硅料、六氟、组件、胶膜、隔膜、钴镍 | 最领先 | 资源 / 材料 / 主链 | 赣锋、天齐、通威、天赐、华友 | 利润弹性最早转弱 |
| 订单信号 | 下游 CapEx、新签订单、合同负债、在手订单 | 最领先 | 设备 / ToB 供应商 | 先导、阳光 | 未来收入可持续性转弱 |
| 毛利率信号 | 分业务毛利率、单车利润、单 GWh 利润 | 较领先 | 全部链条 | 14 股 | 价格和竞争传导到利润率 |
| 现金流信号 | OCF、OCF / 净利润、应收账款 | 同步或略滞后 | 设备 / 平台 / 穿越链 | 先导、比亚迪、阳光、亿纬 | 利润质量开始恶化 |
| 资产负债表信号 | 存货、跌价准备、减值、坏账 | 滞后 | 资源 / 材料 / 设备 | 华友、赣锋、先导 | 退潮后果确认 |
| 估值资金信号 | PE、PB、公募持仓、成交额、换手率 | 最领先或同步 | 全部链条 | 14 股 | 资金拥挤和预期转弱 |

## 三、退潮的典型路径

### 1. 资源价格退潮路径

代表：赣锋、天齐、华友。

```text
资源价格见顶
→ 毛利率见顶
→ 股价不再创新高
→ 利润仍然很好
→ 市场发现价格不可持续
→ 利润下修
→ 存货 / 减值 / 资产负债表压力
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | 碳酸锂 / 钴 / 镍价格 | 第一信号 |
| 2 | 分业务毛利率 | 价格传导确认 |
| 3 | PE TTM / PB | 高利润低 PE 陷阱 |
| 4 | 存货和跌价准备 | 退潮后果 |
| 5 | OCF 和资产负债率 | 质量验证 |

---

### 2. 瓶颈利润退潮路径

代表：通威、六氟链、部分光模块 / HBM 映射。

```text
瓶颈产品涨价
→ 毛利率和利润暴涨
→ 高毛利刺激扩产
→ 新产能释放
→ ASP 下行
→ 毛利率下行
→ 利润中枢重估
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | 瓶颈产品价格 | 硅料、六氟等 |
| 2 | 扩产计划和 CapEx | 高毛利能否持续 |
| 3 | 毛利率 | 利润率拐点 |
| 4 | 库存和存货跌价 | 价格下行后果 |
| 5 | ROIC | 扩产后资本回报 |

---

### 3. 主链价格战退潮路径

代表：隆基、福斯特。

```text
主链扩产
→ 价格战
→ 主链 ASP 下行
→ 客户压价辅材
→ 辅材 ASP 下行
→ 毛利率下行
→ ROIC 重估
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | 组件价格 | 主链第一退潮信号 |
| 2 | 胶膜 ASP / 辅材价格 | 传导信号 |
| 3 | 主链和辅材毛利率 | 财务确认 |
| 4 | 存货和减值 | 退潮后果 |
| 5 | 第二曲线收入和毛利率 | 能否穿越 |

---

### 4. 高壁垒材料周期化路径

代表：恩捷。

```text
高壁垒高毛利
→ 高估值
→ 扩产和客户压价
→ ASP 下行
→ 折旧压力上升
→ 毛利率下行
→ ROIC 重估
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | 隔膜 ASP | 第一价格变量 |
| 2 | 客户压价和产能利用率 | 供需变化 |
| 3 | CapEx 和折旧 | 扩产后压力 |
| 4 | 毛利率 | 周期化确认 |
| 5 | ROIC | 高壁垒是否真实 |

---

### 5. 设备订单退潮路径

代表：先导智能。

```text
下游 CapEx 放缓
→ 新签订单下降
→ 合同负债 / 在手订单回落
→ 收入滞后确认
→ 利润滞后变化
→ 应收和存货压力
→ OCF 恶化
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | 下游 CapEx | 最前置信号 |
| 2 | 新签订单 | 设备股核心领先指标 |
| 3 | 合同负债 / 在手订单 | 未来收入基础 |
| 4 | OCF / 净利润 | 回款质量 |
| 5 | 应收、存货、发出商品 | 退潮后果 |

---

### 6. 核心资产估值退潮路径

代表：隆基、宁德、比亚迪、阳光。

```text
产业仍强
→ 财务仍好
→ 估值和持仓拥挤
→ 股价先见顶
→ 竞争和价格战加剧
→ 毛利率和现金流验证
→ 估值中枢重设
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | PE / PB / 持仓拥挤 | 核心资产顶部常先从估值开始 |
| 2 | 毛利率 / 单车利润 / 电池毛利率 | 竞争压力确认 |
| 3 | OCF | 利润质量 |
| 4 | 海外化 / 第二曲线 | 新估值中枢 |
| 5 | ROIC | 真穿越指标 |

---

### 7. 多业务平台退潮路径

代表：亿纬、华友。

```text
平台叙事扩张
→ 多业务同时高景气
→ 高 CapEx 和估值外推
→ 主线价格或需求转弱
→ 分业务毛利率分化
→ 现金流和 ROIC 暴露质量
→ 真平台 / 伪平台分化
```

监控指标：

| 优先级 | 指标 | 说明 |
|---|---|---|
| 1 | 分业务毛利率 | 拆穿合并口径幻觉 |
| 2 | 主线价格 / 需求 / 路线 | 周期拐点 |
| 3 | CapEx / 在建工程 | 扩张压力 |
| 4 | OCF / 应收 / 存货 | 财务质量 |
| 5 | ROIC | 真平台最终证明 |

## 四、退潮信号的时间排序

| 顺序 | 信号 | 常见表现 | 解释 |
|---|---|---|---|
| 1 | 股价不再创新高 | 基本面仍好但股价走弱 | 资金和预期先变 |
| 2 | 价格 / 订单转弱 | ASP 下行、新签订单下降 | 利润未来基础变差 |
| 3 | 毛利率下行 | 分业务毛利率回落 | 价格和竞争传导 |
| 4 | 合同负债 / OCF 转弱 | 未来收入池和现金质量下降 | 订单和回款变差 |
| 5 | 利润下行 | 归母净利润增速回落 | 财报滞后确认 |
| 6 | 存货 / 应收 / 减值 | 存货跌价、坏账、资产减值 | 退潮后果确认 |
| 7 | 资产负债表修复 | 降 CapEx、去库存、出清 | 进入阴影或新周期前夜 |

## 五、各类信号的误判风险

| 信号 | 容易误判 | 正确判断 |
|---|---|---|
| 价格下跌 | 以为成本下降一定利好 | 对上游和中游是利润压力，对下游未必完全利好 |
| 订单高增 | 以为收入利润必然高增 | 要看交付周期、回款、毛利率和客户质量 |
| 毛利率高 | 以为壁垒强 | 高毛利可能刺激扩产和替代 |
| 利润高增 | 以为估值便宜 | 周期顶部低 PE 往往最危险 |
| 持仓高 | 以为机构认可 | 高持仓也可能意味着拥挤 |
| 第二曲线 | 以为新业务能对冲 | 必须看收入、毛利率、OCF 和 ROIC |
| 减值出现 | 以为最坏已过 | 减值是滞后信号，要看价格和供给是否真正出清 |

## 七、实操检查清单

每次判断一个标的是否进入退潮，按以下顺序检查：

1. 第一价格 / 第一订单变量有没有转弱；
2. 股价是否已经不再创新高；
3. PE / PB / 持仓是否处于拥挤状态；
4. 分业务毛利率是否见顶或下行；
5. OCF / 净利润是否恶化；
6. 应收、存货、合同负债是否变化；
7. CapEx 和 ROIC 是否开始重估；
8. 第二曲线是否有真实利润和现金流；
9. 减值是否只是滞后确认，而不是新信息；
10. 行业供给是否真正出清。

## 八、本文件结论

新能源退潮不是一个信号，而是一组有先后顺序的信号。

最应优先跟踪的是：

```text
价格
订单
下游 CapEx
估值资金
分业务毛利率
合同负债
OCF
```

最容易滞后的信号是：

```text
收入
归母净利润
减值
坏账
资产负债表恶化
```

这套顺序将直接用于 AI 产业链：

> **不要等 AI 公司利润变差才判断退潮，先看价格、订单、CapEx、毛利率、现金流和估值资金。**
