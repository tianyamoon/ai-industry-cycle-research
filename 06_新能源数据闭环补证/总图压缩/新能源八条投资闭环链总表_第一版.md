## 一、总目标

本表不是新增叙事，而是把已完成的 8 条链压缩成一张可执行研究地图。

核心问题：

> **新能源十年周期中，股价到底在交易什么？顶部最早由什么变量提示？退潮时哪些数据先坏，哪些数据滞后坏？**

## 三、八条链的共同结构

```text
产业需求增长
→ 核心变量改善
→ 龙头股价启动
→ 估值和资金强化
→ 报表兑现
→ 市场线性外推
→ 股价高位或顶部
→ 领先变量转弱
→ 报表滞后恶化
→ 估值重估
→ 阴影期 / 出清 / 穿越验证
```

这里最重要的是：

> **股价顶部通常不等财务顶部，财务顶部通常不等产业需求顶部。**

## 五、顶部模式分类

### 1. 价格先见顶型

代表：赣锋、天齐、通威、天赐、多氟多、华友。

```text
价格顶部
→ 毛利率顶部
→ 利润顶部
→ 高利润低 PE 陷阱
→ 利润下修
→ 估值重估
```

核心判断：

> 价格类公司不能只看利润，必须先看价格曲线。

---

### 2. 核心资产先见顶型

代表：隆基、宁德、部分阶段的阳光。

```text
产业仍强
→ 财务仍好
→ 估值和资金先见顶
→ 股价调整
→ 后续财务才被价格战或竞争验证
```

核心判断：

> 核心资产顶部往往发生在基本面最好或仍然很好的阶段。

---

### 3. 高壁垒周期化型

代表：恩捷、福斯特。

```text
高壁垒认知
→ 高毛利和高估值
→ 扩产和客户压价
→ ASP / 毛利率下行
→ ROIC 重估
```

核心判断：

> 高壁垒只能延缓周期，不一定消灭周期。

---

### 4. 订单周期型

代表：先导智能。

```text
下游 CapEx
→ 新签订单
→ 合同负债
→ 收入确认
→ 利润兑现
→ 现金流和应收压力
```

核心判断：

> 设备股不应首先看利润，应先看下游 CapEx、订单和合同负债。

---

### 5. 公司阿尔法验证型

代表：比亚迪、宁德、阳光、亿纬。

```text
赛道贝塔主升
→ 股价高位
→ 贝塔退潮
→ 公司阿尔法验证
→ 毛利率 / OCF / ROIC / 海外化决定是否穿越
```

核心判断：

> 需求高增阶段买贝塔，退潮阶段只认阿尔法。

## 七、后续实填优先级

### 第一优先级：把“顶部先后关系”实填出来

1. 14 只标的前复权 K 线节点；
2. 碳酸锂、硅料、六氟、组件、胶膜、隔膜、钴、镍价格；
3. 2020–2025 季度利润和分业务毛利率；
4. PE / PB / 公募持仓 / 北向持仓。

### 第二优先级：把“退潮后果”实填出来

1. 存货和跌价准备；
2. 资产减值；
3. OCF / 净利润；
4. 应收账款；
5. CapEx、折旧和 ROIC。

### 第三优先级：把“AI 映射指标”落成监控清单

1. AI 硬件 ASP；
2. 云厂商 CapEx；
3. 光模块 / HBM / 电力设备订单；
4. 算力租赁价格；
5. AI 应用收入和毛利率；
6. 现金流和 ROIC。

## 八、本文件结论

新能源 14 只标的已经可以压缩为 8 条投资闭环链。

最重要的结论不是“新能源涨过、AI 也会涨”，而是：

> **产业周期投资中，股价交易的永远不是单一需求，而是需求、价格、供给、利润、现金流、估值和资金拥挤度的先后顺序。**

后续所有 AI 映射，都必须先回答：

1. 这家公司属于哪一条链？
2. 第一领先变量是什么？
3. 股价是否已经领先财务？
4. 报表最好时，价格、订单或估值有没有先坏？
5. 退潮后能不能用 OCF 和 ROIC 证明穿越？
