# K 线节点表｜字段模板

> 所属模块：06｜新能源数据闭环补证  
> 生成日期：2026-06-21  
> 用途：统一记录新能源标杆股的启动、主升、顶部、二次冲顶、退潮和当前状态。

---

## 一、模板目标

K 线节点表用于解决投资复盘中最关键的问题：

> 股价到底在什么时候启动，什么时候第一次见顶，什么时候二次冲顶，什么时候进入不可逆退潮？

不能只说“2020 年上涨”“2021 年见顶”，必须尽量落到：

- 月份；
- 交易日；
- 前复权价格；
- 区间涨跌幅；
- 最大回撤；
- 对应事件；
- 对应价格和财务变量。

## 三、阶段定义

| 阶段 | 定义 | 投资含义 |
|---|---|---|
| 早期概念认知 | 产业逻辑开始被市场发现 | 弹性大，但真假难辨 |
| 第一次出清 | 政策退坡或行业冲击导致回撤 | 筛掉伪成长，真龙头低位验证 |
| 主升前夜 | 价格、政策、需求、成本出现改善 | 最重要的左侧 / 右侧交界 |
| 主升浪 | 业绩、产业、资金共振 | 主要收益阶段 |
| 第一次顶部 | 核心资产高位见顶 | 可能不是基本面见顶，而是估值 / 持仓见顶 |
| 二次冲顶 | 反弹或资源 / 材料补涨带来第二次高位 | 常常是最后叙事外推 |
| 退潮 | 价格、毛利率、估值、资金开始共同反转 | 需要降低仓位或离场 |
| 阴影期 | 产业仍在，但股价长期低迷 | 等待供给出清和新增长曲线 |
