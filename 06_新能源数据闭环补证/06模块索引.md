## 一、模块定位

06 模块不是继续扩写产业叙事，而是补齐投资研究所需的硬证据。

核心目标：

> 每一句“为什么涨、为什么跌”，都必须能落到 K 线、价格、财务、估值、持仓、新闻原文中的至少两个证据。

## 六、当前已形成的五条闭环链

### 1. 锂资源链

```text
碳酸锂价格顶部
→ 赣锋 / 天齐股价顶部
→ 锂盐毛利率顶部
→ 归母净利润顶部
→ 存货 / 减值 / 资产负债表压力
→ 估值重估
```

### 2. 硅料链

```text
硅料价格顶部
→ 通威股价顶部
→ 通威硅料毛利率顶部
→ 通威利润顶部
→ 硅料价格下跌
→ 隆基 / 通威利润中枢重估
→ 组件价格战
→ 隆基长期阴影
```

### 3. 六氟链

```text
六氟价格顶部
→ 天赐 / 多氟多股价顶部
→ 六氟 / 电解液毛利率顶部
→ 利润顶部
→ 六氟价格下跌
→ 材料利润中枢重估
→ 存货 / 减值 / ROIC 压力
→ 估值重估
```

### 4. 组件 / 胶膜链

```text
组件价格战
→ 胶膜 ASP 下行
→ 福斯特胶膜毛利率下行
→ 辅材龙头利润中枢重估
→ 存货 / 减值 / ROIC 压力
→ 电子材料第二曲线验证
```

### 5. 隔膜链

```text
动力电池需求增长
→ 隔膜供需偏紧
→ 恩捷毛利率和份额被重估
→ 扩产 / 折旧 / 客户压价
→ 隔膜 ASP 下行
→ 高壁垒材料周期化
→ ROIC 和估值重估
```

其中：

- 恩捷代表高壁垒材料周期化；
- 重点看隔膜 ASP、隔膜毛利率、出货量、CapEx、折旧、产能利用率、OCF 和 ROIC；
- 这条链用于验证“高壁垒只能延缓周期，不一定消灭周期”。

## 八、执行原则

1. 未核验的数据必须标注“待补”；
2. 同一指标必须尽量统一来源和口径；
3. 价格曲线优先用 SMM、百川、Wind、行业协会、公司公告或可靠媒体；
4. 财务数据优先用公司年报和季报；
5. 股价必须统一前复权口径；
6. 估值和持仓必须标注日期和口径；
7. 结论必须说明是“已验证”还是“待验证假设”。
