## 一、模块定位

06 模块不是继续扩写产业叙事，而是补齐投资研究所需的硬证据。

核心目标：

> 每一句“为什么涨、为什么跌”，都必须能落到 K 线、价格、财务、估值、持仓、新闻原文中的至少两个证据。

第一阶段样本宇宙固定为 14 只：

| 链条 | 样本 |
|---|---|
| 资源链 | 赣锋锂业、天齐锂业、华友钴业 |
| 材料链 | 通威股份、隆基绿能、福斯特、天赐材料、多氟多、恩捷股份 |
| 终端 / 设备 / 平台链 | 比亚迪、宁德时代、阳光电源、先导智能、亿纬锂能 |

每只样本的第一阶段完成门槛：

1. 有单股投资闭环卡；
2. 有 K 线节点表；
3. 有财务季度待补表；
4. 能对应至少一个价格或订单第一变量；
5. 能在总表中归入穿越、半穿越、周期承压、阴影或失败；
6. 缺失的精确值必须显式写成“待补”。

当前阶段已经完成：

1. 14 只标的投资闭环第一版；
2. 价格曲线硬数据框架；
3. 14 只标的 K 线节点待补表；
4. 14 只标的财务季度待补表；
5. 8 条投资闭环链总表；
6. 4 张总图压缩表；
7. 硬数据实填第一版。

因此，第一阶段的“样本底座”已经进入可收口状态；下一阶段应从“锚点级实填”推进到“精确值实填”。

## 九、当前最重要结论

### 1. 股价通常领先财务

```text
价格 / 订单 / 估值资金
→ 股价顶部
→ 毛利率顶部
→ 利润顶部
→ 减值和资产负债表恶化
```

### 2. 不同链条看不同第一变量

| 链条 | 第一变量 |
|---|---|
| 锂资源链 | 碳酸锂价格 |
| 硅料链 | 硅料价格 / 组件价格 |
| 六氟链 | 六氟价格 |
| 组件 / 胶膜链 | 组件价格 / 胶膜 ASP |
| 隔膜链 | 隔膜 ASP / 扩产折旧 |
| 设备订单链 | 下游 CapEx / 新签订单 / 合同负债 |
| 终端 / 核心供应商穿越链 | 毛利率 / OCF / ROIC / 海外化 |
| 多业务平台 / 资源材料复合链 | 分业务毛利率 / CapEx / 资源价格 / ROIC |

### 3. AI 映射不是简单类比行情，而是类比领先指标

后续研究 AI 标的时，优先问：

1. 它属于哪条链；
2. 第一领先变量是什么；
3. 价格、订单、毛利率、OCF、ROIC 有没有转弱；
4. 股价是否已经领先财务；
5. 第二曲线是否有真实利润和现金流。

## 十一、执行原则

1. 未核验的数据必须标注“待补”；
2. 同一指标必须尽量统一来源和口径；
3. 价格曲线优先用 SMM、百川、Wind、行业协会、公司公告或可靠媒体；
4. 财务数据优先用公司年报和季报；
5. 股价必须统一前复权口径；
6. 估值和持仓必须标注日期和口径；
7. 结论必须说明是“已验证”还是“待验证假设”。
