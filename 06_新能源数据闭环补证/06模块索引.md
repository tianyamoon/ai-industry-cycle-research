## 一、模块定位

06 模块不是继续扩写产业叙事，而是补齐投资研究所需的硬证据。

核心目标：

## 七、当前已形成的八条闭环链

### 1. 锂资源链

```text
碳酸锂价格顶部
→ 赣锋 / 天齐股价顶部
→ 锂盐毛利率顶部
→ 归母净利润顶部
→ 存货 / 减值 / 资产负债表压力
→ 估值重估
```

### 2. 硅料链

```text
硅料价格顶部
→ 通威股价顶部
→ 通威硅料毛利率顶部
→ 通威利润顶部
→ 硅料价格下跌
→ 隆基 / 通威利润中枢重估
→ 组件价格战
→ 隆基长期阴影
```

### 3. 六氟链

```text
六氟价格顶部
→ 天赐 / 多氟多股价顶部
→ 六氟 / 电解液毛利率顶部
→ 利润顶部
→ 六氟价格下跌
→ 材料利润中枢重估
→ 存货 / 减值 / ROIC 压力
→ 估值重估
```

### 4. 组件 / 胶膜链

```text
组件价格战
→ 胶膜 ASP 下行
→ 福斯特胶膜毛利率下行
→ 辅材龙头利润中枢重估
→ 存货 / 减值 / ROIC 压力
→ 电子材料第二曲线验证
```

### 5. 隔膜链

```text
动力电池需求增长
→ 隔膜供需偏紧
→ 恩捷毛利率和份额被重估
→ 扩产 / 折旧 / 客户压价
→ 隔膜 ASP 下行
→ 高壁垒材料周期化
→ ROIC 和估值重估
```

### 6. 设备订单链

```text
电池厂扩产
→ 锂电设备订单增长
→ 先导收入和利润确认
→ 合同负债 / 在手订单变化
→ 下游 CapEx 放缓
→ 新签订单下滑
→ 回款和现金流压力
→ 设备股估值重估
```

### 7. 终端 / 核心供应商穿越链

```text
需求高增
→ 终端平台 / 核心供应商 / 基础设施龙头重估
→ 赛道贝塔估值扩张
→ 股价领先利润见顶
→ 价格战和竞争加剧
→ 赛道贝塔退潮
→ 公司阿尔法验证
→ 毛利率 / OCF / ROIC / 海外化 / 第二曲线决定能否穿越
```

### 8. 多业务平台 / 资源材料复合链

```text
多业务平台 / 资源材料复合公司
→ 赛道贝塔重估
→ 多条业务线共同扩张
→ 股价和估值高位
→ 主线价格或需求退潮
→ 各业务毛利率、现金流、CapEx、ROIC 分化
→ 市场重新区分真平台和伪平台
```

其中：

- 亿纬锂能代表多业务电池平台；
- 华友钴业代表资源材料复合周期；
- 这条链用于验证“多业务和一体化到底是护城河，还是上行期叙事”。

## 九、执行原则

1. 未核验的数据必须标注“待补”；
2. 同一指标必须尽量统一来源和口径；
3. 价格曲线优先用 SMM、百川、Wind、行业协会、公司公告或可靠媒体；
4. 财务数据优先用公司年报和季报；
5. 股价必须统一前复权口径；
6. 估值和持仓必须标注日期和口径；
7. 结论必须说明是“已验证”还是“待验证假设”。
