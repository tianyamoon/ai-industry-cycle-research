## 一、一句话结论

> AI 的“更上游”不能由当前市场或媒体正在强调的某一条产业链定义，也不能把所有相关概念股装进一个池。正确入口是先从 `晶圆制造 / 先进封装 / 高速互联 / AIDC` 四条需求链识别制程必需、认证难替换、会影响良率或设备稼动率的原材料和零部件，再单独检验它们是否具备收入、毛利率和 OCF 弹性。
> 因此，本组把“核心卡点”和“核心弹性”拆开：前者回答谁不可轻易替代，后者回答谁最可能把量价变化放大为利润。两者同时成立才进入核心交叉验证池；供给覆盖、工程传导和泛化材料只承担各自边界内的观察或反证角色。

## 二、从全局路由上游，不能局限于当前市场或媒体强调的产业链

| AI 需求链 | 更上游应找的卡点 | 主维护入口 | 本组承担的职责 | 不应混同的对象 |
|---|---|---|---|---|
| 晶圆制造 / 存储扩产 | CMP、湿电子化学品、前驱体 / 特气、硅片、靶材、高纯石英、精密腔体 / 流体 / 真空部件 | 本组、`高端材料兑现组`、`300mm供给安全组`、`半导体设备全景组` | 找制程直接性、认证壁垒与订单到现金流的传导 | 泛气体、泛化工、仅有设备概念的公司 |
| 先进封装 | 载板、键合 / 底填 / 塑封材料、封装基材、测试 / 键合关键部件 | `先进封装组` | 与本组共享卡点评估框架，但经营排序留在封装组 | 把封装材料和前道耗材写成同一利润池 |
| 高速互联 | 光芯片 / 化合物材料、连接器、PCB/CCL 高速材料与精密部件 | `光模块组`、`PCB_CCL组` | 用同一筛选尺找上游瓶颈，不重复建卡 | 把光模块主链龙头与前道制造材料混排 |
| AIDC 电力与液冷 | 电源、热管理材料、管路阀件、冷板 / CDU 核心部件 | `服务器_温控组`、`液冷_散热二线弹性组`、供配电研究入口 | 只审计材料 / 部件的稼动率和订单传导 | 把工程签约直接等同持续耗材利润 |
| 资源与现场供给 | 电子级大宗气体、特种气体、深冷、回收提纯、现场供气 | 本组资源供给观察层 | 验证供给约束、电子级产品和客户收入拆分 | 因政策、稀有气体名词或项目开工直接升为核心卡点 |

这张表是路由，不是把所有上游公司塞进本组。每条链仍由原有组维护经营排序；本组只统一回答“是否构成关键卡点、是否存在核心弹性、证据是否足够升级”。

## 三、双维筛选：卡点强度不等于盈利弹性

### 1. 先过三道门

| 维度 | 要问的问题 | 通过条件 | 未通过时的落位 |
|---|---|---|---|
| `卡点强度` | 是否直接进入关键制程或影响良率、稼动率、可靠性；是否存在纯度、精度、认证、切换成本或供给集中度门槛 | 产品和应用场景可回源，且不能被短期通用替代 | 行业覆盖 / 待核，不叫核心卡点 |
| `经营弹性` | 扩产、利用率、出货或 ASP 变化，能否较快传导到收入、毛利率和 OCF | 有产品级量价 / 订单与报表字段的连续链条 | 只保留技术位置或供给安全观察 |
| `证据成熟度` | 是否已能拆到产品、客户 / 认证、收入、毛利率和现金流 | 至少形成公开来源的产品级证据与两个报告期的经营跟踪 | 不进入正式核心排序，只保留候选位 |

### 2. 用四象限决定角色，不用题材热度决定顺位

| 卡点强度 | 经营弹性 | 组内角色 | 维护动作 |
|---|---|---|---|
| 高 | 高 | `核心交叉验证池` | 优先回源产品、客户、订单、毛利率与 OCF；满足连续验证后才可称“核心弹性” |
| 高 | 待证 | `供给安全 / 长周期卡点池` | 重点看认证、规格升级、ASP、利用率和 CapEx 回报；不能因技术位置直接给高盈利权重 |
| 待证 | 高 | `弹性发现池` | 重点确认收入纯度、客户与现金流，防止泛材料或项目型增长误入主线 |
| 待证 | 待证 | `覆盖 / 反证池` | 只保留产品、客户、收入三项待核清单，不参与核心排序 |

## 四、样本宇宙：原材料、零部件和制程系统分开覆盖

| 样本层 | 代表样本 | 产业位置 | 当前角色与升级条件 |
|---|---|---|---|
| 核心交叉验证池 | 安集科技、鼎龙股份、中船特气、中巨芯、富创精密、新莱应材 | CMP / 湿电子化学品 / 特气 / 前驱体；工艺与结构零部件；高纯流体传输 | 已有主卡或公开产品基础。必须继续验证产品级订单 / 出货、毛利率、OCF 与客户集中度，不能把产业位置直接写成盈利结论 |
| 原材料卡点扩展池 | 沪硅产业、有研硅、立昂微、神工股份；江丰电子、欧莱新材、有研新材；菲利华、石英股份 | 硅片、高纯靶材 / 金属、高纯石英等制造基础材料 | 重点补先进规格、客户认证、产品收入、ASP、利用率与 CapEx 回报。硅片和靶材的供给安全价值不自动等于短期利润弹性 |
| 工艺耗材与化学材料弹性发现池 | 南大光电、雅克科技、上海新阳、晶瑞电材、格林达、江化微、彤程新材 | 前驱体、电子特气、湿电子化学品、光刻及配套材料 | 先拆出半导体产品和客户收入，再看认证转批量、单品放量、毛利率与 OCF；不以“国产替代”或平台储备替代经营验证 |
| 精密零部件扩展池 | 华亚智能、汉钟精机、珂玛科技、富乐德 | 精密结构件、真空、先进陶瓷 / 石英件、设备零部件服务 | 先核产品是否进入设备 / 晶圆厂关键环节，再核设备厂客户、交付、收入、应收和 OCF；未证实前不抢富创 / 新莱的核心位 |
| 制程系统与厂务传导池 | 盛剑环境、正帆科技、至纯科技、京仪装备 | 高纯工艺系统、废气 / 真空 / 流体子系统、附属装备与厂务 | 作为制造 CapEx 向配套外溢的传导器；合同须转交付、验收、回款和 OCF，不直接等同重复耗材利润池 |
| 跨组上游材料 / 部件 | 华海诚科、艾森股份、兴森科技、深南电路；源杰科技、光库科技；飞荣达、鼎通科技等 | 先进封装、高速互联、AIDC 各自的材料和核心部件 | 由 `先进封装组`、`光模块组`、`PCB_CCL组`、温控 / 液冷组维持主排序；本组只提供统一卡点筛选口径，避免重复计数 |
| 资源与电子气体覆盖池 | 华特气体、金宏气体、广钢气体、和远气体、中泰股份、杭氧股份、凯美特气 | 电子级大宗 / 特种气体、深冷、回收提纯与现场供气 | 先证实电子级产品、客户、项目利用率与收入贡献。没有产品和经营拆分时，只是资源覆盖，不是核心卡点或核心弹性 |

### 样本边界

1. 上表中的扩展样本是防漏候选池，不是推荐名单，更不是已完成同等级验证的核心样本。
2. 同一公司若在先进封装、光互联或 AIDC 已有主组，只在原主组维护经营排序；本组不重复计算其“核心弹性”。
3. 气体只是一类原材料。它和硅片、靶材、石英、湿电子化学品、精密零部件一样，都必须回到制程直接性、认证壁垒和经营闭环。

## 五、核心卡点池与核心弹性池的使用方式

### 1. 核心卡点优先验证

1. `重复消耗型工艺材料`：安集科技、鼎龙股份、中船特气、中巨芯。
2. `制造基础材料`：300mm 硅片组、江丰电子 / 欧莱新材 / 有研新材的靶材方向、高纯石英候选。
3. `设备关键接口`：富创精密、新莱应材，以及真空、陶瓷 / 石英件、精密结构件候选。

这些对象先问“缺它会不会影响先进制造的良率、稼动率或合格率”，再问“是否已经赚钱”。

### 2. 核心弹性只在经营闭环后确认

1. `消耗型材料弹性`：认证转量产、订单 / 出货提升、ASP 或产品结构改善，且毛利率与 OCF 同步。
2. `设备部件弹性`：设备厂交付增加，能转化为零部件收入、毛利率、回款与 OCF，且客户集中度不恶化。
3. `系统传导弹性`：项目合同转交付、验收和回款；只能证明 CapEx 外溢，不能替代材料 / 部件的持续消耗逻辑。

### 3. 不允许自动升级的情形

1. 氦气等资源政策、单个项目中标、行业报告预测、市场同涨，都不能单独证明公司盈利弹性。
2. 只有“产品可做”或“已送样”而没有客户认证、收入与现金流时，只能停留在卡点候选。
3. 应收、存货、CapEx 增长长期快于 OCF 时，即使收入增长，也要降低弹性判断。

## 六、当前研究优先级

1. `第一优先：验证卡点与弹性的交集`
   - 安集科技、鼎龙股份、中船特气、中巨芯、富创精密、新莱应材。
   - 固定核对：产品 / 客户 / 订单或出货 -> 收入 / 毛利率 -> OCF。
2. `第二优先：补齐原材料卡点`
   - 硅片、靶材、高纯石英、湿电子化学品和前驱体的产品级客户与收入拆分。
   - 固定核对：先进规格、认证、ASP、利用率、CapEx 回报。
3. `第三优先：补齐设备关键部件`
   - 精密结构件、真空、高纯流体、陶瓷 / 石英件与设备零部件服务。
   - 固定核对：设备厂 / 晶圆厂客户、交付、收入、毛利率、应收和 OCF。
4. `第四优先：用系统和资源做传导与反证`
   - 正帆科技、至纯科技、京仪装备、盛剑环境及电子气体 / 深冷覆盖样本。
   - 固定核对：合同到回款的闭环、电子级产品纯度和收入贡献；不直接列为核心弹性。

## 七、组级验证与证伪

| 层次 | 正向验证 | 降级 / 证伪信号 |
|---|---|---|
| 重复消耗型材料 | 客户认证升级、重复订单、单品收入和毛利率同步改善，OCF 不背离 | 认证停滞、收入无法拆分、存货 / CapEx 先行而 OCF 持续弱 |
| 硅片 / 靶材 / 高纯石英 | 规格升级、客户认证、出货、ASP 与利用率改善 | 扩产领先于真实需求、价格下行、资本强度长期压制利润与 OCF |
| 精密零部件 / 高纯流体 / 真空 | 设备厂客户、订单和交付持续增加，毛利率与回款匹配 | 客户集中度恶化、订单不转收入、应收 / 存货持续抬升 |
| 制程系统 / 厂务 | 合同、交付、验收和回款形成闭环 | 只有中标 / 项目新闻，长期无收入或现金流验证 |
| 资源供气 | 电子级产品、客户与收入可拆分，利用率和毛利改善 | 只有通用工业气体、资源政策或项目投产叙事 |

## 八、可直接复用结论

> 上游研究的顺序应是“全局需求链 -> 制程卡点 -> 样本宇宙 -> 经营弹性 -> 现金流验证”，而不是“一个气体事件 -> 若干概念股”。当前先把原材料、关键零部件和制程系统的样本宇宙铺全，再用 `卡点强度 × 经营弹性 × 证据成熟度` 收缩到少数核心交叉验证样本。这样既不漏掉真正的瓶颈，也不会把所有上游公司写成同一条受益链。
