## 一、一句话结论

> 更上游平台补位组已由原来的 4 个样本扩为 `15 个跟踪样本 + 3 个待核观察位`，覆盖 `电子特气核心 / 电子材料平台 / 深冷与气体运营 / 高纯工艺系统与洁净厂务 / 边界观察` 五层。
> 当前仍以 `中船特气 -> 中巨芯` 为主锚，`华特气体 -> 金宏气体 -> 广钢气体 -> 南大光电 -> 和远气体` 为直接补位，`飞凯材料 / 上海新阳` 保留材料配套定位，其他样本只按各自利润池跟踪，不统一视为特气受益股。
> 扩容的目的，是覆盖产业位置和验证路径，而不是把政策、题材或同涨关系直接升级为盈利结论。

## 二、组别边界

### 事实

1. 本组跟踪样本为：`中船特气 / 中巨芯 / 华特气体 / 金宏气体 / 广钢气体 / 和远气体 / 南大光电 / 飞凯材料 / 上海新阳 / 中泰股份 / 杭氧股份 / 凯美特气 / 正帆科技 / 至纯科技 / 柏诚股份`；待核观察位为 `雅克科技 / 三孚股份 / 昊华科技`。
2. 这些标的不对应同一条兑现路径：
   - `中船特气 / 华特气体 / 金宏气体`：电子特气与特种气体的直接供应平台；
   - `中巨芯 / 南大光电`：电子湿化学品、电子特气、前驱体或光刻胶的材料平台；
   - `广钢气体 / 和远气体`：电子大宗气体、电子特气及园区供气项目承接；
   - `飞凯材料 / 上海新阳`：先进封装和半导体工艺材料配套；
   - `中泰股份 / 杭氧股份 / 凯美特气`：深冷、气体运营、尾气回收和电子化学品边界承接；
   - `正帆科技 / 至纯科技 / 柏诚股份`：高纯工艺系统、气体配套、洁净室和厂务旁证；
   - `雅克科技 / 三孚股份 / 昊华科技`：电子材料或氟化工平台观察位，暂不按纯特气样本处理。
3. 现有仓库已分别完成部分单卡、观察卡或对照稿；新增样本先回写组卡和总表，不平行新建个股卡。

### 判断

1. 这一组的研究价值在于把“更上游”拆成不同利润池和验证链，而不是把更多股票堆到同一结论里。
2. 与 `高端材料兑现组` 相比，本组新增了电子气体、气体运营、深冷装备和高纯厂务的覆盖；重复出现的 `飞凯材料 / 上海新阳 / 雅克科技` 必须保留角色边界。
3. 与 `300mm供给安全组`、`半导体设备全景组` 相比，本组更接近“材料与制程基础设施”的上游入口，但高纯系统和洁净厂务只能作为旁证层。

## 四、为什么这一组值得单独看

### 事实

1. 本轮扩容补上了电子气体供应、电子材料平台、深冷气体与高纯系统几个原先缺口，样本覆盖不再只依赖 `中船特气 / 中巨芯 / 飞凯材料 / 上海新阳`。
2. `中船特气` 已出现供需偏紧、调价、订单强化三者联动的正式线索；`中巨芯` 已在定期报告中把三平台能力、客户基础与利润修复写入报表。
3. `华特气体、金宏气体、广钢气体、和远气体、南大光电` 的公司概况或官网产品页能够确认产业位置，但客户、收入拆分和盈利传导仍需逐一回到年报、IR 和中报。
4. `中泰股份、杭氧股份、凯美特气、正帆科技、至纯科技、柏诚股份` 提供深冷、气体运营、尾气回收、高纯工艺系统和洁净厂务的旁证覆盖，不直接替代电子特气核心样本。

### 判断

1. 这组最有价值的地方，是它能把“更上游”拆成直接电子气体、材料平台、气体资源与系统配套五层，避免继续用泛化的“国产替代受益”覆盖差异。
2. 市场最容易误读的是把平台补位、单品种催化、项目建设、气体运营和洁净工程混成同一利润池。
3. 因此，本组扩容后应转入分层滚动验证：核心样本看价格 / 订单 / 报表，补位样本看客户 / 项目 / 收入，旁证样本看订单 / 回款，观察位只保留证据缺口。

## 五、当前优先级

### 当前排序

1. `第一层：中船特气 / 中巨芯`
2. `第二层：华特气体 / 金宏气体 / 广钢气体 / 南大光电 / 和远气体`
3. `第三层：飞凯材料 / 上海新阳 / 中泰股份 / 杭氧股份`
4. `第四层：凯美特气 / 正帆科技 / 至纯科技 / 柏诚股份`
5. `观察层：雅克科技 / 三孚股份 / 昊华科技`

### 判断

1. `中船特气` 和 `中巨芯` 保持主锚顺序：前者看单品种硬传导，后者看平台转单线突破。
2. `华特气体 / 金宏气体` 是最值得优先补证的直接电子特气平台；`广钢气体 / 和远气体` 更偏电子大宗气体和项目兑现。
3. `南大光电` 的电子特气与前驱体属于材料平台补位，需与光刻胶业务拆开；`飞凯材料 / 上海新阳` 保留既有材料卡的定位，不因扩容改变优先级。
4. `中泰股份 / 杭氧股份 / 凯美特气` 的价值是补足深冷、供气和回收提纯链，不能直接套用六氟化钨的价格传导结论。
5. `正帆科技 / 至纯科技 / 柏诚股份` 与三个观察位只用于扩大覆盖和反证，不直接进入本组主排序。

## 六、待验证点

### 中船特气

1. 六氟化钨供需偏紧与调价是否继续维持；
2. 增量订单能否继续传导到收入、毛利率与利润；
3. 后续是否出现更多客户扩量或其他电子特气品类协同放量。

### 中巨芯

1. 三大业务板块中哪一条最先形成单线突破；
2. 原材料涨价后的成本传导是否顺畅；
3. 经营现金流能否随利润修复同步改善。

### 飞凯材料

1. 临时键合材料是否出现更明确客户验证、量产或订单口径；
2. 先进封装景气受益能否继续传导到半导体材料收入结构；
3. 新基地投产前是否出现新的产品级催化。

### 上海新阳

1. 是否出现头部客户认证、批量订单或单品种放量；
2. 平台扩线是否继续转化为更明确的收入与毛利率改善；
3. 平台储备能否从“方向正确”升级到“订单成立”。

### 华特气体 / 金宏气体 / 广钢气体 / 和远气体 / 南大光电

1. `华特气体`：盯高端电子气体客户认证、供货稳定性、订单与特气收入；氦气政策只能作为旁证，不直接升级公司映射。
2. `金宏气体`：盯电子气体客户和供气项目，拆分大宗气体、特种气体与天然气业务，验证收入、毛利和 OCF。
3. `广钢气体`：盯电子大宗气体现场制气项目的投产、利用率、客户集中度和回款，不能将现场供气等同于单品种特气涨价。
4. `和远气体`：盯电子特气及功能性材料产业园的项目进度、产品认证、投产和收入兑现，先验证建设事实再判断利润弹性。
5. `南大光电`：把前驱体、电子特气、光刻胶拆开跟踪，优先看电子特气 / 前驱体单品放量和客户验证。

### 中泰股份 / 杭氧股份 / 凯美特气

1. `中泰股份`：盯特种气体运营、气体价格、客户与深冷设备订单，区分设备一次性确认与气体运营持续收入。
2. `杭氧股份`：盯空分与特气项目投产、气体利用率、合同质量和 OCF，区分大宗气体运营与半导体纯度需求。
3. `凯美特气`：盯尾气回收、特种气体和电子化学品的产品与客户证据，若无法拆出半导体收入则保持边界观察位。

### 正帆科技 / 至纯科技 / 柏诚股份

1. 盯高纯工艺系统、气体系统、二次配管和洁净室项目订单是否转化为收入与经营现金流。
2. 区分设备 / 工程验收节奏与材料、气体的持续消耗，不把厂务订单直接写成电子特气需求兑现。
3. 若两个报告期只有项目线索而无订单、收入或回款推进，降为厂务题材旁证。

### 雅克科技 / 三孚股份 / 昊华科技

1. `雅克科技`：核对前驱体具体产品、客户和收入拆分，避免与电子特气核心样本重复计数。
2. `三孚股份`：核对特种电子气体、光纤四氯化硅和其他硅化工业务的收入、客户与 OCF。
3. `昊华科技`：核对具体电子化学品 / 氟化工产品、客户和半导体收入贡献；在证据不足前只作观察位。

## 七、组内排序可直接复用结论

> 更上游平台补位组当前应作为半导体上游继续深挖的第一主入口，但必须按利润池分层：`中船特气 / 中巨芯` 看单品种供需与平台转单线突破；`华特气体 / 金宏气体 / 广钢气体 / 和远气体 / 南大光电` 看电子气体或电子材料的客户、项目与收入兑现；`飞凯材料 / 上海新阳` 看材料产品级催化；`中泰股份 / 杭氧股份 / 凯美特气` 看深冷、气体运营和回收提纯；`正帆科技 / 至纯科技 / 柏诚股份` 只作高纯系统与厂务旁证；`雅克科技 / 三孚股份 / 昊华科技` 继续保留待核。扩容后，样本覆盖更完整，但主结论强度仍由订单、客户、收入、毛利率、OCF 和回款决定。

### 2026-07-10 氦气出口管理政策更新

- 事实：商务部、海关总署公告 2026 年第 29 号明确，对氦气（海关商品编号 `2804290010`）实施临时禁止出口管理，公告自公布之日起执行。
- 变量：政策改变的是氦气出口管理状态，暂不能推导国内氦气供给收紧、价格上涨或电子级氦气盈利改善。
- 对本组的影响：可作为电子特气 / 稀有气体方向的政策变量旁证；不改变 `中船特气` 当前以六氟化钨供需、调价和订单为核心的主线，也不直接升级 `广钢气体`、`华特气体` 等公司映射。新增样本的纳入依据是公司公开产品 / 业务资料，而不是这条政策本身。
- 下一步验证：补充中国氦气进出口量、国内产量与进口依赖、电子级产品占比、库存和价格；再核对相关上市公司的氦气产能、纯度、客户认证和收入贡献。
- 证据等级：`A`（一级政府公告）；产业影响状态：待验证。
