# 中船特气｜电子特气预期差卡｜第一版
> 标的：中船特气 / 688146.SH  
> 所属模块：02_桥接层/03_AI标杆股研究卡  
> 生成日期：2026-06-28  
> 所属方向：电子特种气体 / 六氟化钨 / 先进存储与先进逻辑上游材料  
> 研究定位：作为 `AI上游下一批候选标的扩展清单_第一版.md` 中最值得优先深挖的新增候选，把“电子特气供需偏紧、价格传导强化、增量订单洽谈”这条线正式下沉为单公司跟踪卡。  
> 使用边界：本卡重点审计的是 `单品种供需偏紧 -> 调价机制强化 -> 增量订单 -> 收入利润兑现 -> 产能投放前后供需切换`，不把所有电子特气景气笼统等同于公司中期利润已经完全打开。

---

## 一、一句话结论

> 中船特气当前最大的预期差，不在“也是国产替代电子气体公司”，而在它已经通过正式披露把 `六氟化钨供需偏紧、季度/月度调价机制推进、客户增量订单洽谈` 写得足够具体，说明 AI 与先进存储景气对电子特气的传导，已经从方向逻辑走到单品种变量层。

---

## 二、研究边界

1. 研究对象：中船特气在 `六氟化钨` 等电子特气方向的供需、价格与订单强化；
2. 不研究内容：不把公司全部气体产品按同一斜率处理，也不把泛半导体景气直接外推成所有产品同步高弹性；
3. 时间边界：以 `2026Q1 -> 2026-04-23/2026-05-08 投资者关系活动记录` 为主；
4. 证据边界：优先使用 `季报、投资者关系活动记录、交易所正式披露`，不依赖未经核实的产业传闻。

---

## 三、事实

### 1. 2026Q1 已经出现收入与利润同步改善

1. 公司 `2026Q1` 实现营业收入 `7.01亿元`，同比增长 `36.00%`；
2. `2026Q1` 归母净利润 `1.01亿元`，同比增长 `16.86%`；
3. `2026Q1` 扣非归母净利润 `1.05亿元`，同比增长 `30.70%`；
4. 公司季报明确披露，本期收入增长主要系 `市场需求增长，调整部分产品销售价格`。[来源](https://static.cninfo.com.cn/finalpage/2026-04-21/1225127186.PDF)

### 2. 六氟化钨已经从产品名词变成核心变量

1. 公司在 `2026-04-23` 投资者关系活动记录表披露：
   - 当前 `六氟化钨` 有效产能为 `2000吨/年`；
   - 产品纯度均为 `6N级`；
   - 已完全满足 `3D NAND、HBM及先进逻辑芯片制程需求`。[来源](https://dataclouds.cninfo.com.cn/shgonggao/investor/2026/20260423/2155da11a4cf472f9581762895c8a6e1.PDF)
2. 同一记录表披露：
   - 受 `境外部分厂商宣布下半年停产` 影响；
   - 部分客户正积极与公司洽谈 `增量订单`；
   - 在新增产能投放前，公司判断市场将维持 `紧平衡状态`。[来源](https://dataclouds.cninfo.com.cn/shgonggao/investor/2026/20260423/2155da11a4cf472f9581762895c8a6e1.PDF)

### 3. 价格机制正在强化，而不只是静态涨价

1. 公司披露，面对上游钨粉价格快速上涨，正将调价机制由 `年度` 逐步过渡至 `季度`；
2. 同时计划进一步向 `月度调价` 演进，以更及时、真实地反映当期收入与利润贡献；
3. 公司还披露，部分海外客户已提出以 `长协` 锁定订单，价格让步换取增量供应保障。[来源](https://dataclouds.cninfo.com.cn/shgonggao/investor/2026/20260423/2155da11a4cf472f9581762895c8a6e1.PDF)

### 4. 供给扩张已经明确，但兑现窗口还没走完

1. 公司披露，新增 `1000吨/年` 六氟化钨产能预计将于 `2027年` 投产；
2. 这意味着未来最关键的审计点，不只是景气是否存在，而是：
   - `2027年产能投放前` 的供需是否持续紧平衡；
   - `2027年投放后` 是否进入供给缓解或价格回落阶段。[来源](https://dataclouds.cninfo.com.cn/shgonggao/investor/2026/20260423/2155da11a4cf472f9581762895c8a6e1.PDF)

### 5. 风险端已经先在现金流上露头

1. `2026Q1` 经营活动现金流净额为 `-1.28亿元`；
2. 公司解释主要系 `原材料备货支付货款增加`；
3. 这说明当前阶段并非“完全无风险的利润兑现”，而是已经进入：

`收入与利润改善 -> 原材料备货抬升 -> 现金流后验证`

的审计阶段。[来源](https://static.cninfo.com.cn/finalpage/2026-04-21/1225127186.PDF)

## 四、判断

### 1. 当前阶段判断

中船特气当前更接近：

`单品种供需偏紧 -> 调价机制强化 -> 增量订单洽谈 -> 等待利润与现金流继续兑现`

它已经不只是：

1. 泛电子气体国产替代故事；
2. 没有单品种变量支撑的平台型映射位；
3. 还停留在“未来可能受益”的概念候选。

### 2. 为什么它值得单独落卡

1. 相比中巨芯、飞凯材料这类平台补位候选，中船特气最大的优势是已经出现了足够具体的 `单产品级` 变量；
2. 这种变量不是“订单可能增加”，而是：
   - 哪个产品；
   - 当前产能多少；
   - 满足哪些先进制程；
   - 什么原因导致客户来谈增量订单；
   - 下一轮新增产能何时投放。
3. 因此它比一般“上游景气受益”样本更接近可执行跟踪。

### 3. 为什么它还不是最硬主锚

1. 当前最硬的主锚仍是长川科技、鼎龙股份这类已经把利润或订单链压得更实的样本；
2. 中船特气当前最硬的新增量，仍主要集中在：
   - 单品种供需；
   - 调价机制；
   - 增量订单洽谈。
3. 它后续还需要中报、三季报继续验证这条线是否真正向利润质量和现金流传导。

## 六、买点触发

1. 中报或三季报继续披露六氟化钨收入、毛利率或订单强化；
2. 调价机制继续向利润表传导，而不是只停留在投关口径；
3. 客户增量订单进一步从“洽谈”走向更明确的交付或长协；
4. 经营现金流开始跟上利润修复，证明备货压力可控。

---

## 七、证伪信号

1. 六氟化钨增量订单后续没有继续被正式确认；
2. 调价机制推进，但利润率未见改善，说明价格传导不顺；
3. 原材料备货持续吞噬现金流，且无法转化为更高质量收入；
4. 2027 年新增产能投放预期提前压制价格与订单斜率。

---

## 八、待补点

1. `2026H1` 毛利率、存货、经营现金流变化；
2. 六氟化钨之外，其他核心电子特气产品的景气传导强弱；
3. 海外客户长协与国内客户增量订单的后续口径；
4. 2027 年新增产能投放前后的供需切换节奏。
