# 源杰科技｜光芯片与 CW / EML 激光器观察卡
> 标的：源杰科技 / 688498.SH
> 所属模块：02_桥接层/03_AI标杆股研究卡/光模块组
> 生成日期：2026-07-07
> 所属方向：高速互联上游光芯片 / CW 光源 / EML 激光器
> 研究定位：作为 `中际旭创 / 新易盛 / 天孚通信` 三卡之外的更上游补位样本，补上“光模块景气能否继续向激光器芯片与 InP 光源穿透”的执行层观察位。
> 使用边界：本卡重点审 `产品结构切换 -> 客户验证 -> 批量出货 -> 收入 / 利润 / OCF -> 应收 / 存货 / CapEx`，不把 `硅光 / CPO / 1.6T` 这些前沿路线直接写成已经兑现的利润。

---

## 一、一句话结论

> 源杰科技当前更适合定义为 `高速互联上游光芯片兑现观察位`，而不是 `光模块主链龙头` 或 `纯题材硅光期权`：`2025 年报` 已验证收入重心从电信市场转向数据中心市场，`2026Q1` 收入 / 利润 / OCF 继续放大，`2025H1-Q3` 投关记录又把 `CW70mW / 100mW`、`100G / 200G PAM4 EML` 与 `1.6T / CPO` 相关储备写实；但估值极高，应收、存货和扩产资产同步抬升，下一步必须看 `客户导入 -> 批量出货 -> 收入 / 利润 / OCF` 是否连续闭环。

---

## 二、研究边界

1. 研究对象：源杰科技在 `CW 光源 + EML 激光器 + 25/50G PON` 方向的经营兑现强度。
2. 不研究内容：不把所有 `硅光 / CPO / 1.6T` 远期想象直接等同于当前利润。
3. 时间边界：以 `2025H1 / 2025Q3 投关记录 -> 2025年报 -> 2026Q1 -> 2026-06-23 与 2026-07-01 风险提示公告 -> 2026-07-07 行情快照` 为主。
4. 证据边界：优先使用 `公司官网财报页、年报、季报、投资者关系活动记录、风险提示公告` 与 `Luna Stock MCP`；社区帖和题材转述不进入主结论。
5. 技术口径：`2026-07-07` 的价格、估值和两融只作为高波动审计辅助口径，不替代经营结论。

## 三、事实

### 1. 2025 年报先把收入驱动从电信切到数据中心写实

1. 公司 `2025` 年营业收入 `6.01亿元`，同比增长 `138.50%`。
2. `2025` 年归母净利润 `1.91亿元`，同比 `扭亏为盈`；扣非归母净利润 `1.67亿元`。
3. `2025` 年经营活动现金流净额 `1.50亿元`，同比增长 `692.34%`。
4. `2025` 年光芯片主营业务毛利率 `58.15%`，同比提升 `24.62pct`。
5. 其中 `电信市场业务` 收入 `2.06亿元`，同比增长 `2.06%`，毛利率 `31.17%`；`数据中心业务` 收入 `3.93亿元`，同比增长 `719.06%`，毛利率 `72.21%`。
6. 年报明确写到，公司未来主要收入增长来源已转向数据中心市场，并将由“电信市场收入为主”逐步发展成国内领先的 `电信市场 + 数据市场` 协同拓展的光芯片供应商。

### 2. 2026Q1 收入、利润和现金流继续加速，但营运项目也同步抬升

1. `2026Q1` 营业收入 `3.55亿元`，同比增长 `320.94%`。
2. `2026Q1` 归母净利润 `1.79亿元`，同比增长 `1153.07%`；扣非归母净利润 `1.78亿元`，同比增长 `1173.64%`。
3. `2026Q1` 经营活动现金流净额 `3965.08万元`，上年同期为 `-546.20万元`。
4. `2026Q1` 期末应收账款 `4.10亿元`，较 `2025` 年末 `2.64亿元` 明显上升。
5. `2026Q1` 期末存货 `1.62亿元`，较 `2025` 年末 `1.34亿元` 上升；预付款项 `2058.62万元`，较 `2025` 年末 `836.25万元` 上升。
6. `2026Q1` 期末固定资产 `8.20亿元`，较 `2025` 年末 `6.72亿元` 上升；其他非流动资产 `3.72亿元`，较 `2025` 年末 `2.02亿元` 上升；在建工程 `1.38亿元`，较 `2025` 年末 `1.56亿元` 下降，说明部分扩产投入已从在建转向固定资产。
7. `2026Q1` 期末合同负债 `186.17万元`，较 `2025` 年末 `370.72万元` 下降，说明这不是典型靠高预收锁单的商业模式，后续更要看真实交付与回款。

### 3. 投关记录已经把产品推进路径写得比较实

1. `2025-08-31` 披露的半年度业绩交流会记录显示，`2025H1` 电信市场收入 `9987.35万元`，同比下降 `8.93%`；数据中心及其他业务收入 `1.05亿元`，同比增长 `1034.18%`。
2. 同次交流会披露，面向下一代 `25/50G PON` 网络的光芯片产品已实现批量交付并形成规模收入，`10G EML` 在电信市场已成为重要收入组成部分之一。
3. 同次交流会披露，公司已成功量产面向 `400G / 800G` 光模块需求的 `CW70mW` 激光器芯片，并正逐步进入更广泛的客户供应链。
4. 同次交流会披露，`100G PAM4 EML` 已通过客户验证，下一步目标是实现批量出货并逐步提升份额；`200G PAM4 EML` 已完成产品开发，处于客户推广阶段。
5. 同次交流会披露，`100mW CW` 产品验证和测试符合预期，`2025` 年下半年数通收入预计将大幅增长，未来将随客户硅光方案及 `1.6T` 需求逐步出货。
6. `2025-10-30` 披露的三季度业绩交流会记录显示，公司 `2025年前三季度` 营收 `3.83亿元`，同比增长 `115.09%`，归母净利润 `1.06亿元`，同比扭亏为盈；其中 `2025Q3` 营收 `1.78亿元`，同比增长 `207.31%`，环比增长 `47.90%`。
7. 同次交流会披露，公司预计产能将从 `2025年末到 2026 年` 逐步成长；`CPO` 领域光源产品已与客户进行技术规格对接；美国工厂仍处于基础设施改造和设备预定阶段。
8. 同次交流会披露，公司判断当时市场需求仍较旺盛、产品降价压力较小，`CW70mW / CW100mW` 更高功率产品可在兼容产线上直接转产。

### 4. 截至 2026-07-07，它已经是高估值高波动资产，不是低位冷门补涨股

1. `2026-06-23` 与 `2026-07-01` 公司连续披露 `股票交易风险提示公告`，核心指向都是 `估值显著高于行业平均` 与 `交易波动加大`。
2. `Luna Stock MCP` 交叉口径显示，源杰科技在 `2026-07-07` 收盘 `1715.04元`，当日 `+4.58%`，成交额约 `81.98亿元`，换手率约 `4.02%`。
3. 同口径下，`2026-07-07` 对应 `PE 599.71x / PB 86.63x`，静态估值处于极高水平。
4. `2026-07-06` 融资余额约 `35.98亿元`，占流通市值约 `1.79%`；`2026-07-01` 至 `2026-07-06` 两融余额维持高位，说明杠杆博弈不低。
5. 因此当前更应把它放在 `高波动下的兑现审计样本` 里看，而不是把它定义成尚未被市场发现的冷门光芯片资产。

## 四、判断

### 1. 当前阶段判断

源杰科技当前更接近：

`电信 DFB 老业务筑底 -> 数据中心 CW 光源放量成为利润主驱动 -> 100G/200G EML 与 1.6T / 硅光储备打开下一阶段想象 -> 等待 2026H1 / 2026Q3 验证应收、存货、OCF 与客户导入是否同步闭环`

它已经不只是：

1. 纯电信 `10G DFB` 老业务公司；
2. 只靠 `CPO / 硅光` 题材映射的远期概念股；
3. 简单跟涨中际旭创、新易盛的模块二线映射位。

### 2. 为什么它值得单独补一张卡

1. 光模块组三卡主要覆盖 `龙头确定性 / 二线高弹性 / 上游器件平台`，但还缺一张更上游的 `激光器芯片兑现观察位`。
2. 源杰科技正好补上 `InP laser / CW / EML` 这一层，能帮助判断高速互联景气是否真的穿透到更上游光芯片。
3. 它又和 `AI全球芯片制造深链与A股映射总表.md` 里的 `InP 光器件簇 -> A股映射` 直接对应，具备方法论上的补位价值。
4. 对 AI 映射来说，它更像“模块景气向上游激光器芯片传导是否成立”的验证器，而不是新的主升龙头。

### 3. 为什么它暂时不能抬成组内主锚

1. 当前最硬的新增信息主要来自 `产品结构切换 + 客户验证 + 量产推进`，而不是像中际旭创那样的体量级长期化收入确定性。
2. 公司 `2026Q1` 的应收、存货、预付款和扩产资产都在上升，说明经营兑现仍要通过资本周期审计。
3. 估值和价格位置都已很高，市场已经交易了不少 AI 数通和光芯片想象。
4. 因此它更适合定义为 `上游光芯片兑现观察位`，暂不宜直接上调为 `核心算力资产层主锚`。

### 4. 第一性原理下真正该盯的变量传导

1. 先看 `CW70mW / 100mW / 更高功率产品` 是否继续批量交付，而不是先看 CPO 叙事。
2. 再看 `100G / 200G PAM4 EML` 是否从验证、推广走向规模出货。
3. 再看 `应收 / 存货 / OCF / 固定资产` 是否与收入和利润的放大相匹配。
4. 最后看 `1.6T / 硅光 / CPO` 客户导入能否真正写进收入，而不是只停留在技术规格交流。
5. 如果后续只有“路线储备”而没有“批量出货和现金流”，就不应把它抬成更高权重样本。

## 六、继续强化触发器

1. `2026H1` 中报继续验证数据中心业务收入、毛利率和 OCF 强势。
2. `100G PAM4 EML` 从验证转入批量出货，`200G PAM4 EML` 进入更明确客户导入阶段。
3. `100mW` 及更高功率 `CW` 光源正式写进客户放量和收入结构。
4. `1.6T / 硅光 / CPO` 相关需求不只停留在规格交流，而是出现正式出货。
5. 应收、存货和扩产资产增速开始和收入确认、现金回款更匹配。

## 七、证伪信号

1. 数据中心业务增速明显放缓，而电信业务又无法补位。
2. `100G / 200G EML`、`100mW / 300mW CW` 长期停留在验证或测试口径，迟迟不能写进收入。
3. 应收、存货和预付款继续快于收入增长，`OCF` 再度转弱。
4. 美国工厂、扩产设备或新产品导入进度显著滞后。
5. 高估值状态下，如果经营兑现放缓，市场会优先从估值端压缩。
