## 二、分层执行要点

### 核心设备层

1. `长川科技` 更适合按“利润兑现审计”框架跟踪。
2. `盛美上海` 更适合按“订单传导审计”框架跟踪。
3. 核心设备层的重点不是再讲题材，而是继续看订单、交付、利润和现金流。

### 平台扩线观察层

1. `京仪装备` 更适合按“关键子系统配套扩线审计”框架跟踪。
2. 重点不是只看工艺覆盖或客户层级，而是继续看 `合同负债 / 库存 / 在建工程 / 安徽基地` 如何传导到收入、利润和 OCF。

### 旁证扩展层

1. `中微公司 / 北方华创` 更像平台型龙头旁证，不是新增低位预期差。
2. `华峰测控 / 华兴源创 / 中科飞测` 更像测试与量测补位。
3. `拓荆科技 / 芯源微` 更适合盯工艺设备链条是否继续写实。

### 先进封装/TGV 旁证层

1. `长电科技 / 通富微电` 主要看先进封装平台能力是否继续外溢。
2. `帝尔激光 / 莱宝高科 / 兴森科技 / 京东方A` 主要看玻璃基板、TGV、面板级相关验证是否升级。
3. 这层只做旁证，不抢核心设备的结论位。

### 2. 资本周期固定回看项

1. 每次中报、三季报后，固定回看 `OCF / 应收 / 存货 / 合同负债 / 预付款 / CapEx / 在建工程 / 研发费用率`。
2. 若利润、订单、客户验证强化，但上述项目同时恶化，必须在本清单明确写成“资本周期压力上升”，不能只写景气。
3. 若预付款、存货、在建工程抬升能被订单、交付、产能建设解释，则写成“能力建设配套”；若不能解释，则写成“兑现质量待审计”。

### 3. 回写路径

1. 新披露先回写本清单，因为这里负责定义当前样本是强化兑现还是只强化叙事。
2. 再回写 `半导体设备旁证扩展层证据表.md`，把新增事实落到具体样本的证据状态、升级条件与风险信号。
3. 最后再回写 `半导体设备全景组.md`，只上收已经被前两层验证过的阶段结论。
