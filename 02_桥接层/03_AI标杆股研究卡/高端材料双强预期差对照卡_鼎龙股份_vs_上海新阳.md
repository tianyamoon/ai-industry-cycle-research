# 高端材料双强预期差对照卡｜鼎龙股份 vs 上海新阳

> 生成日期：2026-06-28
> 研究定位：在 `鼎龙股份_高端光刻材料CMP预期差卡.md` 与 `上海新阳_高端光刻材料预期差卡.md` 基础上，进一步把高端材料方向里最容易混淆的两种样本放到同一张对照卡里，直接回答：`谁已经形成订单事实链，谁仍处平台储备增强阶段，下一条最关键证伪数据是什么`。
> 使用边界：本文件不是重写两家公司全部业务，而是围绕 `订单硬度、平台深度、兑现路径、证伪位置` 做横向拆分。
> 证据边界：只使用两张单公司卡中已经落地的正式公告、年报、季报、业绩说明会与公司 IR。

## 一、一句话结论

> 鼎龙股份和上海新阳都在高端材料方向上，但预期差类型完全不同：`鼎龙股份` 更像 `订单事实链成立位`，因为光刻胶和 CMP 抛光液已经形成 `量产线 -> 头部客户 -> 批量订单 -> 交付爬坡`；`上海新阳` 更像 `平台储备增强位`，因为集成电路材料收入增长、平台扩线和工艺配套边界扩展都在写实，但仍缺 `头部客户 / 批量订单 / 单品种放量` 的硬催化。
> 如果只从“谁更接近继续被市场上修”看，当前明显更偏 `鼎龙股份`；如果从“谁的方向没错、但市场容易过早拔高定价”看，最该谨慎区分的是 `上海新阳`。

进一步压缩成交易语言：

1. `更接近订单兑现`：鼎龙股份
2. `更接近平台推进`：上海新阳
3. `更适合按收入与利润再定价`：鼎龙股份
4. `更适合等客户与订单催化后再升格`：上海新阳

## 二、先定义这两家公司到底在比什么

这张对照卡真正要比较的，不是“谁都做高端材料”，而是三件事：

1. `兑现路径不同`
   - 鼎龙股份：订单、交付、量产线已写实。
   - 上海新阳：平台扩线、收入增长、项目推进已写实。
2. `预期差类型不同`
   - 鼎龙股份：`订单事实链成立型`
   - 上海新阳：`平台储备增强型`
3. `证伪位置不同`
   - 鼎龙股份：更看订单能否继续传导到收入、毛利率和利润。
   - 上海新阳：更看平台推进能否尽快出现客户、订单或单品种突破。

## 四、为什么鼎龙股份更像“订单事实链成立位”

### 1. 事实

1. `2026-03-20`，年产 `300吨` 的 KrF/ArF 高端晶圆光刻胶产业化项目建成投产。
2. `2026-05-26`，CMP 抛光液获得新客户批量订单，SiC 衬底抛光液落地批量订单。
3. `2026-06-12`，浸没式 ArF / KrF 光刻胶向两家头部晶圆厂交付，并新增近 `1,000` 加仑订单。
4. 截至公告日，已有 `8款` 高端晶圆光刻胶取得多家国内主流晶圆厂批量订单。

### 2. 判断

鼎龙股份更靠前，不是因为“题材更热”，而是因为：

1. 它已经把最关键的东西写成正式订单链；
2. 不只是单一光刻胶，而是 `光刻胶 + CMP 抛光液` 双线推进；
3. 因为订单链条够实，所以后续中报、三季报只要继续兑现，市场更容易直接再定价。

### 3. 它最值得盯的下一条数据

1. 光刻胶与 CMP 材料 `2026H2` 收入兑现；
2. 交付量和毛利率是否继续改善；
3. 头部客户是否继续扩品类、扩份额。

### 4. 它最容易怎么被证伪

1. 订单只停留在公告层，后续收入和利润跟不上；
2. 光刻胶交付节奏低于前期口径；
3. 材料平台内部协同放量不明显。

## 五、为什么上海新阳更像“平台储备增强位”

### 1. 事实

1. `2026Q1` 营业收入 `5.77亿元`，同比增长 `33.05%`，归母净利润同比增长 `102.59%`。
2. 一季报明确，增长主要系 `集成电路材料销售收入增加`。
3. `2026Q1` 在建工程较年初增长 `50.08%`。
4. `2026-06-23` 新阳广东工商变更，经营范围包括 `半导体电镀设备、清洗设备及零配件`。
5. `2026-05-15` 业绩说明会明确，半导体材料受 AI 算力和数据中心建设拉动，相关材料可用于先进封装制程工艺。

### 2. 判断

上海新阳更值得看的地方，不在“会不会立刻被按订单兑现股定价”，而在：

1. 它已经从纯逻辑位变成了 `收入增长 + 平台扩线 + 项目扩边` 的写实样本；
2. 它的平台面比很多单点材料股更宽，后续如果出现单品种突破，弹性会被重新认识；
3. 但现在最大的问题，是这些进展还没有压成 `头部客户 / 批量订单 / 单品种放量`。

### 3. 它最值得盯的下一条数据

1. 湿法蚀刻液、清洗液、电镀液等产品收入占比；
2. 是否出现主流客户验证或批量订单；
3. 平台扩边是否转成更明确的收入和毛利率改善。

### 4. 它最容易怎么被证伪

1. 平台推进继续增强，但长期没有产品级突破；
2. 在建工程和扩边带来资本消耗，却不能转成更实的经营改善；
3. 市场过早按“高端光刻胶订单兑现位”定价，但后续没有硬催化跟上。

原因：

1. 它已经站在订单兑现链上；
2. 后续公告更容易继续强化 `交付量 -> 收入 -> 利润`；

## 2. 如果更看“谁的方向没错，但市场容易过早拔高”

原因：

1. 平台储备和收入增长都是真的；

3. 最容易发生的偏差，是市场把“平台推进位”提前当成“订单兑现位”。

## 九、可直接复用结论

> 高端材料方向里，真正值得拆开的不是“谁都受益于国产替代”，而是兑现路径完全不同。鼎龙股份更像 `订单事实链成立位`，因为光刻胶和 CMP 抛光液已经形成量产线、头部客户、批量订单和交付爬坡；上海新阳更像 `平台储备增强位`，因为集成电路材料收入增长、平台扩线和半导体设备经营范围扩边都在写实，但仍缺头部客户、批量订单和单品种放量这类硬催化。前者更适合按订单、交付、收入和利润再定价，后者更适合等平台推进进一步压缩到客户验证和单品种突破之后再升格。也就是说，高端材料这条线当前不是同质化的“材料股篮子”，而是至少分成“订单事实链成立”和“平台储备增强但仍待催化”两种不同的预期差路径。
