# 半导体上游下一批标的信息组清单｜第一版

> 所属模块：02_桥接层/03_AI标杆股研究卡  
> 生成日期：2026-06-28  
> 研究定位：在 `半导体上游_中日卡脖子近1个月消息与预期差_第一版.md`、`半导体上游近1个月公告及消息面补充_第一版.md`、`中巨芯_上游平台补位观察卡_第一版.md` 与既有多张单卡、对照卡基础上，进一步回答：`如果继续往更上游探，不该再平铺名字，而该怎么按信息组继续推进`。  
> 使用边界：本文件不是重写全部单公司，而是把下一批更上游候选按 `组别 -> 变量 -> 传导 -> 验证 -> 映射` 压成执行清单。  
> 证据边界：只使用正式公告、年报、季报、IR、投资者关系活动记录与本仓库已核验文件；传闻不进入主结论。

## 一、一句话结论

> 如果继续往半导体更上游探，当前最好的推进方式不是再平铺一串“材料国产替代”名字，而是把下一批标的压成四个信息组：`更上游平台补位组`、`高端材料兑现组`、`300mm供给安全组`、`半导体设备全景组`。  
> 其中最值得继续深挖的，不是“所有组一起”，而是 `更上游平台补位组`，核心看 `中巨芯 / 中船特气`；第二顺位是 `高端材料兑现组`，核心看 `飞凯材料 / 上海新阳 / 鼎龙股份` 的兑现层次继续拉开；`300mm供给安全组` 与 `半导体设备全景组` 更适合当验证与补强组，而不是新的第一主线。

进一步压缩成执行语言：

1. `第一优先组`：更上游平台补位组
2. `第二优先组`：高端材料兑现组
3. `第三优先组`：300mm供给安全组
4. `第四优先组`：半导体设备全景组

## 二、为什么这一步不该继续写成“更多名字”

### 事实

1. 当前仓库里，半导体上游主锚已经比较清楚：
   - `长川科技`：利润兑现型硬主锚
   - `鼎龙股份`：订单兑现型硬主锚
2. 中继层与补位层也已经初步形成：
   - `有研硅 / 立昂微 / 沪硅产业`：300mm 硅片分化
   - `华海诚科 / 长电科技 / 通富微电`：先进封装材料与平台分化
   - `盛美上海 / 安集科技 / 中船特气 / 中巨芯`：设备、平台材料、电子特气与更上游平台补位
3. 继续往上游深挖时，仓库既有口径已经把候选收敛到：
   - `中巨芯`
   - `飞凯材料`
   - `上海新阳`
   - 已落卡的 `中船特气`

### 判断

1. 再扩一串名字，研究价值会快速下降；
2. 更合理的做法，是沿着上游链条的底层变量继续分组：
   - `供给约束 / 价格传导`
   - `平台型材料能力`
   - `先进封装景气向材料传导`
   - `300mm 本土供给安全`
   - `测试之外的设备订单强化`
3. 所以下一步最好的产物，不是“新十家公司名单”，而是“哪一组最该继续跟、为什么、下一条看什么”。

## 三、四个信息组总表

| 组别 | 核心样本 | 当前定位 | 当前最该盯的变量 | 当前优先级 |
|---|---|---|---|---|
| 更上游平台补位组 | `中巨芯`、`中船特气` | 更靠上游、最接近新增信息差 | `电子特气供需/调价/订单`、`湿电子化学品/前驱体单线突破` | `第一优先` |

## 四、第一优先组：更上游平台补位组

### 1. 核心样本

1. `中船特气`
2. `中巨芯`

### 2. 为什么这一组最值得继续往上游挖

1. 它们比先进封装平台、300mm 硅片更靠上游；
2. 它们当前的新增信息，不再只是“方向没错”，而是开始出现：
   - `供需偏紧`
   - `价格传导`
   - `增量订单`
   - `收入扩张与利润修复`
3. 更重要的是，它们分别代表两种不同的更上游逻辑：
   - `中船特气`：单品种强催化
   - `中巨芯`：平台型多业务补位

### 3. 组内分工

1. `中船特气`
   - 更像 `单品种供需与调价驱动位`
   - 核心盯：`六氟化钨供需、调价机制、增量订单`
2. `中巨芯`
   - 更像 `三平台并进的上游平台补位位`
   - 核心盯：`电子湿化学品 / 电子特气 / 前驱体` 哪条业务先形成单线突破

### 4. 为什么市场还没完全交易完

1. `中船特气` 虽已出现正式变量，但市场仍未完全按“价格传导能否继续写进毛利与利润表”定价；
2. `中巨芯` 虽已出现收入扩张与利润转正，但市场仍在等待哪条业务先从平台储备走向真正批量突破；
3. 这意味着这一组最有可能继续提供“新增可验证信息差”，而不是只有老逻辑重复。

### 5. 这一组最容易被什么数据证伪

1. `中船特气`
   - 订单只停留在 IR 层，后续收入与毛利没有继续体现；
   - 调价机制弱化，供需紧平衡失效。
2. `中巨芯`
   - 三平台都停留在“客户导入 / 试用”层，始终没有哪条业务形成更硬突破；
   - 收入增长无法继续传导到更高质量利润与现金流。

## 五、第二优先组：高端材料兑现组

### 1. 核心样本

1. `鼎龙股份`
2. `上海新阳`
3. `飞凯材料`

### 2. 这一组为什么值得继续跟

1. 高端材料方向已经不是单线逻辑，而是明确分层：
   - `鼎龙股份`：订单事实链成立位
   - `上海新阳`：平台储备增强位
   - `飞凯材料`：先进封装材料配套补位
2. 这一组适合继续跟，不是因为都属于“国产替代”，而是因为：
   - 市场已经在交易它；
   - 但还没完全交易完不同层次的兑现速度。

### 3. 组内分工

1. `鼎龙股份`
   - 核心盯：光刻胶和 CMP 材料订单是否继续向收入、毛利率和利润传导
2. `上海新阳`
   - 核心盯：是否出现更硬的客户、订单或单品种突破
3. `飞凯材料`
   - 核心盯：功能型湿电子化学品景气延续，临时键合材料是否出现验证或量产催化

### 4. 为什么它不是第一优先组

1. `鼎龙股份` 已经较靠近兑现层，新增信息差更多是继续验证，而不是更上游的新平台挖掘；
2. `上海新阳` 与 `飞凯材料` 目前仍缺更硬的产品级催化；
3. 所以这一组更适合作为“第二优先继续压缩分歧的组”，而不是继续往上游延伸的第一入口。

---

## 六、第三优先组：300mm供给安全组

### 1. 核心样本

1. `有研硅`
2. `立昂微`
3. `沪硅产业`

### 2. 这一组的角色

1. 它们更像 `供给安全与验证升级组`；
2. 核心不是谁讲了新故事，而是谁把：
   - `12英寸出货`
   - `客户认证`
   - `收入结构`
   - `毛利率`
   - `现金流`
   继续写实。

### 3. 组内分工

1. `立昂微`
   - 更像 `直表兑现位`
2. `有研硅`
   - 更像 `平台验证升级位`
3. `沪硅产业`
   - 更像 `逻辑成立但利润传导偏慢位`

### 4. 为什么它不是更上游第一主线

1. 它的研究价值更偏“供给安全验证”，而不是更上游新增认知差；
2. 当前这条线更适合围绕中报和三季报滚动审计，不适合再扩更多名字。

---

## 七、第四优先组：半导体设备全景组

### 1. 核心样本

1. `长川科技`
2. `盛美上海`

### 2. 这一组的角色

1. `长川科技` 已经是测试设备利润兑现主锚；
2. `盛美上海` 则补上了测试之外的湿法清洗设备订单强化位；
3. 两者共同说明：设备方向当前更适合作为“兑现补强组”，而不是“继续往材料更上游延伸”的主入口。

### 3. 为什么优先级放后

1. 设备侧当前新增信息虽然硬，但更接近兑现审计，而不是更上游补位扩展；
2. 如果研究目标是“继续往上游探”，这一组更适合作为辅助验证，而不是第一落点。

---

## 八、当前最适合写进上层结论的话

> 继续往半导体更上游探，当前不该再写成“更多公司名单”，而应压成信息组来推进。最值得继续深挖的是 `更上游平台补位组`，因为 `中船特气` 已把单品种供需、调价和增量订单写成正式变量，`中巨芯` 已把 `电子湿化学品 + 电子特气 + 前驱体` 三平台写成更靠上游的平台补位核心；次优是 `高端材料兑现组`，其中 `鼎龙股份` 已进入订单事实链成立位，`上海新阳` 与 `飞凯材料` 仍处等待更硬产品级催化阶段；`300mm供给安全组` 与 `半导体设备全景组` 更适合围绕中报、三季报和订单兑现继续滚动验证，而不是作为下一轮更上游扩样的第一主线。

## 九、下一步最自然的执行顺序

1. `第一步`
   - 继续围绕 `中巨芯` 做业务线分拆观察：
   - 哪条业务先形成单线突破
2. `第二步`
   - 继续盯 `飞凯材料` 的临时键合材料与先进封装材料景气传导
3. `第三步`
   - 继续盯 `上海新阳` 是否出现客户、订单或单品种硬催化
4. `第四步`
   - 围绕 `300mm供给安全组` 与 `设备补强组` 做中报前滚动验证表，而不是扩更多新名字
