# 数据中心运营_算力租赁组多标的比较小结
> 所属模块：02_桥接层/03_AI标杆股研究卡 -> 数据中心运营_算力租赁组
> 比较对象：润泽科技 / 数据港 / 奥飞数据 / 光环新网 / 宝信软件 / 首都在线
> 生成日期：2026-06-29
> 研究定位：把资源运营层正式样本池压到一张多标的比较卡里，回答“谁是真平台、谁是过渡平台、谁是高波动转型平台”。

---

## 一、一句话结论

> 资源运营层在 A 股当前至少分化为四类：`润泽科技` 代表 AIDC 强释放的重资产平台，`数据港` 与 `奥飞数据` 代表稳态 IDC 现金流向智算弹性迁移的中间层，`光环新网` 与 `宝信软件` 代表云 / IDC 混合或园区型平台的质量分化，`首都在线` 则代表高波动智算转型样本；因此，这一层最重要的不是“谁有机柜”，而是“谁能把上架率、收入口径、OCF、折旧和融资结构放在同一张表里解释清楚”。

---

## 二、六样本定位

| 标的 | 当前定位 | 2025 经营特征 | 当前最硬的验证点 | 当前最敏感审计点 |
|---|---|---|---|---|
| 润泽科技 | AIDC 业绩释放位 | 收入、AIDC 占比与 OCF 强释放 | 交付规模、AIDC 占比、强 OCF | 长期应付款、折旧、资本循环 |
| 数据港 | 稳态 IDC 现金流 + 智算导入位 | 收入平稳、OCF 稳态 | 现金流、运营容量、智算导入 | 智算落地慢、折旧效率 |
| 奥飞数据 | 区域/扩张型 IDC 平台位 | 收入和 OCF 明显改善 | OCF 与利润同步改善 | 固定资产、借款、长期应付款 |
| 光环新网 | 云 / IDC 混合平台转型位 | IDC 增长、云承压、机柜大投放 | 机柜资源、IDC 收入、算力规模 | 上架率、租赁价格、回款 |
| 宝信软件 | 高质量园区/IDC 平台补位 | OCF 稳健、项目扩张有序 | 宝之云交付、强 OCF | 资源运营口径拆分不足 |
| 首都在线 | 高弹性租赁/调度叙事 + 高波动风险位 | 智算占比提升、亏损收窄 | 智算云占比抬升、结构优化 | OCF 转负、借款、融资租赁、诉讼 |

## 三、四类平台如何区分

### 1. AIDC 业绩释放平台

代表样本：

1. `润泽科技`

固定判断：

1. 关键看交付规模、AIDC 收入占比、OCF 与长期应付款。
2. 已经进入“收入兑现之后审计融资与折旧”的阶段。

### 2. 稳态 IDC 现金流 + 智算导入平台

代表样本：

1. `数据港`
2. `奥飞数据`

固定判断：

1. 先看现金流底盘，再看智算增量。
2. 不能把这类平台提前写成润泽式强释放。

### 3. 云 / IDC 混合或园区型平台

代表样本：

1. `光环新网`
2. `宝信软件`

固定判断：

1. 这类平台资源厚度很强，但最容易被利用率、回款、分部拆分和 CapEx 回报率审计。
2. 光环偏“资源投放后审计”，宝信偏“高质量园区补位”。

### 4. 高弹性租赁 / 调度叙事 + 高波动风险平台

代表样本：

1. `首都在线`

固定判断：

1. 这类平台不是不能做正式样本，但必须比其他样本更强调折旧、借款、融资租赁和 OCF。
2. 它补的是风险带，而不是主结论锚点。

---

## 四、统一变量下的组级判断

### 1. 最重要的领先变量

1. 上架率
2. 利用率
3. GPU / 算力租赁价格
4. AIDC / 智算收入占比

### 2. 最重要的同步变量

1. 收入增速
2. OCF
3. 固定资产与在建工程变化
4. 长期应付款 / 借款变化

### 3. 最重要的滞后审计项

1. 折旧吞噬利润
2. 融资扩张快于经营兑现
3. 资产减值或商誉减值
4. 诉讼、冻结与项目收益不达预期

---

## 五、当前最适合写进上层结论的话

1. 资源运营层已经不是单一“算力租赁故事”，而是至少四类平台分化。
2. `润泽科技` 代表强释放样本，`数据港` 与 `奥飞数据` 代表中间层平台，`光环新网` 与 `宝信软件` 代表资源与质量平台分化，`首都在线` 代表高波动转型平台。
3. 真正决定这一层估值持续性的，不是 headline 和机柜规划，而是 `上架率 / 利用率 / 收入口径 / OCF / 折旧 / 融资结构`。
4. 这一层是 AI 当前最接近“资本开支先行、资本回报率后审计”的风险带。

## 六、截至 2026-07-02 的中报前第一轮核验

### 1. 当前核验结论

截至 `2026-07-02`，本组已完成中报前第一轮正式核验。

当前更稳妥的写法是：

1. 本组暂未补到 6 个正式样本明确的 `2026H1` 业绩预告；
2. 因此当前报表硬口径仍以 `2025年报 + 2026Q1` 为主；
3. 本轮新增正式披露主要强化了 `光环新网`、`奥飞数据` 与 `宝信软件` 的已有判断，没有推动组内分层重排。

### 2. 本轮新增正式披露的组级含义

1. `光环新网`
   - 新增 IR 口径进一步确认了 `上架率约60%`、已签约及达成意向机柜比例约 `90%`、算力业务规模超过 `4,000P`。
   - 这强化的不是“高质量释放”，而是“资源很多但仍需审计上架率、租赁价格与回款”的判断。
2. `奥飞数据`
   - 新增 IR 口径确认 `2026年` 仍有多地交付计划，同时持有型 ABS 已获受理。
   - 这强化的是“交付继续推进 + 资本循环工具待落地”，而不是已经形成新一轮财务兑现。
3. `宝信软件`
   - 新增行动方案口径确认继续推进罗泾、华北基地智算业务与标准化智算基础设施云服务。
   - 这更适合作为战略补充，暂不改写其“高质量园区补位样本”的保守定位。

## 七、待补点

1. 多数样本连续上架率口径仍不足。
2. 多数样本连续利用率口径仍不足。
3. GPU / 算力租赁价格仍缺统一可比月度口径。
4. `宝信软件` 的资源运营分部拆分仍有限。
