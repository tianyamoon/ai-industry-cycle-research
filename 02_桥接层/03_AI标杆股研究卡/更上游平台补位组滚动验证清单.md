## 二、分标的执行要点

### 中船特气

1. 优先盯正式公告、业绩说明会、IR 记录中的六氟化钨订单与价格口径。
2. 看调价是否只是短期事件，还是开始传导到季度毛利率与利润表。
3. 若出现新增客户扩量或其他电子特气品类协同放量，视为额外加分。

### 中巨芯

1. 优先盯三大业务板块中是否出现更清晰的单线放量口径。
2. 看利润修复是否伴随经营现金流改善，而不是只靠收入扩张。
3. 看原材料波动是否继续拖累 OCF 或毛利率。

### 飞凯材料

1. 优先盯临时键合材料是否出现客户验证、试产升级、量产或订单级表述。
2. 看先进封装景气是否继续拉动功能湿电子化学品收入与利润。
3. 看新基地投产前后是否出现更硬产品兑现锚点。

### 上海新阳

1. 优先盯头部客户、批量订单与单品种放量口径。
2. 看集成电路材料收入增长是否继续转化为更高质量利润。
3. 看在建工程、CapEx 与经营改善是否匹配，避免只扩不收。

### 电子特气与电子材料补位

1. `华特气体 / 金宏气体`：看高端特气客户认证、电子气体收入拆分、项目利用率、毛利率和 OCF。
2. `广钢气体 / 和远气体`：看现场制气或电子特气产业园投产、客户验证、利用率、回款和项目延期风险。
3. `南大光电`：拆分前驱体、电子特气和光刻胶，只有相关单品的客户与收入连续推进，才可升级本组定位。

### 深冷、气体运营与回收

1. `中泰股份 / 杭氧股份`：区分深冷设备一次性订单与气体运营持续收入，重点看产能利用率、合同、回款和 OCF。
2. `凯美特气`：核对尾气回收、特种气体和电子化学品的客户及收入，若无法拆出半导体业务则保持边界位。

### 高纯工艺系统与洁净厂务

1. `正帆科技 / 至纯科技`：看高纯气体系统、制程设备和厂务订单是否转化为收入、毛利、应收和 OCF。
2. `柏诚股份`：看洁净室、工艺管道、二次配管和大宗化学品供应系统的项目验收与回款。
3. 这三类样本只承担“需求和制程配套旁证”，不把工程订单直接等同于电子特气价格或公司利润增长。

### 观察位

1. `雅克科技`：补前驱体产品与客户回源，避免与中巨芯、南大光电重复计数。
2. `三孚股份`：补特种电子气体收入、客户与 OCF；未拆分前只保留 L0-L1 观察。
3. `昊华科技`：补具体电子化学品 / 氟化工产品与半导体客户，不能用集团业务概况替代公司级验证。

## 三、组级观察原则

### 事实

1. 本组 15 个跟踪样本和 3 个待核观察位对应多种角色，不宜统一按“材料景气股”打分。
2. 扩容解决的是产业链覆盖，不代表所有候选都进入主线；当前最关键的是看分层样本能否出现各自利润池的硬验证。

### 判断

1. `中船特气` 更适合按 `供需 / 调价 / 单品种订单` 监控。
2. `中巨芯` 更适合按 `平台转单线突破` 监控。
3. `飞凯材料` 和 `上海新阳` 更适合按 `客户验证 / 单品种放量 / 订单成立` 监控。
4. `中泰股份 / 杭氧股份 / 凯美特气` 更适合按 `气体运营 / 深冷项目 / 回收提纯 / 电子化学品拆分` 监控。
5. `正帆科技 / 至纯科技 / 柏诚股份` 更适合按 `订单 / 交付 / 验收 / 回款 / OCF` 监控；观察位只在出现产品级证据后升级。

## 四、当前组级动作建议

1. `第一优先`：滚动跟踪 `中船特气 / 中巨芯` 的价格、订单、单线突破和季度利润传导。
2. `第二优先`：补证 `华特气体 / 金宏气体 / 广钢气体 / 和远气体 / 南大光电` 的客户、项目、收入和 OCF。
3. `第三优先`：继续跟踪 `飞凯材料 / 上海新阳` 的材料产品级催化，并核对 `中泰股份 / 杭氧股份` 的气体运营与深冷项目。
4. `第四优先`：把 `凯美特气 / 正帆科技 / 至纯科技 / 柏诚股份` 作为边界和旁证队列，重点看收入、回款和现金流。
5. `观察队列`：`雅克科技 / 三孚股份 / 昊华科技` 只有在产品、客户和收入拆分成立后才升级。
