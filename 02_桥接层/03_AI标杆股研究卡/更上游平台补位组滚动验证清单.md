# 上游关键卡点与核心弹性组滚动验证清单

> 文档类型：滚动验证清单
> 研究对象：AI 硬件制造上游的关键原材料、核心零部件与制程系统
> 所属模块：02_桥接层/03_AI标杆股研究卡
> 所属层级：桥接层
> 第一变量：AI 硬件扩产能否由 `客户认证 -> 重复消耗 / 关键部件交付 -> 收入、毛利率、经营活动现金流` 形成连续经营闭环。
> 使用边界：本清单按 `卡点强度 × 经营弹性 × 证据成熟度` 排序，不按气体概念、政策题材或公司数量排序。
> 更新日期：2026-07-12
> 阅读提示：经营活动现金流指公司主营业务实际收回的现金；平均售价指单位产品的平均卖价；资本开支指扩产和购置设备等投入。

## 二、第一优先：验证“卡点 + 弹性”是否真正重合

### 安集科技 / 鼎龙股份

1. 拆化学机械抛光材料、功能湿电子化学品、抛光垫及相关工艺材料的客户、单品收入和毛利率。
2. 看认证、重复订单、出货和产品结构改善是否同步反映到经营活动现金流。
3. 审计存货、应收、在建工程和资本开支，避免把多产品扩张直接当成已兑现的利润弹性。

### 中船特气 / 中巨芯

1. `中船特气`：回到具体电子特气的价格、订单、收入、毛利率、利润与经营活动现金流连续传导。
2. `中巨芯`：在湿电子化学品、电子特气、前驱体中找出先形成客户扩量、收入与经营活动现金流单线突破的业务。
3. 不用氦气出口管理、行业热度或泛特气概念代替产品级经营验证。

### 富创精密 / 新莱应材

1. `富创精密`：跟踪工艺 / 结构零部件和高纯 / 真空气体传输系统的设备厂客户、订单、交付、毛利率和经营活动现金流。
2. `新莱应材`：跟踪超高纯 / 高纯气体传输、阀门和管件在半导体客户中的认证、收入、订单、毛利率和经营活动现金流，拆开其他行业业务。
3. 两者都审计客户集中度、应收、存货和资本开支，避免“设备国产化”概念超过实际交付。

## 三、第二优先：补齐原材料卡点，不提前给盈利权重

### 硅片、高纯靶材与高纯石英

1. `沪硅产业 / 有研硅 / 立昂微 / 神工股份`：看先进规格认证、出货、平均售价、产能利用率、毛利率、经营活动现金流与资本开支回报。
2. `江丰电子 / 欧莱新材 / 有研新材`：看靶材 / 高纯金属的产品级客户、认证、收入、毛利率和现金流，区分半导体与其他应用。
3. `菲利华 / 石英股份`：先确认高纯石英在半导体制造和设备端的产品、客户、收入及产能利用率，再讨论其是否构成制造卡点。

### 湿电子化学品、前驱体与光刻配套材料

1. `南大光电 / 雅克科技`：拆前驱体、电子特气和其他材料业务，优先看先进制程客户导入、单品收入和毛利率。
2. `上海新阳 / 晶瑞电材 / 格林达 / 江化微 / 彤程新材`：固定回源超净高纯试剂、湿电子化学品、光刻及配套材料的产品认证、客户、收入和经营活动现金流。
3. 仅有产品布局、送样或“国产替代”叙事时，保持弹性发现池，不能升入核心交叉验证池。

---

## 四、第三优先：补齐设备关键部件与系统传导

### 精密零部件 / 真空 / 陶瓷与石英件

1. `华亚智能`：核实精密结构件在半导体设备客户中的产品、收入和交付节奏。
2. `汉钟精机`：核实真空产品在半导体制造场景的客户、收入、订单和毛利率，区分其他下游。
3. `珂玛科技 / 富乐德`：核实先进陶瓷 / 石英件、设备零部件及服务的制程位置、客户、收入与经营活动现金流；无产品级证据前维持待核。

### 高纯系统、厂务与附属装备

1. `盛剑环境 / 正帆科技 / 至纯科技 / 京仪装备`：拆高纯工艺系统、废气 / 真空 / 流体子系统、附属装备和厂务项目的订单、交付、验收、回款与经营活动现金流。
2. 系统订单只能确认资本开支向配套环节外溢；不能替代耗材 / 零部件的重复消耗和经营弹性判断。

## 五、最后处理资源供给与跨组样本

1. `华特气体 / 金宏气体 / 广钢气体 / 和远气体 / 中泰股份 / 杭氧股份 / 凯美特气`：先补电子级产品、客户、利用率和收入拆分；没有这些事实时，只承担资源供给覆盖。
2. `华海诚科 / 艾森股份 / 兴森科技 / 深南电路` 等先进封装上游样本继续在 `先进封装组` 维护；`源杰科技 / 光库科技` 等高速互联上游继续在 `光模块组` 维护；温控和液冷材料 / 部件继续在原组维护。
3. 本组只同步它们的卡点筛选结果，不重复给出经营排序，避免一个公司被多组重复抬升。

## 六、季度维护顺序

1. 先回填核心交叉验证池的产品、客户、订单 / 出货、收入、毛利率、经营活动现金流、应收、存货和资本开支。
2. 再补制造基础材料与工艺耗材候选的产品级认证和收入纯度，决定是否进入核心交叉验证池。
3. 然后补精密零部件和系统传导池，确认设备资本开支是否真的向部件和厂务传导。
4. 最后处理资源供给、政策与跨组旁证；它们只能修正供给或景气判断，不抢核心卡点结论。

## 七、外部消息使用边界

1. 国内算力卡采购额等机构预测，只能作为需求假设；必须回到出货、平均售价、云 / 晶圆厂资本开支和公司订单验证。
2. 氦气出口管理是出口状态变化，不是国内电子级氦气供给、价格或企业盈利的结论。
3. 任何单点政策、项目、研报或股价异动，都不能跳过 `卡点 -> 产品 / 客户 -> 收入 / 毛利率 -> 经营活动现金流` 的验证链。
