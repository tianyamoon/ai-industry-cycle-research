## 一、一句话结论

> 盛美上海当前最大的预期差，已经不只是 `2026Q1` 新签订单同比 `+65%`，而是 `2026-06-16` 投资者关系活动记录进一步把 `2026年收入指引维持82-88亿元`、`年内电镀设备新增订单占比约30%`、`高温单片SPM已获批量订单`、`面板级负压清洗设备已量产` 这些执行层细节正式钉住。
> 这说明它正在从“单一清洗设备订单强化位”升级成“清洗 + 电镀 + 面板级先进封装平台强化位”；但由于 `2026Q1` 利润与 OCF 仍偏弱、当前位置和估值也已不低，它依然更适合定义为 `订单强化补强位`，而不是替代长川科技的 `利润兑现主锚`。

## 三、事实

### 1. 2025 年报先验证了平台能力，但现金兑现已经出现约束

1. 公司 `2025` 年营业收入 `67.86亿元`，同比增长 `44.48%`；
2. `2025` 年归母净利润 `15.31亿元`，同比增长 `26.65%`；
3. `2025` 年主营业务毛利率 `47.48%`；
4. `2025` 年经营活动现金流净额 `2.39亿元`，同比下降 `80.36%`；
5. 这说明盛美上海在 `2025` 年已经不是“只有平台想象”的公司，而是收入、利润都已经被报表验证；但交付增长并没有同步变成强现金兑现，因此资本周期与兑现质量必须一起审。

### 2. 2026Q1 利润与现金流承压，说明当前仍不是利润主锚

1. 公司 `2026Q1` 营业收入 `14.76亿元`，同比增长 `13.06%`；
2. `2026Q1` 归母净利润 `1.04亿元`，同比下降 `57.66%`；利润总额同比下降 `50.51%`；
3. `2026Q1` 经营活动现金流净额 `-1.44亿元`，上年同期为 `+5.33亿元`；
4. 公司在 `2026Q1` 报告中解释，利润与现金流压力主要来自原材料备货增加、研发投入上升和汇率波动损失增加；
5. 这说明盛美上海当前最值得盯的，仍不是静态利润，而是订单与新品扩线能否继续向报表传导。

### 3. 5 月投关先把订单斜率和交付窗口钉住了

1. 公司在 `2026-05-14` 业绩说明会披露，`2026年第一季度新签订单同比增速为65%`；
2. 公司同时披露，订单平均交付周期约 `6个月`；
3. 公司表示将在 `2026年9月底` 自愿披露在手订单情况；
4. 公司预计 `2026年` 中低温及高温硫酸设备将交付 `20余台` 至多个客户。

### 4. 6 月投关把“清洗订单强化”扩展成“平台产品线强化”

1. 公司在 `2026-06-16` 投资者关系活动记录中维持 `2026` 年收入指引 `82-88亿元`；
2. 同一份记录披露，`2026` 年截至当时的新增订单中，电镀设备占比约 `30%`；
3. 公司再次确认，中低温与高温硫酸设备 `2026` 年预计交付 `20余台`，且国内外客户都有需求；
4. 公司披露，高温单片 `SPM` 设备在 `15纳米` 逻辑客户的颗粒指标达到要求后，已经拿到批量订单；
5. 公司披露，改善后的 mini-line 设备通过测试并完成首台 `PECVD SiCN` 设备发货；
6. 公司披露，负压清洗设备已在面板级先进封装方向进入量产；
7. 公司披露，`KrF` 涂胶显影设备已经向一家逻辑客户发货，预计 `2026年7月` 对接生产线，`2026年底` 看全流程验证。

### 5. 截至 2026-07-07，它已经是高位强趋势资产，不是低位冷门股

1. `Luna Stock MCP` 显示，盛美上海在 `2026-07-07` 收盘 `396.50元`，当日 `+6.30%`，开盘 `365.00元`，最高 `400.00元`，最低 `361.00元`，成交量 `820.39万股`；
2. 周线看，`2026-06-26` 当周收于 `414.00元`，`2026-07-03` 当周回撤到 `363.43元`，`2026-07-07` 这一周前两个交易日反弹到 `396.50元`，说明它处在主升后高位震荡而非底部启动；
3. 月线看，`2026-05-29` 月收 `229.04元`，`2026-06-30` 月收 `432.00元`，月度趋势仍明显向上，只是波动已经很大；
4. `validate_cross_source(stock_detail)` 对 `THSDK` 与 `Tencent` 的交叉结果显示，`价格 / 开盘 / 最高 / 最低 / 成交量` 一致，`资金流` 字段冲突，因此本卡只把价格字段当正式技术证据；
5. 本仓库 `2026-07-03` 的历史估值分位表显示，盛美上海 `PE(TTM)` 为 `139.88x`、分位 `84.20`，`3年 / 5年 / 10年` 均在 `>70` 区间；`2026-07-07` 的 MCP 快照口径为 `PE 152.61x / PB 13.96x`，说明它当前并不便宜；
6. 本仓库 `2026Q1` 公募代理口径显示，盛美上海对应 `14` 家基金、持仓市值约 `23.5亿元`、持仓占比 `3.50%`，较上期减仓 `44.85%`，拥挤弱于长川科技，但已不是冷门低位资产。

## 四、判断

### 1. 当前阶段判断

盛美上海当前更接近：

`2025年报先验证平台能力 -> 2026Q1 利润与 OCF 承压 -> 2026-06-16 投关把订单结构与新品扩线继续钉实 -> 等待中报验证订单能否继续写进收入、利润和现金流`

它已经不只是：

1. 泛半导体设备景气的情绪映射位；
2. 没有明确订单斜率的工艺主题样本；
3. 只讲清洗主线、不讲电镀和面板级先进封装平台的单线设备股。

### 2. 为什么它值得升格

1. 上游设备方向里，市场当前最熟的是测试 / 量检测设备，盛美上海补足了“湿法清洗 + 电镀 + 面板级先进封装”这条增量线；
2. 最近两轮正式披露里，`订单同比+65%`、`2026收入指引82-88亿元`、`电镀设备订单占比约30%`、`SPM批量订单` 都是很硬的执行层强化信号；
3. 它的强点已经不只是单一设备验证，而是同一家公司内部多条产品线同时推进，便于做平台型兑现跟踪；
4. `2026年9月底` 在手订单披露、`2026H1` 中报、`KrF` 全流程验证与面板级设备量产，都给了后续滚动跟踪抓手。

### 3. 为什么它仍不是主锚

1. 它目前最硬的新增事实，仍主要停留在 `订单 / 指引 / 新品验证 / 交付节点`，而不是 `中报利润预告`；
2. `2026Q1` 归母净利润和 OCF 都明显转弱，说明短期报表质量还不能简单按高景气重估；
3. 当前估值与位置都已不低，市场已经提前交易了相当一部分景气预期；
4. 因此它更像“高位但仍可跟踪的订单强化补强位”，而不是这一轮上游最核心的高确定性利润主锚。

### 4. 第一性原理下，盛美上海真正该盯的不是题材名字，而是变量传导

1. 公司真正的强点，是 `湿法清洗 -> 电镀 -> 面板级先进封装设备` 这条工艺平台的扩线，而不是“玻璃基板直系受益”这种泛题材标签；
2. 变量上，最重要的是 `新签订单结构`、`先进节点客户验证`、`交付节奏` 与 `收入 / OCF` 的传导，而不是概念热度；
3. 因此若后续只看到 `TGV / TSV / 玻璃基板` 的社区扩散，没有看到正式订单、收入或现金流跟进，就不应把它抬升成新的设备主线。

## 六、继续强化触发器

1. `2026年9月底` 在手订单披露继续强化；
2. `2026H1` 中报里收入、归母净利润、毛利率和 OCF 明显好于 `2026Q1`；
3. 电镀设备订单、`SPM` 批量订单、`KrF` 涂胶显影验证、面板级负压清洗量产继续被正式披露验证；
4. 全年收入指引 `82-88亿元` 被维持甚至上修，而不是在中报后下修；
5. 高位震荡后，价格仍能维持周线强趋势，而不是景气叙事一弱就高位塌陷。

---

## 七、证伪信号

1. 订单高增未能传导到后续收入确认；
2. 毛利率、归母净利润和 OCF 持续弱于订单表现，说明质量不高；
3. 电镀设备、`SPM`、`KrF` 或面板级负压清洗的验证长期停留在投关表述，迟迟不能形成收入与交付证据；
4. 硫酸设备交付台数或节奏明显低于前期口径；
5. `2026年9月底` 在手订单披露不及预期，或中报后全年收入指引被下修；
6. 高估值状态下，如果经营兑现继续落后，市场可能把它从“平台强化位”重新压回“高位题材股”。

---

## 八、待补点

1. `2026年9月底` 在手订单披露；
2. 中报与三季报的收入、毛利率、归母净利润、经营现金流变化；
3. 电镀设备、`SPM`、`KrF` 涂胶显影和面板级负压清洗设备的客户结构与量产节奏；
4. `2026` 年收入指引 `82-88亿元` 的达成路径；
5. 高位估值下，是否出现明显的利润兑现与现金流修复，而不是只剩新品叙事；
6. 公募、两融与大宗交易是否进一步说明它是高位再平衡，还是还能继续吸引增量资金。
