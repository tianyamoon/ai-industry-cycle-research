# 上海新阳｜高端光刻材料预期差卡｜第一版

> 标的：上海新阳 / 300236.SZ  
> 所属模块：02_桥接层/03_AI标杆股研究卡  
> 生成日期：2026-06-28  
> 所属方向：高端光刻材料 / 湿电子化学品 / 电镀液与清洗液平台  
> 研究定位：作为 `鼎龙股份_高端光刻材料CMP预期差卡_第一版.md` 的补位样本，补上 A 股高端材料方向里“平台储备增强、验证推进中、但还缺订单级硬催化”的观察位。  
> 使用边界：本卡重点审计的是 `集成电路材料收入增长 -> 平台扩线推进 -> 是否出现客户/订单/单品种突破` 这条链条，不把平台储备直接写成已兑现的高确定性预期差。

---

## 一、一句话结论

> 上海新阳当前的预期差，不在“也有高端光刻材料概念”，而在它已经把 `集成电路材料收入增长、湿法蚀刻/清洗/电镀平台扩线、子公司半导体设备布局` 逐步写实，但市场如果把它直接等同于鼎龙股份那样的订单兑现位，就会高估其当前兑现强度。更准确的定位是：方向正确、平台在变厚、验证在推进，但还缺 `头部客户 / 批量订单 / 单品种放量` 这一步。

---

## 二、研究边界

1. 研究对象：上海新阳在集成电路材料方向的收入增长、平台扩线与潜在预期差。
2. 不研究内容：把公司所有化工业务混写成同一斜率，不把平台储备直接等同于订单兑现。
3. 时间边界：以 `2025年报摘要 -> 2026Q1 -> 2026-05-15 业绩说明会 -> 2026-06-23 工商变更公告` 为主。
4. 证据边界：优先使用 `年报摘要、季报、业绩说明会、正式公告`；传闻与市场想象不进入主判断。

---

## 三、事实

### 1. 集成电路材料收入增长已经写进报表

1. 公司 `2026Q1` 实现营业收入 `5.77亿元`，同比增长 `33.05%`。
2. 公司 `2026Q1` 归母净利润同比增长 `102.59%`。
3. 一季报明确指出，增长主要系 `集成电路材料销售收入增加`。
4. 这说明上海新阳当前已不是单纯讲材料方向故事，而是相关收入已经开始贡献表内增长。

### 2. 平台储备不是口头逻辑

1. `2025年年报摘要` 显示，公司 `2025年` 实现营业收入 `19.37亿元`。
2. 公司在年报摘要中明确，围绕五大核心技术与产品持续突破，进一步提升了在集成电路材料领域的市场地位。
3. `2024年年报` 对产业位置的描述更具体：
   - 公司开发 `电子电镀、电子清洗、电子光刻、电子研磨、电子蚀刻` 五大系列产品；
   - 位于集成电路产业链上游环节；
   - 中国大陆晶圆厂扩产与湿电子化学品需求增长，有利于国产替代进程。

### 3. 扩产与项目推进在增强

1. 公司 `2026Q1` 在建工程较年初增长 `50.08%`。
2. `2026-06-23`，公司披露全资子公司新阳广东完成工商变更登记并换发营业执照。
3. 公告显示，新阳广东经营范围包括：
   - `半导体电镀设备`
   - `清洗设备及零配件`
   - 相关专用设备与配套业务
4. 这说明公司并非只在材料端加厚，也在围绕工艺配套能力扩边。

### 4. AI 与先进封装映射逻辑存在，但还偏平台层

1. `2026-05-15` 上海辖区集体业绩说明会披露，公司 `2026年` 将继续着重推进 `半导体材料、光纤涂覆材料以及液晶材料` 等板块。
2. 同次说明会明确，半导体材料受 `下游 AI 算力和数据中心建设拉动`，景气度较高。
3. 公司相关材料可用于 `先进封装制程工艺`。
4. 但目前正式披露层面，仍未出现像鼎龙股份那样明确的 `头部客户 / 批量订单 / 交付爬坡` 口径。

## 四、判断

### 1. 当前阶段判断

上海新阳当前更接近：

`集成电路材料收入增长 -> 平台储备增强 -> 等待产品级硬催化`

它已经不是：

1. 与 AI 上游无关的普通材料股；
2. 只有概念、没有表内收入支撑的早期题材位；
3. 已经进入高确定性订单兑现阶段的材料主锚。

### 2. 为什么它值得继续盯

1. 上海新阳是正确方向上的候选位，尤其适合补足“高端材料不只看单点订单，也要看平台储备”的视角。
2. 与鼎龙股份相比，它的优势不在订单硬度，而在平台面更宽：
   - 湿法蚀刻液
   - 清洗液
   - 电镀液
   - 电子光刻相关材料
   - 半导体配套设备边界延展
3. 如果后续出现单品种客户突破或批量订单，市场会从“方向没错但偏慢”重估成“平台推进被低估”。

### 3. 为什么它当前还不能与鼎龙同级

1. 鼎龙股份已经把 `量产线 -> 头部客户 -> 批量订单 -> 交付爬坡` 写成事实链。
2. 上海新阳当前更接近 `收入增长 + 平台推进 + 项目扩边`，缺少订单级硬催化。
3. 所以它更适合归为 `中确定性观察位`，而不是高确定性预期差主样本。

## 六、买点触发

1. 后续中报或三季报进一步披露 `集成电路材料收入结构` 明显提升。
2. 湿法蚀刻液、清洗液、电镀液等具体产品出现主流客户验证或批量订单。
3. 公司继续披露先进封装相关材料在客户侧的导入进展。
4. 在建工程与平台扩边最终能转成真实收入与毛利率改善，而不只是资本开支故事。

---

## 七、证伪信号

1. 集成电路材料收入增长放缓，平台推进没有继续转化为经营改善。
2. 在建工程和项目推进增强，但长期没有产品级突破。
3. 市场把它按“高端光刻胶订单兑现位”定价，但后续没有头部客户或订单级证据跟上。
4. 平台扩边带来资本消耗上升，但利润质量和现金流没有同步改善。

---

## 八、待补点

1. 湿法蚀刻液、清洗液、电镀液等细分材料的收入占比变化。
2. 是否出现更明确的主流客户验证、批量订单或单品种突破。
3. 在建工程与资本开支后续如何向收入与利润传导。
4. 先进封装制程工艺相关材料是否出现更直接的商业化证据。
