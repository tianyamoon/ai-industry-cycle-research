## 一、一句话结论

> 京仪装备当前更适合定义为 `半导体设备平台扩线观察位`，而不是 `利润兑现主锚` 或 `订单强化补强位`：`2025年报` 与 `2026Q1` 已验证其温控 / 工艺废气处理设备需求在提升，`2026Q1` 合同负债进一步升至 `14.88亿元`，同时产品已覆盖 `90nm-14nm` 逻辑与 `64-192层3D NAND` 等工艺；但库存、在建工程与高估值约束并存，下一步核心仍是看 `在手订单 -> 交付 -> 收入 / 利润 / OCF` 是否连续闭环。

## 三、事实

### 1. 2025 年报先把需求扩张、订单前置和资本开支压力一起写实

1. 公司 `2025` 年营业收入 `14.26亿元`，同比增长 `38.95%`。
2. `2025` 年归母净利润 `1.48亿元`，同比下降 `3.26%`；扣非归母净利润 `1.15亿元`，同比下降 `0.47%`。
3. 按 `2025` 年营业收入 `14.26亿元` 与营业成本 `9.61亿元` 口径估算，公司综合毛利率约 `32.64%`。
4. `2025` 年经营活动现金流净额 `3.81亿元`，显著高于上年。
5. `2025` 年末合同负债 `13.51亿元`，较上年末 `7.39亿元` 增长 `82.84%`，公司解释主要为销售业务增长带来的预收货款增加。
6. 年报同时说明，总资产较期初增长 `18.28%`，主要因销售需求增长导致 `存货大幅增加` 与 `在建工程投入增加`。

### 2. 2026Q1 收入和利润仍增，但营运项目继续抬升

1. `2026Q1` 营业收入 `3.92亿元`，同比增长 `16.13%`。
2. `2026Q1` 归母净利润 `0.46亿元`，同比增长 `29.45%`；扣非归母净利润 `0.43亿元`，同比增长 `34.73%`。
3. 按 `2026Q1` 营业收入 `3.92亿元` 与营业成本 `2.54亿元` 口径估算，综合毛利率约 `35.26%`。
4. `2026Q1` 经营活动现金流净额 `-0.37亿元`，虽较上年同期 `-1.84亿元` 明显改善，但仍未转正。
5. `2026Q1` 期末合同负债 `14.88亿元`，较 `2025` 年末 `13.51亿元` 继续抬升。
6. `2026Q1` 期末存货 `25.08亿元`，较 `2025` 年末 `23.95亿元` 上升；在建工程 `3.18亿元`，较 `2025` 年末 `2.87亿元` 上升。

### 3. 年报、招股书与业绩说明会已把工艺位置和客户层级写得比较实

1. 年报披露，公司半导体专用温控设备已批量应用于逻辑芯片 `90nm-14nm`、`64层-192层3D NAND` 存储芯片等工艺需求，并适配 `泛林半导体 / 东京电子 / 应用材料 / 中微公司 / 北方华创 / 屹唐股份` 等设备公司的主工艺设备。
2. 上市招股说明书披露，公司整机类产品已批量应用于 `中芯国际 / 华虹集团 / 长江存储 / 合肥长鑫 / 粤芯半导体 / 长江计算 / 睿力集成` 等国内主要晶圆厂客户。
3. `2026-06-09` 业绩说明会记录披露，公司已掌握 `制冷、加热、控温、工艺废气处理、真空传输` 等核心技术，产品覆盖温控设备、工艺废气处理设备、真空传输设备及关键零部件。
4. 同次说明会披露，公司产品应用已从刻蚀环节延伸至 `薄膜沉积 / 扩散 / 离子注入 / CMP / CVD / 清洗` 等环节。
5. 同次说明会披露，`安徽基地` 将在 `2026年下半年` 部分达到可使用状态。

### 4. 截至 2026-07-07，它更像高位高波动观察资产，而不是低位冷门样本

1. `Luna Stock MCP` 显示，京仪装备在 `2026-07-07` 收盘 `215.50元`，当日 `+6.27%`，换手率 `9.19%`，成交额约 `23.34亿元`。
2. 同一口径下，`2026-07-07` 对应 `PE 228.43x / PB 16.11x`，说明静态估值并不低。
3. `2026-07-06` 融资余额约 `5.77亿元`，较 `2026-07-02` 的 `7.01亿元` 有所回落，但两融参与度仍不低。
4. 因此当前更应把它放在“高波动下的兑现审计样本”里看，而不是把它理解成尚未被市场发现的低位配套资产。

## 四、判断

### 1. 当前阶段判断

京仪装备当前更接近：

`工艺配套层已被年报与 Q1 写实 -> 合同负债继续抬升 -> 业绩说明会把客户层级、工艺覆盖与安徽基地节点钉住 -> 等待中报 / 三季报验证订单能否继续穿透收入、利润与 OCF`

它已经不只是：

1. 泛半导体设备情绪映射位；
2. 只有单点客户验证的零部件样本；
3. 只能依赖“国产替代”泛叙事的低信息密度配套股。

### 2. 为什么它值得单独补一张卡

1. 半导体设备组里，市场更熟悉的是 `刻蚀 / 沉积 / 清洗 / 测试` 等主设备线，京仪装备补的是 `温控 / 工艺废气处理 / 真空传输` 这类关键子系统层。
2. 这类样本如果继续兑现，说明国产替代受益不只停留在主设备，还在向配套层外溢。
3. 它已通过 `招股书客户名单 + 年报工艺覆盖 + 2026Q1 合同负债 + 2026-06-09 说明会` 形成较完整的事实链，信息密度明显高于普通“名字补位”。
4. 对 AI 映射来说，它更像“量产链条向关键子系统扩散”的证据，而不是新的高弹性主题股。

### 3. 为什么它暂时不能抬成主锚

1. 当前最硬的新增信息仍是 `客户 / 工艺验证 + 合同负债 + 产能节点`，而不是正式利润预告。
2. `2026Q1` OCF 仍为负，且存货、在建工程继续抬升，说明资本周期审计还不能放松。
3. 当前估值和位置都不低，市场已经交易了一部分国产替代与订单预期。
4. 因此它更像 `平台扩线观察位`，而不是像长川科技那样的 `利润兑现主锚`，也弱于盛美上海那种 `订单强化补强位`。

### 4. 第一性原理下真正该盯的变量传导

1. 先看 `合同负债 / 客户验证 / 工艺覆盖` 是否继续强化，而不是先看题材热度。
2. 再看 `存货 / 在建工程 / OCF` 是否能被交付节奏和能力建设解释。
3. 最后看收入、利润、毛利率是否持续改善，确认这不是单纯的“扩张前置”而是真兑现。
4. 如果后续只剩“国产替代”和“先进制程配套”的叙事，而没有收入、利润和现金流跟进，就不应把它上调成更高权重样本。

## 六、继续强化触发器

1. `2026H1` 中报里收入、归母净利润、毛利率和 OCF 继续好于 `2026Q1`。
2. 合同负债继续维持高位，且能同步看到收入确认或交付进展。
3. 存货、在建工程的上升能被安徽基地投产、客户交付节奏或产品扩线解释。
4. 半导体专用温控、工艺废气处理、真空传输等品类出现更明确客户扩展或工艺扩展披露。
5. 高位震荡后估值仍能被经营兑现支撑，而不是只剩情绪驱动。

## 七、证伪信号

1. 合同负债高位维持，但后续收入和利润确认明显低于预期。
2. `2026H1` 或三季报里 OCF 继续为负且恶化，同时存货、在建工程继续抬升。
3. 安徽基地节点延后，或投产后未形成更强交付能力。
4. 客户与工艺验证长期停留在说明会或招股书口径，缺少新的正式披露延续。
5. 高估值状态下，若兑现继续落后，市场容易把它重新压回“高位题材配套股”。
