# 玻璃基板 / TGV 三强公告跟踪清单｜第一版

> 所属模块：02_桥接层/03_AI标杆股研究卡  
> 生成日期：2026-06-28  
> 跟踪对象：`莱宝高科`、`帝尔激光`、`京东方A`  
> 研究定位：把三强对决卡进一步下沉到 `公告监控层`，明确后续看到什么公告属于 `加分`、什么属于 `减分`、什么属于 `直接证伪`。  
> 使用边界：本文件不替代三强对决卡，而是服务后续滚动更新和盘中 / 盘后快速判断。
> 口径修正：这里的“三强”指 `正式披露较硬、最值得持续跟踪的三条路径`，不等于当前交易优先级完全并列；其中 `莱宝高科、京东方A` 是核心结论，`帝尔激光` 是高优先级工具链观察位。

---

## 一、一句话结论

> 三强后续最值得盯的不是股价波动本身，而是公告里有没有出现真正改变预期斜率的词。对莱宝高科，要盯 `客户验证 / 产品化`；对帝尔激光，要盯 `连续订单 / 持续出货`；对京东方A，要盯 `中试 / 良率 / 更多客户扩展`。只要关键词不升级，股价涨得越快，越接近接盘风险。

---

## 二、总原则

后续所有公告，先分三类：

1. `加分公告`
   - 说明产业验证在升级，一致预期可以继续上修。
2. `减分公告`
   - 说明逻辑没坏，但进度低于市场预期。
3. `证伪公告`
   - 说明当前市场交易的关键前提不成立，或者距离兑现比原先判断更远。

默认判断顺序：

`先看有没有新变量 -> 再看变量是升级还是原地重复 -> 最后看市场是不是已经提前交易过头`

---

## 三、莱宝高科公告跟踪清单

### 1. 当前最关键变量

`工程样品 -> 客户验证 -> 产品化`

### 2. 加分公告

以下任一出现，视为明显加分：

1. `客户验证`
2. `送样反馈良好`
3. `产品化时间表`
4. `下游应用场景更清晰`
5. `合作对象或验证对象更明确`

### 3. 中性但重要公告

1. 继续披露工艺提升，如孔径比、精度、样品规格升级；
2. 继续披露项目推进，但未提客户验证；
3. 对外说明技术储备仍在推进。

这类公告说明逻辑没坏，但还不足以单独推动大级别预期上修。

### 4. 减分公告

1. 仍停留在 `工程样品 / 研发中 / 待验证`；
2. 继续强调尚未产品化，但没有任何新阶段升级；
3. 项目推进明显慢于市场原先预期。

### 5. 证伪公告

1. 明确表示短中期内没有客户验证安排；
2. 明确表示距离产品化仍非常遥远；
3. 原先市场交易的工艺升级逻辑被公司自己否定。

### 6. 一句话执行规则

> 莱宝高科后续只有看到“客户验证”层面的新词，才算真正加分；如果只是继续重复工程样品和工艺指标，说明逻辑还在，但预期差开始被时间消耗。

## 四、帝尔激光公告跟踪清单

### 1. 当前最关键变量

`新增订单 -> 持续出货 -> 连续兑现`

### 2. 加分公告

1. `新增订单`
2. `连续订单`
3. `更多客户`
4. `持续出货`
5. `相关收入贡献更清晰`

### 3. 中性但重要公告

1. 继续确认已有 TGV 技术能力；
2. 继续确认晶圆级 / 面板级覆盖；
3. 继续说明市场处于中试线阶段。

### 4. 减分公告

1. 后续多个季度都没有再提订单更新；
2. 只有技术表述，没有新增出货与客户；
3. 披露口径反复停留在“小批量”而没有更进一步。

### 5. 证伪公告

1. 原先订单仅为一次性试单；
2. 后续项目终止或暂停；
3. 公司明确表示相关业务对收入贡献极弱且中短期难提升。

### 6. 一句话执行规则

> 帝尔激光最怕的不是慢，而是“订单只是一单”。只要连续订单和持续出货没有被正式确认，它的工具链兑现逻辑就不能当成已经成立，更不能默认上升为整条玻璃路线的核心交易结论。

---

## 五、京东方A公告跟踪清单

### 1. 当前最关键变量

`试验线 / 送样 / 概念认证 -> 中试 / 良率 / 更多客户扩展`

### 2. 加分公告

1. `中试`
2. `良率`
3. `更多客户扩展`
4. `量产节奏更清晰`
5. `应用场景进一步扩大`

### 3. 中性但重要公告

1. 继续确认试验线、送样、概念认证；
2. 继续确认工艺拉通；
3. 继续强调合作范围和技术路线方向。

### 4. 减分公告

1. 只重复“短期难有重大业绩影响”；
2. 后续多个季度没有中试或良率升级；
3. 市场热度高，但公司口径没有向更实质阶段推进。

### 5. 证伪公告

1. 公司明确表示相关路线推进放缓或阶段性停滞；
2. 关键客户验证没有延续；
3. 原本市场认为会推进到中试的逻辑被正式口径否定。

### 6. 一句话执行规则

> 京东方A的核心不是“故事有没有”，而是“中试和良率有没有”。只要公告还停在试验线和送样，逻辑是真的，但不够支持股价无限向远期透支。

## 六、三强统一打分框架

后续看到新公告，可以直接按下面方式快速打分：

| 公司 | 公告级别 | 解释 |
|---|---|---|
| 莱宝高科 | `+2` | 首次出现客户验证 / 产品化节点 |

| 莱宝高科 | `0` | 重复旧口径 |
| 莱宝高科 | `-1` | 明显推迟或继续停留在样品阶段 |

| 帝尔激光 | `+2` | 连续订单 / 持续出货 / 更多客户 |
| 帝尔激光 | `+1` | 新增技术覆盖或单次新增订单 |
| 帝尔激光 | `0` | 重复已有技术口径 |
| 帝尔激光 | `-1` | 后续长期无订单更新 |
| 帝尔激光 | `-2` | 订单被证实只是一次性试单或推进受阻 |

| 京东方A | `-1` | 只强调短期难有业绩影响，节点未升级 |
| 京东方A | `-2` | 路线推进显著慢于原先预期或客户验证受阻 |

## 八、最终执行结论

### 事实

1. 莱宝高科、帝尔激光、京东方A已经形成三条不同的预期差路径。
2. 后续研究的重点不再是“再找谁沾边”，而是盯这三家的公告关键词升级。

### 判断

1. 莱宝高科最适合盯“验证升级”，也是当前核心结论之一。
2. 帝尔激光最适合盯“兑现推进”，但定位应维持在高优先级工具链观察位。
3. 京东方A最适合盯“框架切换是否继续被中试和良率坐实”，也是当前核心结论之一。

### 待补点

1. 三家公司下一条正式公告出现后，需按本清单逐项评分。
2. 若连续两次公告都只重复旧口径，则应下调对应公司的预期差等级。
