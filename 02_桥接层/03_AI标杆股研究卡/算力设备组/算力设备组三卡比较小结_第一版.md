# 算力设备组｜三卡比较小结｜第一版

> 所属模块：02_桥接层/03_AI标杆股研究卡 → 算力设备组  
> 比较对象：金盘科技 / 科华数据 / 科士达  
> 生成日期：2026-07-02  
> 研究定位：把 A 股中三类最有代表性的 `AIDC供配电 / 智算电力基础设施` 资产压到同一张比较卡里，用于判断当前算力设备组更像“订单型高弹性兑现”“平台型基础设施重估”，还是“标准化稳态兑现”。  

---

## 一、一句话结论

> **算力设备组当前已经不能再只写成两类资产，而应拆成三类兑现路径：金盘科技代表 `订单型高弹性电力设备`，最先看订单和 OCF；科华数据代表 `平台型智算供电基础设施`，最先看分部收入、平台交付与应收效率；科士达代表 `标准化智算基础设施`，最先看数据中心核心业务稳增能否持续跑赢新能源波动。三者都受益于 AIDC，但利润兑现顺序、风险暴露方式和市场愿意给的估值框架并不一样。**

---

## 二、比较对象

1. 金盘科技：AIDC 电力设备高弹性兑现样本，核心看 AIDC 订单、海外订单、合同负债、存货和 OCF。
2. 科华数据：智算供电平台型资产，核心看数据中心分部收入、UPS / 高压直流 / 液冷 POD、应收和交付质量。
3. 科士达：标准化智算基础设施样本，核心看数据中心核心业务收入、标准化产品交付、AI 纯度和双主业协同质量。

对应新能源映射：

1. 金盘科技更像 `先导 + 阳光`；
2. 科华数据更像 `阳光 + 宁德`；
3. 科士达更像 `阳光 + 固德威`。

---

## 三、已确认事实

### 1. 三家公司都受益于智算基础设施建设，但所处层级不同

1. 金盘科技 `2025年` 收入 `72.95亿元`，归母净利润 `6.60亿元`，OCF `6.01亿元`。
2. 科华数据 `2025年` 收入 `81.60亿元`，归母净利润 `4.18亿元`，OCF `11.62亿元`。
3. 科士达 `2025年` 收入 `52.70亿元`，归母净利润 `6.11亿元`，OCF `9.11亿元`。
4. 金盘科技更强调 `AIDC 订单 + 海外订单 + 变压器 / 成套设备`。
5. 科华数据更强调 `UPS + 高压直流 + 液冷 POD + IDC 服务 + 平台集成`。
6. 科士达更强调 `UPS + 微模块 + 一体化电力模块 + 液冷 + 数据中心标准化底盘`。

### 2. 金盘科技更像订单型高弹性兑现样本

1. `2025年` AIDC / IDC 数据中心领域销售收入 `13.37亿元`，同比增长 `196.78%`。
2. `2026Q1` 新签销售订单 `33.44亿元`，同比增长 `64.48%`。
3. `2026Q1` 数据中心领域销售订单 `17.35亿元`，同比增长 `278.45%`。
4. `2026Q1` 海外订单 `22.52亿元`，同比增长 `280.73%`。
5. 但 `2026Q1` OCF 转负至 `-1.67亿元`，存货和合同负债同步上升。

### 3. 科华数据更像平台型智算供电基础设施资产

1. `2025年` 数据中心业务分部收入 `35.37亿元`，其中 `IDC 服务` `12.84亿元`、`数据中心产品及集成产品` `22.39亿元`。
2. `2026Q1` 收入同比 `+17.57%`、归母同比 `+13.15%`、OCF 同比 `+160.80%`。
3. `2026Q1` 应收账款下降，但存货、在建工程和合同负债同步抬升。
4. 公司产品链覆盖 `UPS / 高压直流 / 液冷 POD / 预制式电力模组 / IDC 服务`。

### 4. 科士达更像标准化稳态兑现样本

1. `2025年` 数据中心核心业务收入 `29.55亿元`，同比增长 `13.76%`。
2. `2026Q1` 收入同比 `+31.47%`、归母同比 `+24.84%`、OCF 同比 `+186.00%`。
3. `2024年` 到 `2025年` 的业绩修复同时受 `数据中心稳增 + 新能源恢复` 驱动。
4. 其风险不在订单斜率不够陡，而在 AI 纯度和双主业拆分不够清晰。

### 5. 三者被审计的变量并不一样

| 样本 | 当前更像什么 | 当前先看什么 | 后续最敏感风险 |
|---|---|---|---|
| 金盘科技 | 订单型高弹性电力设备资产 | 新签订单、在手订单、海外订单、合同负债、存货、OCF | 备货过重、回款偏慢、海外交付和订单兑现节奏 |
| 科华数据 | 平台型智算供电基础设施资产 | 数据中心分部收入、UPS / 液冷 / 高压直流交付、应收、合同负债、OCF | 应收回款、库存抬升、平台交付效率、预期过满 |
| 科士达 | 标准化智算基础设施资产 | 数据中心核心业务收入、微模块 / 一体化电力模块 / 液冷方案交付、OCF、AI 纯度 | 新能源波动掩盖 AIDC 判断、AI 收入拆分不足、备货与交付效率审计 |

## 四、判断

### 1. 产业位置比较

1. 金盘科技更靠近 `AIDC 电力设备订单层`，逻辑更偏“项目与设备滞后兑现”。
2. 科华数据更靠近 `智算中心基础设施平台层`，逻辑更偏“平台产品与集成能力兑现”。
3. 科士达更靠近 `标准化供配电与微模块底盘层`，逻辑更偏“稳态产品兑现 + 第二曲线协同”。

### 2. 第一变量比较

1. 金盘科技的第一变量是 `AIDC 订单 / 海外订单 / 在手订单` 能否继续放大。
2. 科华数据的第一变量是 `数据中心分部收入 / 平台产品交付 / 应收效率` 能否继续成立。
3. 科士达的第一变量是 `数据中心核心业务稳增能否持续跑赢新能源波动`，以及 `AI 纯度能否逐步拆分清楚`。

### 3. 利润兑现顺序比较

1. 金盘科技更像：`订单放量 -> 合同负债与备货抬升 -> 收入利润兑现 -> OCF 审计`。
2. 科华数据更像：`平台产品导入 -> 数据中心分部收入增长 -> 利润兑现 -> 应收 / 库存 / 交付质量审计`。
3. 科士达更像：`数据中心核心业务稳增 -> 标准化产品放量 -> 双主业业绩修复 -> AI 纯度与营运资本审计`。

### 4. 风险暴露路径比较

1. 金盘科技最敏感的是 `订单兑现节奏、海外交付、存货和 OCF`。
2. 科华数据最敏感的是 `应收账款、库存、合同负债、平台交付效率`。
3. 科士达最敏感的是 `AI 纯度拆分不足、新能源波动干扰、备货与交付效率`。

### 5. 行情与经营错位比较

1. 三家公司当前在结果层统一都属于 `未充分演绎 / 低位观察`，说明市场还没有把这条链整体打满。
2. 但经营层并不弱，尤其是科华数据、科士达的 `收入 + 利润 + OCF` 组合已经更平衡。
3. 因此这条线当前更像 `经营先验证、行情后确认`，而不是 `行情已经先打满再等报表`。

---

## 五、当前最适合写进上层结论的话

1. `AIDC供配电` 当前已经不能再写成单一设备链，而应拆成 `订单型高弹性设备`、`平台型基础设施`、`标准化稳态兑现` 三类路径。
2. 金盘科技代表 `订单斜率最陡` 的一侧，但也是最早进入 `备货 / 合同负债 / OCF` 审计的一侧。
3. 科华数据代表 `平台属性最强` 的一侧，更容易先被 `应收 / 库存 / 交付效率` 审计。
4. 科士达代表 `经营兑现更稳但 AI 纯度较低` 的一侧，更需要通过 `数据中心收入连续拆分` 证明其不是新能源恢复遮蔽下的伪 AI 改善。
5. 从新能源映射看，这一组当前更像“设备订单、平台交付、标准化产品”三条路径并行兑现，但最终都要回到 `现金流、营运资本、收入纯度` 审计。
