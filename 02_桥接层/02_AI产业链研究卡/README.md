# AI 产业链研究卡

本模块当前的固定角色不是“继续堆更多观点”，而是作为：

`桥接层图谱坐标入口 -> 桥接层样本验证坐标轴 -> 结果层正式总论之前的统一骨架`

也就是说，它负责先把 AI 产业链坐标写清楚，再让 AI 标杆股研究卡沿着这些坐标去验证，最后由结果层把图谱和样本压缩成可引用总论。

## 模块边界

本模块当前固定解决这些事：

1. 定义 AI 产业链主坐标，而不是先从个股出发。
2. 区分不同子链条的第一变量和兑现顺序。
3. 区分不同子链条吃到的是总钱包、增量钱包还是利润池。
4. 把已经完成的样本，放回同一张产业图谱里重新定位。
5. 给结果层 AI 总论提供统一坐标，不让样本研究重新滑回“热点堆名单”。
6. 对 `AI样本扩容与偏差控制`，凡涉及样本新增、来源补录和等级迁移，先回写 `AI全景细分样本宇宙总表_第一版.md`；`AI样本扩容与偏差控制主表_第一版.md` 只负责样本分级、偏差判断和结果层权限，再由 `AI全景细分空白审计表_第一版.md` 审计每个细分是否已建样本池。
7. 在关键样本组基本补齐后，提供跨组滚动维护工具，统一资源运营层、液冷二线、AIDC 供配电和应用层的经营质量跟踪。
8. 对 `AI算力基建大组`，桥接层先用 `扩展观察池` 区分核心供配电、电力设备旁证、液冷温控和 IDC / AIDC运营，再回到各自评分表做经营验证。
9. 对 `AIDC供配电`，桥接层默认先走 `滚动评分表 -> 2026H1实填模板 -> 批量回写顺序清单`，结果层再用 `三路径接力判定表` 决定是否升级成基建接力结论。
10. 对 `AI应用方向`，桥接层默认先走 `细分框架 -> 滚动跟踪总表`，结果层再用 `应用方向行情与经营跟踪看板` 统一周更口径。
11. 对 `全球映射`，桥接层默认先走 `AI全球映射框架`，结果层再用 `AI全球映射与A股落点总表` 固定“海外主链 -> A股传导 -> 证伪点”口径。
12. 对 `全球产业链样本`，桥接层默认先走 `AI全球产业链核心样本池`，结果层再用 `AI全球产业链核心样本深度分析` 固定全球主链代表样本与深度结论。
13. 对 `全球芯片制造深链`，桥接层默认先走 `AI全球芯片制造与材料链样本池`，结果层再用 `AI全球芯片制造产业链与技术路线深度分析` 固定日本材料、韩国 HBM 配套、InP 与路线之争口径。
14. 对 `全球芯片制造深链 -> A股回落`，结果层固定入口 `AI全球芯片制造深链与A股映射总表` 专门承接日本材料、韩国 HBM 配套、InP 与 glass / ABF 路线的本土映射。
15. 对 `全球芯片制造深链 -> 赔率审计`，结果层固定入口 `AI全球芯片制造深链估值_预期_拥挤审计表`，只负责制造深链 A 股优先对象的局部赔率，不替代全链条估值总表。

## 当前显性缺口

当前仍保留的显性缺口包括：

1. 样本代表性不足已经升为偏差控制项，需按 `AI全景细分样本宇宙总表_第一版.md` 与 `AI全景细分空白审计表_第一版.md` 逐细分补样本；算力租赁、AI Agent、AI眼镜、AI玩具、半导体设备材料等方向此前未形成统一 L0/L1 防漏池。本轮新增 `AI全产业链财经网站样本资料库_第一版.md` 先补 `170+` 个 L0/L1 候选或复核样本，但 `96+` 只保留为阶段门槛，最终验收仍为 `1000+ L0 / 500+ L1 / 240+ L2 / 120+ S3-S4`。
2. 数据层与安全治理层方向稿已补，但单卡与连续经营验证仍不足。
