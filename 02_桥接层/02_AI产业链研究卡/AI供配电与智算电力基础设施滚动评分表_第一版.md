## 一、一句话结论

> 当前 `AIDC供配电` 更像“市场尚未打满、经营上已经进入兑现审计”的低位观察链条。`科华数据` 更像平台型智算供电基础设施样本，`金盘科技` 更像订单型高弹性电力设备样本，`科士达` 更像标准化 UPS / 微模块 / 液冷一体方案样本。现阶段最该继续做的，不是把这条线提前写成已确认主线，而是持续审计 `订单或项目 -> 收入 -> OCF -> 应收 / 存货 / 合同负债` 这条闭环。

## 二、评分原则

### 1. 三项评分结构

每个样本统一按三项评分：

1. `事实升级分`
   - 看 AIDC 订单、数据中心收入、产品迭代、客户验证、项目交付是否继续升级。
2. `报表传导分`
   - 看收入、利润和 OCF 是否开始同步传导。
3. `风险审计分`
   - 看应收、存货、合同负债、在建工程、回款、海外交付和业务混合度是否仍可控。

### 2. 单项分值

每项默认按 `0-3 分` 处理：

1. `3 分`
   - 已形成连续验证。
2. `2 分`
   - 已验证但仍需继续跟踪。
3. `1 分`
   - 逻辑未破，但经营质量开始承压。
4. `0 分`
   - 已进入高波动、待降级或待证伪状态。

### 3. 总分对应状态

1. `8-9 分`
   - 接近正式经营兑现样本。
2. `6-7 分`
   - 经营验证中，值得优先维护。
3. `4-5 分`
   - 继续观察，不宜提前长期化。
4. `0-3 分`
   - 高波动风险位，优先做反证审计。

## 四、截至 2026-07-02 的中报前正式核验

### 1. 当前核验结论

本轮基于截至 `2026-07-02` 已能定位到的正式年报、一季报和投资者关系活动记录，当前更稳妥的写法是：

1. `AIDC供配电` 这条线在经营层已经进入验证期，但在行情层仍停留在 `低位观察`；
2. `科华数据` 和 `科士达` 的收入、利润与 OCF 组合更平衡；
3. `金盘科技` 的订单斜率最陡，但也是最早暴露 `备货 / 合同负债 / OCF` 审计压力的样本；
4. 三个样本都不能被写成“纯 AI 单一业务股”，都必须区分 `AI 数据中心订单` 与 `传统电力 / 新能源底盘` 的混合影响。

### 2. 已确认的硬口径

| 公司 | 最新补到的正式披露 | 当前新增信息 | 对评分表的处理 |
|---|---|---|---|
| 科华数据 | `2025年报`、`2026Q1` | `2025年` 数据中心业务分部收入 `35.37亿元`，其中 `IDC服务` `12.84亿元`、`数据中心产品及集成产品` `22.39亿元`；`2026Q1` 收入同比 `+17.57%`、归母同比 `+13.15%`、OCF 同比 `+160.80%` | 强化“平台型基础设施兑现”判断，总分维持 `7 分` |

### 3. 当前最重要的组级判断

`AIDC供配电` 现在最容易犯的错，是把“景气开始传导”直接写成“行情已经确认接力”。

更准确的表达是：

1. `经营层` 已经有验证；
2. `行情层` 还没走出统一主升；
3. 因此当前最需要的是 `经营质量滚动审计`，而不是继续放大题材表述。

## 五、三家公司优先维护点

### 1. 当前组级判断

供配电与智算电力基础设施当前真正要盯的不是“谁提了 AIDC”，而是：

`谁能把 AIDC / 数据中心项目 -> 收入确认 -> 利润率 -> OCF -> 营运资本压力`

解释成一条自洽链条。

### 2. 三个样本的优先维护点

| 公司 | 当前最该盯的变量 | 当前最敏感反证点 |
|---|---|---|
| 科华数据 | 数据中心分部收入、UPS / 高压直流 / 液冷 POD 交付、应收、合同负债、OCF | 应收回款重新走弱，库存和在建工程继续抬升 |
| 金盘科技 | 数据中心订单、海外订单、在手订单、合同负债、存货、OCF | 订单强但收入兑现慢，备货过重且 OCF 持续承压 |
| 科士达 | 数据中心核心业务收入、UPS / 微模块 / 一体化电力模组 / 液冷方案交付、OCF、AI 订单纯度 | 数据中心增长继续存在但 AI 纯度始终拆不出来，或新能源波动掩盖智算基础设施判断 |

### 3. 当前统一审计口径

本组统一回到以下变量：

1. 数据中心核心业务收入
2. AIDC / 智算订单或项目纯度
3. 收入确认节奏
4. 毛利率
5. OCF
6. 应收与回款
7. 存货与合同负债
8. 在建工程或海外交付压力

## 六、状态迁移规则

### 1. 升级条件

样本只有在以下三条同时改善时，才从“继续观察”升级到“经营验证中”或“接近经营兑现”：

1. 新披露事实继续升级；
2. 收入、利润和 OCF 至少两项同步强化；
3. 应收、存货、合同负债、在建工程或海外交付压力没有明显恶化。

### 2. 降级条件

若出现以下任一情况，应优先考虑降级或转入反证审计：

1. 事实继续讲故事，但报表传导停滞；
2. OCF 连续走弱，同时存货或应收压力放大；
3. AI 业务纯度迟迟拆不出来，只剩传统电力或新能源估值抬升支撑。

### 3. 本表的纪律

1. 不把 `低位行情` 直接写成 `高胜率接力`。
2. 不把 `订单高增` 直接写成 `利润和现金流已兑现`。
3. 不把 `供配电平台样本` 和 `订单型设备样本` 混写成同一类风险暴露。
4. 不把 `数据中心 + 新能源` 混合业务结构硬写成纯 AI 样本。
