## 一句话结论

全球产业链真正还没补足的，不只是 `NVIDIA / 云厂 / 机器人`，而是 `ASML -> 日本光刻胶与掩模材料 -> 韩国 HBM 与先进封装配套 -> InP 光器件衬底 -> ABF / 玻璃基板路线` 这条制造深链；当前至少应把 `ASML / TOK / JSR / Fujifilm / HOYA / Shin-Etsu / Mitsui Chemicals / ADEKA / SCREEN / TOPPAN / SK hynix / Samsung Electronics / Micron / Hanmi Semiconductor / Samsung Electro-Mechanics / Ibiden / Nepes / ISC / Okins Electronics / SEMES / Sumitomo Electric / Coherent / Broadcom / Intel` 固定为第一版主样本池。

## 三、当前最重要的五个样本簇

### 1. 日本光刻与材料瓶颈簇

1. `ASML`
2. `TOK`
3. `JSR`
4. `Fujifilm`
5. `HOYA`
6. `Shin-Etsu Chemical`
7. `Mitsui Chemicals`
8. `ADEKA`
9. `SCREEN`
10. `TOPPAN`

这组样本回答：

1. 为什么 AI 芯片制造不是只有 GPU 和代工，而是被设备与材料同时卡住。
2. 为什么“日本材料链”不能只被写成一个 `光刻胶` 词条。
3. 为什么 EUV resist / developer、photomask blank、pellicle、silicon wafer、CMP slurry、前驱体、清洗工艺要分开追踪。

### 2. 韩国存储与 HBM 配套簇

1. `SK hynix`
2. `Samsung Electronics`
3. `Micron`
4. `Hanmi Semiconductor`
5. `Samsung Electro-Mechanics`
6. `Nepes`
7. `ISC`
8. `Okins Electronics`
9. `SEMES`

这组样本回答：

1. HBM / DRAM / NAND 之间的利润层级如何分化。
2. 为什么 HBM 不只是存储芯片，而是 `HBM / DRAM -> TC bonding -> memory test socket / burn-in socket / AI system IC socket -> package final test -> bridge interposer / fan-out -> FCBGA / ABF / glass` 的复合链条。
3. 为什么韩国供应链不只是“SK hynix / Samsung 两家存储原厂”，而是要固定拆成 `HBM / DRAM`、`TC bonding`、`memory test socket / burn-in socket / AI system IC socket`、`package final test`、`bridge interposer / fan-out`、`FCBGA / ABF / glass` 六段。

### 3. 封装基板与材料平台簇

1. `Samsung Electro-Mechanics`
2. `Ibiden`
3. `TOPPAN`
4. `Intel`

这组样本回答：

1. 为什么先进封装利润池不能只盯 OSAT。
2. 为什么 `ABF / FCBGA / glass core / bridge interposer` 是下一轮封装路线切换的真实战场。

### 4. 光通信化合物半导体簇

1. `Sumitomo Electric`
2. `Coherent`

这组样本回答：

1. 为什么 InP 衬底和光器件不该只被写成“光模块零件”。
2. 为什么 AI 光互连继续升级时，化合物半导体上游的重要性会上升。

### 5. 路线之争样本簇

1. `Broadcom`
2. `Intel`
3. `SK hynix`
4. `Hanmi Semiconductor`
5. `JSR`
6. `Mitsui Chemicals`
7. `Nepes`

这组样本回答：

1. `CPO` 会不会提前替代可插拔光模块。
2. `Glass substrate` 会不会重构有机基板与先进封装路线。
3. `HBM4` 的持续强化会不会把存储利润更集中到少数厂商与关键配套。
4. `MOR` 会不会升级成 EUV 材料代际切换主线。

## 四、技术路线之争最少要拆成四条

| 路线之争 | 当前更接近的状态 | 已核验事实锚 | 关键证伪点 |
|---|---|---|---|
| `传统 EUV resist` vs `MOR 等下一代 EUV 材料` | 已进入并行验证，不是单边替代 | JSR 已公开推进 MOR，Fujifilm / TOK 继续强化 EUV resist 与 developer | 若 MOR 客户导入低于预期、传统胶体系继续主导，则“材料代际切换”应降级 |
| `HBM4 主线强化` vs `通用 DRAM / NAND 修复` | 短期仍是 HBM 主线更强 | SK hynix、Micron 均公开推进 HBM4，Samsung 继续强化 HBM3E / NAND 迁移 | 若 HBM ASP / 良率 / 客户认证转弱，而通用 DRAM/NAND 周期修复超预期，利润池会重新平衡 |
| `可插拔光模块` vs `CPO / silicon photonics` | 短期更可能共存 | Broadcom 已推至第三代 CPO；InP / 可插拔链仍在升级 | 若 CPO 良率、维护复杂度、热管理迟迟不过关，可插拔主链寿命会延长 |
| `Organic substrate` vs `glass substrate / glass core` | 主线仍是有机基板，玻璃为高阶期权 | Intel 与 SEMCO 均公开推进玻璃路线 | 若玻璃良率 / 成本 / 生态迁移不及预期，则 glass 仍只能停留在观察层 |

## 五、当前最重要的待补点

1. `日本材料链`
   - `TOK / JSR / Fujifilm` 已补，但 `pellicle`、清洗、前驱体与更细二线仍可继续扩样。
2. `韩国存储配套`
   - `HBM / DRAM`、`TC bonding`、`memory test socket / burn-in socket / AI system IC socket`、`package final test`、`bridge interposer / fan-out` 与 `FCBGA / ABF / glass` 六段口径已固定；其中 `contactor` 已由 `ISC iSP` 分类补出官方产品样本，`hybrid bonding` 已由 `SEMES` 补出官方开发样本，但两者连续经营验证仍待补。
3. `路线之争`
   - 还需继续补 `HBM4 vs 其他内存层级`、`InP vs silicon photonics`、`glass vs organic substrate`、`MOR vs 传统 EUV resist` 的持续验证。
4. `A 股映射`
   - 还缺“日本材料 -> A 股高端材料 / 湿化学品 / 掩模配套”、“韩国存储配套 -> A 股设备 / 测试 / 基板映射”的更细回落表。
