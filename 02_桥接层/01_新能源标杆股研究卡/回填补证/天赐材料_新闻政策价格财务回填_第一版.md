## 一、回填目标

天赐材料是锂电材料组中最重要的“材料涨价泡沫”样本之一。

前一版主卡已经建立主线：

**2015–2017 第一轮锂电材料认知 → 2018–2019 补贴退坡和材料价格回落 → 2020 需求修复 → 2021–2022 六氟 / 电解液价格上涨和利润爆发 → 2023–2025 材料价格崩塌、毛利率下行和股价阴影。**

本补证卡要验证的问题是：

> **为什么动力电池需求继续增长，但天赐这类材料龙头在 2022 后仍然进入长期阴影？**

核心证据链：

**动力电池排产上修 → 电解液和六氟供需紧张 → 六氟 / 电解液价格上涨 → 天赐毛利率和利润爆发 → 高利润刺激扩产 → 供给释放和客户压价 → 产品价格回落 → 毛利率下行、存货和减值压力 → 股价从成长切回周期材料。**

## 三、阶段一：2015–2017 第一轮锂电材料认知

### 1. 产业背景

2015–2017 年，新能源车补贴推动动力电池装机增长，锂电材料开始从主题扩散进入价格弹性交易。

天赐早期被市场关注的变量包括：

- 电解液需求增长；
- 六氟磷酸锂供需紧张；
- 材料国产化；
- 动力电池客户绑定；
- 一体化成本优势。

### 2. 待补价格数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 六氟磷酸锂价格 | 验证第一轮涨价 | SMM / 百川 / Wind |
| 电解液价格 | 验证材料价格弹性 | SMM / 百川 / 券商周报 |
| 碳酸锂价格 | 验证上游成本 | SMM / 百川 |
| 动力电池装机 | 验证需求 | GGII / 中汽协 |
| 新能源车销量 | 验证终端 | 中汽协 |

### 3. 待补财务数据

需补 2015–2017：

- 营业收入；
- 归母净利润；
- 电解液业务收入；
- 六氟业务收入；
- 毛利率；
- 客户结构；
- 产能扩张；
- 存货和现金流。

### 4. 事后验证

第一轮逻辑部分成立：锂电材料需求确实随新能源车放量增长。

但 2018–2019 的回撤说明：

> 材料需求增长不等于产品价格和高毛利永久维持。

---

## 四、阶段二：2018–2019 补贴退坡和材料价格回落

### 1. 产业背景

2018–2019，新能源车补贴退坡，低端需求被压缩，动力电池产业链去泡沫，六氟和电解液价格回落。

对天赐的影响包括：

- 材料价格下降；
- 毛利率下行；
- 估值收缩；
- 市场从“材料高成长”转向“材料周期”；
- 成本控制和客户结构成为筛选关键。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 电解液收入 | 看需求是否仍增长 |
| 六氟收入 | 看单品价格影响 |
| 毛利率 | 看价格下跌冲击 |
| 扣非净利润 | 看主业质量 |
| 存货 | 看价格下跌风险 |
| 经营现金流 | 看低谷韧性 |

### 3. 事后验证

这一阶段不是天赐产业逻辑彻底证伪，而是证明：

- 客户结构重要；
- 成本一体化重要；
- 价格周期不可忽视；
- 高景气时扩产会带来后续压力。

---

## 五、阶段三：2020 第二轮材料行情前夜

### 1. 产业背景

2020 年，特斯拉国产、欧洲电动化、中国双碳政策推动电动车需求恢复，电池厂排产改善。

天赐的第二轮行情基础来自：

- 电解液需求恢复；
- 六氟价格底部企稳；
- 电池厂订单改善；
- 一体化产能和客户结构被重新定价；
- 机构资金寻找宁德、比亚迪之外的材料弹性。

### 2. 待补证据

| 证据 | 用途 | 来源 |
|---|---|---|
| 2020 六氟价格底部 | 验证行情前夜 | SMM / 百川 |
| 2020 电解液价格 | 验证需求修复 | SMM / 百川 |
| 2020 动力电池装机 | 验证排产恢复 | GGII / 中汽协 |
| 天赐 2020 年报 | 验证财务修复 | 年报 / Wind |
| 扩产公告 | 验证供给准备 | 公司公告 |

### 3. 事后验证

2020 是材料股第二轮行情的前夜，真正的利润爆发发生在 2021–2022。

---

## 六、阶段四：2021–2022 六氟 / 电解液价格高点和利润泡沫

### 1. 产业背景

2021–2022，新能源车销量和动力电池排产超预期，六氟供需错配，电解液价格上行。

天赐成为材料组核心强势标的，原因是：

- 电解液需求高增；
- 六氟价格上涨；
- 一体化能力提升利润弹性；
- 客户结构支撑出货；
- 市场将阶段性高毛利外推。

### 2. 价格逻辑

必须补三条价格曲线：

| 价格 | 对天赐意义 |
|---|---|
| 六氟磷酸锂 | 利润弹性核心 |
| 电解液 | 主产品价格和收入基础 |
| 碳酸锂 | 上游成本和利润传导 |

### 3. 财务验证

待补 2021–2022：

- 电解液收入；
- 六氟收入；
- 电解液毛利率；
- 六氟毛利率；
- 归母净利润；
- 扣非净利润；
- ROE；
- 经营现金流；
- 存货；
- 资本开支；
- 扩产公告。

### 4. 事后验证

2021–2022 的材料涨价逻辑成立，但它是周期利润，不是永久成长利润。

高点隐含风险：

- 六氟扩产；
- 电解液竞争；
- 下游电池厂压价；
- 碳酸锂成本波动；
- 库存和跌价风险；
- 市场把高毛利线性外推。

### 5. AI 映射

对应 AI 中：

- 液冷材料；
- 高端 CCL；
- 高速连接材料；
- 光模块辅材；
- 电源核心耗材。

如果某个 AI 材料品种价格短期暴涨，要警惕它是否只是供需错配，而不是长期护城河。

---

## 七、阶段五：2023–2025 价格回落、毛利率下行和阴影

### 1. 产业背景

2023–2025，动力电池需求仍增长，但材料环节价格和毛利率显著回落。

压力来自：

- 六氟产能释放；
- 电解液竞争加剧；
- 电池厂压价；
- 材料去库存；
- 碳酸锂价格大幅波动；
- 高点扩产带来折旧和资本开支压力。

### 2. 对天赐的影响

天赐承压的关键不是需求消失，而是：

- 产品价格回落；
- 毛利率下行；
- 利润中枢下降；
- 存货跌价；
- 估值从成长材料切回周期材料；
- 市场等待行业出清和新盈利中枢。

### 3. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 电解液收入、六氟收入、毛利率、存货 | 看价格下跌第一阶段影响 |
| 2024 | 扣非净利润、减值、毛利率、现金流 | 看低价环境压力 |
| 2025 | 中报 / 年报、价格低位、出清迹象 | 看是否具备修复条件 |

### 4. 事后结论

天赐不是出清失败，而是进入材料周期阴影。

准确表述：

> 客户和成本能力仍在，但高点盈利中枢已被证伪；未来修复必须看六氟 / 电解液价格、毛利率和行业出清。

## 九、天赐回填后的核心判断

天赐材料路径可以概括为：

1. **2015–2017：第一轮材料认知。** 新能源车放量带来电解液和六氟需求。
2. **2018–2019：材料周期退潮。** 补贴退坡后价格回落，材料股周期属性显现。
3. **2020：第二轮行情前夜。** EV 需求修复，六氟和电解液价格企稳。
4. **2021–2022：材料涨价泡沫。** 价格上涨和一体化带来极致利润。
5. **2023–2025：价格和毛利率重估。** 需求仍在，但产品价格和利润中枢下行。

### 下一步建议

继续回填：

**多氟多。**

- 天赐代表电解液 / 六氟一体化材料龙头；
- 多氟多更强调六氟单品价格弹性；
- 两者合起来解释材料股“平台型”和“单品型”的差异。
