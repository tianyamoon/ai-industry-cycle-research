## 一、回填目标

宁德时代是新能源车链条中最重要的核心供应商样本。

它和比亚迪不同：

- 比亚迪是终端整车平台；
- 宁德是动力电池和储能电池基础设施供应商；
- 比亚迪看产品周期、品牌、渠道和单车利润；
- 宁德看全球份额、客户结构、毛利率、储能第二曲线和资本开支回报。

本补证卡要验证的问题是：

> **产业核心地位没有被证伪，为什么股价仍会经历估值消化？宁德能否从赛道贝塔进入公司阿尔法？**

核心证据链：

**新能源车渗透率提升 → 动力电池装机高增 → 宁德全球份额和客户绑定强化 → 2020–2021 核心资产重估 → 2022 储能和技术叙事延展 → 2023–2025 电池价格竞争、车企议价、储能利润验证和资本开支回报 → 半穿越 / 穿越观察。**

## 三、阶段一：2015–2017 产业地位形成但无 A 股 K 线

### 1. 产业背景

宁德时代 2018 年才上市，因此不能用 A 股 K 线直接观察 2015–2017 第一轮概念行情。

但宁德的产业地位在这一阶段已经形成：

- 新能源车补贴推动动力电池需求；
- 国产动力电池政策窗口；
- 主流车企客户绑定；
- 动力电池行业从分散走向集中；
- 宁德成为国产电池核心供应商。

### 2. 待补证据

| 证据 | 用途 | 来源 |
|---|---|---|
| 动力电池白名单 / 规范条件 | 验证政策窗口 | 工信部 |
| 动力电池装机数据 | 验证需求增长 | GGII / 工信部 |
| 宁德上市招股书 | 验证上市前客户和收入 | 招股书 |
| 客户绑定资料 | 验证产业地位 | 招股书 / 公司资料 |
| 新能源车销量 | 验证终端需求 | 中汽协 |

### 3. 事后验证

宁德说明一个重要方法论：

> 上市时间不等于产业起点。产业地位往往在股价可见之前已经形成。

## 四、阶段二：2018–2019 上市后估值消化与行业筛选

### 1. 产业背景

2018–2019，新能源车补贴退坡，动力电池行业进入筛选期。

对宁德而言，市场同时看到两件事：

- 行业短期需求和补贴模式承压；
- 头部电池厂份额和客户绑定强化。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 动力电池收入 | 看主业规模 |
| 毛利率 | 看电池价格和成本压力 |
| 归母净利润 / 扣非净利润 | 看盈利质量 |
| 研发费用 | 看技术投入 |
| 资本开支 | 看扩产节奏 |
| 客户集中度 | 看客户绑定和议价 |
| 经营现金流 | 看盈利质量 |

### 3. 事后验证

补贴退坡并没有证伪宁德，反而强化龙头集中。

低端电池厂出清后，真正有客户、技术和产能能力的龙头更强。

宁德被市场定价为：

- 全球动力电池核心供应商；
- 中国新能源车供应链核心资产；
- 长期电动化基础设施；
- 储能潜在第二曲线。

## 六、阶段四：2021 估值高位与机构抱团

### 1. 产业背景

2021 年新能源车渗透率快速提升，动力电池装机高增，宁德作为核心供应商进入极高关注度阶段。

市场相信：

- 动力电池需求长期高增；
- 宁德份额和客户绑定稳固；
- 技术领先；
- 全球扩张空间大；
- 储能提供第二增长曲线。

### 2. 风险开始积累

尽管产业仍强，但股价进入高位后需要注意：

- 估值已经提前反映多年成长；
- 机构持仓集中；
- 上游材料涨价挤压毛利率；
- 车企议价能力增强；
- 竞争对手和车企自研电池形成压力。

### 3. 待补数据

- 2021 动力电池装机和全球份额；
- 2021 动力电池毛利率；
- 2021 资本开支；
- 2021 经营现金流；
- 2021 主要客户结构；
- 2021 公募基金持仓和估值水平。

### 4. 事后验证

宁德的龙头地位没有被证伪，但高估值被消化。

核心结论：

> 核心资产地位和买入价格，是两个问题。

## 七、阶段五：2021H2–2022 储能、技术叙事和高位消化

### 1. 产业背景

2021H2–2022，宁德仍有产业强化：

- 动力电池装机继续高增；
- 储能需求打开；
- 麒麟电池、钠电池等技术叙事；
- 海外产能和客户扩展；
- 但上游锂价高位和下游议价压力也在上升。

### 2. 财务验证重点

| 指标 | 用途 |
|---|---|
| 储能收入 | 判断第二曲线规模 |
| 储能毛利率 | 判断第二曲线质量 |
| 动力电池毛利率 | 判断价格和成本压力 |
| 资本开支 | 判断扩产风险 |
| ROIC | 判断重资产回报 |
| 经营现金流 | 判断利润质量 |

### 3. 事后验证

储能是重要第二曲线，但不能只看收入。

必须看：

- 毛利率；
- 现金流；
- 客户结构；
- 海外认证；
- 系统价格竞争；
- 投入资本回报。

## 八、阶段六：2023–2025 价格战与公司阿尔法验证

### 1. 产业背景

2023–2025，新能源车进入价格战，电池价格下行，车企议价增强，储能系统竞争也加剧。

宁德面临的核心问题：

- 动力电池 ASP 是否下降；
- 毛利率是否企稳；
- 储能毛利率是否能支撑第二曲线；
- 海外市场和客户是否继续拓展；
- 资本开支回报是否合理；
- 车企自研和二线电池厂是否冲击份额。

### 2. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 动力 / 储能收入、毛利率、现金流 | 看价格战第一阶段影响 |
| 2024 | 全球份额、海外收入、ROIC、资本开支 | 看半穿越质量 |
| 2025 | 中报 / 年报、储能利润、客户结构 | 看是否从半穿越走向穿越 |

### 3. 事后判断

宁德不是阴影期失败样本，而是进入公司阿尔法验证期。

准确表述：

> 动力电池全球龙头地位仍在，但市场不再只因电动车销量高增给高估值，而是看毛利率、现金流、海外化、储能利润和 ROIC。

## 十、宁德回填后的核心判断

宁德时代路径可以概括为：

1. **2015–2017：产业地位形成，先于上市 K 线。**
2. **2018–2019：上市后估值消化，补贴退坡反而强化龙头集中。**
3. **2020：全球电动化和双碳推动核心资产重估。**
4. **2021：估值和持仓高位，产业仍强但股价进入压力区。**
5. **2021H2–2022：储能和技术叙事延展，但估值开始消化。**
6. **2023–2025：从赛道贝塔切换到公司阿尔法验证。**
