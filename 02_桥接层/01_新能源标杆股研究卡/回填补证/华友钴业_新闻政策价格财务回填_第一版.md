## 一、回填目标

华友钴业是新能源周期中典型的资源 + 材料复合样本。

它不是单纯锂资源股，也不是单一材料股，而是同时具有：

- 钴资源；
- 镍资源；
- 钴盐 / 镍盐；
- 三元前驱体；
- 正极材料；
- 海外资源和冶炼；
- 高资本开支。

本补证卡要验证的问题是：

> **资源 + 材料平台是否更容易穿越，还是变量更多、风险更复杂？**

核心证据链：

**新能源车需求增长 → 三元电池路线强化 → 钴价 / 镍价和前驱体需求提升 → 华友从钴资源股重估为资源材料平台 → 2021–2022 高景气扩产 → 2023–2025 磷酸铁锂挤压、钴镍价格波动、材料利润下行和资本开支回报再验证。**

## 三、阶段一：2016–2017 第一轮钴资源行情

### 1. 产业背景

2016–2017 年，新能源汽车补贴推动动力电池需求增长，市场开始定价三元电池路线和钴资源稀缺。

华友早期的股价逻辑：

- 钴是三元正极关键资源；
- 新能源车需求提升；
- 钴资源供给受限；
- 钴价上涨带来利润弹性；
- 华友成为 A 股钴资源代表标的。

### 2. 待补数据

| 数据 | 用途 | 待补来源 |
|---|---|---|
| 钴价 | 验证第一轮资源行情 | SMM / Wind / 百川 |
| 三元电池装机 | 验证路线预期 | GGII / 工信部 |
| 新能源车销量 | 验证终端需求 | 中汽协 |
| 华友钴产品收入 | 验证资源弹性 | 年报 |
| 钴产品毛利率 | 验证利润弹性 | 年报 |

### 3. 事后验证

这一阶段说明资源价格弹性真实存在，但后续钴价回落证明：

> 三元需求预期不等于钴价永久高位，资源股仍受价格周期支配。

---

## 四、阶段二：2018–2019 钴价回落和资源周期退潮

2018–2019，补贴退坡和钴价回落使华友承压。

市场关注从“钴资源稀缺”切换为：

- 钴价是否继续下跌；
- 三元需求是否不及预期；
- 低钴化是否影响长期钴需求；
- 公司是否能从钴资源向前驱体和材料平台延伸。

财务验证重点：

| 指标 | 用途 |
|---|---|
| 钴产品收入和毛利率 | 看价格下跌冲击 |
| 存货 | 看跌价风险 |
| 经营现金流 | 看低谷韧性 |
| 前驱体收入 | 看材料转型 |
| 资本开支 | 看扩张压力 |

事后看，这一阶段是华友从资源股向材料平台转型的低位准备期。

## 五、阶段三：2020 平台转型和主升启动

2020 年，EV 需求恢复、双碳和全球电动化推动资源材料链重估。

华友的逻辑不再只是钴价，而是：

- 钴镍资源；
- 三元前驱体；
- 正极材料；
- 海外资源和冶炼；
- 材料平台化。

## 六、阶段四：2021–2022 高景气扩张和复合定价

2021–2022，新能源车需求高增，市场继续交易资源和材料平台。

华友的高景气逻辑：

- 钴镍资源价格支撑；
- 三元前驱体需求；
- 镍资源布局；
- 材料一体化；
- 全球客户和供应链；
- 前驱体 / 正极产能扩张。

但这一阶段也积累风险：

- 磷酸铁锂路线份额提升；
- 三元低钴化；
- 镍价和钴价波动；
- 前驱体竞争加剧；
- 高资本开支带来折旧和 ROIC 压力。

财务验证：

| 指标 | 用途 |
|---|---|
| 钴产品收入和毛利率 | 资源弹性 |
| 镍产品收入和毛利率 | 镍资源布局 |
| 前驱体收入和毛利率 | 材料平台质量 |
| 资本开支 | 扩张压力 |
| ROIC | 投入回报 |
| 存货和减值 | 价格波动风险 |
| 经营现金流 | 盈利质量 |

---

## 七、阶段五：2023–2025 复合周期承压

2023–2025，华友面对的不是单一价格下跌，而是多变量共振：

- 钴价波动；
- 镍价波动；
- 三元路线受磷酸铁锂挤压；
- 前驱体和正极材料竞争；
- 资本开支和折旧；
- ROIC 重估；
- 存货和减值。

这类公司不能简单用“资源股”或“材料股”解释，必须拆分：

1. 资源价格；
2. 材料加工利润；
3. 技术路线；
4. 资本开支回报；
5. 客户结构；
6. 现金流。

准确状态：

> 华友是半穿越 / 周期承压样本。平台化提供修复可能，但并不自动抵消价格、路线和资本开支周期。

## 九、华友回填后的核心判断

华友路径可以概括为：

1. **2016–2017：钴资源第一轮行情。**
2. **2018–2019：钴价回落，资源周期退潮。**
3. **2020：从钴资源股向资源材料平台重估。**
4. **2021–2022：资源和材料双属性充分定价。**
5. **2023–2025：价格、路线、材料利润和资本开支多重承压。**

## 十、对 AI 的映射

华友对应 AI 中资源 + 材料 + 加工复合型公司。

类似方向：

- 同时做上游材料和下游部件；
- 同时做电力设备材料和系统集成；
- 同时做先进封装材料和封装产能；
- 同时做液冷材料和液冷系统。

核心经验：

> 业务版图越大，不代表越能穿越。复合型公司必须逐项验证资源价格、材料利润、技术路线和 ROIC。
