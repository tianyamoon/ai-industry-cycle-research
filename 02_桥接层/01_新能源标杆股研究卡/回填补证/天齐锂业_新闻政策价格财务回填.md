## 一、回填目标

天齐锂业必须和赣锋锂业成组研究。

赣锋更偏：

**锂盐加工 + 全球资源布局 + 产业链延伸。**

天齐更偏：

**优质资源禀赋 + 海外并购 + 高杠杆压力 + 锂价修复带来的资产负债表弹性。**

本补证卡要验证的问题是：

> **为什么天齐在低景气时压力大，在锂价上行时弹性也大？**

核心证据链：

**锂资源禀赋 → 高景气并购和杠杆扩张 → 锂价下行时资产负债表承压 → 2021–2022 锂价暴涨修复利润表和资产负债表 → 2023–2025 锂价回落后重新进入周期压力。**

## 三、阶段一：2015–2017 第一轮锂资源认知

### 1. 产业背景

新能源车补贴推动动力电池需求增长，市场开始系统性定价锂资源价值。

天齐在这一阶段的核心标签是：

- 优质锂矿资源；
- 碳酸锂和氢氧化锂价格弹性；
- 动力电池需求；
- 战略资源稀缺；
- 资源股高弹性。

### 2. 待补价格数据

| 数据 | 用途 | 来源 |
|---|---|---|
| 电池级碳酸锂价格 | 验证第一轮资源行情 | SMM / 百川 / Wind |
| 氢氧化锂价格 | 验证高端锂盐需求 | SMM / 百川 / Wind |
| 锂辉石价格 | 验证资源端供需 | Fastmarkets / Benchmark / SMM |
| 新能源车销量 | 验证需求来源 | 中汽协 |
| 动力电池装机 | 验证电池需求 | GGII / 工信部 |

### 3. 待补财务数据

需补 2015–2017：

- 营业收入；
- 归母净利润；
- 锂资源和锂盐业务收入；
- 毛利率；
- 资源权益；
- 资本开支；
- 有息负债；
- 经营现金流。

### 4. 事后验证

第一轮锂资源认知成立，但随后的 2018–2019 证明：

> 资源禀赋可以支撑长期价值，但无法抵消价格周期和杠杆压力。

## 四、阶段二：2018–2019 锂价下行与杠杆压力

### 1. 产业背景

2018–2019，新能车补贴退坡，锂价回落，资源股估值下行。

天齐较赣锋更值得关注的地方是：

- 高景气期并购带来的有息负债；
- 锂价下跌后利润修复变慢；
- 财务费用和现金流压力；
- 资产负债表对股价的压制。

### 2. 当时市场理解

市场从：

**优质资源禀赋 + 锂资源长期稀缺**

转向：

**锂价下跌 + 高杠杆 + 现金流压力 + 偿债不确定。**

### 3. 待补财务验证

| 指标 | 用途 |
|---|---|
| 有息负债 | 验证并购杠杆压力 |
| 财务费用 | 验证债务成本 |
| 资产负债率 | 验证财务风险 |
| 经营现金流 | 验证偿债能力 |
| 锂盐毛利率 | 验证价格下行冲击 |
| 净利润 / 扣非净利润 | 验证利润下滑 |
| 存货 / 减值 | 验证周期压力 |

### 4. 事后验证

这阶段是天齐区别于赣锋的关键：

> 同样是锂价下行，资产负债表更重的公司，股价压力会被放大。

### 5. AI 映射

AI 高景气阶段如果企业：

- 高杠杆买 GPU；
- 大举建设数据中心；
- 高价并购产能；
- 大规模扩产液冷 / 光模块 / 电力设备；
- 用债务追逐短期高景气；

一旦价格或订单不及预期，资产负债表压力会迅速显性化。

## 五、阶段三：2020 低位修复与锂价反转前夜

### 1. 产业背景

2020 年新能源车需求恢复，双碳提出，锂价底部逐渐企稳。

对天齐而言，2020 年的重点不只是需求恢复，更是：

- 市场预期锂价反转；
- 利润表有望修复；
- 资产负债表压力可能缓解；
- 资源禀赋重新被定价。

### 2. 待补证据

| 证据 | 用途 | 来源 |
|---|---|---|
| 2020 碳酸锂价格底部 | 验证价格反转前夜 | SMM / 百川 |
| 2020 新能源车销量 | 验证需求恢复 | 中汽协 |
| 天齐 2020 年报 | 验证财务压力 | 年报 |
| 债务处置 / 融资 / 股权事项 | 验证资产负债表修复 | 公告 |
| 券商研报 | 验证市场如何理解修复 | 研报 |

### 3. 事后验证

2020 是天齐从“债务压力资源股”转向“锂价反转修复股”的前夜。

## 六、阶段四：2021–2022 锂价泡沫和资产负债表修复

### 1. 产业背景

2021–2022，新能源车需求爆发，锂资源供需错配，碳酸锂价格快速上涨。

天齐受益于双重修复：

1. 锂价上涨带来利润表修复；
2. 利润改善和资源价值重估带来资产负债表预期修复。

### 2. 股价含义

天齐在这一阶段弹性大，原因不只是锂价上涨，而是：

- 前期被债务压制；
- 低位预期极差；
- 锂价上涨后利润弹性极高；
- 债务压力缓解预期极强；
- 市场从“财务风险”切回“资源价值”。

### 3. 待补财务数据

需补 2021–2022：

- 营业收入；
- 锂盐收入；
- 毛利率；
- 归母净利润；
- 扣非净利润；
- ROE；
- 经营现金流；
- 有息负债；
- 财务费用；
- 资产负债率；
- 偿债安排；
- 资源权益价值变化。

### 4. 事后验证

这阶段是天齐最典型的周期弹性阶段。

但风险在于：

> 利润表和资产负债表的修复，都高度依赖锂价高位。一旦锂价回落，估值需要重新定价。

### 5. AI 映射

AI 中类似的公司可能是：

- 前期因债务或资本开支压力被压制；
- 后期因 AI 景气和价格上涨获得极强修复；
- 市场把周期修复当成长期成长；
- 高点时风险被利润掩盖。

## 七、阶段五：2023–2025 锂价回落与重新阴影

### 1. 产业背景

2023–2025，锂价大幅回落，电池厂去库存，车企价格战，下游对材料成本压价。

天齐重新面对：

- 锂盐价格下跌；
- 毛利率回落；
- 利润中枢下降；
- 资产负债表再次被市场审视；
- 高点估值体系破裂；
- 资源股重新切回周期定价。

### 2. 待补财务证据

| 年份 | 必补指标 | 用途 |
|---|---|---|
| 2023 | 收入、毛利率、净利润、有息负债、现金流 | 看锂价下跌初期影响 |
| 2024 | 扣非净利润、减值、财务费用、资产负债率 | 看低价环境压力 |
| 2025 | 中报 / 年报、锂价低位、负债变化 | 看周期修复条件 |

### 3. 事后结论

天齐不是出清失败，而是进入资源周期阴影。

准确表述：

> 资源禀赋仍在，但高景气时修复的利润表和资产负债表，在低价周期中需要重新接受压力测试。

## 九、天齐回填后的核心判断

天齐锂业路径可以概括为：

1. **2015–2017：资源禀赋被发现。** 新能源车需求让优质锂资源被市场战略化定价。
2. **2018–2019：价格周期和杠杆压力共振。** 锂价回落叠加高杠杆，股价压力显著。
3. **2020：低位修复前夜。** 锂价企稳和 EV 需求恢复，让市场重新定价资源价值。
4. **2021–2022：锂价泡沫和资产负债表修复。** 利润表和资产负债表同时改善，弹性极强。
5. **2023–2025：高点逻辑破裂。** 锂价回落后，资源价值仍在，但高估值和高利润不可持续。

## 十一、对 AI 的映射

天齐比赣锋更强调资产负债表风险。

AI 周期中要特别警惕：

1. 高景气时大举扩产；
2. 高价并购核心产能；
3. 高杠杆买 GPU 或建数据中心；
4. 短期高毛利掩盖债务压力；
5. 一旦 ASP 或订单下行，折旧、利息、存货和现金流同时承压。

核心经验：

> 资源禀赋和战略卡位不能替代资产负债表安全。高景气会放大弹性，低景气也会放大风险。
