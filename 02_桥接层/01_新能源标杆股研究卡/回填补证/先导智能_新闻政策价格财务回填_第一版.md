## 一、回填目标

先导智能是新能源周期中设备链的代表样本。

设备股和资源、材料、整车、电池的不同点在于：

> **设备公司往往先受益于下游扩产预期，但当扩产周期结束、订单增速放缓、产能过剩显现时，估值也会提前退潮。**

本补证卡要验证的问题是：

> **设备股最强的时候，到底是产业长期成长，还是下游资本开支周期的高点？**

核心证据链：

**新能源车需求增长 → 动力电池厂扩产 → 锂电设备订单增长 → 设备股主升 → 下游扩产高峰和订单高景气 → 电池价格战和产能过剩 → 客户资本开支放缓 → 新签订单、回款和现金流被重估 → 估值从成长切回资本开支周期。**

## 三、阶段一：2015–2017 设备先行认知

2015–2017 年，新能源车补贴驱动动力电池扩产，锂电设备国产化和自动化需求提升。

先导的早期逻辑：

- 动力电池产线扩张；
- 自动化设备国产替代；
- 头部客户绑定；
- 设备订单先于电池产能释放；
- 下游资本开支带来业绩弹性。

## 四、阶段二：2018–2019 补贴退坡和订单筛选

2018–2019，新能源车补贴退坡，动力电池行业集中度提升，弱电池厂出清。

对设备股而言，压力来自：

- 下游扩产节奏放缓；
- 客户项目推迟；
- 回款压力；
- 应收账款增加；
- 订单质量分化。

但头部设备商也受益于电池厂集中：

- 头部客户更依赖优质设备；
- 自动化和良率要求提高；
- 低端设备厂被淘汰；
- 设备龙头强化客户绑定。

财务验证重点：

| 指标 | 用途 |
|---|---|
| 新签订单 | 看下游资本开支 |
| 在手订单 | 看收入可见度 |
| 应收账款 | 看回款压力 |
| 合同负债 | 看订单质量 |
| 经营现金流 | 看利润质量 |
| 毛利率 | 看设备竞争 |

## 五、阶段三：2020 主升启动与扩产恢复

2020 年，新能源车需求恢复、双碳政策和特斯拉国产推动动力电池链条重估。电池厂重新加速扩产，先导进入主升。

市场相信：

- 下游电池产能扩张持续；
- 宁德、比亚迪等客户资本开支强；
- 储能带来新设备需求；
- 锂电设备龙头订单可持续；
- 自动化和智能制造提升估值。

## 六、阶段四：2021–2022 扩产高峰与订单外推

2021–2022，动力电池和储能扩产进入高峰，先导设备订单高景气。

但设备股高点有一个典型风险：

> 市场容易把下游资本开支高峰，当作公司长期成长中枢。

这一阶段必须补：

| 指标 | 验证问题 |
|---|---|
| 新签订单 | 是否继续高增 |
| 在手订单 | 收入可见度 |
| 收入增速 | 订单是否兑现 |
| 毛利率 | 设备竞争和客户压价 |
| 应收账款 | 回款压力 |
| 经营现金流 | 利润质量 |
| 前五大客户占比 | 客户集中风险 |
| 存货 | 项目交付和订单风险 |

事后看，2021–2022 的设备高景气有真实产业基础，但同时也代表下游扩产最激进阶段。高景气不等于永续增长。

---

## 七、阶段五：2023–2025 订单退潮和资本开支周期重估

2023–2025，电池价格战、产能过剩、车企和电池厂降本，使下游资本开支趋于谨慎。

先导需要验证：

1. 新签订单是否放缓；
2. 在手订单是否仍可兑现；
3. 应收账款是否增加；
4. 经营现金流是否跟得上利润；
5. 毛利率是否被客户压价；
6. 新技术路线是否带来新设备周期。

准确状态：

> 先导不是出清失败，而是锂电设备龙头进入资本开支周期退潮后的半穿越观察期。下一轮机会取决于储能、固态电池、新工艺和海外扩产周期。

## 九、先导回填后的核心判断

先导路径可以概括为：

1. **2015–2017：锂电设备先行。** 下游扩产预期推动设备国产化。
2. **2018–2019：补贴退坡下订单筛选。** 设备股对客户资本开支高度敏感。
3. **2020：扩产恢复，主升启动。** 电池厂重新扩产，设备订单改善。
4. **2021–2022：扩产高峰。** 订单高景气被市场充分外推。
5. **2023–2025：资本开支周期重估。** 下游过剩后，设备股要看新签订单、回款和现金流。
