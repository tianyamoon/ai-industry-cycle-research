## 一、回填目标

通威股份是光伏周期里最典型的上游瓶颈利润样本。

主卡已经建立了通威路径：

**2016–2017 光伏业务结构重估 → 2018 光伏 531 出清 → 2019 平价上网和低成本制造 → 2020 核心资产重估 → 2021–2022 硅料利润上移 → 2023–2025 硅料价格下跌和利润反噬。**

本补证卡要验证的问题是：

> **通威为什么在 2021H2–2022 比部分光伏核心资产更强，又为什么在 2023–2025 被硅料扩产和价格下跌反噬？**

核心证据链：

**硅料供需紧张 → 硅料价格上涨 → 通威毛利率和利润释放 → 高利润刺激扩产 → 硅料产能集中释放 → 硅料价格下跌 → 利润中枢重估 → 股价阴影。**

## 三、阶段一：2016–2017 光伏业务结构重估

### 1. 产业背景

2016–2017 年，光伏装机高增，A 股开始系统性关注光伏制造链条。

通威的特殊性在于，它不是传统意义上的纯组件公司，而是由农牧业务延伸到光伏硅料、电池片等环节。

市场早期买的是：

- 光伏装机增长；
- 通威光伏业务占比提升；
- 电池片和硅料布局；
- 低成本制造能力；
- 从农牧周期股向新能源制造股切换。

### 2. 待补数据

| 数据 | 用途 | 来源 |
|---|---|---|
| 2016–2017 中国新增光伏装机 | 验证行业景气 | 国家能源局 / CPIA |
| 通威光伏业务收入 | 验证业务结构切换 | 年报 / Wind |
| 通威硅料产能 | 验证上游布局 | 年报 / 公告 |
| 通威电池片产能 | 验证制造链布局 | 年报 / 公告 |
| 农牧与光伏业务收入占比 | 验证估值切换 | 年报 |

### 3. 事后验证

这一阶段的核心不是硅料暴利，而是身份切换：

> 通威从农牧企业被市场重新认识为光伏制造链条的重要公司。

## 四、阶段二：2018 光伏 531 出清

### 1. 政策节点

2018 年 531 新政压缩补贴和新增规模，光伏产业链整体承压。

对通威影响包括：

- 光伏需求预期下修；
- 电池片价格承压；
- 硅料价格承压；
- 光伏制造估值下降；
- 板块风险偏好下降。

### 2. 当时市场理解

市场认为光伏制造链条进入出清，通威的光伏业务扩张可能面对盈利压力。

### 3. 事后验证

531 没有证伪通威，而是让低成本制造和资金能力更重要。

通威需要验证的是：

- 是否能在低迷期维持现金流；
- 光伏业务是否具备成本优势；
- 农牧业务能否提供底部安全垫；
- 硅料和电池片产能是否能穿越周期。

### 4. 待补财务

| 指标 | 用途 |
|---|---|
| 2018 营业收入 | 看冲击幅度 |
| 2018 归母净利润 | 看盈利压力 |
| 2018 毛利率 | 看价格下跌影响 |
| 硅料 / 电池片业务毛利率 | 看光伏业务韧性 |
| 农牧业务利润 | 看现金流底盘 |
| 资本开支 | 看是否逆势扩张 |

## 五、阶段三：2019 平价上网与主升前夜

### 1. 产业背景

2019 年平价上网推进，光伏行业从补贴驱动转向成本竞争。

通威的逻辑开始从“光伏参与者”变成“低成本硅料和电池片龙头候选”。

### 2. 当时市场理解

市场开始关注：

- 硅料成本曲线；
- 电池片产能和效率；
- 光伏龙头集中；
- 平价上网后全球装机打开；
- 低成本制造企业的长期优势。

## 七、阶段五：2021–2022 硅料利润上移和高点

### 1. 产业背景

2021 年，光伏产业链利润向上游硅料集中。

原因包括：

- 光伏装机需求强；
- 硅料新增供给释放慢；
- 下游硅片、电池片、组件扩产快；
- 硅料供需错配；
- 硅料价格上涨；
- 通威等上游龙头盈利弹性释放。

### 2. 股价含义

通威在这一阶段比部分光伏主链公司更强，原因是：

- 隆基等核心资产估值已经高；
- 硅片 / 组件受上游成本挤压；
- 硅料价格上涨使通威成为利润集中环节；
- 市场从“光伏核心资产”转向“瓶颈利润环节”。

2021–2022 通威的逻辑成立，但它是供需错配下的周期利润，不是永久高毛利。

高利润阶段同时会引发：

- 同行扩产；
- 自身扩产；
- 下游压价；
- 政策关注产业链成本；
- 未来供给过剩预期。

- 高端封装产能。

> 这是长期定价权，还是供给释放前的阶段性利润？

## 八、阶段六：2023–2025 硅料价格下跌与利润反噬

### 1. 产业背景

2023–2025，光伏行业进入过剩和价格战阶段。

通威受到的压力包括：

- 硅料产能集中释放；
- 硅料价格大幅下跌；
- 电池片和组件价格战；
- 光伏制造全链毛利率下行；
- 市场从高成长估值切回周期制造；
- 存货、减值和现金流压力上升。

### 2. 为什么通威被反噬

通威在 2021–2022 最强，是因为硅料最稀缺。

但 2023–2025 承压，也是因为硅料价格回落最直接打击利润。

这就是瓶颈利润环节的两面性：

> 上涨时利润集中，下跌时利润反噬。

### 5. AI 映射

AI 中的瓶颈环节也会出现类似风险：

- 高毛利导致扩产；
- 大客户扶持第二供应商；
- 产能释放后 ASP 下跌；
- 毛利率回落；
- 估值从成长切回周期；
- 龙头仍是龙头，但股价进入阴影。

## 十、通威回填后的核心判断

通威路径可以被概括为：

1. **2016–2017：业务结构重估。** 从农牧周期股切向光伏制造成长股。
2. **2018：531 压力测试。** 光伏制造承压，但低成本和上游能力保留。
3. **2019：低成本制造和硅料候选。** 平价上网提升成本优势价值。
4. **2020：核心资产重估。** 双碳和光伏需求推动通威进入主升。
5. **2021–2022：硅料利润上移。** 通威成为瓶颈利润环节，后半段强于部分主链公司。
6. **2023–2025：扩产反噬。** 硅料价格下跌后，利润和估值从高位重估。
